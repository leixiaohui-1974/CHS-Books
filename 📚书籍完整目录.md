## 📝 使用建议

### 考研党

1. **时间充足（6个月+）**
   - 按顺序学习：数学(1-4) → 水力学(6-8) → 水文学(10-11) → 专业拓展(13-17)
   - 每天3-4小时，系统学习

2. **时间紧迫（3个月）**
   - 重点突破：数学速成(1) + 水力学核心(6) + 水文学冲刺(10) + 30天冲刺(19)
   - 配合1000题(7)刷题

3. **冲刺阶段（1个月）**
   - 30天冲刺宝典(19) + 模拟考试(20) + 真题解析(9)
   - 每天6-8小时，高强度复习

### 研究生

1. **学术型硕士**
   - 考研系列基础 + 专业系列深入 + 编程实战
   - 重点：专业书(1-6) + Python编程(18)

2. **专业型硕士**
   - 工程应用导向
   - 重点：水力学(6-8) + 数值方法(5) + 专业应用书

3. **博士研究**
   - 前沿方向深入研究
   - 重点：专业系列(1-14) + 科研方法训练

### 工程师

1. **设计院工程师**
   - 水力学核心 + 水工建筑物 + Python编程
   - 重点：实际工程案例

2. **软件工程师**
   - 数值计算 + Python编程 + 专业领域知识
   - 重点：算法实现 + 系统开发

3. **项目管理**
   - 全面了解各专业 + 水资源管理 + 面试求职指南
   - 重点：系统思维 + 管理能力
