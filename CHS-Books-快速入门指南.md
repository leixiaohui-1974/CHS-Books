## 💻 环境配置

### 快速安装（5分钟）

```bash
# 1. 确认Python版本
python --version  # 需要3.8+

# 2. 创建虚拟环境（推荐）
python -m venv chs-env
source chs-env/bin/activate  # Linux/Mac
# chs-env\Scripts\activate    # Windows

# 3. 安装核心依赖
pip install numpy scipy matplotlib pandas

# 4. 安装控制库（如果学习控制相关教材）
pip install control do-mpc

# 5. 安装优化库（如果学习优化相关教材）
pip install PuLP cvxpy pyomo

# 6. 安装机器学习库（如果学习辨识相关教材）
pip install scikit-learn statsmodels
```

## 🆘 常见问题

### Q1：我是零基础，应该从哪里开始？

**A**：推荐从**水系统控制论**开始（案例1-4），这是最容易入门的教材。

### Q2：程序运行报错怎么办？

**A**：
1. 检查Python版本（需要3.8+）
2. 检查依赖包是否安装
3. 查看错误信息
4. 阅读README中的说明

### Q3：看不懂代码怎么办？

**A**：
1. 先看程序注释
2. 查看案例README
3. 从简单案例开始
4. 逐行分析理解

### Q4：需要多久才能学完一本教材？

**A**：
- 快速浏览：1-2周
- 深入学习：1-2个月
- 精通掌握：3-6个月

### Q5：可以跳过某些案例吗？

**A**：
- 基础案例：不建议跳过
- 进阶案例：可以根据兴趣选择
- 高级案例：可以先学基础再回来

### Q6：没有相关专业背景可以学吗？

**A**：可以！教材采用案例驱动学习，从零开始系统教学。建议：
- 从最简单的教材开始
- 循序渐进，不要急于求成
- 多动手实践，多做实验
- 遇到问题及时查资料
