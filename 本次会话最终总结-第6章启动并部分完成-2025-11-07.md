## 💎 第6章技术亮点

### 题1：水泵基本性能参数计算（750行）

**核心算法**:

```python
class PumpPerformance:
    def circumferential_velocity(self, D, n):
        """圆周速度：u = π·D·n/60"""
        u = np.pi * D * n / 60
        return u
    
    def tangential_velocity_component(self, u2, vr2):
        """速度三角形：cu₂ = u₂ - vr₂/tan(β₂)"""
        cu2 = u2 - vr2 / np.tan(self.beta2)
        return cu2
    
    def theoretical_head(self, u2, cu2):
        """欧拉方程：Ht = u₂·cu₂/g"""
        Ht = u2 * cu2 / self.g
        return Ht
    
    def similarity_law_speed_change(self, n_new):
        """相似定律：Q∝n, H∝n², N∝n³"""
        ratio = n_new / self.n
        Q_new = self.Q * ratio
        H_new = H * (ratio ** 2)
        N_new = self.N * (ratio ** 3)
        return Q_new, H_new, N_new
```
