### 2. 地下水数值模拟（Ch7）

**有限差分法**：
```python
# 二维稳态（拉普拉斯方程）
∂²h/∂x² + ∂²h/∂y² = 0

# 五点差分格式
h_{i,j} = (h_{i+1,j} + h_{i-1,j} + h_{i,j+1} + h_{i,j-1})/4

# 非稳定流（隐式格式）
S·∂h/∂t = T·(∂²h/∂x² + ∂²h/∂y²) + W

# 稳定性条件（显式）
Δt ≤ S·(Δx)²·(Δy)² / [4T((Δx)² + (Δy)²)]
```

**迭代法**：
- Jacobi迭代
- Gauss-Seidel迭代  
- SOR超松弛：$\omega_{opt} = 2/(1 + \sqrt{1-\rho})$

**模型评价**：
```python
MAE = Σ|h_obs - h_sim|/n
RMSE = √(Σ(h_obs - h_sim)²/n)
NSE = 1 - Σ(h_obs - h_sim)² / Σ(h_obs - h̄_obs)²
```

### 3. 地下水管理与保护（Ch8）

**超采评估**：
```python
超采量: ΔW = W_extract - W_recharge
超采率: η = (W_extract/W_recharge)×100%

分级：
  轻度：100% < η ≤ 120%
  中度：120% < η ≤ 150%
  严重：η > 150%
```
