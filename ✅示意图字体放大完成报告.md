## 📊 字体放大规则

### 调整方案
```
原字体大小 → 新字体大小 (增量)
─────────────────────────────
   8px    →    10px    (+2px)
   9px    →    11px    (+2px)
  10px    →    12px    (+2px)
  11px    →    13px    (+2px)
  12px    →    14px    (+2px)
  13px    →    15px    (+2px)
  14px+   →    保持    (已足够大)
```

### 设计原则
1. **统一增量**：所有小字体统一增大2px
2. **阈值设定**：14px及以上保持不变（已足够清晰）
3. **避免重叠**：通过适度放大避免过大导致元素重叠
4. **平衡美观**：在可读性和布局美观之间取得平衡
