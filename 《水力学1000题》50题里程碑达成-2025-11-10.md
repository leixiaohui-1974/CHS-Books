### 2. 浮体三心理论 🎯

**核心概念**：
```
G - 重心（Gravity Center）
  ↓ 物体质心，重力作用点

B - 浮心（Buoyancy Center）
  ↓ 浸没部分几何中心，浮力作用点

M - 稳心（Metacenter）
  ↓ 浮体微倾时，浮力作用线与中心轴交点
```

**稳定性判据**：
```
M在G上方 → GM > 0 → 稳定 ✓
M与G重合 → GM = 0 → 临界稳定
M在G下方 → GM < 0 → 不稳定 ✗
```

**计算公式**：
```
KB = T/2           （浮心高度）
BM = I_L/∇         （稳心半径）
KM = KB + BM       （稳心高度）
GM = KM - KG       （初稳性高）
```

**工程意义**：
- 船舶设计：GM通常0.3-1.0m
- 浮筒设计：细长体竖放不稳定
- 海上平台：需加大水线面惯性矩

### 3. 经典物理问题 📚

#### 冰块融化水位变化（题036）

**问题**：冰块漂浮在水中，融化后水位如何变化？

**分析**：
```
漂浮时: V_排 = m_冰 / ρ_水
融化后: V_水 = m_冰 / ρ_水
结论: V_排 = V_水 → 水位不变！
```

**推广**：
- 纯冰融化：水位不变
- 冰中有气泡：水位下降
- 冰上有重物：水位下降

#### 潜艇浮沉控制（题034）

**原理**：调节压载水舱
```
上浮: 排出水 → 减重 → W < F_浮
悬浮: 平衡水 → W = F_浮
下潜: 注入水 → 增重 → W > F_浮
```

**精度**：
- 50吨水 ≈ 490 kN净下沉力
- 精确控制深度
