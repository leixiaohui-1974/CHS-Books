**频率分析核心**：

```python
# P-III型分布
x_p = x̄(1 + Cv·Φp)

# 统计参数

Cs = m3/σ³  # 偏态系数

# 经验频率
```

### 2. 水文预报方法（Ch7）

**单位线法**：
```python
# 单位线推求
U(t) = Q_direct(t) / Σ P_net

# 卷积预报
Q(t) = Σ P_i · U(t-i)

# S曲线法
u(t) = dS(t)/dt
```

**马斯京根河道汇流**：
```python
# 蓄泄方程
S = K[xI + (1-x)Q]

# 演算系数
C0 = (-Kx + 0.5Δt) / [K(1-x) + 0.5Δt]
C1 = (Kx + 0.5Δt) / [K(1-x) + 0.5Δt]
C2 = [K(1-x) - 0.5Δt] / [K(1-x) + 0.5Δt]

# 递推计算
Q_{j+1} = C0·I_{j+1} + C1·I_j + C2·Q_j
```

**水位流量关系**：
