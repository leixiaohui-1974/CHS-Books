## 📚 核心公式汇总

### Top 5必背公式

1. **达西定律**
   ```
   v = k × J
   Q = k × J × A = v × A
   ```

2. **水力梯度**
   ```
   J = Δh / L        (一维渗流)
   J = Q / (2πkrh)   (径向渗流)
   ```

3. **实际流速**
   ```
   u = v / n
   (n为孔隙率)
   ```

4. **潜水完整井（Dupuit公式）**
   ```
   Q = πk(H²-h₀²) / ln(R/r₀)
   h² = h₀² + (H²-h₀²)×ln(r/r₀) / ln(R/r₀)
   ```

5. **承压完整井**
   ```
   Q = 2πkM(H-h₀) / ln(R/r₀)
   (M为承压含水层厚度)
   ```

### 渗透系数k值范围

| 土类 | k值(cm/s) | 渗透性 |
|------|-----------|--------|
| 砾石 | >1 | 强透水 |
| 粗砂 | 1~0.1 | 强透水 |
| 中细砂 | 0.1~0.01 | 中等透水 |
| 粉砂 | 0.01~0.001 | 弱透水 |
| 粘土 | <0.001 | 不透水 |

### 潜水井vs承压井

| 项目 | 潜水井 | 承压井 |
|------|--------|--------|
| 自由水面 | 有 | 无 |
| 流量公式 | H²-h²项 | (H-h)项 |
| 水面形状 | 曲线 | 直线（近似） |
| 出水量 | 较小 | 较大 |

## ⚠️ 记忆口诀与技巧

### 记忆口诀

**🎵 「达西定律」**
```
v等于k乘J（v=kJ）
流量Q是kvA（Q=kvA）
```

**🎵 「潜水承压」**
```
潜水井用H平方差（H²-h²）
承压井用H减h（H-h）
```

**🎵 「虚拟实际」**
```
v是虚拟速度（达西速度）
u是实际流速（u=v/n）
孔隙率n分母站
```

**🎵 「渗透系数」**
```
砾石粗砂强透水（k>0.1cm/s）
中细砂层中透水（0.01~0.1）
粉砂粘土不透水（k<0.01）
```

### 解题技巧

**达西定律4步法**
1. Step 1: 单位统一（k、L、t同一制）
2. Step 2: 计算J=Δh/L
3. Step 3: 应用v=kJ
4. Step 4: Q=vA

**井流计算4步法**
1. Step 1: 判断井类型（潜水/承压）
2. Step 2: 选择公式（H²-h²或H-h）
3. Step 3: 计算Q
4. Step 4: 求h、J、v

### 常见错误

❌ **错误1**：单位不统一
✓ 技巧：k用cm/s，L用m，时必须转换

❌ **错误2**：v和u混淆
✓ 技巧：v是虚拟的（达西速度），u是真实的

❌ **错误3**：潜水井用H-h
✓ 技巧：潜水看到H²-h²，承压看到H-h

❌ **错误4**：对数ln(R-r₀)
✓ 技巧：是ln(R/r₀)，不是ln(R-r₀)
