## 💡 核心公式速查卡

### 静水压强

```
p = p₀ + γh        基本公式
h = p/γ            测压管高度
p_绝 = p_相 + p_atm  绝对压强
```

### 平面总压力

```
P = γh_c × A            总压力
y_p = h_c + I_c/(h_c×A)  压力中心

惯性矩:
I_矩形 = bh³/12
I_圆形 = πD⁴/64
I_三角 = bh³/36
```

### 曲面总压力

```
P_x = γh_c × A_x    水平分力（投影法）
P_z = γV           垂直分力（压力体法）
P = √(P_x² + P_z²)  总压力
α = arctan(P_z/P_x) 方向角

特性: 圆弧面总压力过圆心
```

### 浮力与稳定性

```
F_浮 = ρ_液 g V_排       阿基米德原理
F_浮 = G              漂浮/悬浮平衡
d = h × ρ_物/ρ_液      吃水深度（漂浮）

稳定性:
KB = T/2             浮心高度
BM = I_L/∇           稳心半径
GM = KM - KG         初稳性高
GM > 0  → 稳定 ✓
GM = 0  → 临界
GM < 0  → 不稳定 ✗
```
