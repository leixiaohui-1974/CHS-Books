## 🎯 管流核心知识点

### 雷诺数与流态（30题）

✅ **核心概念**：
$$Re = \frac{vd}{\nu}$$

✅ **流态判别**：
- \(Re < 2000\)：层流
- \(Re > 4000\)：湍流
- \(2000 < Re < 4000\)：过渡区

✅ **速度分布**：
- 层流：抛物线，\(\bar{v} = v_{max}/2\)
- 湍流：对数/1/7次方，\(\bar{v} \approx 0.82 v_{max}\)

### 沿程阻力（40题）⭐⭐⭐⭐⭐

✅ **达西公式**：
$$h_f = \lambda \frac{L}{d} \frac{v^2}{2g}$$

✅ **阻力系数**：
- 层流：\(\lambda = 64/Re\)
- 湍流光滑管：\(\lambda = 0.3164/Re^{0.25}\)
- 湍流粗糙管：科尔布鲁克公式（需迭代）

✅ **工程应用**：
- 管道压降计算
- 功率损失估算
- 经济管径选择

**考研必考**：
- 达西公式：90%试题涉及
- 阻力系数：70%试题涉及
- 莫迪图：50%试题涉及

### 局部阻力（30题）

✅ **基本公式**：
$$h_m = \zeta \frac{v^2}{2g}$$

✅ **常用系数**：
- 突然扩大：\(\zeta = (1 - A_1/A_2)^2\)
- 90°弯头：\(\zeta \approx 1.0\)
- 闸阀全开：\(\zeta \approx 0.2\)

✅ **等价长度**：
$$L_e = \zeta \frac{d}{\lambda}$$

### 管道计算（50题）⭐⭐⭐⭐⭐

✅ **三类基本问题**：

**I类**：已知\(d, L, \lambda, Q\) → 求\(h_f\)
- 直接代入达西公式

**II类**：已知\(d, L, \lambda, h_f\) → 求\(Q\)
- 从达西公式反解\(v\)，再求\(Q\)

**III类**：已知\(Q, L, h_f, \lambda\) → 求\(d\)
- 需要迭代或经验公式

✅ **串并联**：
- 串联：\(h_f = \sum h_{fi}\)，\(Q\)相同
- 并联：\(h_f\)相同，\(\sum Q_i = Q\)

### 管网分析（30题）

✅ **基本原理**：
- 节点连续性：\(\sum Q_{入} = \sum Q_{出}\)
- 回路能量守恒：\(\sum h_f = 0\)

✅ **哈代-克罗斯法**：
$$\Delta Q = -\frac{\sum h_f}{\sum(2h_f/Q)}$$

✅ **应用**：
- 供水管网设计
- 管网流量分配
- 管网优化

### 水击（20题）

✅ **基本理论**：
$$\Delta p = \rho c \Delta v$$

✅ **水击波速**：
$$c = \frac{1}{\sqrt{\rho\left(\frac{1}{K} + \frac{d}{E\delta}\right)}}$$

✅ **防护措施**：
- 缓慢关阀
- 水击保护装置
- 增加管壁厚度
