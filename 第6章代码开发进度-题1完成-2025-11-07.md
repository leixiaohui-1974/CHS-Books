## 🎯 题1：水泵基本性能参数计算（750行）

### 核心功能

**1. 速度计算**:
- 圆周速度：u = π·D·n/60
- 径向速度：vr = Q/(π·D·b)
- 圆周分量：cu = u - vr/tan(β)

**2. 欧拉方程**:
- 理论扬程：Ht = u₂·cu₂/g
- 实际扬程：H = Δp/(ρ·g)
- 水力效率：ηh = H/Ht

**3. 功率效率**:
- 有效功率：Ne = ρ·g·Q·H
- 总效率：η = Ne/N
- 效率分解：η = ηh·ηv·ηm

**4. 相似定律**:
- 流量比：Q'/Q = n'/n
- 扬程比：H'/H = (n'/n)²
- 功率比：N'/N = (n'/n)³

**5. 比转速**:
- ns = n·Q^0.5/H^0.75
- 水泵类型判断

```python
class PumpPerformance:
    def circumferential_velocity(self, D, n):
        """圆周速度：u = π·D·n/60"""
        u = np.pi * D * n / 60
        return u
    
    def theoretical_head(self, u2, cu2):
        """欧拉方程：Ht = u₂·cu₂/g"""
        Ht = u2 * cu2 / self.g
        return Ht
    
    def similarity_law_speed_change(self, n_new):
        """相似定律：Q∝n, H∝n², N∝n³"""
        ratio = n_new / self.n
        Q_new = self.Q * ratio
        H_new = H * (ratio ** 2)
        N_new = self.N * (ratio ** 3)
        return Q_new, H_new, N_new
```
