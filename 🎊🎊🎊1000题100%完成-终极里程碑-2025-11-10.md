## 🚀 核心公式速查

### 静水力学

```
p = p₀ + γh                    静水压强
P = γh_c × A                    平面总压力
y_p = h_c + I_c/(h_c×A)         压力中心
P_x = γh_c × A_x                曲面水平力
P_z = γV                        曲面垂直力
F_浮 = ρ_液 g V_排               浮力
GM = KM - KG                    初稳性高
```

### 水动力学

```
v₁A₁ = v₂A₂                    连续性方程

p₁/γ + v₁²/2g + z₁ = p₂/γ + v₂²/2g + z₂ + h_f    伯努利方程

∑F = ρQ(v₂ - v₁)               动量方程
```

### 管流

```
Re = vd/ν                       雷诺数
h_f = λ(L/d)(v²/2g)            达西公式
λ = 64/Re                       层流阻力系数
λ = 0.3164/Re^0.25             湍流光滑管
```

### 明渠流

```
Q = (1/n)AR^(2/3)i^(1/2)       曼宁公式
h_c = ∛(q²/g)                  临界水深
Fr = v/√(gh)                    弗劳德数
h₂/h₁ = (1/2)(√(1+8Fr₁²)-1)   水跃
```

### 渗流

```
Q = KiA                         达西定律
Q = 2πKM∆H/ln(R/r₀)            承压井
Q = πK(H₀²-Hr²)/ln(R/r₀)      潜水井
```

### 水泵

```
H = H_实 + h_f                  扬程
P = γQH/η                       功率
n_s = nQ^0.5/H^0.75            比转速
```
