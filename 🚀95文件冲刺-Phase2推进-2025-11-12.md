### 2. 明渠临界水深与比能（hydraulics-core-100 第4章）

```python
class OpenChannelCritical:
    def critical_depth_rectangular(self, Q, b):
        """临界水深：h_c = ³√(Q²/gb²)"""
        
    def specific_energy(self, q, h):
        """比能：E = h + q²/(2gh²)"""
        
    def conjugate_depth(self, h1, q):
        """共轭水深：h₂ = (h₁/2)(√(1+8Fr₁²) - 1)"""
        
    def alternate_depths(self, E, q):
        """给定比能求共轭水深（急流、缓流）"""
```

**核心知识**：
- 临界状态：$Fr = 1$，$E_{min} = 1.5h_c$
- 比能曲线：E-h关系，急流区（h<h_c）、缓流区（h>h_c）
- 共轭水深：相同比能的两个水深（急流↔缓流）
- 水跃共轭水深：$h_2 = f(h_1, Fr_1)$

**可视化**：
- 比能曲线（E-h坐标，临界点标注，急流/缓流区域）
- 水跃共轭水深曲线（h₁ vs h₂）
