# 渐变流微分方程

- S1曲线（陡坡壅水）：陡坡进入缓坡

### 4. 假设检验（概率统计-Ch6）

**检验体系**：
```python
# 单样本t检验
T = (x̄ - μ0) / (s/√n) ~ t(n-1)

# 两独立样本t检验
T = (x̄1 - x̄2) / (sw·√(1/n1+1/n2)) ~ t(n1+n2-2)

# 配对样本t检验
T = D̄ / (sD/√n) ~ t(n-1)

# F检验（方差齐性）
F = s1²/s2² ~ F(n1-1, n2-1)
```

**两类错误**：
- α错误（弃真）：拒绝真实H0
- β错误（取伪）：接受错误H0

### 5. 方差分析（概率统计-Ch7）

**单因素ANOVA**：
```python
# 平方和分解
SS_T = SS_A + SS_E

# F统计量
F = MS_A / MS_E = (SS_A/(k-1)) / (SS_E/(n-k))

# 方差分析表
来源    | 平方和 | 自由度 | 均方   | F值
组间    | SS_A   | k-1    | MS_A   | F
组内    | SS_E   | n-k    | MS_E   | -
总计    | SS_T   | n-1    | -      | -
```

**双因素ANOVA**：
- 无交互作用：SS_T = SS_A + SS_B + SS_E
- 有交互作用：SS_T = SS_A + SS_B + SS_A×B + SS_E

### 6. 回归分析（概率统计-Ch8）

**一元线性回归**：
```python
# 最小二乘估计
β1 = Lxy / Lxx
β0 = ȳ - β1·x̄

# 相关系数
r = Lxy / √(Lxx·Lyy)

# 决定系数
R² = SSR / SST = 1 - SSE/SST

# 置信区间（均值）
ŷ0 ± t·σ̂·√(1/n + (x0-x̄)²/Lxx)

# 预测区间（个体）
ŷ0 ± t·σ̂·√(1 + 1/n + (x0-x̄)²/Lxx)
```
