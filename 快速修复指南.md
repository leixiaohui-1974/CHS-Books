# 快速修复指南 - 修复Distributed-Hydrological-Model导入错误

## 问题描述

22个案例失败，都是因为导入错误：
```
ModuleNotFoundError: No module named 'code'
```

## 解决方案

### 方法1：安装为Python包（推荐）⭐⭐⭐⭐⭐

```bash
cd /workspace/books/distributed-hydrological-model
pip install -e .
```

如果没有setup.py，需要创建一个。

### 方法2：修改PYTHONPATH

```bash
export PYTHONPATH=/workspace/books/distributed-hydrological-model:$PYTHONPATH
python3 case_02_thiessen/main.py
```

### 方法3：批量修改导入语句

将所有：
```python
from code.core.interpolation import ...
```

改为：
```python
import sys
sys.path.insert(0, '../../')
from code.core.interpolation import ...
```

## 验证修复

运行测试：
```bash
python3 /workspace/test_all_codes.py
```

预期成功率从54%提升到80%+
