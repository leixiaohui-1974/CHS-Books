## 二、核心知识点覆盖

### 题目1：泥沙沉降速度计算 ⭐⭐⭐⭐⭐

**知识点**：
- **Stokes公式**（层流）：$\omega = \frac{(\rho_s - \rho)gd^2}{18\mu}$
- **Rubey公式**（紊流）：$\omega = \sqrt{\frac{2}{3}gd\left(\frac{\rho_s - \rho}{\rho}\right) + 36\frac{\nu^2}{d^2}} - 6\frac{\nu}{d}$
- **雷诺数校核**：$Re_p = \frac{\omega d}{\nu}$
- **适用范围判别**：$Re_p < 1$（Stokes），$Re_p > 1$（Rubey）

**计算内容**：
1. Stokes公式：$\omega = 0.9$ cm/s（$d = 0.1$ mm）
2. 雷诺数：$Re_p = 0.9 < 1$（层流，Stokes适用）
3. Rubey公式：$\omega = 4.65$ cm/s（高估）
4. 沉降时间：5.55 min（水深3 m）
5. 粒径影响：$d$增大5倍，$\omega$增大11.6倍

**理论深度**：
- 力平衡分析（重力、浮力、阻力）
- 阻力系数$C_D$与雷诺数关系
- 沉降速度与粒径关系：层流区$\omega \propto d^2$，紊流区$\omega \propto \sqrt{d}$

**工程应用**：
- 水库淤积预测（粗沙库尾、细沙库中）
- 沉沙池设计（沉降速度0.5-1.0 cm/s）
- 引水工程（取水口高度设计）

**真题来源**: 河海大学2021年

### 题目2：悬移质浓度分布计算 ⭐⭐⭐⭐⭐

**知识点**：
- **Rouse公式**：$\frac{S}{S_a} = \left[\frac{(h-y)/y}{(h-a)/a}\right]^Z$
- **Rouse数**：$Z = \frac{\omega}{\kappa u_*}$（沉降作用/紊流扩散作用）
- **剪切流速**：$u_* = \sqrt{ghJ}$
- **卡门常数**：$\kappa = 0.4$

**计算内容**：
1. Rouse数：$Z = 1.0$（过渡区）
2. 含沙量分布（$y = 1, 2, 3, 4$ m）：0.89, 0.33, 0.15, 0.056 kg/m³
3. 垂线平均含沙量：$\bar{S} \approx 0.5$ kg/m³
4. 分布特征：底部集中，向上急剧减小
5. 沉降速度影响：$\omega$增大1倍，$Z$增大1倍，上层含沙量减小83-96%

**物理意义**：
- $Z < 0.5$：扩散强，分布均匀（粉沙）
- $0.5 < Z < 1.5$：过渡区（中沙）
- $Z > 1.5$：沉降强，底部集中（粗沙）
- $Z > 2.5$：推移质主导

**工程应用**：
- 取水工程（取水口高度0.5-0.7$h$，避开高含沙层）
- 水库淤积分析（细沙全库淤积、粗沙库尾淤积）
- 引水渠道（底部含沙量大，需定期清淤）

**真题来源**: 清华大学2020年

### 题目3：推移质输沙率计算 ⭐⭐⭐⭐

**知识点**：
- **床面切应力**：$\tau_0 = \gamma h J$
- **起动切应力**：$\tau_c = \theta_c(\rho_s - \rho)gd$（希尔兹公式）
- **希尔兹参数**：$\theta = \tau_0/\tau_c$
- **迈耶-彼得公式**：$q_b = 8(\theta - 1)^{1.5}\sqrt{(s-1)gd^3}$
- **爱因斯坦-布朗公式**：$\Phi = 40\theta^3$（$\theta < 10$时适用）

**计算内容**：
1. 床面切应力：$\tau_0 = 19.6$ Pa
2. 起动切应力：$\tau_c = 1.62$ Pa
3. 希尔兹参数：$\theta = 12.1 > 10$（强烈搬运）
4. 单宽推移质输沙率：$q_b = 0.106$ m²/s
5. 推移质输沙率：22.5 t/s（经验修正后约3.4 t/s）
6. 流速影响：流速增大25%，输沙率增大105%（翻倍）

**经验关系**：
- $G_b \propto v^{3-5}$（输沙率与流速高度非线性）
- 推移质占比：山区河流30-50%，平原河流10-20%
- 洪水期输沙量占全年90%以上

**工程应用**：
- 河床演变预测（推移质是河床变形主要动力）
- 桥梁墩位选择（避开推移质集中区）
- 航道维护（推移质形成沙波、沙垄）
- 水库淤积（推移质主要淤积库尾三角洲）

**真题来源**: 天津大学2020年

### 题目4：床面形态判别 ⭐⭐⭐⭐

**知识点**：
- **水流强度参数**：$\Psi = \frac{v}{\sqrt{gd(s-1)}}$
- **粒径参数**：$D_* = d\left[\frac{(s-1)g}{\nu^2}\right]^{1/3}$
- **床面形态分类**：平整床面（下）、沙纹、沙垄、过渡、平整床面（上）、逆行沙波
- **Simons-Richardson图**：床面形态判别

**计算内容**：
1. 雷诺数：$Re = 1.2 \times 10^6$（紊流）
2. 弗劳德数：$Fr = 0.209 < 1$（缓流）
3. 水流强度参数：$\Psi = 11.5$
4. 粒径参数：$D_* = 7.59$
5. 床面形态：**沙垄**（dunes）
6. 流速增大：$\Psi' = 21.5$ → **平整床面（上）**

**沙垄特征**：
- 几何：高度0.1-0.5$h$，波长5-10$h$，不对称波形
- 运动：向下游移动，速度0.01-0.1 m/s
- 糙率：曼宁系数$n = 0.025-0.035$（是平整床面2-3倍）
- 水力：能量损失大，水面波动

**床面形态演变**：
$$
\text{平整床面(下)} \xrightarrow{流速增大} \text{沙纹} \xrightarrow{} \text{沙垄} \xrightarrow{} \text{过渡} \xrightarrow{} \text{平整床面(上)}
$$

**工程意义**：
- 糙率估算（沙垄使$n$增大50-100%）
- 航道设计（沙波影响水深）
- 桥梁设计（沙波导致局部冲刷）
- 引水工程（取水口淤积预测）

**真题来源**: 武汉大学2019年

### 题目5：河道冲淤平衡计算 ⭐⭐⭐⭐

**知识点**：
- **泥沙平衡方程**：$\Delta W = (G_1 - G_2)t$
- **泥沙连续方程**：$B\frac{\partial z}{\partial t} + \frac{\partial G}{\partial x} = 0$
- **河床冲淤计算**：$\Delta z = \frac{\Delta W}{\gamma_s' BL}$
- **冲淤判别**：$G_1 > G_2$（淤积），$G_1 < G_2$（冲刷），$G_1 = G_2$（平衡）

**计算内容**：
1. 冲淤判别：$G_1 - G_2 = 200$ t/h（淤积）
2. 年净淤积量：175万t/年，135万m³/年
3. 河床平均淤积厚度：1.8 m/年（淤积严重！）
4. 5年河床高程：抬高9 m（50 m → 59 m）
5. 河床稳定措施：5种工程方案

**工程措施**：
1. **上游减沙**：水土保持（减沙50-70%）、水库拦沙
2. **河道整治**：束窄河道、裁弯取直、护岸工程
3. **清淤工程**：机械清淤（15-20元/m³）、水力冲淤
4. **调节输沙**：控制上游来沙、增大下游输沙
5. **监测管理**：定期测量河床高程

**综合方案**：
- 短期（1-2年）：紧急清淤，成本5000万元
- 中期（3-5年）：河道整治（束窄），成本1亿元，减淤60%
- 长期（10年以上）：上游水土保持，成本5亿元，减沙70%

**经济分析**：
- 无措施：5年淤积675万m³，河床抬高9 m，洪灾损失难以估量
- 有措施：总投资6亿元（20年），减淤70%，年淤积0.4 m（可接受）

**典型案例**：
- 黄河下游：年淤积10 cm，河床高出地面10 m（地上河）
- 长江中游：三峡拦沙后下游冲刷，河床下切2-3 m

**真题来源**: 天津大学2019年

## 三、技术亮点

### 3.1 理论深度

✅ **泥沙沉降理论**：
- Stokes公式推导（力平衡：重力=浮力+阻力）
- 雷诺数判别（$Re_p < 1$层流，$Re_p > 1$紊流）
- 阻力系数：$C_D = 24/Re_p$（层流），$C_D = 0.4-0.5$（紊流）
- 沉降速度与粒径关系：$\omega \propto d^2$（层流），$\omega \propto \sqrt{d}$（紊流）

✅ **悬移质浓度分布理论**：
- Rouse公式推导（扩散方程：$\omega S + \varepsilon_s \frac{dS}{dy} = 0$）
- Rouse数物理意义：$Z = \frac{\text{沉降作用}}{\text{紊流扩散作用}}$
- 浓度分布特征：底部集中、向上递减、指数衰减
- 剪切流速$u_* = \sqrt{ghJ}$

✅ **推移质输沙理论**：
- 爱因斯坦-布朗公式：$\Phi = 40\theta^3$（$\theta < 10$）
- 迈耶-彼得公式：$q_b = 8(\theta - 1)^{1.5}\sqrt{(s-1)gd^3}$（更常用）
- 希尔兹参数：$\theta = \tau_0/\tau_c$（水流强度/起动条件）
- 输沙率与流速：$G_b \propto v^{3-5}$（高度非线性）

✅ **床面形态理论**：
- Simons-Richardson判别图
- 水流强度参数：$\Psi = v/\sqrt{gd(s-1)}$
- 粒径参数：$D_* = d[(s-1)g/\nu^2]^{1/3}$
- 床面形态演变序列

✅ **河床演变理论**：
- 泥沙连续方程：$B\frac{\partial z}{\partial t} + \frac{\partial G}{\partial x} = 0$
- 冲淤计算：质量守恒、体积转换
- 河床稳定条件：$G_1 = G_2$

### 3.2 计算方法

**1. 泥沙沉降速度**：
- 选择公式：$Re_p < 1$用Stokes，$Re_p > 1$用Rubey
- 雷诺数校核：$Re_p = \omega d / \nu$
- 粒径敏感性分析：$\omega' = \omega(d'/d)^2$（层流）

**2. 悬移质浓度分布**：
- Rouse数计算：$Z = \omega/(\kappa u_*)$
- 逐点计算：$S/S_a = [(h-y)/y \cdot a/(h-a)]^Z$
- 垂线平均：积分法或数值法
- 参数敏感性：$Z$增大→泥沙更集中底部

**3. 推移质输沙率**：
- 床面切应力：$\tau_0 = \gamma h J$
- 起动切应力：$\tau_c = \theta_c(\rho_s - \rho)gd$
- 希尔兹参数：$\theta = \tau_0/\tau_c$
- 输沙率计算：迈耶-彼得公式
- 经验修正：考虑隐蔽度、级配等因素

**4. 床面形态判别**：
- 水流强度参数$\Psi$计算
- 粒径参数$D_*$计算
- 查Simons-Richardson图
- 判别床面形态类型
- 估算糙率系数

**5. 河道冲淤**：
- 泥沙平衡：$\Delta W = (G_1 - G_2)t$
- 体积转换：$\Delta V = \Delta W/\gamma_s'$（注意单位）
- 河床抬高：$\Delta z = \Delta V/(BL)$
- 长期预测：线性外推或考虑反馈
