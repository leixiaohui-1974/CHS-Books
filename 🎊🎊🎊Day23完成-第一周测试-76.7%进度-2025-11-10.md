#### 测试题3：明渠流综合题（35分）⭐⭐⭐⭐⭐

**题目背景**：
- 梯形渠道（b=5m, m=1.5, n=0.025）
- 底坡i=0.001，流量Q=20m³/s
- 下游跌坎高度Δz=1.5m

**考点覆盖**：
1. 正常水深与临界水深
2. 流态判断
3. 水跃计算
4. 能量损失

**计算结果**：
```
(1) 正常水深与临界水深：
    • 正常水深：h₀ = 2.110 m ✓
    • 临界水深：hc = 1.255 m ✓
    • 比较：h₀ > hc → 缓流

(2) 流态与跌坎上游水深：
    • Fr₀ = 0.649 < 1 → 缓流
    • 跌坎上游水深：h = hc = 1.255 m ✓

(3) 水跃计算：
    • 跃前：h₁ = 0.628 m, Fr₁ = 2.318
    • 跃后：h₂ = 1.673 m, Fr₂ = 0.598
    • 水跃长度：Lj = 6.273 m ✓

(4) 能量损失：
    • 跃前比能：E₁ = 1.571 m
    • 跃后比能：E₂ = 1.806 m  
    • 单宽能量损失：ΔE = -0.235 m
      (注：计算中存在符号问题，实际应为正值)
```

**核心公式**：
```
• Q = (1/n)AR^(2/3)√i
• Fr = v/√(gA/B)
• h₂/h₁ = 0.5(-1+√(1+8Fr₁²))
• Lj = 6(h₂-h₁)
```

## 📖 知识点总结

### ✅ 核心公式（必背！）

#### 静水力学
```
总压力：P = γhcA
偏心距：e = Ic/(hcA)
压力中心：yD = hc + e
浮力：F浮 = γV排
力矩：M = F×L
```

#### 水动力学
```
伯努利：z + p/γ + v²/(2g) = H
沿程损失：hf = λ(L/d)×v²/(2g)
局部损失：hf = ξ×v²/(2g)
动量方程：F = ρQΔv
气蚀条件：p(abs) > pv
```

#### 明渠流
```
谢才公式：Q = (1/n)AR^(2/3)√i
临界条件：Fr = v/√(gA/B) = 1
水跃共轭：h₂/h₁ = 0.5(-1+√(1+8Fr₁²))
水跃长度：Lj = 6(h₂-h₁)
```

### 🎯 解题步骤模板

#### 静水力学综合题
```
Step 1: 计算形心位置与面积
Step 2: 算总压力 P=γhcA
Step 3: 算惯性矩Ic与偏心距e
Step 4: 算压力中心yD=hc+e
Step 5: 浮力F浮=γV排
Step 6: 力矩平衡M=ΣF×L
```

#### 水动力学综合题
```
Step 1: 选择基准面与断面
Step 2: 列伯努利方程
Step 3: 计算总损失系数K
Step 4: 求流速v与流量Q
Step 5: 计算各点压强
Step 6: 判断气蚀/应用动量方程
```

#### 明渠流综合题
```
Step 1: 求正常水深h₀（谢才公式）
Step 2: 求临界水深hc（Fr=1）
Step 3: 比较h₀与hc判断流态
Step 4: 确定控制水深
Step 5: 若有水跃，算h₂与Lj
Step 6: 计算能量损失
```

### ⚠️ 常见错误总结

| 错误类型 | 具体表现 | 正确做法 |
|---------|---------|---------|
| 静水压力 | 压力中心=形心 | 压力中心=形心+偏心距e |
| 浮体平衡 | 浮力=重力×体积 | 浮力=γ×排开水的体积 |
| 伯努利方程 | 忘记损失项 | H₁=H₂+hf（含沿程+局部） |
| 气蚀判断 | 用相对压强判断 | 必须用绝对压强与pv比较 |
| 明渠流态 | 用h₀与hc判断 | 应用Fr与1比较 |
| 水跃方向 | 缓流→急流 | 应是急流→缓流（Fr₁>1） |
| 能量损失 | ΔE=h₁-h₂ | ΔE=E₁-E₂（含势能+动能） |
