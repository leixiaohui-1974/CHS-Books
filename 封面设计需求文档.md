## 🎯 必须元素

### 1. 书名（主标题）⭐⭐⭐⭐⭐

**文字**：
```
水力学考研核心100题
```

**要求**：
- 字体：黑体或思源黑体（加粗）
- 大小：最大，占封面30-40%
- 位置：上部1/3或居中
- 颜色：深蓝色或白色（看背景）
- 效果：可加轻微立体感

**强调**：
- "100题"要醒目
- 可用不同颜色或大小强调

---

### 2. 副标题 ⭐⭐⭐⭐

**文字**：
```
Python代码可视化详解
```

**要求**：
- 字体：细黑体或楷体
- 大小：主标题的30-40%
- 位置：主标题下方
- 颜色：与主标题协调

---

### 3. 卖点标签（3-5个）⭐⭐⭐⭐⭐

**必须包含**：
```
✓ 985/211名校真题精选
✓ 39个Python程序
✓ 三级难度分层
✓ 详细踩分点解析
```

**样式**：
- 可用图标+文字
- 或标签形式
- 或列表形式
- 颜色：橙色或黄色（醒目）

---

### 4. Python元素 ⭐⭐⭐⭐⭐

**位置**：
- Logo：右上角或左下角
- 代码：背景（半透明）
- 图表：底部装饰

### 5. 水利/水流元素 ⭐⭐⭐⭐
