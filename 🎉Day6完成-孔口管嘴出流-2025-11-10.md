## 📚 本日核心内容

### 1. 托里拆利公式 ⭐⭐⭐⭐⭐

**核心公式**：
```
v理论 = √(2gH)
```

**物理意义**：
- 理想流体从容器底部孔口流出的理论流速
- 等于自由落体从高度H落下时的速度
- 不考虑能量损失的极限情况

### 2. 孔口出流三个系数 ⭐⭐⭐⭐⭐

**流速系数 φ**：
```
φ = v实际/v理论 ≈ 0.97
```
- 反映能量损失
- 实际流速小于理论流速

**收缩系数 Cc**：
```
Cc = Ac/A0 ≈ 0.64（孔口）
Cc = 1.0（管嘴）
```
- 反映射流收缩程度
- 孔口有明显收缩，管嘴无收缩

**流量系数 μ**：
```
μ = φ × Cc
μ孔 ≈ 0.62
μ管 ≈ 0.82
```
- 综合系数，最实用
- 直接用于流量计算

### 3. 流量计算公式 ⭐⭐⭐⭐⭐

**统一公式**：
```
Q = μ × A0 × √(2gH)
```

**应用**：
- 孔口：Q孔 = 0.62 × A0 × √(2gH)
- 管嘴：Q管 = 0.82 × A0 × √(2gH)
- 管嘴流量约为孔口的1.3倍

### 例题1：薄壁孔口出流（基础题）⭐⭐⭐⭐⭐

**题目设置**：
- 孔口直径 d = 0.05 m
- 水头高度 H = 2.0 m
- 流速系数 φ = 0.97
- 收缩系数 Cc = 0.64

**计算结果**：
```
(1) 理论流速：
    v理论 = √(2gH) = 6.264 m/s ✓

(2) 实际流速：
    v = φ√(2gH) = 6.076 m/s ✓

(3) 断面积：
    A0 = 0.001963 m²
    Ac = Cc×A0 = 0.001257 m² ✓

(4) 流量系数：
    μ = φ×Cc = 0.621 ✓

(5) 出流流量：
    Q = μA0√(2gH) = 0.007636 m³/s ✓
    Q = 7.636 L/s
```

### 2. 物理分析深度

**托里拆利公式推导**：
```python
# 能量守恒：z1 + p1/(ρg) + v1²/(2g) = z2 + p2/(ρg) + v2²/(2g)
# 水面：z1=H, p1=p2=pa, v1≈0
# 出口：z2=0, v2=v
# 得到：H = v²/(2g)
# 解得：v = √(2gH)
v_theory = np.sqrt(2 * g * H)
```

**三个系数计算**：
```python
# 流速系数
phi = v_actual / v_theory  # ≈0.97

# 收缩系数
Cc = Ac / A0  # 孔口≈0.64, 管嘴=1.0

# 流量系数
mu = phi * Cc  # 孔口≈0.62, 管嘴≈0.82
```

**管嘴负压分析**：
```python
# 收缩断面流速
vc = v_theory / Cc  # 增大

# 能量方程：H + pc/(ρg) = vc²/(2g)
# 负压水头
hc = H - vc**2 / (2*g)  # 负值

# 负压值（真空度）
pc = rho * g * abs(hc)  # Pa
```

## 🎯 易错点与技巧

### 常见错误

1. ❌ **托里拆利公式少写系数2**
   - 错误：v = √(gH)
   - 正确：v = √(2gH)

2. ❌ **流量系数用加法**
   - 错误：μ = φ + Cc
   - 正确：μ = φ × Cc

3. ❌ **孔口管嘴系数搞混**
   - 错误：μ孔=0.82, μ管=0.62
   - 正确：μ孔=0.62, μ管=0.82

4. ❌ **管嘴长度无限制**
   - 错误：管嘴越长越好
   - 正确：l = (2~4)d 最佳

### 解题技巧

1. ✅ **记住「2、6、8」口诀**
   - 2：托里拆利有系数2（√2gH）
   - 6：孔口流量系数0.62
   - 8：管嘴流量系数0.82

2. ✅ **孔口有收缩，管嘴无收缩**
   - 孔口：Cc = 0.64（明显收缩）
   - 管嘴：Cc = 1.0（充满断面）

3. ✅ **管嘴流量约为孔口1.3倍**
   - 0.82/0.62 ≈ 1.32
   - 增加率约32%

4. ✅ **管嘴内有负压（真空吸力）**
   - 收缩断面处流速大
   - 产生负压（真空度）
   - 吸力增大流量
