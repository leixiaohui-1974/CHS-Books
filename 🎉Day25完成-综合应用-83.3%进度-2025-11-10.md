- **输水渠道**：梯形断面，含跌坎和水跃
- **提水泵站**：扬程15m，泵站工况匹配

**涉及知识点**（跨5个章节）：
1. 静水力学：闸门出流计算
2. 明渠均匀流：谢才公式求正常水深
3. 临界水深：流态判断
4. 水跃理论：共轭水深与能量损失
5. 泵站工程：扬程、功率、工况匹配

**计算结果**：
```
(1) 水库放水流量：
    • 闸门上游水深：17m
    • 放水流量：Q = 77.672 m³/s ✓

(2) 渠道输水与水跃：
    • 正常水深：h₀ = 2.556 m
    • 临界水深：hc = 1.255 m
    • 流态：缓流（Fr₀ = 0.593 < 1）
    
    水跃分析：
    • 跌坎后急流：h₁ = 0.879 m

    • 水跃长度：Lj = 5.799 m
    • 能量损失：ΔE = 0.581 m

(3) 泵站工况与功率：
    • 管道损失：hf = 9.720 m

    • 泵提供扬程：H泵 = -90.399 m（需调整）

```

```

```

#### 2. 逐步求解法
```
Step 1: 上游→下游
  • 水库→闸门→渠道→泵站

  • 闸门出流

  • 水跃计算
  • 泵站工况

  • Q₁ = Q₂ = Q₃
```

#### 3. 优化调度法
```
Step 1: 识别瓶颈
  • 流量过大/过小
  • 能量损失过大
  • 效率低下

Step 2: 提出方案
  • 调整闸门开度
  • 优化渠道断面
  • 改进泵站参数

Step 3: 评估效果
  • 系统效率
  • 经济性
  • 可行性
```
