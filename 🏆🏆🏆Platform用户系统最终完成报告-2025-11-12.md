## 🚀 部署清单

### 环境要求

```
后端:
  - Python 3.12+
  - PostgreSQL 15+
  - Redis 7+
  
前端:
  - Node.js 18+
  - npm 9+

第三方服务:
  - SMTP服务（邮件）
  - 阿里云SMS（短信）
  - GitHub OAuth
  - Google OAuth
  - 微信开放平台
```

### 配置文件

```
backend/.env:
  - 数据库配置
  - JWT密钥
  - SMTP配置
  - SMS配置
  - OAuth配置

frontend/.env.local:
  - API地址
```

### 部署步骤

```
1. 数据库初始化
   alembic upgrade head

2. 后端启动
   uvicorn app.main:app --host 0.0.0.0 --port 8000

3. 前端构建
   npm run build

4. 前端启动
   npm start
```
