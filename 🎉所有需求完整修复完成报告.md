## ⚠️ 重要提示

### 浏览器缓存
由于浏览器可能缓存了旧图片，您需要：

**方法1**：强制刷新
```
按 Ctrl+F5 (Windows)
按 Cmd+Shift+R (Mac)
```

**方法2**：清除缓存
```
1. 打开开发者工具(F12)
2. 右键刷新按钮
3. 选择"清空缓存并硬性重新加载"
```

**方法3**：重启服务器
```bash
# 停止当前服务器
Ctrl+C

# 重新启动
cd platform/backend
python full_server.py
```
