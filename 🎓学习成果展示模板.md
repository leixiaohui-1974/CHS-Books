### 项目3：地下水流场数值模拟

**项目简介**：
```
基于有限差分法的二维地下水稳定流和非稳定流数值模拟系统。
```

**技术栈**：
- NumPy（矩阵运算）
- SciPy（稀疏矩阵求解）
- Matplotlib（流场可视化）
- Numba（JIT加速）

**数学模型**：

**控制方程**（二维稳定流）：
```
∂/∂x(K ∂h/∂x) + ∂/∂y(K ∂h/∂y) + W = 0
```

**数值离散**（五点差分格式）：
```
K(h[i+1,j] - 2h[i,j] + h[i-1,j])/Δx² + 
K(h[i,j+1] - 2h[i,j] + h[i,j-1])/Δy² + W[i,j] = 0
```

**求解方法**：
- Gauss-Seidel迭代
- SOR超松弛迭代
- 共轭梯度法（CG）

**核心代码**：
```python
class GroundwaterSimulation:
    def solve_steady_state(self, h_boundary, W=None):
        """求解稳定流"""
        h = np.copy(h_boundary)
        
        for iter in range(self.max_iter):
            h_old = h.copy()
            
            for i in range(1, self.nx-1):
                for j in range(1, self.ny-1):
                    h[i,j] = 0.25 * (h[i+1,j] + h[i-1,j] + 
                                     h[i,j+1] + h[i,j-1])
                    if W is not None:
                        h[i,j] -= W[i,j] * self.dx**2 / (4*self.K)
            
            if np.max(np.abs(h - h_old)) < self.tol:
                break
        
        return h, iter
```

**项目成果**：
- 模拟区域：任意矩形区域
- 网格精度：可调（推荐50×50）
- 收敛速度：通常100-1000次迭代
- 支持边界：第一类、第二类、混合边界
- 支持源汇：抽水井、回灌井

**应用案例**：
1. 某地区地下水资源评价
2. 抽水井影响范围计算
3. 地下水污染扩散模拟
