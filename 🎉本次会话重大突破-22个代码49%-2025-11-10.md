### 核心知识点（全面覆盖）

#### 1. 达西定律（渗流基础）
```
公式: v = k·i
  • v: 渗流速度（m/s）
  • k: 渗透系数（m/s，土壤特性）
  • i: 水力坡度（dh/dr，无量纲）

物理意义:
  • 渗流速度与水力坡度成正比
  • 比例系数为渗透系数
  • 线性定律（层流条件下）
```

#### 2. 承压井流量计算
```
承压井特点:
  • 上下有不透水层
  • 水头降低但饱和厚度H不变
  • 压力承压，不形成自由水面

推导过程:
  1. 达西定律: v = k·(dh/dr)
  2. 流量连续: Q = 2πrH·v
  3. 积分: Q·ln(R/r₀) = 2πkH·s
  
公式: Q = 2πkHs / ln(R/r₀)

特点: Q与s成正比（线性关系）
```

#### 3. 无压井流量计算
```
无压井特点:
  • 上部为自由水面
  • 水头降低且饱和厚度h减小
  • 无压力承压，形成降落漏斗

推导过程:
  1. 达西定律: v = k·(dh/dr)
  2. 流量连续: Q = 2πrh·v（h变化）
  3. 积分: Q·ln(R/r₀) = πk(H² - h₀²)
  
公式: Q = πk(H² - h₀²) / ln(R/r₀)

特点: Q与(H²-h₀²)成正比（非线性）
```

#### 4. 水位降落曲线
```
承压井:
  s(r) = s · ln(R/r) / ln(R/r₀)
  • s与ln(r)成反比
  • 井壁处降深最大（s）
  • 影响半径处降深为0

无压井:
  h²(r) = h₀² + (H² - h₀²)·ln(R/r)/ln(R/r₀)
  s(r) = H - h(r)
  • h²与ln(r)成关系
  • 降深曲线非线性
```

#### 5. 渗流速度
```
公式: v = Q / (2πrh)

承压井:
  v = Q / (2πr₀H)
  • h = H（饱和厚度不变）

无压井:
  v = Q / (2πr₀h₀)
  • h₀ = H - s（饱和厚度减小）
  • 井壁速度更大
```

#### 6. 影响半径R
```
定义: 抽水影响的范围半径

影响因素:
  • 渗透系数k（越大，R越大）
  • 抽水时间t（越长，R越大）
  • 抽水量Q（越大，R越大）

经验公式:
  R = C√(k·t)
  C: 经验系数
```

### 计算步骤（详细9步）

#### 承压井计算（4步）
1. **应用承压井公式计算流量Q**
   - Q = 2πkHs / ln(R/r₀)
   - 代入H, k, s, R, r₀

2. **计算井壁渗流速度v**
   - v = Q / (2πr₀H)
   - 饱和厚度保持为H

3. **计算水位降落曲线s(r)**
   - s(r) = s · ln(R/r) / ln(R/r₀)
   - 给出任意r处的降深

4. **绘制降落漏斗剖面图**
   - 显示含水层、不透水层
   - 水位线、井筒

#### 无压井计算（4步）
1. **计算井壁水头h₀**
   - h₀ = H - s

2. **应用无压井公式计算流量Q**
   - Q = πk(H² - h₀²) / ln(R/r₀)

3. **计算井壁渗流速度v**
   - v = Q / (2πr₀h₀)
   - 饱和厚度为h₀

4. **计算水位降落曲线**
   - h²(r) = h₀² + (H² - h₀²)·ln(R/r)/ln(R/r₀)
   - s(r) = H - h(r)

### 技术难点突破

#### 1. 承压井与无压井区分
```
物理机制:
  • 承压井：压力承压，H不变
  • 无压井：自由水面，h变化

数学表达:
  • 承压井：Q∝s（一次）
  • 无压井：Q∝(H²-h₀²)（二次）

工程意义:
  • 承压井：流量大，但需要承压层
  • 无压井：流量相对小，但更常见
```

#### 2. 水位降落曲线计算
```
承压井（简单）:
  s(r) = s · ln(R/r) / ln(R/r₀)
  • 直接代入计算

无压井（复杂）:
  h²(r) = h₀² + (H² - h₀²)·ln(R/r)/ln(R/r₀)
  s(r) = H - √[h²(r)]
  • 需要开方运算
  • 注意h²的计算
```
