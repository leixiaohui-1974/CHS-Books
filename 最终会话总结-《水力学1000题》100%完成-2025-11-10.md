### 静水力学（7个核心公式）

```
1. p = p₀ + γh                     静水压强
2. P = γh_c × A                     平面总压力

4. P_x = γh_c × A_x                 曲面水平力
5. P_z = γV                         曲面垂直力
6. F_浮 = ρ_液 g V_排                浮力
7. GM = KM - KG                     初稳性高
```

### 水动力学（5个核心公式）

```
1. v₁A₁ = v₂A₂                     连续性方程
2. p/γ + v²/2g + z = C             伯努利方程

4. v = √(2gH)                       托里拆利
5. v = √(2gΔh)                      毕托管
```

### 管流（6个核心公式）

```
1. Re = vd/ν                        雷诺数
2. h_f = λ(L/d)(v²/2g)             达西公式
3. λ = 64/Re                        层流
4. λ = 0.3164/Re^0.25              湍流光滑管
5. h_m = ζ(v²/2g)                   局部阻力
6. Δp = ρcΔv                        水击
```

### 明渠流（6个核心公式）

```
1. Q = (1/n)AR^(2/3)i^(1/2)        曼宁公式
```
