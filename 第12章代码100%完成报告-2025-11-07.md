### 2.1 河床演变计算（题1）

**工程背景**：
河床演变是河流自然作用的结果，冲淤变化直接影响河道防洪、航运和水资源利用。

**核心计算**：
```
1. 挟沙能力：S* = K·V^m（张瑞瑾公式，K=0.05, m=2.5）
2. 冲淤量：ΔV = (S₀-S*)·Q·T/γₛ
3. 床面剪切力：τ₀ = γ·R·i
4. 起动判别：τ₀ > τc
```

**技术亮点**：
- 水流参数精确计算（曼宁公式反算水深）
- 挟沙能力与来沙量对比
- 时间序列演变过程
- 河床形态分类（Fr数）
- 平衡坡度分析

**计算实例**：
- 流量Q=1000m³/s，河宽B=200m
- 上游含沙量S₀=5kg/m³
- 挟沙能力S*=1.39kg/m³
- 冲淤类型：淤积（S₀>S*）
- 日淤积量：30.7万m³

### 2.2 弯曲河道水流分析（题4）

**工程背景**：
天然河道多呈弯曲形态，弯道水流具有横向水面比降和环流特征，凹岸冲刷、凸岸淤积。

**核心计算**：
```
1. 横向水面比降：J_s = v²/(g·R)
2. 环流强度：Γ = K·v²/R（K=0.25）
3. 弯曲度：P = L_c/L_v
4. 凹岸冲刷深：h_max = h·(1+α·P)（α=0.5）
```

**技术亮点**：
- 弯道几何参数精确建模
- 横向水位差计算
- 环流示意图可视化
- 冲淤分布分析
- 离心力与弯曲半径关系

**计算实例**：
- 弯曲半径R=500m，弯道角θ=90°
- 横向比降J_s=0.0002
- 横向水位差Δh=2cm
- 凹岸冲刷深h_max=5.14m
- 凸岸淤积深h_min=4.87m

### 2.4 泥沙输移计算（题10）

**工程背景**：
泥沙输移是河流的基本特性，分为推移质（床面运动）和悬移质（水体悬浮）。

**核心计算**：
```
1. 起动流速：v_c = K·√[(γₛ-γw)/γw·g·d]
2. Shields参数：θ = τ/(γₛ-γw)·d
3. 推移质：Q_b = K·√[(γₛ-γw)/γw·g·d³]·(θ-θ_c)^1.5
4. 悬移质：Q_s = S·Q
```

**技术亮点**：
- 泥沙起动判别（Shields图）
- 推移质公式（Einstein-Brown）
- 悬浮判别（Rouse数）
- 输沙率组成分析
- 粒径影响参数化

**计算实例**：
- 流速v=1.0m/s，起动流速vc=0.018m/s（泥沙起动）
- Shields参数θ=0.11 > θc=0.05（运动）
- 推移质Q_b=13.8kg/s（0.5%）
- 悬移质Q_s=3000kg/s（99.5%）
- 总输沙率Q_t=3013.8kg/s

## 四、知识体系

### 4.1 河床演变

**冲淤判别**：
```
S < S*：冲刷（来沙不足）
S = S*：平衡
S > S*：淤积（来沙过剩）
```

**演变类型**：
- 溯源冲刷：上游向下游发展
- 逐渐淤积：下游向上游发展
- 局部冲淤：河弯、汇流口

**影响因素**：
- 流量：Q↑ → S*↑ → 冲刷
- 坡度：i↑ → τ↑ → 冲刷
- 粒径：d↑ → τc↑ → 淤积

### 4.2 弯道水流

**形成机理**：
- 离心力：使水流指向凹岸
- 横向比降：凹岸水位高
- 环流：表层向凹岸，底层向凸岸

**冲淤特征**：
```
凹岸：主流顶冲 → 冲刷 → 深槽
凸岸：流速减小 → 淤积 → 边滩
```

**演变趋势**：
- 弯曲发展：P↑ → 河弯加剧
- 裁弯取直：P过大 → 新河道

### 4.3 河相关系

**经验系数（因地区而异）**：

| 河流类型 | K_B | b | K_h | f | K_v | m |
|---------|-----|---|-----|---|-----|---|
| 冲积平原 | 4.7 | 0.5 | 0.56 | 0.36 | 0.38 | 0.14 |
| 山区河流 | 3.5 | 0.45 | 0.40 | 0.40 | 0.70 | 0.15 |

**应用**：
- 河道规划：估算断面尺寸
- 洪水演算：推算水位
- 河势分析：稳定性评价

### 4.4 泥沙输移

**起动判别**：

| 方法 | 判别式 | 临界值 |
|-----|--------|--------|
| 流速法 | v vs vc | v > vc |
| 剪切力法 | τ vs τc | τ > τc |
| Shields法 | θ vs θc | θ > 0.05 |

**输沙组成**：
- 推移质：0-10%（粗颗粒）
- 悬移质：90-100%（细颗粒）
- 全沙：推移质+悬移质

**影响因素**：
- 流速：v↑ → Q_t↑
- 水深：h↑ → Q_s↑
- 含沙量：S↑ → Q_s↑

### 4.5 河流治理

**治理原则**：
1. **防洪安全第一**：K ≥ 1.2
2. **生态优先**：缓坡护岸
3. **因地制宜**：适应当地条件
4. **综合治理**：上中下游统筹

**护岸类型**：

| 类型 | 适用条件 | 优点 | 缺点 |
|-----|---------|------|------|
| 硬质护岸 | 冲刷强烈 | 强度高 | 造价高、不生态 |
| 软质护岸 | 冲刷较弱 | 造价低 | 强度低 |
| 生态护岸 | 一般河段 | 生态好 | 需维护 |

**泥沙治理**：
- 上游：水土保持、植树造林
- 中游：淤地坝、梯田
- 下游：疏浚清淤、调水调沙
