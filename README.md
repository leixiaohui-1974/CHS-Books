# CHS-Books

水利系统建模、仿真与控制系列丛书（Canal and Hydraulic Systems Series）

## 系列简介

本系列丛书旨在系统地介绍水利系统（包括明渠系统和有压管道系统）的建模、仿真与控制理论及工程实践。全系列由浅入深，从基础的单元系统到复杂的大规模系统，从经典建模方法到现代智能控制技术，为水利工程、自动化控制等领域的研究人员、工程师和学生提供全面的理论指导和实践参考。

## 系列书籍

### 第一部：水箱系统控制
**关键词**：基础控制、单输入单输出、经典方法

本书作为系列的入门篇，以水箱系统为对象，介绍控制理论的基本概念和方法。水箱系统具有结构简单、物理意义清晰的特点，是学习控制理论的理想入门案例。

**主要内容**：
- 水箱系统的数学建模
- PID控制设计与调试
- 状态空间方法
- 数字控制实现
- 实验系统搭建

**适用对象**：
- 控制工程初学者
- 水利工程本科生
- 自动化专业学生

---

### 第二部：明渠和有压系统的仿真
**关键词**：水力学建模、数值仿真、分布参数系统

本书深入介绍明渠系统和有压管道系统的数学建模与数值仿真方法，是从简单集总参数系统过渡到复杂分布参数系统的桥梁。

**主要内容**：
- 圣维南方程组及其简化
- 有压管流方程与水锤理论
- 数值离散方法（有限差分、有限体积、有限元）
- 仿真软件应用（MIKE、InfoWorks、EPANET等）
- 边界条件处理
- 复杂工况模拟
- 模型验证与校准

**适用对象**：
- 水利工程研究生
- 仿真工程师
- 水利系统设计人员

**目录**：见 [book2-simulation/outline.md](book2-simulation/outline.md)（待完善）

---

### 第三部：明渠和有压系统的控制
**关键词**：先进控制、智能控制、数字孪生

本书是系列的核心篇章，系统介绍明渠和有压系统的控制理论与工程实践，涵盖从经典控制到现代智能控制的完整知识体系，并引入数字孪生技术，代表了水利系统控制的最新发展方向。

**主要内容**：
- **模型降阶**：POD、DMD、Koopman算子等现代降阶方法
- **系统辨识**：从经典方法到子空间辨识、神经网络辨识
- **经典控制**：PID、频域设计、根轨迹法
- **现代控制**：最优控制（LQR/LQG/MPC）、鲁棒控制（H∞、滑模）、自适应控制
- **智能控制**：模糊控制、神经网络控制、强化学习
- **数字孪生**：孪生建模、数据融合、预测性维护、智能运维
- **工程实践**：大量实际工程案例

**适用对象**：
- 水利工程研究生、博士生
- 控制工程研究人员
- 水利系统工程师
- 智慧水务从业者

**详细目录**：见 [book3-control-systems/outline.md](book3-control-systems/outline.md)

## 系列特色

### 1. 体系完整
从基础到前沿，从理论到实践，形成完整的知识体系：
- **第一部**：控制理论入门，建立基本概念
- **第二部**：水利系统建模与仿真，掌握系统特性
- **第三部**：先进控制方法，解决实际工程问题

### 2. 循序渐进
- 从简单集总参数系统（水箱）到复杂分布参数系统（明渠、管道）
- 从单一控制目标到多目标优化
- 从单一系统到大规模互联系统

### 3. 理论与实践结合
- 严谨的数学推导
- 丰富的工程案例
- 实用的仿真代码
- 详细的实施步骤

### 4. 紧跟前沿
- 模型降阶技术
- 深度学习与强化学习
- 数字孪生技术
- 智能运维

### 5. 多学科交叉
融合了以下学科的知识：
- 水力学
- 自动控制理论
- 系统工程
- 计算机科学
- 人工智能
- 运筹学

## 学习路径建议

### 路径一：控制理论学习者

2. **第二部**（选读）：了解水利系统特性

1. **第二部** → **第三部**：全面学习
2. 重点关注：模型降阶、智能控制、数字孪生章节
