## 📚 项目简介

CHS-Books是一个**多领域工程教材系列**，采用**案例驱动学习**（Case-Based Learning）方法，涵盖**水利水电**和**新能源**两大领域，目前已完成**15本教材**，包含**500+工程案例**，**100,000+行代码**。

### 🌟 核心特色

- ✅ **案例驱动**: 每个知识点通过真实工程案例学习
- ✅ **代码实现**: 所有案例都有完整Python代码
- ✅ **系统完整**: 从基础到高级，从单一到综合
- ✅ **开源共享**: 全部代码和文档开源
- ✅ **教学导向**: 可直接用于本科生/研究生教学
