## 📚 本日核心内容

### 1. 连续性方程（质量守恒）⭐⭐⭐⭐⭐

**核心公式**：
```
Q = A₁v₁ = A₂v₂ = 常数
```

**物理意义**：
- 质量守恒：单位时间通过各断面的流量相等
- 断面积与流速成反比：A↓ → v↑
- 适用条件：不可压缩流体

**应用场景**：
- ✅ 渐变管流速计算
- ✅ 文丘里管流量测量
- ✅ 管道系统流速分布

### 2. 动量方程（矢量形式）⭐⭐⭐⭐⭐

**核心公式**：
```
F⃗ = ρQ(v⃗₂ - v⃗₁)

分量形式：
Fₓ = ρQ(v₂ₓ - v₁ₓ)
Fᵧ = ρQ(v₂ᵧ - v₁ᵧ)

合力：
F = √(Fₓ² + Fᵧ²)
```

**物理意义**：
- 动量守恒：流体对边界的作用力
- 矢量方程：必须分解为分量
- 作用力与反作用力

**应用场景**：
- ✅ 弯管受力计算
- ✅ 射流冲击力
- ✅ 水轮机叶片受力

## 🎯 易错点与技巧

### 常见错误

1. ❌ **连续性方程用质量流量**
   - 错误：Qₘ = ρ₁A₁v₁ = ρ₂A₂v₂
   - 正确：Q = A₁v₁ = A₂v₂（不可压缩流体）

2. ❌ **动量方程忘记分解速度矢量**
   - 错误：F = ρQ|v₂ - v₁|（标量）
   - 正确：Fₓ = ρQ(v₂ₓ - v₁ₓ), Fᵧ = ρQ(v₂ᵧ - v₁ᵧ)

3. ❌ **力的方向判断错误**
   - 错误：不看Fₓ、Fᵧ正负
   - 正确：Fₓ<0向左，Fₓ>0向右；Fᵧ<0向下，Fᵧ>0向上

4. ❌ **作用力与反作用力搞混**
   - 错误：不区分水对管 vs 管对水
   - 正确：作用力与反作用力方向相反

### 解题技巧

1. ✅ **连续性方程：A↓→v↑**
   - 断面积减小 → 流速增加
   - 成反比关系：A₂/A₁ = v₁/v₂

2. ✅ **动量方程一定要画坐标系**
   - 建立直角坐标系
   - 明确x、y方向

3. ✅ **速度矢量要分解**
   - 水平：vₓ = v cosθ
   - 竖直：vᵧ = v sinθ

4. ✅ **力的方向看Fₓ、Fᵧ的正负**
   - Fₓ正负 → 左右方向
   - Fᵧ正负 → 上下方向
