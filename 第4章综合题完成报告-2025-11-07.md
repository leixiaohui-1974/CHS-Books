### 示例1：题目11（地下水资源综合评价）节选

**题目描述**：
某冲积平原地下水水源地，面积100 km²，含水层厚度$M = 50$ m，渗透系数$k = 30$ m/d，给水度$\mu = 0.15$。年均降雨$P = 600$ mm，入渗系数$\alpha = 0.20$。现状开采井群100眼，单井流量$Q_0 = 50$ m³/h，常年抽水。已观测到水位下降$s = 5$ m，降落漏斗半径$R = 5$ km。下游城市规划新增水量需求$Q_{需} = 5000$万m³/年。环境要求保证河道生态基流$Q_{生态} = 0.5$ m³/s...

**解答要点**（部分）：

**(1) 地下水资源总量评估**

**降雨入渗补给**：
$$W_{补} = P \times \alpha \times A = 600 \times 0.20 \times 100 = 1.2 \times 10^7 \text{ m}^3/\text{年}$$

**侧向径流补给**（假设上游来水$Q_{侧} = 200$ L/s）：
$$W_{侧} = Q_{侧} \times T = 0.2 \times 365 \times 86400 = 6.3 \times 10^6 \text{ m}^3/\text{年}$$

**总补给量**：
$$W_{总补} = W_{补} + W_{侧} = 1.2 + 0.63 = 1.83 \times 10^7 \text{ m}^3/\text{年}$$

**(2) 可开采资源量**

考虑生态用水约束：
$$Q_{生态} = 0.5 \text{ m}^3/\text{s} = 1.58 \times 10^7 \text{ m}^3/\text{年}$$

可开采量（不考虑储存量变化）：
$$W_{可采} = W_{总补} - W_{生态} = 1.83 - 1.58 = 0.25 \times 10^7 \text{ m}^3/\text{年} = 250万m³/年$$

**(考虑到总补给量1830万m³/年 < 生态需求1580万m³/年，说明该地区地下水资源已经不足以维持当前生态用水，需要重新评估。实际上这里应该是数据设定有误，或者需要多水源联合调度。)**

### 示例2：题目14（地下水污染修复）节选

**题目描述**：
某化工厂地下储罐泄漏，污染物为苯（C₆H₆），初始浓度$C_0 = 500$ mg/L，污染源面积$A_0 = 100$ m²。含水层厚度$M = 10$ m，渗透系数$k = 15$ m/d，水力坡度$I = 0.002$，孔隙度$n = 0.30$。地下水流速$u = kI/n = 0.1$ m/d。污染羽已向下游扩散$L = 200$ m，最大宽度$W = 50$ m，最大浓度$C_{max} = 100$ mg/L...

**解答要点**（部分）：

**(1) 污染羽迁移模拟**

**实际流速**：
$$u = \frac{k \times I}{n} = \frac{15 \times 0.002}{0.30} = 0.1 \text{ m/d} = 36.5 \text{ m/年}$$

**一维对流-扩散方程**（稳态）：
$$u \frac{\partial C}{\partial x} = D_L \frac{\partial^2 C}{\partial x^2} - \lambda C$$

其中：
- $D_L = \alpha_L \times u$ (纵向弥散系数，$\alpha_L = 10$ m)
- $\lambda = 0.1$/年（生物降解速率常数）

**解析解**（一维）：
$$C(x,t) = C_0 \exp\left(-\frac{(x-ut)^2}{4D_L t} - \lambda t\right)$$

**二维高斯羽流模型**（更实用）：
$$C(x,y) = \frac{M_0}{4\pi n M \sqrt{D_L D_T}} \exp\left(-\frac{(x-ut)^2}{4D_L t} - \frac{y^2}{4D_T t} - \lambda t\right)$$

...(详细计算省略)...

**(2) 抽出处理方案（Pump & Treat）**

**抽水井布置**：
- 在污染羽下游边界布置3眼抽水井
- 井距$d = 20$ m
- 单井流量$Q = 20$ m³/h（总抽水量60 m³/h）

**修复时间估算**：
$$T_{修复} = \frac{V_{污染} \times n \times R_f}{Q_{总}} = \frac{200 \times 50 \times 10 \times 0.30 \times 3}{60 \times 24} = 62.5 \text{ 天}$$

其中$R_f = 3$为冲洗倍数（需抽取3倍孔隙体积）
