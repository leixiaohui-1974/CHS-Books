## 💡 核心知识点

### 1. 重力坝稳定分析 ⭐⭐⭐⭐⭐

#### 关键公式
```
• 水压力：P = (1/2)γwh²
  作用点：y = h/3

• 抗滑稳定：Ks = fW/(P₁-P₂) ≥ 1.05
  - f: 摩擦系数
  - W: 坝体自重
  - P₁, P₂: 上下游水压力

• 抗倾覆稳定：K₀ = ΣMS/ΣM₀ ≥ 1.50
  - MS: 稳定力矩（对坝踵）
  - M₀: 倾覆力矩（对坝踵）
```

#### 解题步骤
```
Step 1: 计算坝体自重W
        - 几何形状（三角形、梯形）
        - W = γc×A×L

Step 2: 计算水压力P₁、P₂
        - P = (1/2)γwh²
        - 作用点y = h/3

Step 3: 抗滑验算
        - FR = fW (抗滑力)
        - FS = P₁-P₂ (滑动力)
        - Ks = FR/FS ≥ 1.05

Step 4: 抗倾覆验算
        - 对坝踵取矩
        - MS = W×xW
        - M₀ = P₁×y₁ - P₂×y₂
        - K₀ = MS/M₀ ≥ 1.50
```

### 2. 土石坝渗流计算 ⭐⭐⭐⭐⭐

#### 关键公式
```
• 浸润线出逸点：
  y₀ ≈ 0.4√(h₁L)
  - h₁: 上游水深
  - L: 渗流长度
  - y₀: 出逸点高度（经验公式）

• 单宽渗流量：
  q = k(h₁²-y₀²)/(2L)
  - k: 渗透系数
  - q: 单位坝长渗流量 [m³/(s·m)]

• 排水设施影响：
  - 排水棱体：y₀' ≈ y₀/2
  - 降低出逸点
  - 增加下游坡稳定
```

#### 解题步骤
```
Step 1: 确定渗流长度
        L = B₀ + m₁H + m₂H
        - B₀: 坝顶宽
        - m₁, m₂: 上下游坡度

Step 2: 计算出逸点
        y₀ = 0.4√(h₁L)

Step 3: 计算渗流量
        q = k(h₁²-y₀²)/(2L)

Step 4: 排水设施分析
        - 新出逸点y₀'
        - 新渗流量q'
        - 对比分析
```

## 💪 关键知识点记忆

### 重力坝稳定（必背！）
```
抗滑口诀：
  「摩擦抗滑，净压推动」
  FR = fW（摩擦力）
  FS = P₁-P₂（净水平力）
  Ks = FR/FS ≥ 1.05

抗倾口诀：
  「对踵取矩，稳定除倾」
  MS = W×xW（稳定力矩）
  M₀ = P₁×y₁-P₂×y₂（倾覆力矩）
  K₀ = MS/M₀ ≥ 1.50
```

### 土坝渗流（必背！）
```
浸润线口诀：
  「四分根号，水深长度」
  y₀ = 0.4√(h₁L)
  - 0.4是经验系数
  - h₁是上游水深
  - L是渗流长度

渗流量口诀：
  「平方差除二倍长，乘以系数得流量」
  q = k(h₁²-y₀²)/(2L)
  - k是渗透系数
  - 平方差：h₁²-y₀²
  - q是单宽流量
```

## ⚠️ 易错点总结

### 重力坝题易错点
1. **水压力作用点**
   - ❌ 错误：在h/2处
   - ✅ 正确：在h/3处（距底部）

2. **抗滑公式**
   - ❌ 错误：Ks = (P₁-P₂)/(fW)
   - ✅ 正确：Ks = fW/(P₁-P₂)

3. **抗倾覆取矩点**
   - ❌ 错误：对坝趾取矩
   - ✅ 正确：对坝踵取矩

4. **规范要求**
   - ❌ 错误：Ks≥1.5, K₀≥1.05
   - ✅ 正确：Ks≥1.05, K₀≥1.50

### 土坝渗流易错点
1. **出逸点公式**
   - ❌ 错误：y₀=0.4(h₁+L)
   - ✅ 正确：y₀=0.4√(h₁L)

2. **渗流量公式**
   - ❌ 错误：q=k(h₁-y₀)/(2L)
   - ✅ 正确：q=k(h₁²-y₀²)/(2L)（注意平方）

3. **渗透系数单位**
   - ❌ 错误：不转换，直接用cm/s
   - ✅ 正确：转换为m/s（×0.01）

4. **排水效果**
   - ❌ 错误：降低出逸点且减小渗流量
   - ✅ 正确：降低出逸点但增大水力梯度
