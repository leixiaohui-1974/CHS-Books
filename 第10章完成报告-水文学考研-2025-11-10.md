### 题目1：水资源量计算 ★★★★★（28分）

**知识点覆盖**：
1. **水资源量计算原理**
   - 水资源量定义：W_total = Ws + Wg - W_repeat
   - 地表水资源量：河川径流总量
   - 地下水资源量：地下水补给总量
   - 重复水量：地表地下重复计算部分（基流）

2. **地表水资源量**
   - 计算方法：Ws = R × A × 10⁶
   - 径流组成法：Ws = 地表径流 + 地下径流
   - 水量平衡验证

3. **地下水资源量**
   - 补给量计算：Wg = 降雨入渗 + 河道渗漏
   - 动态储量变化：ΔSg = 补给 - 排泄
   - 水量平衡验证

4. **重复水量**
   - 基流法：W_repeat = 基流
   - 取小值法：W_repeat = min(地下→河, 河→地下)
   - 避免双重计算

5. **水资源模数**
   - 定义：单位面积水资源量
   - 单位：mm 或 万m³/km²
   - 丰富程度评价分级

**案例数据**：
- 流域面积：5000 km²
- 年均降水：800 mm
- 年均径流深：300 mm
- 水资源总量：15.0×10⁸ m³
- 水资源模数：30 万m³/km²（中等）

**核心结论**：
- 地表水资源量：14.0×10⁸ m³
- 地下水资源量：5.0×10⁸ m³
- 重复水量：4.0×10⁸ m³
- 水资源总量：15.0×10⁸ m³
- 丰富程度：中等（半干旱区）

### 题目2：水资源承载力分析 ★★★★★（28分）

**知识点覆盖**：
1. **承载力概念**
   - 定义：水资源对人口和经济的最大支撑能力
   - 核心要素：承载对象、承载主体、约束条件、时空特征
   - 类型：人口承载力、经济承载力、生态承载力、综合承载力

2. **承载力计算**
   - 人口承载力：P_cap = W_usable / q_capita
   - 经济承载力：G_cap = W_usable / q_gdp × 10⁴
   - 可用水量：W_usable = W_available + W_transfer - W_eco

3. **承载指数法**
   - 人口承载指数：I_pop = P_current / P_cap
   - 经济承载指数：I_gdp = G_current / G_cap
   - 评价标准：
     - I < 0.6：未超载
     - 0.6 ≤ I < 0.8：轻度超载
     - 0.8 ≤ I < 1.0：中度超载
     - I ≥ 1.0：严重超载

4. **承载潜力分析**
   - 承载潜力 = 承载力 - 现状值
   - 潜力率 = 潜力 / 现状值 × 100%

5. **对策建议**
   - 需求管理：节水、产业调整
   - 供给管理：水源开发、外调水
   - 生态保护：生态需水保障
   - 管理措施：用水总量控制、水价改革

**案例数据**：

**2020年现状**：
- 水资源总量：5.0×10⁸ m³
- 可用水量：3.5×10⁸ m³
- 人口：100万人
- GDP：800亿元
- 人口承载力：140万人（I=0.714，轻度超载）
- 经济承载力：1120亿元（I=0.714，轻度超载）

**2030年预测**：
- 可用水量：3.2×10⁸ m³（生态需水增加）
- GDP：1432.6亿元（年增6%）
- 人口承载力：160万人（节水成效）
- 经济承载力：1600亿元
- 经济承载指数：I=0.895（中度超载，压力增大！）

**核心结论**：
- 节水成效显著（用水定额下降）
- 经济快速增长导致水资源压力增大
- 从轻度超载向中度超载发展
- 存在水资源约束经济发展的风险
