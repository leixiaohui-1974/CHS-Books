## 🚀 部署说明

### 系统要求

**最低要求**
- Python 3.8+
- 2GB RAM
- 1GB 磁盘空间

**推荐配置**
- Python 3.10+
- 4GB+ RAM
- 2GB+ 磁盘空间
- 现代浏览器

### 部署步骤

1. **安装依赖**
```bash
cd platform/backend
pip install -r requirements.txt
```

2. **启动服务**
```bash
python full_server.py
```

3. **访问系统**
```
http://localhost:8000/
```

### 端口配置

默认端口：8000  
可在 `full_server.py` 中修改
