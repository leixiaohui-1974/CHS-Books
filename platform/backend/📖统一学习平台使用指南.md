## 📚 使用教程

### 第一步：选择教材

在左侧教材目录中，点击任意教材名称：

```
📖 运河模拟系统 - 功能索引 ▶  ← 点击展开
📖 明渠水力学计算 ▶
📖 水系统控制论 ▶
...
```

**提示**: 
- ▶ 表示未展开
- ▼ 表示已展开

---

### 第二步：浏览章节

教材展开后，会看到完整的章节树：

```
📖 运河模拟系统 - 功能索引 ▼
  ├─ 1 运河模拟系统 - 功能索引
  ├─ 1.1 核心仿真功能
  ├─ 1.2 控制算法 💡3        ← 有3个配套案例
  │  ├─ 1.2.1 基础控制 💡3   ← 点击查看
  │  ├─ 1.2.2 高级控制 💡3
  │  └─ 1.2.3 分布式控制 💡3
  ├─ 1.3 智能功能
  └─ ...
```

**标识说明**:
- 💡 图标：表示该章节有配套案例
- 💡 后的数字：配套案例的数量

---

### 第三步：学习内容

点击任意章节后，右侧会显示：

#### 📝 章节信息
```

标题: 1.2.1 基础控制
元数据: 📝 127字 | 🎯 初级 | 💡 3个配套案例
```

#### 📖 章节正文
- Markdown格式的理论知识
- 支持标题、列表、代码块、表格等

#### 💡 配套实践案例（如果有）
每个案例卡片包含：
- 🎨 图标和标题
- 🏷️ 类型标签（理论验证/实践练习/扩展学习）
- ⭐ 相关度评分
- 📝 简短描述
- 🔘 操作按钮：
  - 📖 查看文档
  - ▶️ 运行代码

## 🛠️ 功能说明

### 教材目录面板（左侧）
- **展开/折叠**: 点击教材名称
- **查看章节**: 点击章节标题
- **识别案例**: 寻找💡图标
- **折叠面板**: 点击顶部"📁 目录"按钮或按`Ctrl+B`

### 内容显示区域（中间）
- **面包屑导航**: 显示当前位置
- **章节元数据**: 字数、难度、案例数
- **Markdown渲染**: 自动格式化内容

### AI助手面板（右侧）
- **智能问答**: 输入问题获取解答
- **学习建议**: 根据内容提供指导
- **折叠面板**: 点击顶部"🤖 AI助手"按钮或按`Ctrl+J`

## ❓ 常见问题

### Q1: 为什么有些章节没有内容？
**A**: 部分章节正在开发中，会显示"暂无内容"。这不影响其他章节的学习。

### Q2: 💡图标的数字是什么意思？
**A**: 数字表示该章节配套的案例数量。例如"💡3"表示有3个相关案例。

### Q3: 如何知道哪些章节有实践案例？
**A**: 查看章节标题旁边是否有💡图标。有图标的章节都配有实践案例。

### Q4: 案例按钮点击后会怎样？
**A**: 
- 📖 查看文档：跳转到案例的详细说明页面
- ▶️ 运行代码：打开代码编辑器并运行案例
（注：当前版本按钮功能待完善）

### Q5: 可以搜索章节吗？
**A**: 章节搜索功能正在开发中。当前建议通过教材目录浏览查找。

### Q6: 如何返回首页？
**A**: 点击顶部的"🏠 返回首页"按钮。
