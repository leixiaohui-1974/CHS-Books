## 🐛 常见问题

### Q: 启动失败怎么办？
**A**: 检查端口8000是否被占用
```bash
# Windows
netstat -ano | findstr :8000

# Linux/Mac
lsof -i :8000
```

### Q: 找不到某个模块？
**A**: 安装依赖
```bash
pip install fastapi uvicorn sqlalchemy
```

### Q: 代码执行超时？
**A**: 正常，默认30秒超时保护机制

### Q: 图片不显示？
**A**: 清除浏览器缓存，强制刷新（Ctrl+Shift+R）

### Q: 进度没有保存？
**A**: 检查 `platform/backend/data/progress/` 目录权限
