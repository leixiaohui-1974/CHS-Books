## 🎓 平台简介

### 什么是Platform?

Platform是一个现代化的在线学习平台，专为工程类专业学生设计。它集成了：

- 📚 **专业教材** - 18本精选教材，622个章节
- 💻 **代码运行器** - 在线编写和运行Python代码
- 🤖 **AI助手** - 智能辅导和答疑
- 📖 **知识库** - 专业知识图谱

### 适用对象

- 🎓 **学生**: 水系统控制、水力学等专业学生
- 👨‍🏫 **教师**: 课程教学辅助工具
- 💼 **工程师**: 专业知识查询和学习
- 🔬 **研究人员**: 理论学习和代码实验

#### 示例代码

**示例1: Hello World**
```python
print("Hello, Platform!")
```

**示例2: 水池液位计算**
```python
import numpy as np

# 水池参数
length = 10  # 米
width = 8    # 米
height = 5   # 米

# 计算容积
volume = length * width * height
print(f"水池容积: {volume} 立方米")

# 当前液位
current_level = 3.5  # 米
current_volume = length * width * current_level
print(f"当前水量: {current_volume} 立方米")
print(f"充满度: {(current_level/height)*100:.1f}%")
```

**示例3: 数据可视化**
```python
import matplotlib.pyplot as plt
import numpy as np

# 生成数据
t = np.linspace(0, 10, 100)
level = 2 * np.sin(t) + 5

# 绘制图形
plt.figure(figsize=(10, 6))
plt.plot(t, level, 'b-', linewidth=2)
plt.xlabel('时间 (s)')
plt.ylabel('液位 (m)')
plt.title('水池液位变化曲线')
plt.grid(True)
plt.show()
```

## ❓ 常见问题

### Q1: 代码运行失败怎么办？

**A**: 检查以下几点：
1. 语法是否正确
2. 是否缺少导入语句
3. 变量名是否拼写错误
4. 查看错误信息提示

### Q2: 如何保存学习进度？

**A**: 
- 使用浏览器书签
- 代码运行器自动保存
- 记录笔记和代码

### Q3: AI助手无响应？

**A**: 
1. 检查网络连接
2. 刷新页面重试
3. 简化问题重新提问

### Q4: 页面加载缓慢？

**A**:
1. 清理浏览器缓存
2. 检查网络速度
3. 关闭不必要的标签页

### Q5: 如何导出代码？

**A**:
1. 选中代码
2. 复制到剪贴板 (Ctrl+C)
3. 粘贴到本地文件 (Ctrl+V)
