## 🚀 快速开始

### 1分钟体验

```bash
# 克隆项目
git clone <repository>
cd platform

# 诊断系统
python3 backend/system_diagnostics.py

# 查看教程
cat TUTORIAL.md
```

### 5分钟部署

```bash
# 1. 配置
python3 backend/setup_wizard.py

# 2. 部署
python3 deploy.py

# 3. 验证
python3 backend/health_check.py
```

### 10分钟学习

## 💻 技术栈

### 后端

- **框架**: FastAPI
- **语言**: Python 3.11+
- **数据库**: PostgreSQL, MongoDB
- **缓存**: Redis
- **容器**: Docker

### 前端

- **框架**: React + TypeScript
- **UI**: Ant Design
- **状态**: React Hooks
- **编辑器**: Monaco Editor

### DevOps

- **容器化**: Docker Compose
- **监控**: 自研工具链
- **日志**: 结构化日志
- **测试**: pytest + 集成测试
