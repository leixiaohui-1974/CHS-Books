## 💻 技术亮点

### 1. 复杂水跃-堰流耦合
```python
# 水跃共轭
h2 = h1/2 * (-1 + np.sqrt(1 + 8*Fr1**2))

# 堰流计算
H_weir = h2 - P
Q_weir = m_weir * b * np.sqrt(2*g) * H_weir**1.5

# 连续性验证
error = abs(Q_weir - Q) / Q * 100
```

### 2. 地下水井降落漏斗
```python
# 裘布依公式
s = Q / (2 * np.pi * k * M) * np.log(R / r)

# 降落漏斗可视化
r_array = np.linspace(r0, R, 100)
s_array = Q / (2*np.pi*k*M) * np.log(R / r_array)
```

### 3. BOD-DO模型
```python
# 临界时间
tc = 1/(k2-k1) * np.log((k2/k1) * (1 - D0*(k2-k1)/(k1*BOD0)))

# DO曲线
for t in t_array:
    D_t = (k1*BOD0)/(k2-k1) * (np.exp(-k1*t) - np.exp(-k2*t)) + D0*np.exp(-k2*t)
    DO_t = DOs - D_t
```

浸润线：h = √[H₁²-(H₁²-H₂²)x/L]

#### 泵站环境
```
泵站扬程：H = Hs + hf + v²/(2g)
轴功率：N = γQH/η
混合浓度：Cm = (QrCr + QpCp)/Qm
BOD-DO：D(t) = (k₁BOD₀)/(k₂-k₁) × (e^(-k₁t) - e^(-k₂t))
临界时间：tc = 1/(k₂-k₁) × ln[(k₂/k₁)×(1-D₀(k₂-k₁)/(k₁BOD₀))]
```
