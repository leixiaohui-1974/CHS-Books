## 📚 核心公式汇总

### Top 5必背公式

1. **串联管道**
   ```
   Q = Q₁ = Q₂ = Q₃ = ...
   H = h₁ + h₂ + h₃ + ...
   ```

2. **并联管道**
   ```
   Q = Q₁ + Q₂ + Q₃ + ...
   h₁ = h₂ = h₃ = ... = H
   ```

3. **能量方程**
   ```
   z₁ + p₁/(ρg) + v₁²/(2g) = z₂ + p₂/(ρg) + v₂²/(2g) + hf
   ```

4. **达西公式**
   ```
   hf = λ(L/d)(v²/2g)
   ```

5. **流量分配**
   ```
   Q₁/Q₂ = √(K₂/K₁)
   其中 K = λ(L/d)/A²
   ```

### 串并联对比表

| 类型 | 流量关系 | 阻力关系 | 特点 |
|------|----------|----------|------|
| 串联 | Q₁=Q₂=Q | H=h₁+h₂ | 阻力大 |
| 并联 | Q=Q₁+Q₂ | h₁=h₂=H | 阻力小 |

## ⚠️ 记忆口诀与技巧

### 记忆口诀

**🎵 「串并联速记」**
```
串联：Q等H加（Q₁=Q₂, H=h₁+h₂）
并联：Q加H等（Q=Q₁+Q₂, h₁=h₂）
```

**🎵 「并联阻力小」**
```
串联阻力大，一段接一段
并联阻力小，分流更通畅
```

### 解题技巧

**串联管道4步法**
1. Step 1: 由Q计算各段流速 vi=Q/Ai
2. Step 2: 计算各段阻力 hi=λi(Li/di)(vi²/2g)
3. Step 3: 累加总阻力 H=Σhi
4. Step 4: 应用能量方程求未知量

**并联管道4步法**
1. Step 1: 建立h₁=h₂方程
2. Step 2: 推导流量分配 Q₁/Q₂=√(K₂/K₁)
3. Step 3: 结合Q₁+Q₂=Q求解Q₁、Q₂
4. Step 4: 计算总阻力 H=h₁=h₂

### 常见错误

❌ **错误1**：串联并联特点搞反
✓ 记忆：串Q并H（串联Q相等，并联H相等）

❌ **错误2**：能量方程漏项或符号错误
✓ 技巧：从左到右逐项写出，阻力项用减号

❌ **错误3**：并联流量分配公式记错
✓ 技巧：从h₁=h₂推导，Q₁/Q₂=√(K₂/K₁)

❌ **错误4**：忘记质量守恒Q₁+Q₂=Q
✓ 技巧：画图标注流量，检查守恒
