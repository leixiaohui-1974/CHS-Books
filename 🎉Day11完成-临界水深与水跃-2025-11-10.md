## 📊 核心知识点

### 1. 临界水深公式（必背！）

**矩形断面**：
```
hc = (q²/g)^(1/3) = (Q²/(gb²))^(1/3)
```

**临界流条件**：
- Fr = 1
- v = √(gh)
- E = Emin

**流态判断**：
- h > hc → 缓流（Fr < 1）
- h < hc → 急流（Fr > 1）
- h = hc → 临界流（Fr = 1）

### 2. 水跃共轭水深公式（必考！）⭐⭐⭐⭐⭐

```
h₂ = (h₁/2) × [-1 + √(1 + 8Fr₁²)]
```

**注意**：
- 公式中是 **[-1 + √...]**，不是 [1 + √...]
- 必须满足 Fr₁ > 1 才能发生水跃
- 这是考研必考公式，必须背熟！

### 3. 水跃能量损失

```
ΔE = E₁ - E₂ = (h₂ - h₁)³/(4h₁h₂)
```

## 🎯 例题精选

### 例题1：临界水深计算

**题目**：矩形明渠，b=3m，Q=15m³/s，求hc

**解答要点**：
1. 计算单宽流量：q = Q/b = 5.000 m²/s
2. 应用公式：hc = (q²/g)^(1/3) = **1.3659 m** ✓
3. 临界流速：vc = √(ghc) = 3.661 m/s
4. 流态判断：h₀=2m > hc → 缓流

**考点**：临界水深公式、Froude数、流态判断  
**分值**：12分

### 例题2：水跃共轭水深

**题目**：矩形明渠，b=4m，h₁=0.5m，Q=20m³/s，求h₂和ΔE

**解答要点**：
1. 计算Fr₁：v₁ = 10.00 m/s，Fr₁ = 4.5152（Fr₁>1✓）
2. 共轭水深：h₂ = (h₁/2)[-1+√(1+8Fr₁²)] = **2.3983 m** ✓
3. 水跃损失：ΔE = E₁ - E₂ = **2.3520 m**
4. 相对损失：ΔE/E₁ = 47.55%（消能效率高）

**考点**：水跃共轭水深、能量损失  
**分值**：15分

## 💡 易错点总结

### ❌ 易错点1：临界水深公式
- **错误**：用Q²代替q²
- **正确**：hc = (q²/g)^(1/3)，其中 q = Q/b

### ❌ 易错点2：共轭水深公式符号
- **错误**：h₂ = (h₁/2)[1 + √(1+8Fr₁²)]
- **正确**：h₂ = (h₁/2)[-1 + √(1+8Fr₁²)]

### ❌ 易错点3：水跃条件
- **错误**：忘记判断Fr₁>1
- **正确**：必须先验证Fr₁>1，才能计算水跃

### ❌ 易错点4：能量损失方向
- **错误**：ΔE = E₂ - E₁
- **正确**：ΔE = E₁ - E₂（损失为正）
