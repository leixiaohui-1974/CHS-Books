## 🎯 二、核心功能详解

### 2.1 题目服务封装 (`questionService.ts`)

#### 功能模块
1. **题目管理**
   - `createQuestion()` - 创建题目
   - `getQuestions()` - 获取题目列表（带筛选）
   - `getQuestion()` - 获取题目详情
   - `updateQuestion()` - 更新题目
   - `deleteQuestion()` - 删除题目

2. **提交和判题**
   - `submitAnswer()` - 提交答案
   - `getSubmissions()` - 获取提交记录

3. **错题本**
   - `getWrongQuestions()` - 获取错题列表
   - `addWrongQuestionNote()` - 添加/编辑笔记
   - `removeWrongQuestion()` - 移除错题

4. **题集管理**
   - `createQuestionSet()` - 创建题集
   - `getQuestionSets()` - 获取题集列表
   - `getQuestionSet()` - 获取题集详情
   - `getQuestionsBySet()` - 获取题集中的题目

#### 技术特点
- 🔐 自动Token注入
- 📦 完整的TypeScript类型定义
- 🔄 Axios拦截器
- 🎯 RESTful API封装

#### 2. 题型自适应渲染
```typescript
// 根据题型动态渲染输入组件
const renderQuestionInput = (question: Question) => {
  switch (question.question_type) {
    case 'single_choice': return <Radio.Group />
    case 'multiple_choice': return <Checkbox.Group />
    case 'true_false': return <Radio.Group />
    default: return <TextArea />
  }
```
