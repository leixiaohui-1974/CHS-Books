## 🚀 快速安装

### 方式一：一键安装（推荐）

```bash
# 1. 克隆或下载项目
cd /path/to/your/directory

# 2. 安装依赖
pip install -r 📦requirements.txt

# 3. 运行启动脚本
./🚀一键启动脚本.sh

```

### 方式二：使用国内镜像（更快）

```bash

```

### 方式三：使用虚拟环境（隔离环境）

```bash

python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 安装依赖
pip install -r 📦requirements.txt
```

## 📖 详细安装步骤

### 步骤1：检查Python版本

```bash
python --version
# 或
python3 --version
```

**期望输出**：`Python 3.7.0` 或更高版本

**如果Python未安装**：

- **Windows**: 访问 [python.org](https://www.python.org/downloads/) 下载安装
- **macOS**: `brew install python3`
- **Linux**: `sudo apt install python3 python3-pip` (Ubuntu/Debian)

### 步骤2：下载项目

#### 方法A：Git克隆（推荐）

```bash
git clone <your-repo-url>
cd 15-books-series
```

#### 方法B：直接下载

1. 下载ZIP压缩包
2. 解压到目标目录
3. 进入项目目录

### 步骤3：创建虚拟环境（可选但推荐）

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows CMD:
venv\Scripts\activate.bat

# Windows PowerShell:
venv\Scripts\Activate.ps1

# macOS/Linux:
source venv/bin/activate
```

**激活成功标志**：命令提示符前出现 `(venv)`

### 步骤4：安装Python依赖

```bash
# 基础安装
pip install -r 📦requirements.txt

# 或使用国内镜像（推荐）
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r 📦requirements.txt

# 或使用阿里云镜像
pip install -i https://mirrors.aliyun.com/pypi/simple/ -r 📦requirements.txt
```

**依赖包列表**：
- numpy >= 1.20.0
- scipy >= 1.7.0
- pandas >= 1.3.0
- matplotlib >= 3.4.0
- seaborn >= 0.11.0
- tqdm >= 4.62.0

### 步骤5：设置脚本执行权限（Linux/macOS）

```bash
chmod +x 🚀一键启动脚本.sh
chmod +x 🔄快速更新脚本.sh
```

### 步骤6：验证安装

```bash
# 运行项目验证脚本
python 🔧项目验证脚本.py
```

**期望输出**：所有检查项显示 ✅

## ⚙️ 环境配置

### Python环境配置

#### 配置pip镜像（永久）

**Windows**:
```bash
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

**macOS/Linux**:
```bash
# 创建配置目录
mkdir -p ~/.pip

# 创建配置文件
cat > ~/.pip/pip.conf << EOF
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
[install]
trusted-host = pypi.tuna.tsinghua.edu.cn
EOF
```

#### 配置Matplotlib中文显示

**Windows**:
```python
# 在Python代码中添加
import matplotlib.pyplot as plt
plt.rcParams['font.sans-serif'] = ['SimHei']  # 黑体
plt.rcParams['axes.unicode_minus'] = False
```

**macOS**:
```python
plt.rcParams['font.sans-serif'] = ['Arial Unicode MS']
```

**Linux**:
```bash
# 安装中文字体
sudo apt install fonts-wqy-zenhei

# 在Python代码中
plt.rcParams['font.sans-serif'] = ['WenQuanYi Zen Hei']
```

## 🐛 常见问题

### 问题1：pip安装速度慢

**症状**：`pip install` 很慢或超时

**解决方案**：
```bash
# 使用国内镜像
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r 📦requirements.txt

# 或设置永久镜像
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题2：权限错误（Windows）

**症状**：`Permission Denied` 错误

**解决方案**：
```bash
# 以管理员身份运行命令提示符
# 或使用 --user 标志
pip install --user -r 📦requirements.txt
```

### 问题3：虚拟环境激活失败（Windows PowerShell）

**症状**：无法运行 `Activate.ps1`

**解决方案**：
```powershell
# 设置执行策略
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# 然后激活
venv\Scripts\Activate.ps1
```

### 问题4：Matplotlib中文显示为方框

**症状**：图表中文显示为 `□□□`

**解决方案**：
```python
# 在代码开头添加
import matplotlib.pyplot as plt
plt.rcParams['font.sans-serif'] = ['SimHei']  # Windows
plt.rcParams['axes.unicode_minus'] = False
```

### 问题5：找不到模块

**症状**：`ModuleNotFoundError: No module named 'xxx'`

**解决方案**：
```bash
# 确认虚拟环境已激活
which python  # macOS/Linux
where python  # Windows

# 重新安装依赖
pip install -r 📦requirements.txt
```

### 问题6：Python版本太低

**症状**：`SyntaxError` 或版本警告

**解决方案**：
```bash
# 安装新版Python
# Windows: 从python.org下载
# macOS:
brew install python@3.10

# Linux:
sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt update
sudo apt install python3.10
```

## 📦 Docker部署（可选）

### 创建Dockerfile

```dockerfile
FROM python:3.10-slim

WORKDIR /app

COPY 📦requirements.txt .
RUN pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r 📦requirements.txt

COPY . .

CMD ["./🚀一键启动脚本.sh"]
```

### 构建和运行

```bash
# 构建镜像
docker build -t 15-books-series .

# 运行容器
docker run -it 15-books-series
```

## 🗑️ 卸载方法

### 完整卸载

```bash
# 1. 停用虚拟环境（如果激活）
deactivate

# 2. 删除虚拟环境
rm -rf venv

# 3. 删除项目目录
cd ..
rm -rf 15-books-series

# 4. 清理pip缓存（可选）
pip cache purge
```

### 仅卸载依赖

```bash
# 卸载项目依赖
pip uninstall -r 📦requirements.txt -y
```
