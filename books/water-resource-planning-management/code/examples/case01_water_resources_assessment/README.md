# 案例1.1：流域水资源调查与评价

## 工程背景

某中型流域（流域面积 5000 km²）需要进行水资源本底调查，为后续的水资源规划与管理提供基础数据支撑。本案例将对该流域的地表水资源、地下水资源进行全面评价，分析水资源的时空分布特征。

## 案例目标

1. 掌握水资源评价的基本方法
2. 熟悉频率分析与统计方法
3. 理解水资源时空分布特征
4. 能够计算不同保证率下的可利用水量

## 主要内容

### 1. 地表水资源量计算
- 多年平均径流量计算
- 不同保证率（P=50%, 75%, 95%）径流量
- 径流年际变化分析
- 径流年内分配

### 2. 频率分析
- 采用Pearson-III型曲线拟合
- 计算不同频率的设计值
- 绘制频率曲线
- 适线检验

### 3. 水资源可利用量分析
- 考虑生态基流
- 考虑调节能力
- 不同保证率下的可供水量

### 4. 水资源时空分布
- 年内分配不均匀性分析
- 空间分布特征（如有分区数据）

## 核心技术

### 1. 频率分析（Pearson-III型分布）

Pearson-III型分布的概率密度函数：

$$
f(x) = \\frac{(x - a_0)^{\\alpha - 1} e^{-(x-a_0)/\\beta}}{\\beta^\\alpha \\Gamma(\\alpha)}
$$

参数关系：
- $\\alpha = 4 / C_s^2$
- $\\beta = \\bar{x} C_v C_s / 2$
- $a_0 = \\bar{x}(1 - 2C_v / C_s)$

### 2. 水资源可利用量

$$
W_{\\text{可利用}} = W_{\\text{total}} - W_{\\text{生态}} - W_{\\text{损失}}
$$

其中：
- $W_{\\text{total}}$: 多年平均径流量
- $W_{\\text{生态}}$: 生态基流（一般取10-20%）
- $W_{\\text{损失}}$: 蒸发渗漏损失
