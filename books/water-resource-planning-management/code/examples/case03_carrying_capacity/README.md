# 案例1.3：水资源承载能力评价

## 工程背景

某区域面临水资源紧缺问题，需要评价水资源承载能力，识别超载风险，为水资源管理决策提供科学依据。本案例将构建综合评价指标体系，运用多准则决策方法，对该区域的水资源承载能力进行全面评价。

## 案例目标

1. 掌握多指标综合评价方法
2. 理解水资源承载力内涵
3. 学习AHP、熵权法、TOPSIS等决策方法
4. 能够进行承载状态分析

## 主要内容

### 1. 指标体系构建

**五个维度，15个指标**：

#### 水资源量（3个指标）
- 人均水资源量（m³/人·年）
- 水资源开发利用率（%）
- 地下水开采系数

#### 水环境（3个指标）
- 水功能区水质达标率（%）
- 污水处理率（%）
- COD排放强度（kg/万元GDP）

#### 经济发展（3个指标）
- 万元GDP用水量（m³/万元）
- 农田灌溉水有效利用系数
- 工业用水重复利用率（%）

#### 社会发展（3个指标）
- 城镇供水保证率（%）
- 农村自来水普及率（%）
- 生态用水比例（%）

#### 管理能力（3个指标）
- 水资源管理制度完善度（分）
- 监测站网密度（个/万km²）
- 节水宣传教育覆盖率（%）

### 2. 权重确定方法

#### 主观赋权：AHP层次分析法
- 构造判断矩阵
- 计算权重
- 一致性检验

#### 客观赋权：熵权法
- 基于数据离散程度
- 客观确定权重

#### 组合赋权
- 主客观权重结合
- α·W_AHP + (1-α)·W_Entropy

### 3. 综合评价方法

#### TOPSIS法
- 正理想解和负理想解
- 综合得分计算
- 承载状态判定

#### 模糊综合评价
- 模糊关系矩阵
- 模糊合成
- 等级判定

### 4. 承载状态分级

| 等级 | 综合得分 | 状态 | 说明 |
|------|---------|------|------|
| I | ≥0.8 | 可载 | 水资源承载能力强 |
| II | 0.6-0.8 | 临界 | 基本平衡，需注意 |
| III | 0.4-0.6 | 超载 | 已超载，需采取措施 |
| IV | <0.4 | 严重超载 | 水资源短缺严重 |

## 核心技术

### 1. AHP层次分析法

**判断矩阵标度**：
- 1：同等重要
- 3：稍微重要
- 5：明显重要
- 7：强烈重要
- 9：极端重要
- 2,4,6,8：中间值

**一致性检验**：
- CI = (λ_max - n) / (n - 1)
- CR = CI / RI
- CR < 0.10 通过检验

### 2. 熵权法

信息熵：$H_j = -k \sum_{i=1}^{m} p_{ij} \ln p_{ij}$

权重：$w_j = \frac{1 - H_j}{\sum_{j=1}^{n}(1 - H_j)}$

### 3. TOPSIS法

距离正理想解：$D^+ = \sqrt{\sum_{j=1}^{n}(v_{ij} - v_j^+)^2}$

距离负理想解：$D^- = \sqrt{\sum_{j=1}^{n}(v_{ij} - v_j^-)^2}$

综合得分：$C_i = \frac{D_i^-}{D_i^+ + D_i^-}$

### 4. 模糊综合评价

$B = W \circ R = (b_1, b_2, ..., b_m)$

其中：
- W：权重向量
- R：模糊关系矩阵
- ∘：模糊合成算子

- 人均水资源量偏低

1. 加强节水管理
2. 优化水资源配置
3. 增加生态用水
