# 案例6.1：洪水风险评估

## 工程背景

洪水风险管理是水资源规划的重要内容。通过蒙特卡洛模拟和概率分析，评估不同设计标准下的洪水风险，为防洪工程设计和应急预案提供决策依据。

## 案例目标

1. 理解洪水风险的概率表达
2. 掌握蒙特卡洛风险模拟
3. 学习VaR/CVaR风险度量
4. 能够进行风险-成本权衡分析

## 主要内容

### 1. 洪水风险要素

**致灾因子**：
- 洪峰流量
- 洪量
- 洪水过程

**承灾体**：
- 保护区人口
- 经济资产
- 基础设施

**风险度量**：
- 失效概率
- 期望损失
- 最大可能损失（VaR）
- 条件期望损失（CVaR）

## 核心算法

### 1. Pearson-III频率分析

**概率密度函数**：
```
f(x) = (β^α / Γ(α)) * (x - a0)^(α-1) * exp(-β(x - a0))
```

**参数估计**：
```
均值 μ = E[X]
标准差 σ = sqrt(Var[X])
偏态系数 Cs = E[(X-μ)³] / σ³
```

### 2. 蒙特卡洛风险模拟

```
for i = 1 to N:
    Q_i = sample_from_P3_distribution()
    if Q_i > Q_design:
        Loss_i = damage_function(Q_i, Q_design)
    else:
        Loss_i = 0
    
Risk = mean(Loss_i)
P_failure = count(Q_i > Q_design) / N
```

### 3. VaR和CVaR

**VaR** (Value at Risk):
```
VaR_α = inf{l : P(Loss ≤ l) ≥ α}
```

**CVaR** (Conditional VaR):
```
CVaR_α = E[Loss | Loss ≥ VaR_α]
```
