## 📚 项目定位

这是一本**案例驱动**的水环境模拟教材，专注于培养学生的**建模能力**和**工程实践能力**。

### 核心特色

```
理论严谨 + 案例丰富 + 代码完整 + 工程导向
```

- 📖 **30个工程案例**：从简单到复杂，循序渐进
- 💻 **10,000+行代码**：完整Python实现，可直接运行
- 🎯 **324学时设计**：本科/研究生教学适用
- 🌊 **全面覆盖**：河流、湖库、地下水、流域尺度
- 🔬 **方法先进**：FDM、FVM、FEM多种数值方法

## 🔬 技术亮点

### 数学模型全覆盖

```
基础方程：
✓ Fick扩散定律
✓ 对流-扩散方程
✓ 质量守恒定律

反应动力学：
✓ 零阶反应
✓ 一阶反应
✓ 二阶反应
✓ Monod动力学

水质过程：
✓ BOD降解
✓ DO平衡（S-P模型）
✓ 氮循环（硝化、反硝化、氨化）
✓ 磷循环（沉降、释放）
✓ 藻类生长（营养限制、光照限制）

生态模型：
✓ 食物链模型
✓ 多稳态理论
✓ 生态系统动力学
```

### 数值方法详解

```
有限差分法（FDM）：
✓ 显式格式（Forward Euler）
✓ 隐式格式（Backward Euler）
✓ Crank-Nicolson格式
✓ 迎风格式（Upwind）
✓ QUICK格式

有限体积法（FVM）：
✓ 守恒性保证
✓ Riemann求解器
✓ 激波捕捉（TVD/WENO）

有限元法（FEM）：
✓ Galerkin方法
✓ 不规则网格
✓ 地下水应用

特殊技术：
✓ ADI方法（多维问题）
✓ 算子分裂
✓ 自适应网格
✓ 并行计算
```

### 参数率定与验证

```
率定方法：
✓ 试错法
✓ 最小二乘法
✓ 贝叶斯方法
✓ PEST自动率定

验证技术：
✓ 解析解对比
✓ 实验数据验证
✓ 实测资料验证
✓ 模型间交叉验证

不确定性分析：
✓ 敏感性分析
✓ 蒙特卡洛模拟
✓ 置信区间估计
```
