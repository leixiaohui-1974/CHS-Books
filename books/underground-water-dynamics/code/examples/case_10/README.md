## 不确定性来源

地下水模型的不确定性主要来自：

1. **参数不确定性**: 水力参数估计误差
2. **观测误差**: 测量噪声
3. **模型结构不确定性**: 简化假设
4. **初始/边界条件不确定性**: 边界值误差

## 方法对比

| 方法 | 计算成本 | 参数分布 | 模型不确定性 | 推荐度 |
|------|---------|---------|------------|--------|
| Monte Carlo | 中 | 需要 | 否 | ⭐⭐⭐⭐ |
| GLUE | 高 | 不需要 | 是 | ⭐⭐⭐⭐⭐ |
| Bootstrap | 低 | 不需要 | 否 | ⭐⭐⭐⭐ |
| 贝叶斯MCMC | 高 | 先验 | 是 | ⭐⭐⭐⭐⭐ |

## 关键概念

### Monte Carlo方法
- 参数分布采样
- 不确定性传播
- 置信区间估计

### GLUE方法
- 行为参数集
- 似然函数
- 非正式贝叶斯

### Bootstrap方法
- 残差重采样
- 观测误差量化
- 免分布假设
