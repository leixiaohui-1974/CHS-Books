## 📈 案例详解

### 案例29：湖泊风生流与水质模拟

**工程价值**：
- 预测风暴期间的水动力响应
- 评估藻华爆发风险
- 优化取水口位置
- 指导水质管理

**核心公式**：
```
风应力: τ = Cd × ρ_air × U²
表层流速: u_s = 0.03 × U
风浪高: H_s = f(U, F, g)
藻华风险 = f(温度, 营养盐, 光照)
```

### 案例30：人工湿地HRT优化

**工程价值**：
- 确定湿地最优尺寸
- 预测处理效果
- 成本效益分析
- 运行参数优化

**核心公式**：
```
HRT = V / Q
去除率 = 1 - exp(-k × HRT)
长宽比优化: L/W = 3~10
有效容积比: e = 0.5~0.9
```

### 案例31：湖滨带生态缓冲

**工程价值**：
- 确定缓冲带宽度
- 评估削减效果
- 植被配置设计
- 经济效益分析

**核心公式**：
```
径流流速: v = (1/n) × R^(2/3) × S^0.5
泥沙拦截: η = 1 - exp(-0.05 × W × V)
污染物削减: C_out = C_in × (1 - η)
```

### 案例32：湖泊分层与内波

**工程价值**：
- 预测季节性分层
- 评估底层缺氧风险
- 优化曝气系统
- 指导取水调度

**核心公式**：
```
N = √(g/ρ₀ × dρ/dz)  # 布伦特-韦萨拉频率
c = √(g' × h₁h₂/(h₁+h₂))  # 内波波速
T = 2L / c  # 内涌周期
```

### 案例33：退化湿地补水

**工程价值**：
- 确定年度需水量
- 优化月度补水
- 预测恢复时间
- 植被配置指导

**核心公式**：
```
Q_eco = A × (ET + S)
恢复时间 = ΔV / (Q_in - Q_out)
植被适宜性 = f(水深, 淹没天数)
```
