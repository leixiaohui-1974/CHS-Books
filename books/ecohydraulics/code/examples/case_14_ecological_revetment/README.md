# 案例14：生态护岸水力设计

## 问题描述

设计生态护岸，既要满足防洪安全要求，又要为水生生物提供栖息空间，实现工程防护与生态功能的有机结合。

## 理论背景

### 1. 生态护岸类型

**植被护坡**：
- 草皮、灌木、乔木护坡
- 粗糙系数高（n ≈ 0.06）
- 流速削减效果好
- 根系加固土壤

**石笼护岸**：
- 装填石料的金属网笼
- 孔隙率高（40-50%）
- 透水性好
- 利于植物生长

**生态袋护岸**：
- 装土植生的编织袋
- 可种植植被
- 柔性结构
- 适应变形

**格宾网护岸**：
- 铁丝网石笼
- 结构稳定
- 生态功能好
- 景观效果佳

### 2. 水流与护岸相互作用

**近岸流速分布**：
```
v_岸边 = β × v_主流

其中β为流速削减系数：
- 植被护坡：β ≈ 0.3-0.5
- 石笼护岸：β ≈ 0.4-0.6
- 混凝土护岸：β ≈ 0.8-0.9
```

**多孔介质阻力**：
```
Cd = 1.2 × (1 - ε)²

其中：
Cd = 阻力系数
ε = 孔隙率
```

### 3. 庇护所功能

**低流速区**：
- 护岸前流速降低区域
- 为鱼类幼体提供庇护
- 减少能量消耗
- 提高生存率

**庇护所面积**：
```
A_庇护 = W_庇护 × L

其中：
W_庇护 = 庇护所宽度
L = 河段长度
```

### 4. 护岸稳定性

**剪应力**：
```
τ = 0.5 × ρ × Cd × v²

其中：
ρ = 水密度
Cd = 阻力系数
v = 近岸流速
```

**安全系数**：
```
SF = v_临界 / v_实际

评判标准：
SF > 2.0：安全
SF = 1.5-2.0：基本安全
SF < 1.5：不安全
```

### 5. 植被根系加固

根系可提高土壤抗剪强度：
```
c_总 = c_土 + Δc_根

其中：
c_土 = 土壤粘聚力
Δc_根 = 根系贡献
```

## 工程应用

### 1. 城市河道治理

**传统护岸问题**：
- 混凝土硬化
- 生境单一
- 景观性差
- 水陆隔离

**生态护岸优势**：
- 自然亲水
- 生境多样
- 景观优美
- 生态连通

### 2. 河流生态修复

**护岸选择原则**：
- 优先植被护坡（缓坡）
- 石笼/格宾（中等坡度）
- 生态袋（灵活适应）
- 分段组合设计

### 3. 防洪与生态兼顾

**设计要点**：
- 满足防洪安全要求（SF > 1.5）
- 最大化生态功能
- 考虑长期维护
- 兼顾景观效果

### 4. 鱼类栖息地营造

**庇护所设计**：
- 营造低流速区
- 提供隐蔽空间
- 增加栖息地面积
- 保护幼鱼育肥

## 关键参数

### 粗糙系数
- **植被护坡**：n = 0.050-0.080
- **石笼护岸**：n = 0.035-0.050
- **生态袋**：n = 0.030-0.040
- **格宾网**：n = 0.035-0.045
- **混凝土**：n = 0.012-0.018

### 流速削减系数
- **植被护坡**：β = 0.3-0.5
- **石笼护岸**：β = 0.4-0.6
- **生态袋**：β = 0.5-0.7
- **混凝土**：β = 0.8-0.9

### 安全系数要求
- **重要河段**：SF > 2.5
- **一般河段**：SF > 2.0
- **次要河段**：SF > 1.5

### 边坡角度
- **植被护坡**：< 35°
- **石笼护岸**：< 45°
- **混凝土**：可达60°

## 注意事项

1. **防洪安全**
   - 不得降低泄洪能力
   - 安全系数满足要求
   - 考虑洪水冲刷

2. **植被选择**
   - 适应当地气候
   - 根系发达
   - 耐水淹
   - 易维护

3. **施工质量**
   - 护岸基础牢固
   - 植被成活率高
   - 排水措施完善

4. **维护管理**
   - 定期检查
   - 补植补种
   - 清理杂物
   - 修复破损

5. **生态效果监测**
   - 植被覆盖度
   - 生物多样性
   - 护岸完整性
