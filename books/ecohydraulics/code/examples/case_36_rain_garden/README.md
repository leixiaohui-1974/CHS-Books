# 案例36:雨水花园净化系统设计

## 案例简介

本案例研究雨水花园(生物滞留系统)的设计与净化功能评估。雨水花园是最常用的低影响开发(LID)设施之一,通过土壤-植物-微生物系统滞留、渗透和净化雨水径流。通过优化蓄水深度、介质层设计、植物配置等参数,最大化雨水花园的调蓄容量和污染物去除效率,为海绵城市建设提供关键技术支撑。

## 理论背景

雨水花园工作原理:

**1. 调蓄容量**
V_total = V_surface + V_media
- V_surface = A * d_pond (表面蓄水)
- V_media = A * d_media * n (介质层蓄水,n为孔隙率)

**2. 排空时间**(达标标准<24小时)
t = d_pond / i + d_media * n / K
- i: 渗透速率(mm/h)
- K: 介质层渗透系数(mm/h)

不同介质渗透系数:
- 砂土: 50-250mm/h (快)
- 砂壤土: 25-75mm/h (中快)
- 壤土: 13-25mm/h (中)
- 黏土: <13mm/h (慢,不推荐)

**3. 污染物去除机制**
- TSS: 物理过滤(80-95%)
- TN: 植物吸收(30%)+微生物硝化反硝化(40%)
- TP: 介质吸附(50%)+植物吸收(20%)
- 重金属: 介质吸附固定(70-85%)

去除率随HRT(水力停留时间)增加而提高,但存在边际效应递减。

**4. 植物选择原则**
- 耐水性: 能承受短期淹没(1-3天)
- 耐旱性: 能承受长期干旱(>7天)
- 根系发达: 促进土壤渗透
- 景观价值: 四季有景,多样搭配

**5. 成本效益**
建设成本200-400元/m²,远低于传统排水管网扩容
运行维护成本低,主要为植物养护
综合效益包括径流控制、水质净化、景观提升、生物多样性等

## 参数说明

### 设计参数
- `garden_area`: 雨水花园面积,典型值50-200m²
- `media_depth`: 介质层深度,建议0.6-1.0m
- `ponding_depth`: 蓄水层深度,建议0.2-0.4m
- `porosity`: 介质孔隙率,典型值0.35-0.45

### 介质配比
- 砂: 50-60% (保证渗透性)
- 壤土/表土: 20-30% (提供养分)
- 有机质(堆肥): 10-20% (吸附污染物)

### 汇水面积比
雨水花园面积 / 汇水面积 = 5-10%
例: 1000m²停车场需配置50-100m²雨水花园

## 生态意义与环境影响

### 生态意义

1. **径流削减**:滞留雨水,减少管网负荷,缓解城市内涝
2. **水质净化**:去除初期雨水中的污染物,保护受纳水体
3. **地下水补给**:雨水渗透补充地下水,缓解地下水下降
4. **生物栖息地**:为昆虫、小型动物提供栖息空间
5. **景观美化**:四季花卉植物,提升社区环境
6. **碳汇固碳**:植被生长固定CO2,缓解温室效应
7. **环境教育**:展示绿色基础设施,提高公众环保意识

### 设计要点

1. **选址原则**
   - 地势低洼处,便于汇水
   - 距建筑物>3m,避免渗水影响地基
   - 地下水位低于花园底部>1m
   - 避开树根密集区

2. **尺寸设计**
   - 面积 = 汇水面积 × 5-10% × 径流系数 / 控制降雨深度
   - 深度 = 蓄水层(0.3m)+ 介质层(0.6m)+ 砾石层(0.2m)= 1.1m

3. **溢流设计**
   - 设置溢流口,超标降雨溢流至市政管网
   - 溢流口标高 = 蓄水层顶部

4. **防渗与排水**
   - 底部铺设透水土工布,防止淤堵
   - 不设防渗层(除非地下水污染风险)
   - 设置排水管连接市政管网,应急排水

### 应用场景

1. **居住区**
   - 小区绿地改造为雨水花园
   - 道路两侧绿化带
   - 案例: 北京通州海绵城市示范区

2. **公共建筑**
   - 学校、医院、政府大楼
   - 停车场周边绿化
   - 案例: 清华大学第六教学楼雨水花园

3. **道路交通**
   - 道路中央分隔带
   - 人行道与车行道之间
   - 案例: 上海临港新城道路雨水花园

4. **工业园区**
   - 厂区绿化带
   - 初期雨水净化
   - 案例: 深圳光明新区工业园

### 维护管理

1. **植物养护**(每年)
   - 春季补种,夏季浇灌,秋季收割
   - 控制杂草,防止外来入侵种

2. **设施维护**(每季度)
   - 清理进水口垃圾
   - 检查溢流口通畅
   - 及时修复破损

3. **介质更换**(5-10年)
   - 介质堵塞时部分更换
   - 保持渗透性能
