# 案例17:河口感潮河段生态水力学

## 案例简介

本案例研究河口感潮河段的水动力过程及其生态效应。河口是淡水与海水的交汇区域,受潮汐、径流双重影响,具有独特的水文水力特征。通过模拟潮汐水位变化、盐水楔侵入、生态需水量等过程,揭示河口水动力对生态系统的影响机制,为河口生态保护和水资源管理提供科学依据。

## 理论背景

河口感潮河段具有以下特征:
1. **潮汐作用**:水位呈周期性涨落,典型半日潮周期约12.4小时
2. **盐水楔**:由于海水密度大于淡水,海水以楔形沿河床向上游侵入
3. **双向流**:涨潮时向上游流动,落潮时向下游流动
4. **河海相互作用**:径流量影响盐水楔侵入距离,潮汐影响水位和流速

**潮汐水位公式**: H(t) = A * sin(2πt/T)
其中A为潮差的一半,T为潮周期(约12.4小时)

**盐水楔侵入长度**(van der Burgh方程):
L = K * (潮差 / 径流量)^(1/3) * B^(1/3)
其中K为经验系数,B为河宽

生态需水量需保证盐水不过度上溯,维持河口咸淡水交界的动态平衡,保护河口湿地和水生生物栖息地。

## 生态意义与环境影响

### 生态意义

1. **河口湿地保护**:保持适度的咸淡水梯度,维护湿地植被和鸟类栖息地
2. **洄游通道**:为中华鲟、鲥鱼等洄游性鱼类提供适宜的盐度环境
3. **水质保护**:充足的径流稀释污染物,维持河口水质

### 环境影响

1. **上游水库调度影响**
   - 枯季径流减少导致盐水楔上溯
   - 威胁取水口水质安全
   - 影响河口湿地生态系统

2. **气候变化影响**
   - 海平面上升增强盐水楔侵入
   - 极端天气事件频发

3. **管理措施**
   - 上游水库生态调度,保障基本径流
   - 取水口优化布局,避开高盐度区
   - 建立河口湿地保护区
