## 涉及理论

### 1. 热分层

水体因密度差异形成垂向分层：
- 表温层（Epilimnion）：温度高
- 温跃层（Thermocline）：温度梯度最大
- 底温层（Hypolimnion）：温度低

### 2. DO动力学
