# 案例37:城市内涝生态化防治方案

## 案例简介

本案例研究城市内涝的生态化防治策略。传统"灰色"基础设施(排水管网、泵站)面临暴雨强度增加和用地紧张的挑战,绿色基础设施(LID设施)通过源头减排、过程控制、系统治理,提供更可持续的内涝防治方案。通过模拟设计暴雨强度、径流计算、调蓄池设计、LID设施效能评估,为城市内涝综合治理提供决策支持。

## 理论背景

城市内涝生态化防治核心技术:

**1. 设计暴雨强度**(芝加哥雨型公式)
i = A * (1 + C * lgP) / (t + B)^n
- A, B, C, n: 地区参数
- P: 设计重现期(年)
- t: 降雨历时(min)

北京参数: A=2881, B=9.03, C=0.726, n=0.798

**2. 径流计算**(推理公式)
Q_p = ψ * i * F
- ψ: 径流系数(与不透水率相关)
- i: 暴雨强度(L/s/ha)
- F: 汇水面积(ha)

**3. 调蓄池设计**
V = Q_in * t - Q_out * t
需考虑入流过程线和出流能力

**4. LID设施效能**
- 径流削减率: (Q_0 - Q_LID) / Q_0 × 100%
- 峰值削减率: (Q_p0 - Q_pLID) / Q_p0 × 100%
- 污染物削减率: 综合考虑各类设施

不同覆盖率下效能:
- 10%: 削减20-30%
- 30%: 削减50-60%
- 50%: 削减70-80%

**5. 综合管理策略**
- 源头减排(LID设施): 生物滞留、透水铺装、绿色屋顶、雨水罐
- 过程控制(调蓄设施): 调蓄池、湿塘、植草沟
- 系统治理(排水系统): 管网提标、泵站增容、河湖连通

优先级: 源头 > 过程 > 末端

## 代码说明

### 主要类和函数

1. **UrbanFloodControl类** - 内涝防治设计
   - `design_rainfall_intensity()`: 计算设计暴雨强度
   - `runoff_calculation()`: 计算径流
   - `detention_basin_design()`: 调蓄池设计
   - `green_infrastructure_effectiveness()`: LID设施效能评估
   - `integrated_management_strategy()`: 综合管理策略

### 核心算法

```python
# 设计暴雨强度(芝加哥公式)
i = A * (1 + C * log10(P)) / (t + B)**n

# 径流系数
psi = 0.15 + 0.75 * imperviousness

# 峰值流量
Q_p = psi * i * area / 360

# 调蓄容积
V_storage = inflow_volume * (1 - outlet_capacity / peak_inflow)
```

## 参数说明

### 区域参数
- `district_area`: 区域面积,如2km²
- `design_return_period`: 设计重现期,一般20-50年
- `imperviousness`: 不透水率,老城区可达80-90%

### 暴雨参数
- `duration`: 降雨历时,短历时(15-120min)决定峰值
- `location`: 地区(影响公式参数)

### LID设施
- `lid_coverage`: 覆盖率(%),目标20-40%
- `facility_types`: 设施类型组合

### 调蓄设施
- `storage_volume`: 调蓄容积(m³)
- `outlet_capacity`: 出流能力(m³/s)

## 生态意义与环境影响

### 生态意义

1. **内涝防治**:减少城市内涝风险,保护人民生命财产安全
2. **水生态修复**:恢复自然水文循环,改善城市水环境
3. **气候适应**:提高城市应对极端降雨的韧性
4. **生态服务**:提供多重生态系统服务(调节、支持、文化)

### 综合效益

1. **防洪安全**
   - 削减径流峰值,降低管网压力
   - 分散调蓄,提高系统韧性
   - 降低内涝积水深度和时长

2. **环境改善**
   - 削减面源污染,改善受纳水体水质
   - 地下水补给,缓解地面沉降
   - 缓解热岛效应,降低城市温度

3. **经济效益**
   - 避免内涝损失(直接损失+间接损失)
   - 节约管网扩容投资
   - 提升房地产价值

4. **社会效益**
   - 改善市民生活环境
   - 提供休闲游憩空间
   - 展示生态文明建设成果

### 实施案例

1. **武汉青山区**
   - 问题:老城区内涝严重,积水点30多处
   - 方案:LID设施+管网改造+调蓄设施
   - 效果:内涝点减少80%,年径流总量控制率达75%

2. **厦门马銮湾新城**
   - 特点:新建区,高标准建设
   - 设计:雨水花园、透水铺装、湿塘系统
   - 目标:年径流总量控制率80%以上

3. **北京通州副中心**
   - 定位:千年大计,国家大事
   - 标准:小雨不积水,大雨不内涝,暴雨保安全
   - 措施:海绵+智慧排水系统

4. **国际经验**
   - 美国费城:绿色雨水基础设施计划,25年减少85%溢流
   - 日本东京:地下调蓄隧道+屋顶绿化
   - 荷兰鹿特丹:水广场,平时休闲,雨时蓄水
   - 丹麦哥本哈根:气候适应计划,应对海平面上升+暴雨

### 挑战与对策

1. **老城区改造困难**
   - 空间受限,难以大规模建设LID
   - 对策:见缝插针,小型分散设施

2. **维护管理**
   - 设施多而分散,养护工作量大
   - 对策:智慧化监测,社区参与

3. **效益难量化**
   - 生态效益、社会效益难以货币化
   - 对策:建立综合评价体系

4. **多部门协调**
   - 涉及规划、建设、水务、园林等多部门
   - 对策:建立联席会议制度,统筹协调
