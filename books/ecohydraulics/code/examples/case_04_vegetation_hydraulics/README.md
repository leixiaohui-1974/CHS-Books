# 案例4：涉水植物生长水力条件分析

## 案例概述

**难度等级**: ⭐⭐ 基础  
**工程背景**: 河道生态修复中的植被恢复设计  
**核心理论**: 植物-水流相互作用、Manning糙率、抗冲稳定性

## 问题描述

某河道生态修复项目，需要设计植被恢复方案，确定不同植物的最佳种植位置和水力条件。

### 基本参数

- 河段长度: L = 500 m
- 河道宽度: b = 20 m
- 河床坡度: S0 = 0.001
- Manning糙率: n = 0.025（裸露河床）
- 边坡系数: m = 2.0
- 设计流量范围: 5-50 m³/s
- 候选植物: 芦苇、香蒲、柳树灌木

## 计算目标

1. 分析不同植物对水流的影响
2. 评估植物的抗冲稳定性
3. 计算植物生长的淹水条件
4. 提出植物配置方案

## 涉及理论

### 1. 植被糙率

植被增加河道糙率，影响水流速度。

**综合糙率公式**（植物未淹没）:
```python
n_eff = n_bare * (1 + k * coverage * (h_veg/h))
```

**综合糙率公式**（植物淹没）:
```python
n_eff = (n_lower * h_veg + n_upper * h_clear) / h
```

其中：
- n_bare: 裸露河床糙率
- k: 经验系数（通常2-3）
- coverage: 植被覆盖率（0-1）
- h_veg: 植物高度
- h: 水深

### 2. 植被阻力

植物对水流产生阻力。

**阻力公式**:
```python
F_d = 0.5 * ρ * C_d * A * v²
```

其中：
- ρ: 水的密度（1000 kg/m³）
- C_d: 阻力系数
- A: 迎流面积
- v: 流速

**单位面积迎流面积**:
```python
a = n * d * h
```

其中：
- n: 植物密度（stems/m²）
- d: 茎干直径（m）
- h: 有效高度（m）

### 3. 抗冲稳定性

判断植物是否会被冲刷。

**安全系数**:
```python
SF = v_critical / v_actual
```

**稳定性判据**:
- SF ≥ 2.0: 安全
- 1.5 ≤ SF < 2.0: 基本安全
- 1.0 ≤ SF < 1.5: 临界
- SF < 1.0: 不稳定

### 4. 淹水条件

植物生长需要适宜的淹水条件。

**淹水比例**:
```python
R_sub = 淹水天数 / 总天数
```

**生长适宜性**:
- 挺水植物偏好: 0.3-0.7淹水比例
- 沉水植物偏好: >0.8淹水比例
- 陆生植物偏好: <0.3淹水比例

## 推荐配置方案

### 1. 低滩区（0.5m高程）
- **推荐植物**: 芦苇、香蒲
- **理由**: 耐淹能力强，适应高淹水比例（79.5%）
- **覆盖率**: 40-60%
- **预期效果**: 提供鱼类栖息地，净化水质

### 2. 中滩区（1.0m高程）
- **推荐植物**: 芦苇、柳树灌木
- **理由**: 适中淹水比例（34.5%），生长条件最佳
- **覆盖率**: 50-70%
- **预期效果**: 增加生物多样性，稳固岸坡

### 3. 高滩区（1.5m高程）
- **推荐植物**: 柳树灌木
- **理由**: 低淹水比例（0.3%），适合耐旱植物
- **覆盖率**: 30-50%
- **预期效果**: 岸坡防护，景观绿化
