# 案例6：底栖生物栖息地水力条件评价

## 案例概述

**难度等级**: ⭐⭐⭐ 中等  
**工程背景**: 河床生境修复  
**核心理论**: 床面剪切应力、Shields数、底质稳定性

## 问题描述

评估河道底质的稳定性，为底栖生物栖息地保护提供依据。

## 涉及理论

### 床面剪切应力

```python
τ = ρ * g * R * S
```

### Shields数

```python
θ = τ / ((ρ_s - ρ) * g * d50)
```

临界值：θ_c ≈ 0.05
