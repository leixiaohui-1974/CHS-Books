# 案例38:河口盐水楔三维模拟

## 案例简介

本案例研究河口盐水楔的三维分布特征和动力学过程。盐水楔是河口最重要的水动力-盐度现象,海水因密度大于淡水而沿河床向上游侵入,形成楔形分布。盐水楔影响河口水质、生态系统和取水安全,是河口管理的关键问题。通过模拟盐度三维分布、混合类型、内波动力学、生态分区等,为河口水资源管理和生态保护提供科学依据。

## 理论背景

河口盐水楔关键过程:

**1. 密度弗劳德数**(混合类型判据)
Fr_d = u / sqrt(g' * h)
- g' = g * Δρ / ρ (折算重力)
- Δρ: 密度差(淡水-海水约25kg/m³)

分类:
- Fr_d < 0.08: 盐水楔型(强分层,弱混合)
- 0.08 < Fr_d < 1.0: 部分混合型
- Fr_d > 1.0: 高度混合型

**2. 盐水楔长度**(van der Burgh公式)
L = K * (a / Q)^(1/3) * B^(1/3)
- a: 潮差
- Q: 径流量
- B: 河宽
- K: 经验系数(约15)

典型值: 低径流期可达20-50km,威胁上游取水口

**3. 盐度分布**(纵向+垂向)
S(x,z) = S_0 * exp(-x/L) * f(z/h)
沿程指数衰减,垂向分层(表层淡,底层咸)

**4. 分层参数**
α = ΔS / S_avg
- ΔS: 垂向盐度差
- S_avg: 平均盐度
α > 0.5为强分层, 0.1-0.5为中等分层, <0.1为弱分层

**5. 停留时间**
T = V / Q
河口水体体积 / 径流量,影响污染物滞留时间和生态过程

**6. 生态影响**
不同盐度区对应不同生态系统:
- 淡水区(<0.5 ppt): 淡水生物
- 低盐区(0.5-5 ppt): 河口特有种,生物多样性低
- 半咸水区(5-18 ppt): 适应性强的广盐种
- 高盐区(>18 ppt): 海洋生物

## 代码说明

### 主要类和函数

1. **SaltWedge类** - 盐水楔模型

   - `mixing_type_classification()`: 混合类型分类
   - `salt_wedge_length()`: 计算盐水楔长度
   - `salinity_distribution()`: 计算盐度分布
   - `stratification_parameter()`: 计算分层参数
   - `residence_time()`: 计算停留时间
   - `ecological_impact_assessment()`: 生态影响评估

### 核心算法

```python
# 密度弗劳德数
g_prime = 9.81 * 25 / 1000
u = Q / (W * h)
Fr_d = u / sqrt(g_prime * h)

# 盐水楔长度(km)
L = 15 * (tidal_range / Q)**0.33 * W**0.33

# 盐度分布
S(x, z_frac) = 35 * exp(-3 * x / L) * (1 - 0.7 * z_frac**2)

# 分层参数
alpha = (S_bottom - S_surface) / S_avg
```

## 生态意义与环境影响

### 生态意义

1. **洄游通道**:中华鲟、鲥鱼等洄游性鱼类需要合适的盐度梯度
2. **河口湿地**:盐沼植被(互花米草、芦苇)适应不同盐度
3. **生物多样性**:河口是海淡水生物交汇区,生物多样性高
4. **渔业资源**:青蟹、对虾、河豚等经济物种的繁育场

### 环境影响

1. **取水安全**
   - 枯季盐水楔上溯,威胁上游取水口
   - 上海、珠三角每年发生咸潮入侵
   - 对策:上移取水口、水库蓄淡、跨流域调水

2. **生态影响**
   - 过度上溯压缩淡水生物栖息地
   - 湿地盐碱化,植被退化
   - 鱼类繁殖受阻

3. **农业影响**
   - 灌溉水盐度过高,影响农作物
   - 土壤盐碱化加剧

### 管理措施

1. **上游水库调度**
   - 枯季增加下泄流量
   - 抑制盐水楔上溯
   - 案例:长江三峡水库压咸调度

2. **取水口优化**
   - 多水源联合调度
   - 建立应急水源
   - 分层取水(避开底层高盐水)

3. **生态调度**
   - 洄游期(3-5月)保证适宜盐度
   - 维持河口咸淡水动态平衡

4. **监测预警**
   - 实时监测盐度、流量
   - 建立盐水楔预报模型
   - 提前应对

### 实际案例

1. **长江口**
   - 问题:枯季咸潮入侵,上溯至吴淞口(距河口70km)
   - 影响:上海市陈行水库等取水口水质超标
   - 对策:青草沙水库(外海取水)+三峡水库压咸调度

2. **珠江口**
   - 问题:多年枯季咸潮,影响珠三角供水
   - 特点:八大口门,情况复杂
   - 对策:西江、东江、北江联合调度

3. **钱塘江河口**
   - 特点:强潮河口,涌潮发育
   - 盐水楔动态变化剧烈
   - 研究:盐度实时监测网络

4. **国际经验**
   - 荷兰:三角洲工程,风暴潮防御
   - 美国旧金山湾:生态流量保障
   - 孟加拉国:恒河-布拉马普特拉河口盐碱治理

### 气候变化影响

1. **海平面上升**
   - 21世纪末可能上升0.5-1.0m
   - 盐水楔上溯加剧

2. **径流变化**
   - 降雨时空分布改变
   - 枯季径流可能减少

3. **适应策略**
   - 长远规划考虑海平面上升因素
   - 建立更大容量的淡水储备
   - 海水淡化作为补充水源
