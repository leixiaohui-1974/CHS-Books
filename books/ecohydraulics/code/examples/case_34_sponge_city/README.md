# 案例34:海绵城市雨洪管理设计

## 案例简介

本案例研究海绵城市低影响开发(LID)设施的雨洪管理功能。海绵城市通过渗、滞、蓄、净、用、排等措施,实现城市雨水的自然积存、自然渗透、自然净化。通过模拟不同降雨情景下的径流产生、LID设施削减效果、年径流总量控制率等,为海绵城市建设提供量化设计依据,实现城市雨洪资源化利用和水生态环境改善。

## 理论背景

海绵城市核心技术原理:

**1. 径流计算**(综合径流系数法)
Q = ψ * i * A
- ψ:径流系数,与不透水率相关
- i:降雨强度(mm/h)
- A:汇水面积(ha)

径流系数 ψ ≈ 0.05 + 0.9 * 不透水率

**2. LID设施类型及功能**
- 生物滞留设施(雨水花园):径流削减70-90%,污染物去除60-90%
- 透水铺装:径流削减40-60%,补充地下水
- 绿色屋顶:削减50-70%,延缓峰现时间
- 雨水罐:收集利用,节约水资源

**3. 年径流总量控制率**
α = (控制的降雨事件数 / 总降雨事件数) × 100%
设计目标:控制70-85%的降雨(对应设计降雨量20-30mm)

**4. 水质改善**
LID设施通过过滤、吸附、生物降解去除污染物:
- TSS去除率: 80-90%
- TN去除率: 40-60%
- TP去除率: 50-70%
- COD去除率: 50-65%

**5. 综合效益评估**
径流控制、峰值削减、水质改善、生态价值、景观提升等多维度评价

```python
# 径流系数
ψ = 0.05 + 0.9 * imperviousness

# 径流量

ratio = (target_control / 100) * catchment_area
```

## 参数说明

### 区域参数
- `catchment_area`: 汇水面积,如50公顷
- `imperviousness`: 不透水率,城区典型值0.6-0.8
- `design_return_period`: 设计重现期,一般采用2-5年

### 设计降雨
- 小雨: <10mm
- 中雨: 10-25mm
- 大雨: 25-50mm
- 暴雨: 50-100mm
- 大暴雨: >100mm

### LID设施参数
- 生物滞留: 面积比5-10%,深度0.6-1.0m
- 透水铺装: 面积比10-20%,透水系数>10mm/min
- 绿色屋顶: 建筑屋顶的30-50%
- 雨水罐: 容积50-200L/户

## 生态意义与环境影响

### 生态意义

1. **水生态修复**:恢复城市自然水文循环,减少径流外排
2. **生物多样性**:LID设施为昆虫、鸟类提供栖息地
3. **热岛缓解**:绿色屋顶、雨水花园降低城市温度
4. **碳汇功能**:植被固碳,减缓气候变化

### 环境效益

1. **内涝防治**
   - 削减径流峰值,降低管网压力
   - 分散调蓄,提高排水系统韧性
   - 案例:武汉市青山区海绵城市建设,内涝点减少80%

2. **水质改善**
   - 削减面源污染,保护受纳水体
   - 初期雨水净化,TSS去除率达80%以上
   - 案例:深圳光明新区,河流水质从劣V类提升至IV类

3. **水资源利用**
   - 雨水收集用于绿化灌溉、道路冲洗等
   - 节约自来水,年节水率可达20-40%
   - 案例:北京奥林匹克公园,年回用雨水100万m³

4. **经济效益**
   - 减少灰色基础设施投资
   - 提升房地产价值(绿色建筑溢价5-10%)
   - 减少内涝损失

### 示范工程

1. **国内**
   - 萍乡市(老工业区):首批试点,内涝防治效果显著
   - 西咸新区沣西新城:新建区海绵城市典范
   - 厦门市(高密度城区):海绵+水系综合整治
   - 迁安市(北方缺水区):雨水资源化利用

2. **国际**
   - 美国费城:绿色基础设施计划,25年减少85%溢流
   - 德国柏林:雨水管理法规,强制绿色屋顶
   - 澳大利亚墨尔本:水敏感城市设计(WSUD)
   - 新加坡:ABC水计划(活跃、美丽、清洁)

### 政策支持

- 国务院2015年发布《关于推进海绵城市建设的指导意见》
- 30个试点城市,中央财政补贴12-15亿元/城市
- 目标: 2030年城市建成区80%达标
