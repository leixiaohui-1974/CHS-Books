# 案例5: 尾流效应

## 简介

本案例研究风电场中的尾流效应，这是影响风电场整体性能和布局优化的关键因素。通过Jensen尾流模型，本案例演示单台风力机下游的风速分布特征、多台风力机的尾流叠加效应以及尾流对风电场发电量的影响。掌握尾流效应对于风电场微观选址、机位排布优化和发电量评估具有重要意义。典型的尾流损失可达5%-15%，是风电场设计中不可忽视的因素。

## 理论背景

当风流经风力机时，风轮从气流中提取能量，导致下游风速降低并产生尾流区域。Jensen尾流模型是工程中最常用的尾流计算方法，它假设尾流以线性速率扩展，尾流半径r_wake = D/2 + αx，其中D为风轮直径，x为下游距离，α为尾流扩展系数(平坦地形0.075，粗糙地形0.1)。尾流中心轴线的风速亏损为Δv/v₀ = (1-√(1-Ct))/(1+2αx/D)²，其中Ct为推力系数。多台风力机的尾流叠加通常采用平方和方法或线性叠加方法。尾流恢复距离通常需要5-10倍风轮直径，这决定了风电场的最小机组间距。

## 代码说明

### 主要类

**JensenWakeModel**: Jensen尾流模型类
- `__init__(D, Ct, alpha)`: 初始化模型参数
  - D: 风轮直径 (m)
  - Ct: 推力系数 (典型值0.8)
  - alpha: 尾流扩展系数 (0.075平坦地形, 0.1粗糙地形)
- `wake_velocity_deficit(x, v0)`: 计算下游x距离处的风速
- `wake_profile(x, y, v0)`: 计算尾流横截面上(x,y)点的风速

### 演示函数

- `demo_single_wake()`: 单台风力机尾流特性分析
  - 轴线风速恢复曲线
  - 风速亏损随距离的变化

- `demo_wake_field()`: 尾流流场可视化
  - 2D流场等值线图
  - 展示尾流扩展和恢复过程

- `demo_multiple_wakes()`: 多台风力机尾流叠加
  - 3台机组串联布局
  - 尾流亏损叠加计算
  - 总功率损失评估

## 参数说明

### 风轮参数
- **D**: 风轮直径 (90 m)
- **Ct**: 推力系数 (0.8)
- **v0**: 上游自由风速 (10 m/s)

### 尾流模型参数
- **alpha**: 尾流扩展系数
  - 平坦地形: 0.075
  - 粗糙地形: 0.10
  - 海上风电: 0.04-0.05

### 布局参数
- **机组间距**: 通常5D-10D
  - 5D: 紧密布置，尾流损失大
  - 8D: 常用布置
  - 10D以上: 尾流影响小

## 工程应用

尾流效应分析的实际应用：
1. **风电场微观选址**: 优化机位排布，减少尾流影响
2. **发电量评估**: 考虑尾流损失的AEP计算
3. **经济性分析**: 权衡机组间距与土地利用率
4. **控制策略**: 尾流转向控制(Wake Steering)
5. **功率预测**: 考虑尾流的短期功率预测

### 典型机组间距
- 主导风向: 8-10D
- 垂直风向: 3-5D
- 海上风电: 7-9D (尾流恢复慢)
