# 案例8: 并网连接

## 简介

本案例研究风力发电机组的并网动态特性，这是风电大规模接入电网的关键技术问题。案例演示电网电压跌落响应、频率波动响应、有功/无功功率动态控制以及低电压穿越(LVRT)能力。随着风电渗透率不断提高，电网对风电机组的并网性能提出了越来越严格的要求，包括电压支撑、频率响应、故障穿越等能力。掌握这些并网特性对于满足电网规程、保障系统稳定运行具有重要意义。

## 理论背景

风电并网的核心要求包括：(1)低电压穿越(LVRT)：电网故障导致电压跌落时，风电机组需保持并网运行并提供无功支撑，中国国标GB/T 19963.1-2011规定0.15秒内电压可跌落至20%，0.625秒需恢复至90%；(2)频率响应：电网频率偏差时，风电机组需按下垂特性调整有功功率，典型下垂系数3-5%；(3)有功功率控制：需具备快速功率爬坡速率限制功能，防止风速波动对电网冲击；(4)无功电压控制：可提供动态无功支撑，维持电网电压稳定，功率因数通常要求在0.95超前至0.95滞后范围内可调。这些功能主要由变流器控制系统实现。

## 代码说明

### 主要函数

本案例采用简化的并网动态模型，通过仿真演示：

1. **demo_voltage_dip()**: 电压跌落响应
   - 模拟电网电压跌落至50%，持续1秒
   - 有功功率随电压平方关系下降
   - 无功功率注入支撑电压恢复

2. **demo_frequency_response()**: 频率响应
   - 模拟电网频率跌落至49.5 Hz
   - 下垂控制：df = -0.5 Hz → ΔP ≈ +5%
   - 功率响应公式：P = P0 - K_droop × (f - f0)

3. **demo_lvrt_curve()**: LVRT要求分析
   - GB/T 19963.1-2011标准曲线
   - 不同严重程度的电压跌落场景
   - 必须保持并网区域vs允许脱网区域

### 关键参数

- **电压正常范围**: 0.9-1.1 pu
- **频率正常范围**: 49.5-50.5 Hz (中国)
- **下垂系数**: 3-5% (频率响应)
- **无功支撑系数**: 电压跌落时Q ≈ 0.3 pu

## 参数说明

### 电压跌落场景
- **跌落时刻**: 1.0 s
- **跌落幅度**: 50% (0.5 pu)
- **持续时间**: 1.0 s
- **恢复时间**: ~1.0 s

### 频率响应参数
- **额定频率**: 50 Hz
- **频率偏差**: -0.5 Hz
- **下垂系数**: 5%
- **响应时间**: < 2 s

### LVRT标准 (GB/T 19963.1-2011)
- **t = 0.00 s**: 0% 电压 (最严重)
- **t = 0.15 s**: 20% 电压
- **t = 0.625 s**: 90% 电压
- **t = 3.0 s**: 100% 电压恢复

## 工程应用

并网性能的实际要求：

### 中国电网规程
1. **GB/T 19963.1-2011**: 风电场接入电力系统技术规定
2. **Q/GDW 392-2009**: 风电场接入电网技术规定
3. **功率因数**: 0.95超前~0.95滞后可调
4. **电压调节范围**: ±10%额定电压

### 国际标准
1. **IEC 61400-21**: 风电机组并网特性测量
2. **IEEE 1547**: 分布式资源并网标准
3. **各国电网规程**: 德国E.ON、西班牙PO 12.3等

### 关键技术
- **Crowbar保护**: DFIG短路保护
- **动态电压支撑**: 快速无功注入
- **储能辅助**: 平滑功率波动
- **虚拟惯量**: 提供频率支撑
