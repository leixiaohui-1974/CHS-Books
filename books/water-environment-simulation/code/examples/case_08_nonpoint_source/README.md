# 案例8：非点源污染模拟

## 案例背景

某流域接纳农业面源、城市径流等非点源污染，需要评估降雨径流对河流水质的影响，制定面源污染控制方案。

**流域参数**:
- 河段长度: L = 15 km
- 河流流量: Q = 30 m³/s
- 流速: u = 0.6 m/s
- COD降解系数: k = 0.18 day⁻¹

**土地利用**:
- 农田: 30 km², CN=78
- 城市居住区: 15 km², CN=85
- 森林: 25 km², CN=55

**降雨情景**:
- 降雨量: P = 50 mm
- 降雨强度: I = 10 mm/hr
- 持续时间: 5 hours

**水质目标**: III类水（COD ≤ 20 mg/L）

## 学习目标

1. 理解非点源污染特点和机理
2. 掌握SCS-CN径流计算方法
3. 学会EMC污染负荷估算
4. 掌握分布式源项处理
5. 理解降雨-径流-水质耦合

## 核心理论

### 1. 非点源污染特点

**定义**: 污染物以分散、面状方式进入水体的污染源。

**特点**:
- **分散性**: 污染源分布广泛，难以确定具体位置
- **随机性**: 污染负荷与降雨、季节等随机因素相关
- **间歇性**: 主要在降雨期间产生
- **滞后性**: 污染峰值滞后于降雨峰值

**主要类型**:
- 农业面源：化肥、农药、畜禽养殖
- 城市径流：路面冲刷、雨水管道
- 大气沉降：酸雨、颗粒物
- 林地径流：腐殖质、营养盐

### 2. SCS-CN径流计算

#### 基本方程

**径流深**:
```
Q = (P - Ia)² / (P - Ia + S)    (P > Ia)
Q = 0                            (P ≤ Ia)
```

**参数关系**:
```
S = 25400 / CN - 254    (S in mm)
Ia = 0.2 * S            (初损)
```

**径流系数**:
```
α = Q / P
```

#### 曲线数（CN）

**定义**: 反映流域产流能力的参数（0-100）

**影响因素**:
1. **土地利用**: 不透水面积大 → CN高
2. **土壤类型**: 
   - A类（砂土）: 渗透性强，CN低
   - B类（壤土）: 渗透性中等
   - C类（粘壤土）: 渗透性差
   - D类（粘土）: 渗透性很差，CN高
3. **前期土壤湿度（AMC）**:
   - AMC-I: 干燥
   - AMC-II: 正常（标准条件）
   - AMC-III: 湿润

**典型CN值**（AMC-II, B类土壤）:

| 土地利用 | CN值 |
|---------|------|
| 森林 | 55 |
| 草地 | 69 |
| 农田（旱作） | 78 |
| 牧草地 | 69 |
| 城市居住区 | 85 |
| 城市商业区 | 92 |
| 工业区 | 88 |
| 水体 | 100 |

### 3. 事件平均浓度（EMC）

#### 定义

**EMC**: 一次降雨径流过程中污染物的平均浓度。

#### 污染负荷计算

**基本方程**:
```
L = EMC * V * 10⁻⁶
```

其中：
- L: 污染负荷 (kg)
- EMC: 事件平均浓度 (mg/L)
- V: 径流量 (m³)

#### 典型EMC值

**城市径流**:

| 污染物 | EMC (mg/L) | 范围 |
|--------|-----------|------|
| TSS | 150 | 50-300 |
| COD | 90 | 40-200 |
| BOD | 15 | 5-25 |
| TN | 2.5 | 1-5 |
| TP | 0.4 | 0.1-1.0 |
| NH₃-N | 1.2 | 0.5-3.0 |

**农业径流**:

| 污染物 | EMC (mg/L) | 范围 |
|--------|-----------|------|
| TSS | 500 | 200-1000 |
| COD | 80 | 30-150 |
| TN | 5.0 | 2-10 |
| TP | 1.2 | 0.5-3.0 |

### 4. 初期冲刷效应

#### 现象

降雨初期污染物浓度远高于后期。

**机理**:
- 地表累积污染物在降雨初期快速冲刷
- 后期地表污染物减少，浓度降低

**经验规律**:
```
初期30%径流 → 携带60-80%污染物
```

#### 初期冲刷系数

```
FF = C_initial / C_average
```

**典型值**:
- 短时强降雨: FF = 2-3
- 中等降雨: FF = 1.5-2
- 长时降雨: FF = 1.2-1.5

### 5. 累积-冲刷模型

#### 累积阶段

**指数累积**:
```
P(t) = P_max * (1 - e^(-k₁*t))
```

其中：
- P(t): t时刻地表污染物累积量
- P_max: 最大累积量
- k₁: 累积速率常数
- t: 距上次降雨时间

#### 冲刷阶段

**冲刷负荷**:
```
W = k₂ * P * I^n
```

其中：
- W: 冲刷负荷
- k₂: 冲刷系数
- P: 累积量
- I: 降雨强度
- n: 指数（通常1-2）

### 6. 河流分布式源项

#### 源项表达

**分布式源项**:
```
S(x) = dL/dx    (kg/m/s)
```

**河流水质方程**:
```
u * ∂C/∂x = D * ∂²C/∂x² - k * C + S(x)/Q
```

#### 浓度增量

**单位河段浓度增量**:
```
ΔC = S(x) * Δx / Q
```
