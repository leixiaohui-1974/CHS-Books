# 案例9：热污染扩散模拟

## 案例背景

某火电厂冷却水排入河流，排放温度35°C，流量20 m³/s。需要评估温排水扩散过程、混合区范围和对水生生物的影响。

**河流参数**:
- 流量: Q = 100 m³/s
- 流速: u = 0.8 m/s
- 河宽: B = 80 m
- 水深: H = 1.6 m
- 环境温度: 20°C

**温排水参数**:
- 排放流量: 20 m³/s
- 排放温度: 35°C
- 温升: 15°C
- 排放位置: 侧向排放

**环境标准**: 温升≤3°C（GB 3838-2002）

## 学习目标

1. 理解温排水扩散机理
2. 掌握二维热扩散模拟
3. 学会混合区评估方法
4. 理解生物热耐受性
5. 掌握冷却系统设计

## 核心理论

### 1. 热污染概念

**定义**: 工业冷却水、核电站等向水体排放的高温废水造成的水温升高现象。

**来源**:
- 火电厂冷却水（最主要）
- 核电站冷却水
- 钢铁厂冷却水
- 化工厂工艺废水

**危害**:
- 降低溶解氧
- 影响水生生物
- 改变生态系统
- 促进藻类生长

### 2. 温度场控制方程

#### 二维热传输方程

```python
∂T/∂t + u*∂T/∂x + v*∂T/∂y = Kx*∂²T/∂x² + Ky*∂²T/∂y² - λ*(T-Ta)
```

**物理意义**:
- 第一项: 时间变化
- 第二、三项: 对流输运
- 第四、五项: 扩散输运
- 第六项: 表面热交换

#### 稳态简化

```python
u*∂T/∂x = Kx*∂²T/∂x² + Ky*∂²T/∂y² - λ*(T-Ta)
```

### 3. 高斯羽流模型

#### 解析解（简化）

对于侧向排放的温排水：

```python
T(x,y) = Ta + ΔT0 * exp(-y²/(2σy²)) * exp(-λ*x/u)
```

其中：
- ΔT0: 初始温升
- σy: 横向扩散宽度, σy = √(2*Ky*x/u)
- λ: 表面热交换系数

**特点**:
- 横向呈高斯分布
- 纵向呈指数衰减

### 4. 表面热交换

#### 热交换机制

**散热方式**:

1. **蒸发散热**（最重要）:
```python
Qe = f(u) * (es - ea)
f(u) = 9.2 + 0.46*u²  (Brady公式)
```

2. **对流散热**:
```python
Qc = h * (Tw - Ta)
h ≈ 4.5*(1 + 0.2*u)
```

3. **辐射散热**:
```python
Qr ≈ 5.5 * (Tw - Ta)
```

4. **太阳辐射加热**:
```python
Qs = Rs * (1 - α)
α ≈ 0.06 (水面反射率)
```

#### 净热通量

```python
Qnet = -(Qe + Qc + Qr) + Qs
```

**热交换系数**:
```python
λ = |Qnet| / (ρ*Cp*H*ΔT)
```

典型值: 0.05-0.2 day⁻

### 5. 混合区（Mixing Zone）

#### 定义

温排水与受纳水体混合，温度逐渐降低的区域。

#### 标准要求（GB 3838-2002）

**温升限值**:
- 温排水造成的温升 ≤ 3°C
- 夏季最高水温 ≤ 35°C

**混合区规定**:
- 面积应尽可能小
- 不得妨碍航运、渔业
- 不得影响取水口

#### 计算方法

**混合区长度**:
```python
找到 x 使得 max(T(x,y)) - Ta ≤ 3°C
```

**混合区宽度**:
```python
找到 y 使得 T(x,y) - Ta ≤ 3°C
```

### 6. 生物热耐受性

#### 温度对生物的影响

**鱼类**:

| 类别 | 最适温度 | 胁迫温度 | 致死温度 |
|------|---------|---------|---------|
| 冷水鱼 | 10-15°C | 20°C | 24°C |
| 温水鱼 | 20-28°C | 32°C | 35°C |

**无脊椎动物**:
- 最适温度: 15-25°C
- 胁迫温度: 28°C
- 致死温度: 32°C

**藻类**:
- 最适温度: 20-30°C
- 胁迫温度: 35°C
- 致死温度: 40°C

#### 温度效应

**急性效应**:
- 高温致死
- 热休克

**慢性效应**:
- 生长受抑制
- 繁殖能力下降
- 免疫力降低
- 疾病增加

### 7. 冷却系统

#### 直流冷却

**特点**:
- 取水 → 冷却 → 排放
- 温升大（8-15°C）
- 水量大

**优点**: 简单、投资少  
**缺点**: 热污染大

#### 循环冷却

**冷却塔**:
- 水循环使用
- 蒸发散热
- 补充新鲜水

**优点**: 热污染小  
**缺点**: 投资大、能耗高

#### 混合冷却

结合直流和循环优点。

**考虑因素**:
