# 案例6：河流自净能力与环境容量评估

## 案例背景

某河流接纳多个污染源排放，需要评估河流的自净能力和环境容量，确定合理的污染物排放总量，保证河流水质达标。

**河流参数**:
- 流量: Q = 20 m³/s
- 流速: u = 0.5 m/s
- 断面面积: A = 40 m²
- 河宽: B = 15 m
- 水深: H = 2.7 m
- BOD降解系数: kd = 0.2 day⁻¹
- 复氧系数: ka = 0.6 day⁻¹

**水质现状**:
- 本底BOD: 5 mg/L
- 本底DO: 7 mg/L
- 饱和DO: 9 mg/L

**水质目标**: III类水（DO ≥ 5 mg/L, BOD ≤ 4 mg/L）

## 学习目标

1. 理解河流自净能力的概念和影响因素
2. 掌握环境容量计算方法
3. 学会水质综合评价方法
4. 理解河流功能区划分
5. 掌握污染物总量控制方法

## 核心理论

### 1. 自净能力（Self-Purification Capacity）

**定义**: 水体通过物理、化学和生物作用，降解和转化污染物，恢复到原有水质状态的能力。

**主要机制**:

#### 物理自净
- **稀释作用**: 污染物浓度降低
- **沉降作用**: 悬浮物和颗粒物沉降
- **挥发作用**: 挥发性物质逸散

#### 化学自净
- **氧化还原**: 有机物氧化分解
- **吸附作用**: 污染物被吸附
- **沉淀作用**: 形成难溶物沉淀

#### 生物自净
- **微生物分解**: 细菌分解有机物
- **植物吸收**: 水生植物吸收营养盐
- **生物链**: 食物链净化作用

### 2. 自净系数

**自净系数** (f):
```
f = ka / kd
```

**物理意义**:
- f > 1: 复氧 > 耗氧，DO逐渐恢复
- f = 1: 复氧 = 耗氧，DO维持不变
- f < 1: 复氧 < 耗氧，DO持续下降

**分级标准**:
- f < 1: 自净能力弱（污染加重）⚠️
- 1 ≤ f < 2: 自净能力一般
- 2 ≤ f < 5: 自净能力较强
- f ≥ 5: 自净能力强 ✓

**影响因素**:
1. **水力条件**: 流速↑ → 复氧↑ → f↑
2. **水深**: 水深↑ → 复氧↓ → f↓
3. **温度**: 温度↑ → ka和kd都↑
4. **污染负荷**: BOD↑ → 耗氧↑ → DO↓

### 3. 环境容量（Environmental Capacity）

**定义**: 在满足水质目标的前提下，水体能够容纳的污染物最大量。

#### 稀释容量（Dilution Capacity）

**仅考虑稀释作用**:
```
W = Q × (C_standard - C_background)
```

**单位**: kg/day

**适用**: 保守性污染物（不降解）

#### 降解容量（Degradation Capacity）

**考虑降解作用**:
```
W = W_dilution + W_degradation

W_degradation = Q × kd × C_standard × t
```

其中：
- t: 行程时间 (day)
- t = L / (u × 86400)

**适用**: 可降解污染物（BOD、COD等）

#### 同化容量（Assimilative Capacity）

**基于S-P模型**:

在保证DO达标的前提下，能够接纳的最大BOD量。

```
临界条件: DOc ≥ DO_standard
反推: L0_max
同化容量: W = Q × L0_max
```

### 4. 水质综合评价

#### 单因子指数法

```
P_i = C_i / S_i

P_i < 1: 达标 ✓
P_i ≥ 1: 超标 ⚠️
```

**特点**: 简单直观，"一票否决"

#### 综合污染指数法

```
P = Σ(w_i × P_i) / Σw_i
```

**分级**:
- P ≤ 0.2: 清洁
- 0.2 < P ≤ 0.4: 尚清洁
- 0.4 < P ≤ 1.0: 轻度污染
- 1.0 < P ≤ 2.0: 中度污染
- P > 2.0: 重度污染

#### 水质标识指数法（WQII）

```
WQII = X1.X2X3

X1: 水质类别（1-5）
X2X3: 污染分指数（01-99）
```

**示例**:
- WQII = 3.05 → III类水，轻度超标
- WQII = 2.00 → II类水，达标

### 5. 河流功能区划分

**GB 3838-2002 地表水环境质量标准**:

| 类别 | 主要功能 | DO | BOD₅ | NH₃-N | TP |
|------|---------|-----|------|-------|-----|
| I类 | 源头水、保护区 | ≥7.5 | ≤3 | ≤0.15 | ≤0.02 |
| II类 | 饮用水一级保护区 | ≥6 | ≤3 | ≤0.5 | ≤0.1 |
| III类 | 饮用水二级保护区、渔业 | ≥5 | ≤4 | ≤1.0 | ≤0.2 |
| IV类 | 工业用水、非接触娱乐 | ≥3 | ≤6 | ≤1.5 | ≤0.3 |
| V类 | 农业用水、景观 | ≥2 | ≤10 | ≤2.0 | ≤0.4 |

### 6. 污染物总量控制

**总量控制**是环境容量管理的核心。

**计算步骤**:
1. 确定水质目标（功能区划）
2. 计算环境容量（W_total）
3. 调查现状排放量（W_current）
4. 确定削减量（ΔW = W_current - W_total）
5. 分配排放指标（各污染源）

**分配原则**:
- 公平性原则
- 效率原则
- 可达性原则

## 预期结果

### 自净能力

**本案例**:
- f = ka/kd = 0.6/0.2 = 3.0
- **自净能力较强** ✓

**改善措施**（如果f < 2）:
- 增加曝气（提高ka）
- 减少污染负荷（降低kd）
- 改善水力条件（增加湍流）

### 环境容量

**稀释容量**（保守估计）:
- BOD: ~200 kg/day
- NH₃-N: ~100 kg/day

**总环境容量**（考虑降解，50km）:
- BOD: ~400 kg/day
- **降解作用使容量翻倍**

### 水质评价

**单因子**:
- BOD: P = 5/4 = 1.25（超标25%）⚠️
- DO: P = 7/5 = 1.4（达标）✓

**综合评价**:
- 水质类别: IV类
- 适用功能: 工业用水、景观

## 工程意义

### 1. 排污许可证管理

**总量控制**:
- 计算流域环境容量
- 分配各企业排污指标
- 核发排污许可证

**动态管理**:
- 枯水期：削减排放
- 丰水期：适当放宽

### 2. 水功能区划

**功能定位**:
- 上游：饮用水源（I-II类）
- 中游：工业/农业（III-IV类）
- 下游：景观（IV-V类）

**分区管理**:
- 不同功能区不同标准
- 优先保护饮用水源

### 3. 污染源布局

**选址原则**:
- 避开饮用水源
- 布置在自净能力强的河段
- 考虑稀释和混合条件

### 4. 应急管理

**污染事故**:
- 根据环境容量评估影响
- 确定需要削减的排放量
- 制定应急处置方案

**预警阈值**:
- 实时排放量接近环境容量的80%
- 启动应急响应

### 5. 生态修复

**提升自净能力**:
- 河道生态化改造（增加湍流）
- 种植水生植物（吸收营养盐）
- 建设生态湿地（净化功能）

## 关键概念

### 自净能力

## 扩展知识

### 1. 环境容量模型

**一维稳态模型**:
```
C(x) = C0 × exp(-k × x/u)
```

**二维模型**:
- 考虑横向扩散
- 适用于宽浅河流

**动态模型**:
- 考虑流量变化
- 考虑季节因素

### 2. 总量分配方法

**公平分配法**:
- 按人口分配
- 按产值分配

**优化分配法**:
- 最小成本
- 最大效益

**协商分配法**:
- 利益相关方协商
- 达成共识

### 3. 国际经验
