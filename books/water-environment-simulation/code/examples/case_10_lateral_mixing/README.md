# 案例10：河流横向混合模拟

## 案例背景

某化工厂从河流一侧岸边排放废水，需要评估污染物横向扩散混合过程和对岸水质影响。

**河流参数**:
- 河宽: B = 100 m
- 平均水深: H = 3 m
- 平均流速: u = 0.5 m/s
- 河流流量: Q = 150 m³/s
- 水力坡降: I = 0.0002

**排放参数**:
- 排放位置: 左岸（y = 0）
- 排放流量: 2 m³/s
- 排放浓度: 100 mg/L
- 污染物: COD

**环境标准**: COD ≤ 20 mg/L（GB 3838-2002 III类）

## 学习目标

1. 理解河流横向混合机理
2. 掌握二维对流-扩散模拟
3. 学会横向扩散系数计算
4. 理解完全混合概念
5. 掌握侧向排放影响评估

## 核心理论

### 1. 横向混合过程

**定义**: 污染物从排放点向河流横断面扩散，最终达到横向均匀分布的过程。

**驱动力**:
- 横向湍流扩散
- 二次流
- 河床形态影响

**特征**:
- 近区：浓度梯度大，横向不均匀
- 远区：浓度梯度小，接近均匀

### 2. 控制方程

#### 二维对流-扩散方程

```
∂C/∂t + u*∂C/∂x + v*∂C/∂y = Ex*∂²C/∂x² + Ey*∂²C/∂y²
```

**物理意义**:
- 第一项: 时间变化
- 第二、三项: 对流输运
- 第四、五项: 扩散输运

#### 稳态简化

对于稳定排放：
```
u*∂C/∂x = Ex*∂²C/∂x² + Ey*∂²C/∂y²
```

进一步简化（Ex << u*dx）：
```
u*∂C/∂x ≈ Ey*∂²C/∂y²
```

### 3. 横向扩散系数

#### 经验公式

**Elder公式（1959）**:
```
Ey = 0.23 * H * u*
```

**Fischer公式（1979）**:
```
Ey = 0.15 * H * u*
```

其中，u* 为摩阻流速：
```
u* = √(g*H*I)
```

**典型值**:
- 天然河流: 0.1-1.0 m²/s
- 规则渠道: 0.05-0.3 m²/s
- 宽浅河流: 0.5-2.0 m²/s

#### 影响因素

1. **水深H**: Ey ∝ H
2. **摩阻流速u***: Ey ∝ u*
3. **河床形态**: 不规则河床增大Ey
4. **弯道**: 二次流增强混合
5. **植被**: 减小Ey

### 4. 横向混合长度

#### 定义

从排放点到横断面浓度达到均匀分布（变异系数<5%）的纵向距离。

#### 经验公式

**Fischer公式（1979）**:
```
L_mix ≈ 0.4 * u * B² / Ey
```

**简化估算**:
```
L_mix ≈ (50-100) * B
```

对于B=100m的河流：
```
L_mix ≈ 5-10 km
```

#### 混合时间

```
T_mix = B² / (6 * Ey)
```

对于Ey=0.5 m²/s, B=100m：
```
T_mix ≈ 3333 s ≈ 0.93 hour
```

### 5. 侧向排放浓度分布

#### 近区（未完全混合）

**高斯分布模型**:
```
C(x,y) = (M / (4*π*Ey*x/u)^0.5) * exp(-u*y² / (4*Ey*x))
```

其中，M为排放强度（kg/s）。

**特点**:
- 排放侧浓度高
- 对岸浓度低
- 纵向衰减

#### 远区（完全混合）

**均匀分布**:
```
C_mix = Q_discharge * C_discharge / Q_river
```

对于本案例：
```
C_mix = 2 * 100 / 150 = 1.33 mg/L
```

### 6. 对岸影响评估

#### 岸边浓度

**解析解**:
```
C(x, y=B) = C0 * exp(-π²*Ey*x / (u*B²))
```

**影响距离**:

当 C(x,B) = 0.1*C_mix 时的距离：
```
x_impact = u*B² / (π²*Ey) * ln(10*C0/C_mix)
```

#### 保护距离

**取水口保护**:
- 排放口下游距离 > L_mix
- 对岸侧距离 > 0.5*B
- 考虑完全混合前的高浓度区

## 预期结果

### 横向扩散系数

**计算结果**:
- 摩阻流速: u* ≈ 0.077 m/s
- Elder公式: Ey ≈ 0.053 m²/s
- Fischer公式: Ey ≈ 0.035 m²/s
- 推荐值: Ey ≈ 0.044 m²/s

### 混合长度

**估算结果**:
- 理论公式: L_mix ≈ 11,360 m (≈11.4 km)
- 经验公式: L_mix ≈ 5,000-10,000 m
- 推荐值: L_mix ≈ 10 km (100倍河宽)

### 浓度分布

**纵向分布**:
- 0-2 km: 排放侧浓度高（>10 mg/L）
- 2-10 km: 逐渐混合，横向差异减小
- >10 km: 接近完全混合（1.33 mg/L）

**横向分布**:
- 排放侧（y=0）: 初始最高，逐渐降低
- 中线（y=50m）: 浓度适中
- 对岸（y=100m）: 初始最低，逐渐升高

### 对岸影响

**影响距离**:
- 2 km处: 对岸浓度 < 1 mg/L
- 5 km处: 对岸浓度 ≈ 1.2 mg/L
- 10 km处: 对岸浓度 ≈ 1.33 mg/L（完全混合）

**达标评估**:
- 排放侧: 长距离超标（>20 mg/L）
- 对岸: 全程达标（< 20 mg/L）

## 工程意义

### 1. 排污口选址

**选址原则**:
- 避开取水口上游
- 考虑横向混合距离
- 优先选择对岸无取水的河段
- 考虑河流弯道（增强混合）

**安全距离**:
- 取水口下游 > L_mix
- 对岸侧距离 > 0.5*B
- 考虑水文变化

### 2. 取水口布置

**保护措施**:
- 取水口设在排放口上游
- 取水口设在排放口对岸
- 增加横向距离
- 深度取水（表层污染物浓度高）

**监测要求**:
- 取水口实时监测
- 排放口在线监控
- 应急切换系统

### 3. 排放优化

**优化策略**:
- 增加排放扩散器（多点排放）
- 使用扩散管（增强初始混合）
- 选择枯水期之外排放
- 分时段排放

**效果提升**:
- 多点排放可减少50%混合长度
- 扩散器可提高3-5倍混合效率

### 4. 环境影响评估

**评估内容**:
- 横向混合区范围
- 对岸水质影响
- 取水口安全保障
- 生态敏感区保护

**环评要求**:
- 2D水质模拟
- 不同水文条件分析
- 突发事故情景模拟
- 应急响应预案

### 5. 水质监测布点

**监测断面**:
- 排放口断面（近场）
- 混合过程断面（0.5*L_mix）
- 完全混合断面（L_mix）
- 对照断面（上游）

**横向布点**:
- 排放侧（y=0, B/4）
- 中线（y=B/2）
- 对岸侧（y=3B/4, B）

## 关键概念

### 横向混合
污染物从排放点向河流横断面扩散达到均匀分布的过程

### 横向扩散系数
表征横向湍流扩散强度的系数（Ey）

### 混合长度
从排放点到完全混合的纵向距离（L_mix）

### 摩阻流速
表征河床摩阻强度的特征流速（u*）

### 完全混合
横断面浓度分布达到均匀（变异系数<5%）

### 二次流
河流横断面上的环流，增强横向混合

## 扩展知识

### 1. 三维模拟

**考虑垂向分层**:
- 表层流速快，浓度高
- 底层流速慢，浓度低
- 垂向混合（Ez）

### 2. 非稳态模拟

**时间变化**:
- 排放流量变化
- 河流流量变化
- 瞬时排放（事故）

### 3. 弯道效应

**二次流**:
- 增强横向混合
- 减少混合长度30-50%
- 弯曲系数K=1.5-3.0

**修正公式**:
```
L_mix = L_mix_直道 / K
```

### 4. 多点排放

**优化布置**:
- 多个排放口
- 交错排列
- 增强初始混合

**效果**:
- 减少混合长度
- 降低峰值浓度
- 改善水质分布

### 5. 扩散器设计

**射流扩散器**:
- 动量射流
- 浮力羽流
- 近场快速稀释

**设计参数**:
- 喷口直径
- 喷口间距
- 喷射角度
- 佛劳德数
