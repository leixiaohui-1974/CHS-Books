# 案例14：湖泊营养盐输入响应分析

## 案例背景

**工程场景**：评估点源削减对湖泊水质的改善效果  
**模拟对象**：中型湖泊（面积10 km²，平均水深8 m），接受污水处理厂排放

**湖泊参数**：
- 面积：10 km²
- 水深：8 m
- 停留时间：500天
- 当前磷浓度：60 μg/L（富营养）

**管理目标**：
- 目标：≤ 30 μg/L（中营养）
- 评估削减方案
- 制定控制目标

---

## 核心理论

### 1. Vollenweider模型

**控制方程**：
```
dP/dt = L/V - P*Q/V - σ*P*A/V
```

其中：
- P: 磷浓度 (μg/L)
- L: 外源负荷 (mg/d)
- V: 体积 (m³)
- Q: 出流 (m³/d)
- σ: 沉降速率 (m/d)
- A: 面积 (m²)

**稳态浓度**：
```
P_ss = L / (Q + σ*A)
```

### 2. 磷滞留系数

**定义**：
```
R = 1 - P_ss*Q/L = σ*A/(Q + σ*A)
```

**物理意义**：
- R = 0: 无滞留
- R = 1: 完全滞留
- 典型值：0.1-0.9

### 3. 营养状态分类

根据OECD标准：
- 贫营养：TP < 10 μg/L
- 中营养：10-30 μg/L
- 富营养：30-100 μg/L
- 超富营养：TP ≥ 100 μg/L
