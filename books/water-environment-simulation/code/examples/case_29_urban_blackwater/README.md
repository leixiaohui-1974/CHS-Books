# 案例29：城市黑臭水体治理方案评估

## 案例简介

本案例演示城市黑臭水体综合治理方案的评估方法，通过建立黑臭评价体系和治理效果模型，分析不同治理措施对水质改善的贡献，优化治理方案以实现经济有效的黑臭消除目标。案例采用城市黑臭水体模型（UrbanBlackwaterModel）基于溶解氧、氨氮、透明度等指标评估黑臭程度，模拟源头控制、底泥清淤、曝气增氧、生态修复等多种措施的协同作用，并在预算约束下优化治理方案，为黑臭水体治理工程提供技术支撑。

## 理论背景

城市黑臭水体是指城市河道因有机污染严重而呈现黑色、散发臭味的水体，是城市水环境的顽疾。黑臭成因主要包括：外源污染输入（生活污水直排、雨污混流）、内源污染释放（底泥中有机质厌氧分解）、水体流动性差（断头浜、闸坝阻隔）。判定标准为：透明度<25cm、溶解氧<2mg/L、氧化还原电位<50mV、氨氮>8mg/L（满足任意一项即为黑臭）。治理措施分为四类：源头控制（截污纳管、雨污分流）、内源治理（底泥清淤、原位钝化）、补水活化（引清释污、曝气增氧）、生态修复（水生植物、微生物）。综合治理强调多措施协同：源头控制是根本，内源治理是关键，补水活化是辅助，生态修复是长效保障。方案优化需考虑技术可行性、经济性、时效性和可持续性。

## 代码说明

### 主要类和函数

1. **UrbanBlackwaterModel类**
   - `__init__(river_length, width, depth)`: 初始化模型，设置河道几何参数
   - `assess_current_status(DO, NH3_N, transparency)`: 评估黑臭现状
     - DO: 溶解氧浓度
     - NH3_N: 氨氮浓度
     - transparency: 透明度
     - 返回：黑臭等级（轻度/重度）、综合评分
   - `simulate_treatment_effects(measures)`: 模拟治理措施效果
     - measures: 措施字典（source_control, dredging, aeration, ecological_restoration）
     - 返回：DO改善量、氨氮削减量

2. **optimize_treatment_plan函数**
   - 在预算约束下优化治理方案
   - 参数：budget（预算）、target_quality（目标水质等级）
   - 返回：最优措施组合

3. **主程序流程**
   - 创建河道模型（长2000m，宽15m，深2m）
   - 评估现状（DO=1.5mg/L，NH3-N=10mg/L，透明度=0.15m）
   - 模拟四种治理措施的效果
   - 优化治理方案（预算800万元，目标III类水）
   - 生成治理前后对比图
   - 生成措施效果评估图
