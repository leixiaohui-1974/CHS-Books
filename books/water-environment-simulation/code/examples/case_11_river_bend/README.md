# 案例11：河流弯道水质模拟

## 案例背景

某化工厂排污口位于河流弯道段，需要评估弯道二次流对污染物横向分布的影响，以及与直道的差异。

**河流参数**:
- 河宽: B = 80 m
- 平均水深: H = 2.5 m
- 平均流速: u = 0.6 m/s
- 河流流量: Q = 120 m³/s
- 直道横向扩散系数: Ey = 0.05 m²/s

**弯道参数**:
- 弯道长度: 1500 m
- 转角: 90°
- 曲率半径: R = 955 m
- 宽曲比: B/R = 0.084

**排放参数**:
- 排放流量: 1.5 m³/s
- 排放浓度: 80 mg/L
- 排放位置: 弯道内侧（凸岸）

## 学习目标

1. 理解弯道二次流机理
2. 掌握弯道增强混合效应
3. 学会浓度偏向现象分析
4. 理解弯道与直道差异
5. 掌握弯道水质优化设计

## 核心理论

### 1. 弯道二次流

#### 成因

**离心力不平衡**:
- 表层流速快，离心力大
- 底层流速慢，离心力小
- 产生横向压力梯度
- 形成横向环流

**环流特征**:
- 表层：流向凹岸（外岸）
- 底层：流向凸岸（内岸）
- 形成螺旋流动

#### 强度

**影响因素**:
1. 流速u：流速越大，二次流越强
2. 曲率半径R：半径越小，二次流越强
3. 宽深比B/H：比值越大，二次流越强

**最大横向流速**:
```python
v_max ≈ 0.1 * u * (B/R)
```

对于本案例：
```python
v_max ≈ 0.1 * 0.6 * (80/955) ≈ 0.005 m/s
```

### 2. 弯道对混合的影响

#### 横向混合增强

**增强机制**:
- 二次流横向对流
- 螺旋流强化湍流
- 有效扩散系数增大

**增强系数**:
```python
K_bend = 1 + C * (B/R)^n
```

典型值：
- C = 2.0
- n = 1.0
- K_bend = 1.5-3.0

**有效扩散系数**:
```python
Ey_eff = Ey_straight * K_bend
```

#### 混合长度缩短

**Fischer公式**:
```python
L_mix_bend = L_mix_straight / K_bend
```

**缩短比例**:
- K_bend = 2.0 → 缩短50%
- K_bend = 3.0 → 缩短67%

### 3. 浓度偏向现象

#### 物理原因

**二次流输运**:
- 表层二次流携带污染物向凹岸
- 底层二次流较弱
- 净输运指向凹岸

**结果**:
- 凹岸侧浓度高于凸岸侧
- 偏向比通常1.5-3.0
- 排放在凸岸，污染物向凹岸富集

#### 工程意义

**排污口选址**:
- 排放在凸岸：污染物向凹岸偏移
- 排放在凹岸：污染物更快扩散
- 需考虑下游取水口位置

**取水口布置**:
- 弯道内应避开凹岸侧
- 或布置在弯道上游
- 充分考虑浓度偏向

### 4. 弯道与直道对比

#### 主要差异

| 特征 | 直道 | 弯道 | 比值 |
|------|------|------|------|
| 横向扩散系数 | Ey | Ey×K | ×2-3 |
| 混合长度 | L | L/K | ×0.3-0.5 |
| 横向流速 | ~0 | 0.1×u×B/R | 新增 |
| 浓度分布 | 对称 | 偏向凹岸 | 非对称 |

#### 适用条件

**弯道效应显著**:
- B/R > 0.05
- 弯道角度 > 45°
- Fr < 1（亚临界流）

**弯道效应较小**:
- B/R < 0.02
- 弯道角度 < 30°
- 高Fr数流动

### 5. 弯道水质模拟方法

#### 简化方法（本案例）

**增强扩散系数法**:
- 计算增强系数K_bend
- Ey_eff = Ey × K_bend
- 叠加横向对流

**优点**:
- 计算简单快速
- 参数容易获取
- 工程实用性强

**缺点**:
- 无法精确描述二次流结构
- 浓度偏向估算粗略

#### 完整方法

**三维模型**:
- 求解3D N-S方程
- 计算完整二次流场
- 耦合输运方程

**特点**:
- 计算复杂
- 精度高
- 计算成本大

## 工程意义

### 1. 排污口选址优化

**弯道位置选择**:
- **凸岸排放**（推荐）：
  - 污染物向凹岸偏移
  - 避开凸岸取水
  - 利用二次流输运

- **凹岸排放**（谨慎）：
  - 污染物快速扩散
  - 但凹岸常有取水
  - 需详细评估

**与直道对比**:
- 弯道混合快50%
- 达标距离缩短
- 环境容量提升

### 2. 取水口保护

**布置原则**:
- 避开凹岸侧
- 考虑浓度偏向
- 设置保护距离
- 监测预警

**保护距离**:
- 纵向: > L_mix_bend
- 横向: 凸岸侧优先
- 深度: 表层浓度高

### 3. 利用弯道优势

**增强混合**:

**效果提升**:

- 监测成本降低

### 4. 河流整治

## 关键概念

### 二次流
弯道中垂直于主流方向的横向环流

### 横向环流
表层向凹岸、底层向凸岸的螺旋流动

### 弯道增强系数
弯道横向扩散系数相对直道的增强倍数

### 浓度偏向
污染物浓度向凹岸富集的现象

### 曲率半径
弯道中心线的曲率半径

### 宽曲比
河宽与曲率半径的比值，表征弯道强度
