## 📚 教材概述

### 什么是生态水力学？

**生态水力学（Ecohydraulics）** 是研究水流特性与水生生态系统相互作用的交叉学科，将水力学计算与生态学原理相结合，为生态保护与修复工程提供定量化设计依据。

### 教材特色

1. **交叉融合**: 水力学计算 + 生态学指标 + 工程设计
2. **案例驱动**: 28个真实生态水利工程案例
3. **定量分析**: 基于数值模拟的生境评价方法
4. **工程导向**: 从生态诊断到工程设计的完整流程
5. **Python实现**: 提供完整的计算代码和可视化工具

### 与《明渠水力学》的关系

```
明渠水力学（基础）
    ↓ 提供水力计算方法
生态水力学（应用）
    ↓ 水力学 + 生态学
生态修复工程设计
```

**知识互补**:
- **明渠水力学**: 告诉你水怎么流（水力计算）
- **生态水力学**: 告诉你水生生物需要什么样的水流（生境营造）
- **生态修复工程**: 将理论应用于河流、湖泊、湿地修复

### 案例1：河流生态基流计算 ⭐
**难度等级**: 入门  
**工程背景**: 水库下游生态保障流量确定

#### 问题描述
某水库拟建在山区河流上，需确定下游河道生态基流，保障鱼类生存繁殖。河流多年平均流量Q=15 m³/s，枯水期流量Q=3 m³/s。

#### 涉及理论
- **Tennant法**: 基于多年平均流量百分比
- **湿周法**: 基于河道湿周-流量关系
- **R2-Cross法**: 基于水深-流量关系
- **栖息地模拟法**: 基于生境适宜性曲线

#### 学习目标
1. 理解生态基流的概念和重要性
2. 掌握多种生态流量计算方法
3. 学会不同方法的适用条件
4. 理解流量-生境关系

#### 计算内容
- 4种方法计算生态基流
- 绘制流量-湿周曲线
- 分析不同方法的差异
- 提出推荐值及依据

#### 预期结果
- Tennant法: 3.0 m³/s (20%MAF)
- 湿周法: 2.8 m³/s
- R2-Cross法: 3.2 m³/s
- 推荐生态基流: 3.0 m³/s

### 案例2：鱼类栖息地适宜性评价 ⭐⭐
**难度等级**: 基础  
**工程背景**: 河流生境质量评估

#### 问题描述
评估某河段对目标鱼类（如鲤鱼）的栖息地适宜性。河段长度L=500m，流量Q=5-20 m³/s变化。

#### 涉及理论
- **栖息地适宜性指数（HSI）**: 水深、流速、底质综合评价
- **加权可利用面积（WUA）**: 适宜生境面积计算
- **物理栖息地模拟（PHABSIM）**: 一维水力-生境模型
- **适宜性曲线**: 不同生活史阶段的偏好曲线

#### 学习目标
1. 理解生境适宜性评价方法
2. 掌握HSI和WUA计算
3. 学会水力模型与生态模型耦合
4. 分析流量对生境质量的影响

#### 计算内容
- 不同流量下水深、流速分布计算
- 各网格栖息地适宜性指数计算
- WUA随流量变化曲线
- 最优生态流量推荐

#### 关键输出
- 流量-WUA关系曲线
- 栖息地分布图
- 最优流量区间
- 生境质量评价报告

## 💡 教学建议

### 对教师的建议

1. **案例教学法**
   - 从实际工程案例引入理论
   - 鼓励学生提出自己的解决方案
   - 组织案例讨论课

2. **实践教学**
   - 安排野外实习（河流调查）
   - 组织参观生态工程现场
   - 开展动手实验（水槽试验）

3. **跨学科融合**
   - 邀请生态学专家讲座
   - 与鱼类学、植物学课程联动
   - 引入最新研究进展

4. **软件教学**
   - HEC-RAS上机实习
   - Python编程训练
   - GIS制图练习

5. **批判性思维培养**
   - 讨论生态工程的不确定性
   - 分析失败案例
   - 辩论不同技术路线

### 对学生的建议

1. **基础扎实**
   - 复习明渠水力学知识
   - 补充生态学基础
   - 加强数学建模能力

2. **动手实践**
   - 认真完成每个案例作业
   - 学习HEC-RAS等软件
   - 掌握Python编程

3. **文献阅读**
   - 阅读经典论文
   - 跟踪最新研究
   - 学习批判性阅读

4. **野外观察**
   - 观察自然河流形态
   - 记录鱼类行为
   - 识别水生植物

5. **团队合作**
   - 课程设计分组协作
   - 讨论交流学习心得
   - 互相审阅作业
