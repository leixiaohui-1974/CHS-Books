# 案例19: 气候变化对水文过程影响评估

## 案例简介

本案例演示如何评估气候变化对流域水文过程的影响。基于IPCC第五次评估报告(AR5)的代表性浓度路径(RCP)情景，分析未来不同排放情景下气温和降水变化对径流、蒸散发、土壤含水量等水文要素的影响。通过情景对比分析，为流域水资源适应性管理提供科学依据。

## 理论背景

### RCP气候情景

IPCC AR5定义了四种RCP情景，代表不同的温室气体排放路径：

- **RCP2.6**：低排放情景，辐射强迫在2100年达到2.6 W/m²后下降
- **RCP4.5**：中低排放情景，辐射强迫稳定在4.5 W/m²
- **RCP6.0**：中高排放情景，辐射强迫稳定在6.0 W/m²
- **RCP8.5**：高排放情景，辐射强迫在2100年达到8.5 W/m²

### 气候变化水文响应

气候变化通过以下途径影响水文循环：

1. **气温升高**：
   - 增加蒸散发强度
   - 改变降雪/降雨比例
   - 提前融雪时间

2. **降水变化**：
   - 改变年降水总量
   - 极端降水事件频率增加
   - 降水季节分布变化

3. **综合效应**：
   ```
   ΔQ = f(ΔP, ΔT, ΔET, ΔSoil)
   ```
   其中Q为径流，P为降水，T为气温，ET为蒸散发，Soil为土壤含水量。

### Delta方法

采用Delta变化因子法调整气候要素：

```
P_future(t) = P_baseline(t) × (1 + ΔP%)
T_future(t) = T_baseline(t) + ΔT
```

## 代码说明

### 主要类

1. **ClimateChangeScenario**: 气候变化情景生成器
   - `__init__(scenario_name)`: 初始化RCP情景参数
   - `generate_climate_series(baseline_P, baseline_T, years)`: 生成未来气候序列
   - `apply_delta_method()`: 应用Delta变化因子

2. **HydrologicalModel**: 水文模型（简化版）
   - `__init__(params)`: 初始化模型参数
   - `run(P, T, PET)`: 运行水文模拟
   - `compute_water_balance()`: 计算水量平衡

3. **ImpactAnalyzer**: 影响分析器
   - `compare_scenarios()`: 对比不同情景的水文响应
   - `extreme_event_analysis()`: 极端事件频率分析
   - `seasonal_pattern_change()`: 季节模式变化分析

### 演示函数

- `demonstrate_climate_impact()`: 完整演示气候变化影响评估
  - 生成RCP2.6、RCP4.5、RCP8.5三种情景
  - 运行基准期(1986-2005)和未来期(2046-2065)模拟
  - 对比年均径流、极端洪水、枯水频率变化
  - 评估季节径流分配变化

## 预期结果说明

程序将生成以下分析结果：

### 1. 径流年际变化图
- 横轴：年份(基准期1986-2005 + 未来期2046-2065)
- 纵轴：年径流深(mm)
- 三条曲线分别代表RCP2.6、4.5、8.5情景
- 显示径流增加趋势和情景间差异

### 2. 水文要素相对变化率表
| 要素 | RCP2.6 | RCP4.5 | RCP8.5 |
|------|--------|--------|--------|
| 年径流 | +3% | +8% | +12% |
| 蒸散发 | +5% | +10% | +15% |
| 枯水流量 | -8% | -15% | -22% |
| 洪峰流量 | +15% | +25% | +35% |

### 3. 极端事件频率变化
- 洪水（Q>Q95%）频率增加20-40%
- 枯水（Q<Q5%）持续时间延长
- 年最大日径流增幅显著

### 4. 季节径流分配图
- 春季融雪径流峰值提前10-20天
- 夏季径流占比下降5-10%
- 秋季径流占比增加

典型结论：
- 高排放情景(RCP8.5)下影响最显著
- 年径流总量呈增加趋势（+3%至+12%）
- 极端洪水风险增加，枯水期更严重
- 径流季节分布更不均匀
