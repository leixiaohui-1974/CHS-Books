## 💡 学习技巧

### 调试技巧

```python
# 1. 使用print调试
print(f"变量值: {variable}")

# 2. 使用断言
assert water_level > 0, "水位必须为正"

# 3. 使用try-except
try:
    result = calculate()
except Exception as e:
    print(f"错误: {e}")

# 4. 分步骤调试
step1 = calculate_step1()
print(f"Step 1: {step1}")
step2 = calculate_step2(step1)
print(f"Step 2: {step2}")
```

### 代码优化

```python
# 1. 向量化计算（用NumPy代替循环）
# 慢
result = []
for x in data:
    result.append(x ** 2)

# 快
result = data ** 2

# 2. 列表推导式
result = [x**2 for x in data]

# 3. 使用内置函数
sum(data)  # 比自己写循环快
max(data)
min(data)
```

### 可视化技巧

```python
import matplotlib.pyplot as plt

# 1. 设置中文字体
plt.rcParams['font.sans-serif'] = ['SimHei']

# 2. 多子图
fig, (ax1, ax2) = plt.subplots(2, 1)

# 3. 保存高清图
plt.savefig('figure.png', dpi=300, bbox_inches='tight')
```

## 🎯 常见问题

### Q1: 零基础能学吗？

**A**: 可以！Part 1专门为零基础设计。

建议：
- 先学Part 1前5个项目
- 多练习，多运行代码
- 遇到问题查文档
- 2-3周可以入门

### Q2: 需要什么数学基础？

**A**: 不同Part要求不同

- Part 1: 中学数学即可
- Part 2: 需要微积分、线性代数
- Part 3-4: 需要水力学、水文学基础
- Part 5: 需要优化理论基础

### Q3: 如何选择学习内容？

**A**: 根据目标选择

- 考研/课程：Part 1-3必学
- 科研：Part 1-2 + Part 4-5
- 工程应用：Part 1 + Part 3 + Part 5

### Q4: 代码运行出错怎么办？

**A**: 系统化排查

1. 检查Python版本（建议3.8+）
2. 检查库是否安装
3. 检查代码是否完整复制
4. 查看错误信息
5. Google/百度搜索错误
6. 查阅官方文档

### Q5: 如何深入学习？

**A**: 多种途径

1. 阅读源码（NumPy/SciPy）
2. 参与开源项目
3. 实现论文算法
4. 编写自己的库
5. 发表技术博客
