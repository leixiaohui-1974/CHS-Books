## 📖 项目简介

《水利工程Python编程实战》是一本面向水利、土木、环境类专业学生和工程师的**完全代码驱动**的实战教材。

### 核心定位

- **100%代码实战**：70个完整项目，可直接运行
- **工程导向**：从实际工程问题出发
- **毕设利器**：10个完整毕设项目模板
- **职业技能**：Python+工程计算+可视化
- **高复用率**：基于215个已有代码，80%复用

## 🚀 快速开始

### 环境配置

```bash
# 克隆仓库
git clone [repository-url]
cd python-practice

# 安装依赖
pip install -r requirements.txt

# 或使用conda
conda env create -f environment.yml
conda activate hydraulics-python
```

### 运行示例

```bash
# 示例1：管流计算
cd codes/part3_hydraulics
python3 26_simple_pipe.py

# 示例2：明渠流计算
python3 31_uniform_flow.py

# 示例3：洪水预报
cd ../part4_hydrology
python3 56_muskingum.py
```

### Jupyter Notebook

```bash
# 启动Jupyter
jupyter notebook

# 打开笔记本

```
