## 🚀 快速开始

### 环境配置

```bash
# 1. 安装Python（建议3.8+）
# 从 python.org 下载安装

# 2. 安装依赖包
pip install numpy matplotlib pandas scipy sympy jupyter

# 3. 下载教材代码
git clone [repository_url]
cd python-practice

# 4. 运行第一个项目
python codes/part1_basics/project_01_data_types.py
```
