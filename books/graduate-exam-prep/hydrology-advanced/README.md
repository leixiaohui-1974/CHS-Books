## 📖 项目简介

《水文学考研高分突破》是继《水力学考研高分突破》之后的**第二本旗舰考研教材**。

### 核心定位

🎯 **水文学专业课**：全面覆盖水文学考研大纲  
📊 **330题完整体系**：15章×22题  
🐍 **Python可视化**：30个核心算法代码  
🏆 **985/211真题**：近10年名校真题改编

5. `mle_estimation.py` - 极大似然估计
6. `weight_function.py` - 权函数适线法
7. `frequency_curve.py` - 频率曲线绘制
8. `cv_cs_analysis.py` - Cv-Cs关系分析

14. `time_area_method.py` - 等流时线法
15. `xinanjiang_model.py` - 新安江模型
16. `scs_cn_model.py` - SCS-CN模型
17. `topmodel.py` - TOPMODEL
18. `runoff_routing.py` - 汇流计算

### 水文预报部分（6个）

22. `parameter_calibration.py` - 参数率定
23. `forecast_accuracy.py` - 预报精度

### 水资源计算部分（6个）

26. `annual_regulation.py` - 年调节
27. `multi_year_regulation.py` - 多年调节
