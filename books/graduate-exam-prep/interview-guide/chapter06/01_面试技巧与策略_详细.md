## 一、面试准备策略

### 1.1 简历优化

**核心原则**：
- 简洁明了（1-2页）
- 结果导向
- 量化成果
- 关键词匹配

**结构建议**：
```
个人信息
教育背景（时间倒序）
科研经历（重点突出）
项目经验（3-5个代表性项目）
技能清单（专业技能+工具）
获奖荣誉
发表论文/专利
```

**项目描述模板**：
```
项目名称 + 时间
背景：1-2句话说明项目意义
职责：你的具体工作（动词开头）
方法：使用的技术/工具
成果：量化结果（提升xx%，节省xx时间）
```

### 1.2 目标公司研究

**必做功课**：
- 公司业务与产品
- 最近新闻动态
- 企业文化价值观
- 岗位职责详解
- 技术栈要求

**信息来源**：
- 官网、公众号
- 招聘JD详细分析
- 知乎、脉脉、看准网
- 在职员工交流
- 技术博客/GitHub

## 二、自我介绍技巧

### 2.1 结构化表达

**经典三段式**（1-2分钟）：

**(1) 基本情况**（15秒）：
```
"您好，我是XXX，XX大学XX专业硕士研究生，
研究方向是XX，导师是XX教授。"
```

**(2) 核心优势**（45秒）：
```
"在研究生期间，我主要从事XX方向研究：
- 第一，完成了XX项目，使用XX方法，实现了XX效果
- 第二，发表了XX篇论文，其中SCI XX篇
- 第三，掌握了XX、XX等技术栈，有XX实践经验"
```

**(3) 求职动机**（30秒）：
```
"我对贵公司XX业务非常感兴趣，因为XX原因。
我相信我的XX能力能够为团队带来XX价值。
希望能有机会加入贵公司，实现XX目标。"
```

### 2.2 常见误区

**❌ 错误示范**：
- 背诵简历（重复信息）
- 泛泛而谈（无亮点）
- 过于谦虚（不自信）
- 冗长啰嗦（超时）

**✓ 正确示范**：
- 突出核心竞争力
- 量化成果数据
- 结构清晰逻辑
- 控制时间节奏

## 三、技术问题应答

### 3.1 STAR法则

**Situation**（情境）：
- 简要描述项目背景

**Task**（任务）：
- 你的具体职责是什么

**Action**（行动）：
- 你采取了哪些措施
- 为什么选择这种方法

**Result**（结果）：
- 取得了什么成果
- 量化数据支撑

**示例**：
```
问：介绍一下你的水力学计算项目？

答：(S) 在XX水库工程设计中，需要进行溢洪道水力计算。
    (T) 我负责建立数值模型，优化堰型参数。
    (A) 采用了FLOW-3D软件进行三维模拟，
        通过调整堰面曲线，对比了5种方案。
    (R) 最终推荐的WES堰型方案，流量系数提高8%，
        工程造价降低15%，成果应用于实际工程。
```

### 3.2 不知道时的应对

**诚实原则**：
```
"这个问题我目前了解不深，但我可以谈谈我的理解..."
"我在XX方面有一定基础，这个具体问题需要进一步学习。"
"如果我来解决这个问题，我会从XX角度入手..."
```

**展示学习能力**：
```
"虽然我没做过这个，但我做过类似的XX项目，
原理是相通的，我相信能快速上手。"
```

## 四、行为面试技巧

### 4.1 团队协作

**常见问题**：
- "描述一次团队合作经历"
- "如何处理团队冲突？"
- "你在团队中的角色定位？"

**回答要点**：
- 强调贡献而非抱怨
- 展现沟通能力
- 体现责任担当

**示例**：
```
问：如何处理与导师意见不一致的情况？

答：在XX项目中，我提出用XX方法，导师建议用XX。
    我首先认真思考导师建议的合理性，
    然后做了对比实验，用数据说话。
    实验表明我的方法在XX指标上更优，
    但导师方法在XX方面更稳定。
    最后我们采取了折中方案，结合两种方法优点。
    这次经历让我学会了尊重经验，同时坚持创新。
```

### 4.2 压力应对

**常见问题**：
- "遇到过最大的困难是什么？"
- "如何平衡科研与生活？"
- "deadline临近还没完成怎么办？"

**回答策略**：
- 正面积极
- 问题→行动→成长
- 展现抗压能力

## 五、薪资谈判策略

### 5.1 市场调研

**信息来源**：
- 校招薪资统计
- 行业报告（猎聘、BOSS）
- 师兄师姐经验
- Offer比较法

**薪资构成**：
```
总包 = 月薪×月数 + 奖金 + 股票/期权 + 福利
```

### 5.2 谈判技巧

**时机**：
- HR询问期望薪资时
- 收到口头offer后
- 正式offer前的最后协商

**话术**：
```
❌ "我要XX万"（生硬）
✓ "根据我的了解，这个岗位市场范围是XX-XX，
   结合我的XX背景和XX能力，
   我的期望是XX左右，不知是否在贵司范围内？"
```

**策略**：
- 先让对方报价
- 留有谈判空间（期望略高于心理价位）
- 强调价值而非索取
- 考虑整体package

## 六、常见陷阱问题

### 6.1 优缺点

**错误回答**：
```
❌ 优点：我很努力、责任心强（空泛）
❌ 缺点：我太追求完美（伪缺点）
```

**正确回答**：
```
✓ 优点：结合具体事例
   "我的优势是快速学习能力。在XX项目中，
    我用2周时间掌握了陌生的XX技术，
    并成功应用到实际问题中。"

✓ 缺点：真实且可改进
   "我在公众演讲方面还需提升，
    之前组会汇报会紧张。
    为此我主动申请了3次大会报告机会，
    现在已经有明显进步。"
```

### 6.2 职业规划

**5年规划模板**：
```
短期（1-2年）：
  技术上：深耕XX领域，成为技术专家
  项目上：独立负责完整项目

中期（3-5年）：
  横向：拓展XX相关技术栈
  纵向：带小团队，担任技术lead

长期：
  保持开放心态，根据公司发展调整
```

**避免**：
- 过于具体（"2年后升XX职位"）
- 过于模糊（"先干着看"）
- 与公司无关（"我想创业"）

```python
import matplotlib.pyplot as plt
import numpy as np
from datetime import datetime, timedelta

class InterviewPreparation:
    """面试准备工具"""
    
    def __init__(self):
        self.companies = []
        self.timeline = []
    
    def salary_comparison(self, offers):
        """
        Offer对比分析
        
        参数:
            offers: 字典列表，每个包含公司名、月薪、月数、奖金等
        """
        fig, axes = plt.subplots(2, 2, figsize=(16, 12))
        
        # 1. 总包对比
        ax1 = axes[0, 0]
        
        companies = [o['company'] for o in offers]
        total_comp = []
        
        for o in offers:
            total = o['monthly_salary'] * o['months'] + o['bonus']
            total_comp.append(total)
        
        colors_bar = plt.cm.Set3(np.linspace(0, 1, len(companies)))
        bars = ax1.bar(companies, total_comp, color=colors_bar,
                      edgecolor='black', linewidth=2)
        
        # 标注数值
        for bar, total in zip(bars, total_comp):
            height = bar.get_height()
            ax1.text(bar.get_x() + bar.get_width()/2, height * 1.02,
                    f'{total:.0f}k',
                    ha='center', va='bottom', fontsize=11, fontweight='bold')
        
        ax1.set_ylabel('年总包 (k)', fontsize=11)
        ax1.set_title('Offer总包对比', fontsize=13, fontweight='bold')
        ax1.grid(True, alpha=0.3, axis='y')
        
        # 2. 薪资结构
        ax2 = axes[0, 1]
        
        base_salary = [o['monthly_salary'] * o['months'] for o in offers]
        bonuses = [o['bonus'] for o in offers]
        
        x_pos = np.arange(len(companies))
        width = 0.35
        
        bars1 = ax2.bar(x_pos - width/2, base_salary, width,
                       label='基本工资', color='skyblue',
                       edgecolor='black', linewidth=1.5)
        bars2 = ax2.bar(x_pos + width/2, bonuses, width,
                       label='奖金', color='lightcoral',
                       edgecolor='black', linewidth=1.5)
        
        ax2.set_xticks(x_pos)
        ax2.set_xticklabels(companies)
        ax2.set_ylabel('金额 (k)', fontsize=11)
        ax2.set_title('薪资结构对比', fontsize=13, fontweight='bold')
        ax2.legend()
        ax2.grid(True, alpha=0.3, axis='y')
        
        # 3. 综合评分（雷达图）
        ax3 = axes[1, 0]
        
        categories = ['薪资', '发展', '平台', '团队', '地点']
        N = len(categories)
        
        angles = np.linspace(0, 2 * np.pi, N, endpoint=False).tolist()
        angles += angles[:1]
        
        ax3 = plt.subplot(2, 2, 3, projection='polar')
        
        colors_radar = plt.cm.tab10(np.linspace(0, 1, len(offers)))
        
        for offer, color in zip(offers, colors_radar):
            scores = offer.get('scores', [8, 7, 9, 8, 7])
            scores += scores[:1]
            
            ax3.plot(angles, scores, 'o-', linewidth=2, color=color,
                    label=offer['company'])
            ax3.fill(angles, scores, alpha=0.15, color=color)
        
        ax3.set_xticks(angles[:-1])
        ax3.set_xticklabels(categories)
        ax3.set_ylim(0, 10)
        ax3.set_title('综合评分对比', fontsize=13, fontweight='bold',
                     pad=20)
        ax3.legend(loc='upper right', bbox_to_anchor=(1.3, 1.0))
        ax3.grid(True)
        
        # 4. 决策矩阵
        ax4 = axes[1, 1]
        
        # 计算综合得分
        weights = {'salary': 0.4, 'development': 0.25, 
                  'platform': 0.2, 'team': 0.1, 'location': 0.05}
        
        final_scores = []
        for offer in offers:
            scores = offer.get('scores', [8, 7, 9, 8, 7])
            score_dict = dict(zip(['salary', 'development', 'platform',
                                  'team', 'location'], scores))
            
            final_score = sum(score_dict[k] * weights[k] 
                            for k in weights.keys())
            final_scores.append(final_score)
        
        # 排序
        sorted_indices = np.argsort(final_scores)[::-1]
        
        y_pos = np.arange(len(companies))
        bars = ax4.barh(y_pos, [final_scores[i] for i in sorted_indices],
                       color=colors_bar, edgecolor='black', linewidth=2)
        
        ax4.set_yticks(y_pos)
        ax4.set_yticklabels([companies[i] for i in sorted_indices])
        ax4.set_xlabel('综合得分', fontsize=11)
        ax4.set_title('决策推荐排序', fontsize=13, fontweight='bold')
        ax4.grid(True, alpha=0.3, axis='x')
        
        # 标注推荐
        max_idx = sorted_indices[0]
        ax4.text(final_scores[max_idx] * 0.5, 0,
                '★ 推荐', fontsize=14, color='red',
                weight='bold', ha='center')
        
        plt.tight_layout()
        plt.savefig('offer_comparison.png', dpi=300)
        plt.show()
    
    def interview_timeline(self, events):
        """
        面试时间线可视化
        
        参数:
            events: 列表，每项包含日期、公司、轮次
        """
        fig, ax = plt.subplots(figsize=(14, 8))
        
        # 按日期排序
        events_sorted = sorted(events, key=lambda x: x['date'])
        
        dates = [datetime.strptime(e['date'], '%Y-%m-%d') 
                for e in events_sorted]
        companies = [e['company'] for e in events_sorted]
        rounds = [e['round'] for e in events_sorted]
        
        # 绘制时间线
        ax.plot(dates, range(len(dates)), 'k-', linewidth=2, alpha=0.3)
        
        # 标注事件
        colors_event = {'简历投递': 'blue', '笔试': 'green',
                       '一面': 'orange', '二面': 'red', 'HR面': 'purple',
                       'Offer': 'gold'}
        
        for i, (date, company, round_type) in enumerate(zip(dates, companies, rounds)):
            color = colors_event.get(round_type, 'gray')
            
            ax.scatter(date, i, s=200, c=color, marker='o',
                      edgecolors='black', linewidth=2, zorder=5)
            
            ax.text(date, i + 0.3, f'{company}\n{round_type}',
                   ha='center', fontsize=9,
                   bbox=dict(boxstyle='round', facecolor=color, alpha=0.3))
        
        ax.set_xlabel('日期', fontsize=11)
        ax.set_ylabel('进度', fontsize=11)
        ax.set_title('面试时间线', fontsize=13, fontweight='bold')
        ax.grid(True, alpha=0.3, axis='x')
        ax.set_yticks([])
        
        plt.tight_layout()
        plt.savefig('interview_timeline.png', dpi=300)
        plt.show()
```

**角色选择**：
- Leader（组织者）：适合有经验、表达强

**注意事项**：
- 不抢话、不打断
