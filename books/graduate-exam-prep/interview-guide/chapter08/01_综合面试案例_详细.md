- 拱坝：河谷稍宽，不是最优
- 支墩坝：受力复杂，地震区不利

**题2：堰流计算**
```
宽顶堰，b=30m，上游水位H=3m，
堰顶高程与河床齐平，求泄流量。

解：
1) 判别：宽顶堰自由溢流
   
2) 流量公式
   Q = m·b·√(2g)·H^(3/2)
   
   其中m=0.385（查表或0.32~0.48）

3) 计算
   Q = 0.385×30×√(2×9.81)×3^1.5
     = 0.385×30×4.43×5.20
     = 267 m³/s

4) 校核
   单宽流量q = Q/b = 8.9 m²/s
   临界水深hc = (q²/g)^(1/3) = 2.0 m
   H = 3 > hc = 2，满足溢流条件✓
```

**Q1：LeetCode风格题目**
```
题目：给定一组水位数据，找出最长的连续上涨区间。

输入：h = [1.2, 1.5, 1.8, 1.6, 1.9, 2.1, 2.3, 2.0]
输出：[4, 5, 6]（索引，对应1.9, 2.1, 2.3）

代码（10分钟完成）：
```python
def longest_rising_interval(h):
    """找出最长连续上涨区间"""
    if not h or len(h) < 2:
        return []
    
    max_start = 0
    max_length = 1
    current_start = 0
    current_length = 1
    
    for i in range(1, len(h)):
        if h[i] > h[i-1]:
            current_length += 1
            if current_length > max_length:
                max_length = current_length
                max_start = current_start
        else:
            current_start = i
            current_length = 1
    
    return list(range(max_start, max_start + max_length))

# 测试
h = [1.2, 1.5, 1.8, 1.6, 1.9, 2.1, 2.3, 2.0]
print(longest_rising_interval(h))  # [4, 5, 6]
```

**追问：如果允许有一次下降呢？**
```python
def longest_rising_with_one_drop(h):
    """允许一次下降的最长上涨区间"""
    if len(h) < 2:
        return []
    
    # 动态规划
    # dp[i][0]: 到i为止，未使用容错的最长长度
    # dp[i][1]: 到i为止，使用了一次容错的最长长度
    
    n = len(h)
    dp = [[1, 1] for _ in range(n)]
    
    max_length = 1
    max_end = 0
    
    for i in range(1, n):
        if h[i] > h[i-1]:
            # 上涨
            dp[i][0] = dp[i-1][0] + 1
            dp[i][1] = dp[i-1][1] + 1
        else:
            # 下降，使用容错
            dp[i][0] = 1
            dp[i][1] = dp[i-1][0] + 1
        
        if max(dp[i]) > max_length:
            max_length = max(dp[i])
            max_end = i
    
    start = max_end - max_length + 1
    return list(range(start, max_end + 1))
```
**复杂度分析**：
- 时间：O(n)
- 空间：O(n)，可优化到O(1)

**Q2：Explain shallow water equations**
```
"Shallow water equations are depth-averaged form 
of Navier-Stokes equations, applicable when 
horizontal scale >> vertical scale.

Governing equations:
∂h/∂t + ∂(uh)/∂x + ∂(vh)/∂y = 0  (continuity)
∂(uh)/∂t + ∂(u²h)/∂x + ∂(uvh)/∂y = -gh∂z/∂x - τx  (momentum-x)
∂(vh)/∂t + ∂(uvh)/∂x + ∂(v²h)/∂y = -gh∂z/∂y - τy  (momentum-y)

Where:
h: water depth
u, v: depth-averaged velocities
z: bed elevation
τ: friction terms

Key assumptions:
- Hydrostatic pressure
- Uniform velocity profile
- Negligible vertical acceleration

Applications:
- River flooding
- Dam break
- Tsunami propagation
- Coastal inundation
"
```

**Q3：Code Review (给一段代码，找bug)**
```python
# Bug版本
def manning_flow_depth(Q, b, i, n):
    """计算矩形渠道正常水深"""
    h = 1.0
    for iter in range(100):
        A = b * h
        P = b + 2 * h
        R = A / P
        Q_calc = (1/n) * A * R**(2/3) * i**0.5
        error = Q_calc - Q
        if abs(error) < 0.001:
            return h
        h = h - error / (Q_calc / h)  # BUG!

# 问题：
# 1. 牛顿法求导不准确
# 2. 可能震荡不收敛
# 3. 没有处理h<0的情况
# 4. 迭代步长固定可能过大

# 正确版本
def manning_flow_depth_fixed(Q, b, i, n):
    """计算矩形渠道正常水深（改进版）"""
    from scipy.optimize import fsolve
    
    def equation(h):
        if h <= 0:
            return 1e10
        A = b * h
        P = b + 2 * h
        R = A / P
        Q_calc = (1/n) * A * R**(2/3) * i**0.5
        return Q_calc - Q
    
    # 初值估计
    h0 = (Q * n / (b * i**0.5))**(3/5)
    
    # 求解
    h = fsolve(equation, h0)[0]
    
    return h if h > 0 else None
