### Q1：什么是雷诺数？其物理意义是什么？

**【标准答案】**

**定义**：
$$Re = \frac{vD}{\nu} = \frac{\rho vD}{\mu}$$

**物理意义**：
- 惯性力与粘滞力的比值
- 判别层流与湍流的无量纲数

**临界值**：
- 管流：Re < 2000（层流），Re > 4000（湍流）
- 明渠：Re > 500（湍流）

**回答技巧**：
1. 先说定义和公式（30秒）
2. 解释物理意义（30秒）
3. 举临界值实例（20秒）

### Q2：伯努利方程及其应用

**【标准答案】**

**方程**：
$$\frac{p}{\gamma} + \frac{v^2}{2g} + z = C$$

**三项含义**：
- 压强水头：$p/\gamma$
- 流速水头：$v^2/(2g)$
- 位置水头：$z$

**应用**：
1. 孔口出流计算
2. 文丘里流量计
3. 管道水击分析
4. 水工建筑物水力设计

**常见追问**：
- "实际流动如何修正？" → 加入水头损失项
- "适用条件？" → 理想流体、定常流、同一流线

### Q3：临界水深的概念及计算

**【标准答案】**

**定义**：特定流量下，比能量最小时的水深

**判别**：
$$Fr = \frac{v}{\sqrt{gh}} = 1$$

**计算**（矩形断面）：
$$h_c = \sqrt[3]{\frac{q^2}{g}}$$

**工程意义**：
- 急流与缓流分界
- 测流断面设计
- 驼峰水位计设计

**展示深度**：画出比能曲线图

---

### Q4：水跃现象及其应用

**【标准答案】**

**定义**：急流突然过渡到缓流，水面急剧壅高

**共轭水深**：
$$\frac{h_2}{h_1} = \frac{-1 + \sqrt{1 + 8Fr_1^2}}{2}$$

**能量损失**：
$$\Delta E = \frac{(h_2 - h_1)^3}{4h_1 h_2}$$

**工程应用**：
- 消能工设计
- 防止冲刷
- 稳定水流

---

### Q5：达西定律及地下水应用

**【标准答案】**

$$Q = KiA$$

- $K$：渗透系数
- $i$：水力梯度
- $A$：过水面积

**应用**：
1. 坝基渗流
2. 地下水井流
3. 排水设计
4. 渗流稳定分析

## 二、专业计算篇（21-40题）

### Q21：快速计算矩形明渠正常水深

**【题目】**：Q=10m³/s，b=5m，i=0.001，n=0.025，求h₀

**【快速解法】**：

**步骤**：
1. 谢才公式：$Q = AC\sqrt{Ri}$
2. 矩形：$A = bh$，$R = \frac{bh}{b+2h}$
3. 曼宁公式：$C = \frac{1}{n}R^{1/6}$

**数值求解**（Python）：

```python
from scipy.optimize import fsolve

def normal_depth(h, Q, b, i, n):
    """正常水深方程"""
    A = b * h
    R = A / (b + 2*h)
    C = (1/n) * R**(1/6)
    Q_calc = A * C * (R * i)**0.5
    return Q_calc - Q

# 求解
Q, b, i, n = 10, 5, 0.001, 0.025
h0 = fsolve(normal_depth, 1.0, args=(Q, b, i, n))[0]

print(f"正常水深: {h0:.3f} m")
```

**答案**：h₀ ≈ 1.2m

**评分技巧**：
- 写对公式：5分
- 正确求解：3分
- 验证合理性：2分
