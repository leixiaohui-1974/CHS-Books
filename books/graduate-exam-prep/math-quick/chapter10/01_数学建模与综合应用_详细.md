### æ¡ˆä¾‹1ï¼šLogisticäººå£æ¨¡å‹

**èƒŒæ™¯**ï¼šè€ƒè™‘èµ„æºæœ‰é™çš„äººå£å¢é•¿

**å‡è®¾**ï¼š
- å¢é•¿ç‡ä¸ç°æœ‰äººå£æˆæ­£æ¯”
- å­˜åœ¨ç¯å¢ƒå®¹é‡$K$

**æ¨¡å‹**ï¼š
$$\frac{dN}{dt} = rN\left(1 - \frac{N}{K}\right)$$

å…¶ä¸­ï¼š
- $N(t)$ï¼šæ—¶åˆ»$t$çš„äººå£
- $r$ï¼šå›ºæœ‰å¢é•¿ç‡
- $K$ï¼šç¯å¢ƒå®¹é‡

**æ±‚è§£**ï¼š

åˆ†ç¦»å˜é‡ï¼š
$$\frac{dN}{N(K-N)} = \frac{r}{K}dt$$

éƒ¨åˆ†åˆ†å¼ï¼š
$$\frac{1}{N} + \frac{1}{K-N} = \frac{r}{K}dt$$

ç§¯åˆ†ï¼š
$$\ln N - \ln(K-N) = rt + C$$

$$\frac{N}{K-N} = Ce^{rt}$$

è§£å¾—ï¼š
$$N(t) = \frac{K}{1 + Ae^{-rt}}$$

å…¶ä¸­$A$ç”±åˆå€¼$N(0)$ç¡®å®š

**ç‰¹æ€§åˆ†æ**ï¼š
- $t \to \infty$æ—¶ï¼Œ$N \to K$
- æ‹ç‚¹ï¼š$N = K/2$æ—¶å¢é•¿æœ€å¿«

### æ¡ˆä¾‹2ï¼šSIRä¼ æŸ“ç—…æ¨¡å‹

**èƒŒæ™¯**ï¼šä¼ æŸ“ç—…ä¼ æ’­åŠ¨åŠ›å­¦

**åˆ†ç±»**ï¼š
- Sï¼ˆSusceptibleï¼‰ï¼šæ˜“æ„Ÿè€…
- Iï¼ˆInfectedï¼‰ï¼šæ„ŸæŸ“è€…
- Rï¼ˆRecoveredï¼‰ï¼šåº·å¤è€…/å…ç–«è€…

**æ¨¡å‹**ï¼š
$$\begin{cases}
\frac{dS}{dt} = -\beta SI \\
\frac{dI}{dt} = \beta SI - \gamma I \\
\frac{dR}{dt} = \gamma I
\end{cases}$$

å…¶ä¸­ï¼š
- $\beta$ï¼šæ¥è§¦ç‡Ã—ä¼ æŸ“æ¦‚ç‡
- $\gamma$ï¼šåº·å¤ç‡ï¼ˆ$1/\gamma$ä¸ºå¹³å‡æ„ŸæŸ“æœŸï¼‰
- $R_0 = \beta S_0 / \gamma$ï¼šåŸºæœ¬å†ç”Ÿæ•°

**ä¸´ç•Œæ¡ä»¶**ï¼š
- $R_0 > 1$ï¼šç–¾ç—…çˆ†å‘
- $R_0 < 1$ï¼šç–¾ç—…æ¶ˆé€€

class EpidemicModel:
    """ä¼ æŸ“ç—…æ¨¡å‹"""
    
    @staticmethod
    def SIR_model(S0, I0, R0, beta, gamma, t_max=200):
        """
        SIRä¼ æŸ“ç—…æ¨¡å‹
        
        å‚æ•°:
            S0, I0, R0: åˆå§‹æ˜“æ„Ÿã€æ„ŸæŸ“ã€åº·å¤äººæ•°
            beta: æ¥è§¦ç‡Ã—ä¼ æŸ“æ¦‚ç‡
            gamma: åº·å¤ç‡
            t_max: æ—¶é—´èŒƒå›´
        """
        def sir_derivatives(y, t):
            S, I, R = y
            dS = -beta * S * I
            dI = beta * S * I - gamma * I
            dR = gamma * I
            return [dS, dI, dR]
        
        y0 = [S0, I0, R0]
        t = np.linspace(0, t_max, 1000)
        
        solution = odeint(sir_derivatives, y0, t)
        
        return t, solution[:, 0], solution[:, 1], solution[:, 2]
    
    @staticmethod
    def plot_SIR_model():
        """ç»˜åˆ¶SIRæ¨¡å‹"""
        fig, axes = plt.subplots(2, 2, figsize=(14, 10))
        
        N = 1000  # æ€»äººå£
        I0 = 10   # åˆå§‹æ„ŸæŸ“è€…
        R0_init = 0  # åˆå§‹åº·å¤è€…
        S0 = N - I0 - R0_init
        
        beta = 0.4 / N  # æ¥è§¦ç‡
        gamma = 0.1     # åº·å¤ç‡
        
        # åŸºæœ¬å†ç”Ÿæ•°
        R0_number = beta * S0 / gamma
        
        print(f"åŸºæœ¬å†ç”Ÿæ•° Râ‚€ = {R0_number:.2f}")
        
        # æ—¶é—´æ¼”åŒ–
        ax1 = axes[0, 0]
        
        t, S, I, R = EpidemicModel.SIR_model(S0, I0, R0_init, beta, gamma)
        
        ax1.plot(t, S, 'b-', linewidth=2.5, label='æ˜“æ„Ÿè€… S')
        ax1.plot(t, I, 'r-', linewidth=2.5, label='æ„ŸæŸ“è€… I')
        ax1.plot(t, R, 'g-', linewidth=2.5, label='åº·å¤è€… R')
        ax1.set_xlabel('æ—¶é—´ t (å¤©)', fontsize=11)
        ax1.set_ylabel('äººæ•°', fontsize=11)
        ax1.set_title(f'SIRæ¨¡å‹æ¼”åŒ– (Râ‚€={R0_number:.2f})',
                     fontsize=12, fontweight='bold')

        ax1.grid(True, alpha=0.3)
        
        # ä¸åŒRâ‚€çš„å½±å“
        ax2 = axes[0, 1]
        
        for gamma_var in [0.05, 0.1, 0.2]:
            R0_var = beta * S0 / gamma_var
            t, S, I, R = EpidemicModel.SIR_model(S0, I0, R0_init, beta, gamma_var)
            ax2.plot(t, I, linewidth=2, label=f'Râ‚€={R0_var:.2f}')
        
        ax2.set_xlabel('æ—¶é—´ t (å¤©)', fontsize=11)
        ax2.set_ylabel('æ„ŸæŸ“è€…äººæ•° I', fontsize=11)
        ax2.set_title('åŸºæœ¬å†ç”Ÿæ•°Râ‚€çš„å½±å“', fontsize=12, fontweight='bold')
        ax2.legend()
        ax2.grid(True, alpha=0.3)
        
        # ç›¸å›¾ (S vs I)
        ax3 = axes[1, 0]
        
        t, S, I, R = EpidemicModel.SIR_model(S0, I0, R0_init, beta, gamma)
        
        ax3.plot(S, I, 'b-', linewidth=2.5)
        ax3.plot(S[0], I[0], 'go', markersize=12, label='åˆå§‹çŠ¶æ€')
        ax3.plot(S[-1], I[-1], 'rs', markersize=12, label='æœ€ç»ˆçŠ¶æ€')
        
        # é˜ˆå€¼çº¿
        threshold = gamma / beta
        ax3.axvline(threshold, color='r', linestyle='--', linewidth=1.5,
                   label=f'é˜ˆå€¼ S={threshold:.0f}')
        
        ax3.set_xlabel('æ˜“æ„Ÿè€… S', fontsize=11)
        ax3.set_ylabel('æ„ŸæŸ“è€… I', fontsize=11)
        ax3.set_title('ç›¸å›¾', fontsize=12, fontweight='bold')
        ax3.legend()
        ax3.grid(True, alpha=0.3)
        
        # ç–«æƒ…å³°å€¼åˆ†æ
        ax4 = axes[1, 1]

        I_max_list = []
        
        for gamma_val in gamma_range:
            t, S, I, R = EpidemicModel.SIR_model(S0, I0, R0_init, beta, gamma_val)
            I_max_list.append(I.max())
        
        R0_range = beta * S0 / gamma_range
        
        ax4.plot(R0_range, I_max_list, 'b-', linewidth=2.5)
        ax4.axvline(1, color='r', linestyle='--', linewidth=1.5,
                   label='Râ‚€=1 (ä¸´ç•Œ)')
        ax4.set_xlabel('åŸºæœ¬å†ç”Ÿæ•° Râ‚€', fontsize=11)
        ax4.set_ylabel('å³°å€¼æ„ŸæŸ“äººæ•°', fontsize=11)
        ax4.set_title('Râ‚€ä¸ç–«æƒ…å³°å€¼å…³ç³»', fontsize=12, fontweight='bold')

        ax4.grid(True, alpha=0.3)
