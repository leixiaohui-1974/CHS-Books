## 高等数学（60个）

### 极限与连续（10个）

**1. 洛必达法则** 🔥
```
lim f(x)/g(x) = lim f'(x)/g'(x)  (0/0或∞/∞型)
```

**2-7. 等价无穷小（x→0）** 🔥
```
sin x ~ x
tan x ~ x
arcsin x ~ x
arctan x ~ x
ln(1+x) ~ x
e^x - 1 ~ x
1 - cos x ~ x²/2
(1+x)^α - 1 ~ αx
```

**8. 第一重要极限**
```
lim_{x→0} sin x / x = 1
```

**9. 第二重要极限** 🔥
```
lim_{x→∞} (1 + 1/x)^x = e
lim_{x→0} (1 + x)^{1/x} = e
```

**10. 连续性**
```
lim_{x→x₀} f(x) = f(x₀)
```

### 导数与微分（12个）

**11-17. 基本导数公式**
```
(x^n)' = nx^{n-1}
(sin x)' = cos x
(cos x)' = -sin x
(e^x)' = e^x
(a^x)' = a^x ln a
(ln x)' = 1/x
(arctan x)' = 1/(1+x²)
```

**18. 复合函数求导（链式法则）** 🔥
```
[f(g(x))]' = f'(g(x)) · g'(x)
```

**19. 乘积求导**
```
(uv)' = u'v + uv'
```

**20. 商的求导**
```
(u/v)' = (u'v - uv') / v²
```

**21. 参数方程求导**
```
dy/dx = (dy/dt) / (dx/dt)
```

**22. 隐函数求导**
```
对F(x,y)=0两边对x求导
```

### 导数应用（8个）

**23. 单调性**
```
f'(x) > 0  ⇒  f(x)↑
f'(x) < 0  ⇒  f(x)↓
```

**24-25. 极值判定** 🔥
```
f'(x₀) = 0 且 f''(x₀) > 0  ⇒  极小值
f'(x₀) = 0 且 f''(x₀) < 0  ⇒  极大值
```

**26-27. 凹凸性**
```
f''(x) > 0  ⇒  凹（开口向上）
f''(x) < 0  ⇒  凸（开口向下）
```

**28. 拐点**
```
f''(x₀) = 0 且 f''(x)变号  ⇒  拐点
```

**29. 拉格朗日中值定理** 🔥
```
f(b) - f(a) = f'(ξ)(b - a)
```

**30. 泰勒公式**
```
f(x) = f(x₀) + f'(x₀)(x-x₀) + f''(x₀)(x-x₀)²/2! + ...
```

### 积分（12个）

**31-37. 基本积分公式**
```
∫ x^n dx = x^{n+1}/(n+1) + C  (n≠-1)
∫ 1/x dx = ln|x| + C
∫ e^x dx = e^x + C
∫ sin x dx = -cos x + C
∫ cos x dx = sin x + C
∫ 1/(1+x²) dx = arctan x + C
∫ 1/√(1-x²) dx = arcsin x + C
```

**38. 分部积分法** 🔥
```
∫ u dv = uv - ∫ v du
```
口诀："反对幂指三"

**39. 换元积分法**
```
∫ f(φ(x))φ'(x) dx = ∫ f(u) du
```

**40. 牛顿-莱布尼茨公式** 🔥
```
∫[a,b] f(x) dx = F(b) - F(a)
```

**41. 变限积分求导** 🔥
```
d/dx [∫[a,x] f(t) dt] = f(x)
d/dx [∫[a,φ(x)] f(t) dt] = f(φ(x))φ'(x)
```

**42. 分段积分**
```
∫[a,b] f(x) dx = ∫[a,c] f(x) dx + ∫[c,b] f(x) dx
```

### 多元函数（8个）

**48. 偏导数**
```
∂f/∂x = lim[Δx→0] [f(x+Δx,y) - f(x,y)] / Δx
```

**49. 全微分**
```
dz = (∂z/∂x)dx + (∂z/∂y)dy
```

**50. 复合函数偏导** 🔥
```
∂z/∂x = (∂z/∂u)(∂u/∂x) + (∂z/∂v)(∂v/∂x)
```

**51. 方向导数**
```
∂f/∂l = (∂f/∂x)cosα + (∂f/∂y)cosβ
```

**52. 梯度** 🔥
```
grad f = (∂f/∂x, ∂f/∂y)
```

**53. 多元函数极值**
```
∂f/∂x = 0,  ∂f/∂y = 0
判别式 D = f_{xx}f_{yy} - f_{xy}²
```

**54. 二重积分（直角坐标）**
```
∫∫_D f(x,y) dσ = ∫[a,b] dx ∫[y₁,y₂] f(x,y) dy
```

**55. 二重积分（极坐标）** 🔥
```
∫∫_D f(x,y) dσ = ∫[α,β] dθ ∫[r₁,r₂] f(r cosθ, r sinθ) r dr
```

### 微分方程（5个）

**56. 可分离变量**
```
dy/dx = f(x)g(y)  ⇒  dy/g(y) = f(x) dx
```

**57. 一阶线性方程** 🔥
```
dy/dx + P(x)y = Q(x)
y = e^{-∫P dx}[∫Q e^{∫P dx} dx + C]
```

**58. 二阶常系数齐次** 🔥
```
y'' + py' + qy = 0
特征方程：r² + pr + q = 0
```

**59-60. 特征根情形**
```
两不等实根：y = C₁e^{r₁x} + C₂e^{r₂x}
两相等实根：y = (C₁ + C₂x)e^{rx}

```

### 分布（7个）

**86. 二项分布** 🔥
```
X ~ B(n,p)
P(X=k) = C_n^k p^k (1-p)^{n-k}
E(X) = np,  Var(X) = np(1-p)
```

**87. 泊松分布**
```
X ~ P(λ)
P(X=k) = λ^k e^{-λ} / k!
E(X) = Var(X) = λ
```

**88. 均匀分布**
```
X ~ U(a,b)
f(x) = 1/(b-a),  a<x<b
E(X) = (a+b)/2,  Var(X) = (b-a)²/12
```

**89. 指数分布**
```
X ~ Exp(λ)
f(x) = λe^{-λx},  x>0
E(X) = 1/λ,  Var(X) = 1/λ²
```

**90. 正态分布** 🔥
```
X ~ N(μ, σ²)
f(x) = 1/(σ√(2π)) exp[-(x-μ)²/(2σ²)]
E(X) = μ,  Var(X) = σ²
```

**91. 标准正态分布**
```
X ~ N(0,1)
Φ(x) = P(X ≤ x)
Φ(-x) = 1 - Φ(x)
```

**92. 3σ原则**
```
P(|X-μ| < σ) ≈ 0.6827
P(|X-μ| < 2σ) ≈ 0.9545
P(|X-μ| < 3σ) ≈ 0.9973
```

### 数字特征（5个）

**93. 期望** 🔥
```
离散：E(X) = Σ x_i p_i
连续：E(X) = ∫ x f(x) dx
```

**94. 方差**
```
Var(X) = E(X²) - [E(X)]²
```

**95. 期望性质**
```
E(aX + b) = aE(X) + b
E(X + Y) = E(X) + E(Y)
若X,Y独立：E(XY) = E(X)E(Y)
```

**96. 方差性质**
```
Var(aX + b) = a²Var(X)
若X,Y独立：Var(X ± Y) = Var(X) + Var(Y)
```

**97. 协方差与相关系数** 🔥
```
Cov(X,Y) = E(XY) - E(X)E(Y)
ρ_{XY} = Cov(X,Y) / (σ_X σ_Y)
```

### 统计（3个）

**98. 样本均值与方差**
```
x̄ = (1/n)Σ x_i
s² = 1/(n-1) Σ(x_i - x̄)²
```

**99. 置信区间（μ，σ已知）** 🔥
```
(x̄ - z_{α/2} σ/√n,  x̄ + z_{α/2} σ/√n)
```

**100. 假设检验统计量**
```
u检验：u = (x̄ - μ₀)/(σ/√n) ~ N(0,1)
t检验：t = (x̄ - μ₀)/(s/√n) ~ t(n-1)
χ²检验：χ² = (n-1)s²/σ₀² ~ χ²(n-1)
```
