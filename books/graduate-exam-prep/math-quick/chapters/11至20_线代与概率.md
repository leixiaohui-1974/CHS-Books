### 专题11：行列式 ⭐⭐⭐⭐

**核心公式**：
```
二阶行列式：
|a b|
|c d| = ad - bc

三阶行列式（对角线法则）：
|a11 a12 a13|
|a21 a22 a23| = a11a22a33 + a12a23a31 + a13a21a32
|a31 a32 a33|   - a13a22a31 - a11a23a32 - a12a21a33
```

**行列式性质** 🔥：
1. 转置行列式值不变：\(|A^T| = |A|\)
2. 互换两行，行列式变号
3. 某行乘\(k\)，行列式乘\(k\)
4. 某行加上另一行的\(k\)倍，行列式不变
5. 两行成比例，行列式为0

**克拉默法则**：
若\(|A| \neq 0\)，则方程组\(Ax = b\)有唯一解：
```
x_i = |A_i| / |A|
```
其中\(A_i\)是将\(A\)的第\(i\)列换成\(b\)得到的矩阵

### 工程应用：管网方程组
管网流量方程用克拉默法则求解。

### 专题12：矩阵 ⭐⭐⭐⭐⭐

**矩阵运算**：
```
加法：(A + B)_ij = a_ij + b_ij
数乘：(kA)_ij = k·a_ij
乘法：(AB)_ij = Σ a_ik·b_kj
```

**矩阵性质**：
```
(AB)C = A(BC)
A(B + C) = AB + AC
(AB)^T = B^T A^T
(AB)^(-1) = B^(-1) A^(-1)
|AB| = |A|·|B|
```

**逆矩阵** 🔥：
若\(|A| \neq 0\)，则\(A^{-1}\)存在：
```
A^(-1) = A*/|A|
```
其中\(A*\)为伴随矩阵

**矩阵的秩**：
\(r(A)\)为矩阵的秩（最高阶非零子式的阶数）

### Python代码：
```python
import numpy as np

A = np.array([[1, 2], [3, 4]])
A_inv = np.linalg.inv(A)  # 逆矩阵
A_rank = np.linalg.matrix_rank(A)  # 秩
```

### 专题13：线性方程组 ⭐⭐⭐⭐⭐

**高斯消元法** 🔥：
将增广矩阵化为行阶梯形，求解。

**解的判定**：
对于\(Ax = b\)：
```
r(A) = r(A|b) = n  =>  唯一解
r(A) = r(A|b) < n  =>  无穷多解
r(A) < r(A|b)      =>  无解
```

**通解结构**：
齐次方程组\(Ax = 0\)：
- 若\(r(A) = n\)，只有零解
- 若\(r(A) < n\)，有\(n - r(A)\)个线性无关的解

非齐次方程组\(Ax = b\)：
```
通解 = 特解 + 齐次通解
```

### Python代码：
```python
# 求解线性方程组
x = np.linalg.solve(A, b)
```

### 专题14：特征值与特征向量 ⭐⭐⭐⭐⭐

**定义**：
若\(Ax = \lambda x\)（\(x \neq 0\)），则\(\lambda\)为特征值，\(x\)为特征向量

**特征方程** 🔥：
```
|λE - A| = 0
```

**性质**：
```
Σλ_i = tr(A)（迹，对角元素之和）
Πλ_i = |A|
```

**矩阵对角化**：
若\(A\)有\(n\)个线性无关的特征向量，则：
```
P^(-1)AP = Λ
```
其中\(\Lambda = diag(\lambda_1, \lambda_2, ..., \lambda_n)\)

### 工程应用：结构振动模态分析 🔥
结构振动方程：
```
Mẍ + Kx = 0
```
特征值问题：\(|K - \omega^2 M| = 0\)  
求得固有频率和振型。

### Python代码：
```python
# 特征值分解
eigenvalues, eigenvectors = np.linalg.eig(A)
```

### 专题15：二次型 ⭐⭐⭐⭐

**二次型定义**：
```
f(x) = x^T A x = Σ Σ a_ij x_i x_j
```

**标准化**：
通过正交变换\(x = Py\)，化为标准形：
```
f = λ_1 y_1² + λ_2 y_2² + ... + λ_n y_n²
```

**正定性判断** 🔥：
- **正定**：所有特征值\(> 0\)，或所有顺序主子式\(> 0\)
- **负定**：所有特征值\(< 0\)
- **半正定**：所有特征值\(\geq 0\)
- **不定**：特征值有正有负

### 工程应用：优化问题约束
目标函数为二次型，判断正定性确定极值点。

### 专题16：随机事件与概率 ⭐⭐⭐⭐

**基本公式**：
```
加法公式：P(A∪B) = P(A) + P(B) - P(AB)
乘法公式：P(AB) = P(A)P(B|A)
```

**条件概率** 🔥：
```
P(B|A) = P(AB) / P(A)
```

**全概率公式**：
```
P(B) = Σ P(A_i)P(B|A_i)
```

**贝叶斯公式**：
```
P(A_i|B) = P(A_i)P(B|A_i) / Σ P(A_j)P(B|A_j)
```

**独立性**：
\(A, B\)独立 \(\Leftrightarrow P(AB) = P(A)P(B)\)

### 工程应用：洪水频率分析 🔥
某年发生洪水概率\(p\)，\(n\)年内至少发生一次：
```
P = 1 - (1-p)^n
```

### 专题17：随机变量及其分布 ⭐⭐⭐⭐⭐

**离散型随机变量**：
- **0-1分布**：\(P(X=1) = p\)，\(P(X=0) = 1-p\)
- **二项分布**：\(X \sim B(n,p)\)，\(P(X=k) = C_n^k p^k (1-p)^{n-k}\)
- **泊松分布**：\(X \sim P(\lambda)\)，\(P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}\)

**连续型随机变量**：
- **均匀分布**：\(X \sim U(a,b)\)
  ```
  f(x) = 1/(b-a),  a < x < b
  ```

- **指数分布**：\(X \sim Exp(\lambda)\)
  ```
  f(x) = λe^(-λx),  x > 0
  ```

- **正态分布** 🔥：\(X \sim N(\mu, \sigma^2)\)
  ```
  f(x) = (1/(σ√(2π))) exp[-(x-μ)²/(2σ²)]
  ```
  
  标准正态分布：\(X \sim N(0,1)\)
  ```
  Φ(x) = P(X ≤ x)
  ```

### 工程应用：降雨量分布
降雨量常服从对数正态分布或P-III型分布。

### Python代码：
```python
from scipy import stats

# 正态分布
mu, sigma = 0, 1
x = stats.norm.ppf(0.95, mu, sigma)  # 95%分位数

# 泊松分布
lambda_ = 3
p = stats.poisson.pmf(2, lambda_)  # P(X=2)
```

### 专题18：多维随机变量 ⭐⭐⭐⭐

**联合分布**：
```
F(x,y) = P(X≤x, Y≤y)
```

**边缘分布**：
```
F_X(x) = F(x, +∞)
F_Y(y) = F(+∞, y)
```

**条件分布** 🔥：
```
P(Y≤y|X=x) = f(x,y) / f_X(x)
```

**独立性**：
\(X, Y\)独立 \(\Leftrightarrow f(x,y) = f_X(x) f_Y(y)\)

### 工程应用：降雨-径流关系

### 专题19：数字特征 ⭐⭐⭐⭐⭐

**期望** 🔥：
```
离散：E(X) = Σ x_i p_i
连续：E(X) = ∫ x f(x) dx
```

**方差**：
```
Var(X) = E[(X - E(X))²] = E(X²) - [E(X)]²
```

**性质**：
```
E(aX + b) = aE(X) + b
Var(aX + b) = a²Var(X)
若X,Y独立：E(XY) = E(X)E(Y)
```

**协方差**：
```
Cov(X,Y) = E[(X-E(X))(Y-E(Y))] = E(XY) - E(X)E(Y)
```

**相关系数** 🔥：
```
ρ_{XY} = Cov(X,Y) / (√Var(X) · √Var(Y))
```
\(-1 \leq \rho \leq 1\)

### 工程应用：水文统计特征
计算年径流量的均值、方差、变异系数\(C_v = \sigma / \mu\)

### Python代码：
```python
import numpy as np

data = np.array([...])
mean = np.mean(data)
var = np.var(data)
std = np.std(data)
cov = np.cov(x, y)[0,1]  # 协方差
corr = np.corrcoef(x, y)[0,1]  # 相关系数
```

### 专题20：参数估计与假设检验 ⭐⭐⭐⭐⭐

**点估计**：

矩估计法：
```
用样本矩估计总体矩
μ̂ = x̄
σ̂² = (1/n)Σ(x_i - x̄)²
```

最大似然估计(MLE) 🔥：
```
L(θ) = Π f(x_i; θ)
求 max L(θ) 或 max ln L(θ)
```

**区间估计**：
正态总体均值\(\mu\)的置信区间（\(\sigma\)已知）：
```
(x̄ - z_{α/2} σ/√n,  x̄ + z_{α/2} σ/√n)
```

**假设检验**：

\(u\)检验（\(\sigma\)已知）：
```
H_0: μ = μ_0
检验统计量：u = (x̄ - μ_0)/(σ/√n) ~ N(0,1)
```

\(t\)检验（\(\sigma\)未知）：
```
检验统计量：t = (x̄ - μ_0)/(s/√n) ~ t(n-1)
```

\(\chi^2\)检验（方差检验）：
```
χ² = (n-1)s² / σ_0² ~ χ²(n-1)
```

### 工程应用：水质监测数据分析 🔥
检验某断面水质是否达标：
- \(H_0\)：浓度均值\(\mu \leq \mu_0\)（达标）
- \(H_1\)：浓度均值\(\mu > \mu_0\)（超标）

### Python代码：
```python
from scipy import stats

# t检验
t_stat, p_value = stats.ttest_1samp(data, mu_0)

# 卡方检验
chi2_stat, p_value = stats.chisquare(observed, expected)

# 正态性检验
stat, p_value = stats.normaltest(data)
```
