## 专题2：导数与微分 ⭐⭐⭐⭐⭐

### 核心公式

**基本导数公式**：
```
(x^n)' = nx^(n-1)
(sin x)' = cos x
(cos x)' = -sin x
(e^x)' = e^x
(a^x)' = a^x ln a
(ln x)' = 1/x
(log_a x)' = 1/(x ln a)
```

**导数四则运算**：
```
(u ± v)' = u' ± v'
(uv)' = u'v + uv'
(u/v)' = (u'v - uv')/v²
```

**复合函数求导（链式法则）** 🔥：
```
[f(g(x))]' = f'(g(x)) · g'(x)
```

**隐函数求导**：
对\(F(x,y) = 0\)两边对\(x\)求导

**参数方程求导**：
```
dy/dx = (dy/dt)/(dx/dt)
```

### 典型例题

**例1**：求\(y = \ln(\sin^2 x)\)的导数

解：
```
y' = [ln(sin^2 x)]' = 1/(sin^2 x) · (sin^2 x)'
   = 1/(sin^2 x) · 2sin x · cos x
   = 2cot x
```

**例2**：求由\(x^3 + y^3 = 3xy\)确定的隐函数\(y'(x)\)

解：两边对\(x\)求导：
```
3x² + 3y²y' = 3y + 3xy'
y'(3y² - 3x) = 3y - 3x²
y' = (y - x²)/(y² - x)
```

### Python代码片段

```python
import sympy as sp
x = sp.Symbol('x')
# 符号求导
y = sp.ln(sp.sin(x)**2)
dy = sp.diff(y, x)
print(f"导数：{dy}")  # 输出：2*cos(x)/sin(x)
```

### 工程应用：水位变化率
水库水位\(h(t)\)随时间变化，流速\(v = \sqrt{2gh}\)  
求流速变化率：
```
dv/dt = dv/dh · dh/dt = g/√(2gh) · dh/dt
```

## 专题3：微分中值定理 ⭐⭐⭐⭐

### 核心定理

**罗尔定理**：
若\(f(x)\)在\([a,b]\)连续，\((a,b)\)可导，且\(f(a) = f(b)\)，  
则至少存在一点\(\xi \in (a,b)\)，使\(f'(\xi) = 0\)

**拉格朗日中值定理** 🔥：
```
f(b) - f(a) = f'(ξ)(b - a)  (ξ ∈ (a,b))
```
推论：若\(f'(x) = 0\)，则\(f(x) = C\)（常数）

**柯西中值定理**：
```
[f(b) - f(a)]/[g(b) - g(a)] = f'(ξ)/g'(ξ)
```

**泰勒公式** 🔥：
```
f(x) = f(x₀) + f'(x₀)(x-x₀) + f''(x₀)(x-x₀)²/2! + ... + Rn
```
拉格朗日余项：\(R_n = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-x_0)^{n+1}\)

### 典型例题

**例**：证明当\(x > 0\)时，\(e^x > 1 + x\)

证明：设\(f(x) = e^x - 1 - x\)，\(f(0) = 0\)  
由拉格朗日中值定理：
```
f(x) - f(0) = f'(ξ)x  (0 < ξ < x)
f(x) = (e^ξ - 1)x > 0  (因为 e^ξ > 1)
```
所以\(e^x > 1 + x\)成立。

## 专题4：导数应用 ⭐⭐⭐⭐⭐

### 核心内容

**单调性**：
```
f'(x) > 0  =>  f(x)单调增
f'(x) < 0  =>  f(x)单调减
```

**极值**：
必要条件：\(f'(x_0) = 0\)  
充分条件：
- 若\(f'(x_0) = 0\)且\(f''(x_0) > 0\)，则\(x_0\)为极小值点
- 若\(f'(x_0) = 0\)且\(f''(x_0) < 0\)，则\(x_0\)为极大值点

**凹凸性**：
```
f''(x) > 0  =>  凹函数（开口向上）
f''(x) < 0  =>  凸函数（开口向下）
```

**拐点**：\(f''(x) = 0\)且二阶导数变号

**渐近线** 🔥：
- 水平渐近线：\(\lim_{x\to\infty} f(x) = A\)  =>  \(y = A\)
- 垂直渐近线：\(\lim_{x\to x_0} f(x) = \infty\)  =>  \(x = x_0\)
- 斜渐近线：\(y = kx + b\)，其中\(k = \lim_{x\to\infty}\frac{f(x)}{x}\)，\(b = \lim_{x\to\infty}[f(x) - kx]\)

### 工程应用：水库优化设计

**问题**：设计矩形蓄水池，体积\(V\)一定，求表面积最小的尺寸。

设底面边长\(x\)，高\(h\)，则\(V = x^2h\)，\(h = V/x^2\)  
表面积\(S = x^2 + 4xh = x^2 + 4V/x\)

求导：\(S' = 2x - 4V/x^2 = 0\)  =>  \(x = \sqrt[3]{2V}\)

当\(x = \sqrt[3]{2V}\)时，\(S\)最小。

## 专题5：不定积分 ⭐⭐⭐⭐

### 基本积分公式

```
∫ x^n dx = x^(n+1)/(n+1) + C  (n ≠ -1)
∫ 1/x dx = ln|x| + C
∫ e^x dx = e^x + C
∫ a^x dx = a^x/ln a + C
∫ sin x dx = -cos x + C
∫ cos x dx = sin x + C
∫ 1/cos²x dx = tan x + C
∫ 1/sin²x dx = -cot x + C
∫ 1/√(1-x²) dx = arcsin x + C
∫ 1/(1+x²) dx = arctan x + C
```

### 积分方法

**第一换元法（凑微分）** 🔥：
```
∫ f(φ(x))φ'(x) dx = ∫ f(u) du  (令 u = φ(x))
```

**第二换元法**：
```
∫ f(x) dx = ∫ f(φ(t))φ'(t) dt  (令 x = φ(t))
```

**分部积分法** 🔥：
```
∫ u dv = uv - ∫ v du
```
口诀：**"反对幂指三"**（按此顺序选\(u\)）
- 反：反三角函数
- 对：对数函数
- 幂：幂函数
- 指：指数函数
- 三：三角函数

### 典型例题

**例**：计算\(\int x e^x dx\)

解：分部积分，取\(u = x\)，\(dv = e^x dx\)
```
∫ x e^x dx = x e^x - ∫ e^x dx = x e^x - e^x + C = (x-1)e^x + C
```

## 专题6：定积分 ⭐⭐⭐⭐⭐

### 核心公式

**牛顿-莱布尼茨公式** 🔥：
```
∫[a,b] f(x) dx = F(b) - F(a)
```
其中\(F'(x) = f(x)\)

**定积分性质**：
```
∫[a,b] [f(x) ± g(x)] dx = ∫[a,b] f(x) dx ± ∫[a,b] g(x) dx
∫[a,b] kf(x) dx = k∫[a,b] f(x) dx
∫[a,b] f(x) dx = ∫[a,c] f(x) dx + ∫[c,b] f(x) dx
```

**变限积分** 🔥：
```
d/dx [∫[a,x] f(t) dt] = f(x)
d/dx [∫[a,φ(x)] f(t) dt] = f(φ(x))φ'(x)
```

**反常积分**：
```
∫[a,+∞) f(x) dx = lim[b→+∞] ∫[a,b] f(x) dx
```

### 典型例题

**例**：计算\(\int_0^\pi x \sin x dx\)

解：分部积分：
```
∫ x sin x dx = -x cos x + ∫ cos x dx = -x cos x + sin x + C
∫[0,π] x sin x dx = [-x cos x + sin x]₀^π = π
```

### 工程应用：水量平衡计算

入库流量\(Q_{in}(t)\)，出库流量\(Q_{out}(t)\)  
\(T\)时段内蓄水量变化：
```
ΔV = ∫[0,T] [Q_in(t) - Q_out(t)] dt
```

## 专题7：定积分应用 ⭐⭐⭐⭐⭐

### 核心应用

**平面图形面积**：
```
S = ∫[a,b] |f(x) - g(x)| dx
```

**旋转体体积**：
```
V_x = π∫[a,b] y² dx  (绕x轴)
V_y = π∫[c,d] x² dy  (绕y轴)
```

**曲线弧长**：
```
s = ∫[a,b] √(1 + (y')²) dx
```

**平均值**：
```
ȳ = 1/(b-a) ∫[a,b] f(x) dx
```

### 工程应用：库容曲线计算 🔥

水库库容\(V\)与水位\(h\)的关系：
```
V(h) = ∫[h₀,h] A(z) dz
```
其中\(A(z)\)为高程\(z\)处的水面面积

## 专题8：多元函数微分 ⭐⭐⭐⭐⭐

### 核心公式

**偏导数**：
```
∂f/∂x = lim[Δx→0] [f(x+Δx,y) - f(x,y)]/Δx
```

**全微分**：
```
dz = (∂z/∂x)dx + (∂z/∂y)dy
```

**复合函数偏导** 🔥：
若\(z = f(u,v)\)，\(u = u(x,y)\)，\(v = v(x,y)\)，则：
```
∂z/∂x = (∂f/∂u)(∂u/∂x) + (∂f/∂v)(∂v/∂x)
```

**方向导数**：
```
∂f/∂l = (∂f/∂x)cosα + (∂f/∂y)cosβ
```

**梯度** 🔥：
```
grad f = (∂f/∂x, ∂f/∂y)
```
方向导数在梯度方向达到最大值，最大值为\(|grad f|\)

### 工程应用：地下水位场分析

地下水位场\(h(x,y)\)，渗流方向为梯度方向的反向：
```
v = -k grad h = -k(∂h/∂x, ∂h/∂y)
```

## 专题9：二重积分 ⭐⭐⭐⭐⭐

### 计算方法

**直角坐标系**：
```
∫∫_D f(x,y) dσ = ∫[a,b] dx ∫[y₁(x),y₂(x)] f(x,y) dy
```

**极坐标系** 🔥：
```
x = r cosθ,  y = r sinθ
∫∫_D f(x,y) dσ = ∫[α,β] dθ ∫[r₁(θ),r₂(θ)] f(r cosθ, r sinθ) r dr
```

**适用极坐标的情形**：
1. 积分区域为圆或圆的一部分
2. 被积函数含\(x² + y²\)

### 典型例题

**例**：计算\(\int\int_D e^{x²+y²} dσ\)，其中\(D: x² + y² \leq 1\)

解：使用极坐标：
```
∫∫_D e^{x²+y²} dσ = ∫[0,2π] dθ ∫[0,1] e^{r²} r dr
                   = 2π · [e^{r²}/2]₀¹
                   = π(e - 1)
```

### 工程应用：降雨量分布计算

降雨强度\(p(x,y)\)（mm/h），区域\(D\)内总降雨量：
```
P = ∫∫_D p(x,y) dσ
```

## 专题10：微分方程 ⭐⭐⭐⭐⭐

### 一阶微分方程

**可分离变量型**：
```
dy/dx = f(x)g(y)  =>  dy/g(y) = f(x) dx
```

**齐次方程**：
```
dy/dx = f(y/x)  =>  令 u = y/x
```

**一阶线性方程** 🔥：
```
dy/dx + P(x)y = Q(x)
通解：y = e^(-∫P dx)[∫Q e^(∫P dx) dx + C]
```

### 二阶常系数线性微分方程 🔥

**齐次方程**：
```
y'' + py' + qy = 0
```
特征方程：\(r² + pr + q = 0\)

**根的情况**：
1. 两个不等实根\(r_1, r_2\)：\(y = C_1 e^{r_1 x} + C_2 e^{r_2 x}\)
2. 两个相等实根\(r\)：\(y = (C_1 + C_2 x)e^{rx}\)
3. 共轭复根\(r = α ± iβ\)：\(y = e^{αx}(C_1 \cos βx + C_2 \sin βx)\)

**非齐次方程**：
```
y'' + py' + qy = f(x)
通解 = 齐次通解 + 非齐次特解
```

### 工程应用：水库调洪演算 🔥

水库蓄水量\(V\)，入库流量\(Q_{in}\)，出库流量\(Q_{out}\)：
```
dV/dt = Q_in - Q_out
```
若\(Q_{out} = kV\)（线性水库），则：
```
dV/dt + kV = Q_in
```
这是一阶线性微分方程。

**Python代码片段**：
```python
from scipy.integrate import odeint
import numpy as np

def reservoir_model(V, t, k, Q_in):
    """水库调洪模型"""
    dVdt = Q_in - k * V
    return dVdt

# 参数设置
k = 0.01  # 出流系数
Q_in = 100  # 恒定入流
t = np.linspace(0, 500, 1000)

# 求解ODE
V0 = 0  # 初始蓄水量
V = odeint(reservoir_model, V0, t, args=(k, Q_in))

# V趋于平衡值 Q_in/k = 10000
```
