## 技巧1：等价无穷小替换

**口诀**：同阶替换快又准

$$x \to 0 \text{时}:$$
$$\sin x \sim x, \quad \tan x \sim x, \quad \arcsin x \sim x$$
$$1 - \cos x \sim \frac{x^2}{2}, \quad e^x - 1 \sim x, \quad \ln(1+x) \sim x$$

**【例】**：
$$\lim_{x \to 0} \frac{\sin 3x}{x} = \lim_{x \to 0} \frac{3x}{x} = 3$$

---

## 技巧2：洛必达速算

**适用**：$\frac{0}{0}$或$\frac{\infty}{\infty}$型

**【例】**：
$$\lim_{x \to 0} \frac{e^x - 1}{x} \overset{L}{=} \lim_{x \to 0} \frac{e^x}{1} = 1$$

---

## 技巧3：泰勒展开快速法

**常用展开**（$x \to 0$）：
$$e^x = 1 + x + \frac{x^2}{2!} + o(x^2)$$
$$\sin x = x - \frac{x^3}{6} + o(x^3)$$
$$\ln(1+x) = x - \frac{x^2}{2} + o(x^2)$$

**【例】**：
$$\lim_{x \to 0} \frac{e^x - 1 - x}{x^2} = \lim_{x \to 0} \frac{\frac{x^2}{2}}{x^2} = \frac{1}{2}$$

---

## 技巧4：重要极限秒杀

**两个重要极限**：

$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$

$$\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$$

**变形**：
$$\lim_{x \to 0} (1+x)^{1/x} = e$$

---

## 技巧5：无穷小阶数比较

**口诀**：高阶除低阶趋于零

**【例】**：
$$\lim_{x \to 0} \frac{x^2}{\sin x} = \lim_{x \to 0} \frac{x^2}{x} = 0$$

## Python速算工具

```python
import sympy as sp

class LimitQuickCalc:
    """极限快速计算"""
    
    @staticmethod
    def equivalent_infinitesimal(expr, x=sp.Symbol('x'), x0=0):
        """等价无穷小替换"""
        # 常用替换
        replacements = {
            sp.sin(x): x,
            sp.tan(x): x,
            1 - sp.cos(x): x**2/2,
            sp.exp(x) - 1: x,
            sp.log(1 + x): x
        }
        
        expr_simplified = expr
        for old, new in replacements.items():
            expr_simplified = expr_simplified.subs(old, new)
        
        return sp.limit(expr_simplified, x, x0)
    
    @staticmethod
    def quick_limit(expr_str, var='x', point=0):
        """快速计算极限"""
        x = sp.Symbol(var)
        expr = sp.sympify(expr_str)
        
        # 尝试直接计算
        result = sp.limit(expr, x, point)
        
        # 显示步骤
        print(f"lim(x→{point}) {expr} = {result}")
        
        return result

# 示例
calc = LimitQuickCalc()

# 例1
calc.quick_limit("sin(3*x)/x", point=0)

# 例2  
calc.quick_limit("(exp(x) - 1)/x", point=0)

# 例3
calc.quick_limit("(exp(x) - 1 - x)/x**2", point=0)
```

## 技巧6-10：连续性快判

### 技巧6：连续定义

$$\lim_{x \to x_0} f(x) = f(x_0)$$

### 技巧7：间断点分类

- **第一类**：左右极限存在
  - 可去：左=右≠f(x₀)
  - 跳跃：左≠右
- **第二类**：至少一个极限不存在

### 技巧8：分段函数连续

在分界点：左极限=右极限=函数值

### 技巧9：零点定理

$f(a)f(b) < 0$ ⇒ 存在$\xi \in (a,b)$使$f(\xi)=0$

### 技巧10：最值定理

闭区间连续函数必有最大最小值
