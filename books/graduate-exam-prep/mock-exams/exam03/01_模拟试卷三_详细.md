### 1. 简述雷诺数的物理意义及其在水力学中的应用

**【参考答案】**

**(1) 物理意义**（5分）

雷诺数$Re = \frac{vd}{\nu}$表示惯性力与粘性力的比值：

- $Re$大：惯性力占主导，流动不稳定，易形成湍流
- $Re$小：粘性力占主导，流动稳定，保持层流

**(2) 应用**（5分）

1. **流态判别**：
   - 圆管：$Re < 2000$层流，$Re > 4000$湍流
   - 明渠：$Re < 500$层流，$Re > 2000$湍流

2. **模型实验**：雷诺数相似准则，$Re_m = Re_p$

3. **阻力计算**：层流$\lambda = 64/Re$，湍流查莫迪图

### 3. 简述水库调洪的基本原理

**【参考答案】**

**(1) 基本方程**（4分）

**水量平衡方程**：
$$\frac{dV}{dt} = Q(t) - q(t)$$

或差分形式：
$$V_{t+\Delta t} - V_t = (\bar{Q}_t - \bar{q}_t)\Delta t$$

- $V$：库容
- $Q(t)$：入库流量
- $q(t)$：出库流量

**(2) 调洪作用**（3分）

- **削减洪峰**：$q_{max} < Q_{max}$
- **滞后洪峰**：峰现时间推迟
- **拉长历时**：洪水过程展宽

**(3) 计算方法**（3分）

- 列表试算法
- 图解法（调洪图）
- 数值法（编程求解）

### 计算题1：管道水力计算（20分）

**题目**：

水平圆管，L=1000m，d=0.3m，Q=0.15m³/s，沿程阻力系数λ=0.025。求：

(1) 管中平均流速  
(2) 沿程水头损失  
(3) 若上游断面压强为300kPa，求下游断面压强

**【解答】**

**(1) 流速**（5分）

$$A = \frac{\pi d^2}{4} = \frac{\pi \times 0.3^2}{4} = 0.0707 \text{ m}^2$$

$$v = \frac{Q}{A} = \frac{0.15}{0.0707} = 2.12 \text{ m/s}$$

**(2) 沿程损失**（8分）

达西公式：
$$h_f = \lambda \frac{L}{d} \frac{v^2}{2g} = 0.025 \times \frac{1000}{0.3} \times \frac{2.12^2}{2 \times 9.81}$$

$$h_f = 0.025 \times 3333.3 \times 0.229 = 19.1 \text{ m}$$

**(3) 下游压强**（7分）

能量方程（水平管，流速相同）：
$$\frac{p_1}{\gamma} = \frac{p_2}{\gamma} + h_f$$

$$\frac{p_2}{\gamma} = \frac{300000}{9810} - 19.1 = 30.6 - 19.1 = 11.5 \text{ m}$$

$$p_2 = 11.5 \times 9810 = 112.8 \text{ kPa}$$

**Python验证**：

```python
import numpy as np

class PipeHydraulics:
    """管道水力计算"""
    
    @staticmethod
    def pipe_flow_calculation(L, d, Q, lam, p1):
        """
        管道水力计算
        
        参数:
            L: 管长 (m)
            d: 直径 (m)
            Q: 流量 (m³/s)
            lam: 沿程阻力系数
            p1: 上游压强 (Pa)
        """
        g = 9.81
        gamma = 9810
        
        # 流速
        A = np.pi * d**2 / 4
        v = Q / A
        
        # 沿程损失
        h_f = lam * (L/d) * (v**2 / (2*g))
        
        # 下游压强
        p2 = p1 - gamma * h_f
        
        results = {
            'velocity': v,
            'head_loss': h_f,
            'p2': p2
        }
        
        return results

# 示例计算
pf = PipeHydraulics()

results = pf.pipe_flow_calculation(
    L=1000, d=0.3, Q=0.15, lam=0.025, p1=300000)

print("管道水力计算结果:")
print(f"  流速: {results['velocity']:.2f} m/s")
print(f"  沿程损失: {results['head_loss']:.1f} m")
print(f"  下游压强: {results['p2']/1000:.1f} kPa")
```

### 计算题2：设计洪水计算（20分）

**题目**：

某站30年洪峰流量资料，统计参数：$\bar{Q}=1500$ m³/s，$C_v=0.40$，$C_s=2C_v$。求：

(1) 百年一遇设计洪峰流量  
(2) 五十年一遇设计洪峰流量

**【解答】**

**(1) 参数**（4分）

$$C_s = 2C_v = 2 \times 0.40 = 0.80$$

**(2) 百年一遇**（8分）

$P=1\%$，查Pearson-III表：

$C_v=0.40$，$C_s=0.80$ → $\Phi_{1\%} = 2.35$

$$Q_{1\%} = \bar{Q} \times \Phi_{1\%} = 1500 \times 2.35 = 3525 \text{ m}^3\text{/s}$$

**(3) 五十年一遇**（8分）

$P=2\%$，查表：$\Phi_{2\%} = 2.15$

$$Q_{2\%} = 1500 \times 2.15 = 3225 \text{ m}^3\text{/s}$$

**Python实现**：

```python
class FloodFrequencyAnalysis:
    """设计洪水计算"""
    
    @staticmethod
    def pearson3_phi_approx(Cv, Cs, P):
        """Pearson-III模比系数（近似）"""
        from scipy import stats
        
        u_p = stats.norm.ppf(1 - P)
        
        phi = 1 + Cv * (u_p + (u_p**2 - 1)*Cs/6 - 
                        (u_p**3 - 6*u_p)*Cs**2/36)
        
        return phi
    
    @staticmethod
    def design_flood(mean, Cv, Cs, P_values):
        """设计洪水"""
        results = {}
        
        for P in P_values:
            phi = FloodFrequencyAnalysis.pearson3_phi_approx(Cv, Cs, P)
            Q_p = mean * phi
            
            results[P] = Q_p
        
        return results

# 计算
ffa = FloodFrequencyAnalysis()

mean_Q = 1500
Cv = 0.40
Cs = 0.80

design_cases = [0.01, 0.02, 0.05, 0.1]

results = ffa.design_flood(mean_Q, Cv, Cs, design_cases)

print("设计洪水计算:")
for P, Q_p in results.items():
    T = 1 / P
    print(f"  P={P*100:.1f}% ({T:.0f}年一遇): Q_p = {Q_p:.0f} m³/s")
```

### 计算题3：明渠均匀流计算（20分）

**题目**：

梯形断面明渠，底宽b=5m，边坡系数m=2，水深h=2m，坡度i=0.001，糙率n=0.025。求：

(1) 过流断面积  
(2) 湿周  
(3) 水力半径  
(4) 流量（Manning公式）

**【解答】**（见详细计算）

```python
class OpenChannelFlow:
    """明渠流计算"""
    
    @staticmethod
    def trapezoidal_channel(b, m, h, i, n):
        """
        梯形断面均匀流
        
        Manning公式: Q = (1/n) * A * R^(2/3) * i^(1/2)
        
        参数:
            b: 底宽 (m)
            m: 边坡系数
            h: 水深 (m)
            i: 坡度
            n: 糙率
        """
        # 面积
        A = (b + m*h) * h
        
        # 湿周
        chi = b + 2*h*np.sqrt(1 + m**2)
        
        # 水力半径
        R = A / chi
        
        # 流量
        Q = (1/n) * A * R**(2/3) * i**(1/2)
        
        # 流速
        v = Q / A
        
        results = {
            'area': A,
            'wetted_perimeter': chi,
            'hydraulic_radius': R,
            'discharge': Q,
            'velocity': v
        }
        
        return results

# 计算
ocf = OpenChannelFlow()

results = ocf.trapezoidal_channel(
    b=5, m=2, h=2, i=0.001, n=0.025)

print("梯形明渠均匀流计算:")
print(f"  过流面积: {results['area']:.2f} m²")
print(f"  湿周: {results['wetted_perimeter']:.2f} m")
print(f"  水力半径: {results['hydraulic_radius']:.2f} m")
print(f"  流量: {results['discharge']:.2f} m³/s")
print(f"  流速: {results['velocity']:.2f} m/s")
```

### 题目：水资源可持续利用综合分析

某市水资源总量12亿m³，现状人口180万，生态需水3.5亿m³。

现状用水：生活2亿m³，工业4亿m³，农业5亿m³。

规划2030年人口达到250万。

**问题**：

(1) 计算现状水资源开发利用率、人均水资源量、缺水率（8分）

(2) 判断现状承载状态（4分）

(3) 分析2030年承载压力，提出对策（8分）

**【参考答案】**

**(1) 现状指标**

```python
class WaterResourcesAnalysis:
    """水资源综合分析"""
    
    @staticmethod
    def current_status_analysis(W_total, W_eco, W_use, pop):
        """现状分析"""
        # 可利用水量
        W_avail = W_total - W_eco
        
        # 开发利用率
        util_rate = W_use / W_total * 100
        
        # 人均水资源
        per_capita = W_total / pop * 1e4  # m³/人·年
        
        # 缺水量
        W_deficit = max(0, W_use - W_avail)
        
        # 缺水率
        deficit_rate = W_deficit / W_use * 100 if W_use > 0 else 0
        
        return {
            'utilization_rate': util_rate,
            'per_capita': per_capita,
            'deficit_rate': deficit_rate,
            'W_avail': W_avail,
            'W_deficit': W_deficit
        }

# 计算
wra = WaterResourcesAnalysis()

W_total = 12.0  # 亿m³
W_eco = 3.5
W_use = 2 + 4 + 5  # 11亿m³
pop = 180  # 万人

results = wra.current_status_analysis(W_total, W_eco, W_use, pop)

print("现状指标:")
print(f"  开发利用率: {results['utilization_rate']:.1f}%")
print(f"  人均水资源: {results['per_capita']:.0f} m³/人·年")
print(f"  缺水率: {results['deficit_rate']:.1f}%")

# (2) 承载状态
if results['utilization_rate'] > 40 or results['deficit_rate'] > 10:
    print("\n承载状态: 超载")
else:
    print("\n承载状态: 未超载")

# (3) 2030年预测
pop_2030 = 250
W_demand_2030 = W_use * (pop_2030 / pop)  # 简化预测

print(f"\n2030年预测:")
print(f"  需水量: {W_demand_2030:.1f} 亿m³")
print(f"  缺水量: {W_demand_2030 - results['W_avail']:.1f} 亿m³")

print(f"\n对策建议:")
print("  1. 节水：农业节水灌溉，工业循环利用")
print("  2. 开源：再生水利用，雨洪资源化")
print("  3. 调水：跨流域引水")
print("  4. 治污：提高水质，增加可用水量")
```
