**无机污染物**：

- 多环芳烃（PAHs）

## 二、污染物迁移理论

### 2.1 对流-弥散方程

**一维稳态**：

$$v\frac{\partial C}{\partial x} = D_L\frac{\partial^2 C}{\partial x^2}$$

**解析解**（连续点源）：

$$C(x,t) = \frac{C_0}{2}\text{erfc}\left(\frac{x-vt}{2\sqrt{D_L t}}\right)$$

**三维瞬时点源**：

$$C(x,y,z,t) = \frac{M}{8(\pi t)^{3/2}\sqrt{D_x D_y D_z}}\exp\left[-\frac{(x-vt)^2}{4D_x t} - \frac{y^2}{4D_y t} - \frac{z^2}{4D_z t}\right]$$

### 2.2 弥散系数

**纵向弥散系数**：

$$D_L = \alpha_L v + D_m$$

**横向弥散系数**：

$$D_T = \alpha_T v + D_m$$

**参数**：
- $\alpha_L$：纵向弥散度（m）
- $\alpha_T$：横向弥散度（m）
- $D_m$：分子扩散系数（m²/d）

**经验关系**：

$$\alpha_L = 0.1 L^{1.2}$$

其中$L$为特征长度

### 2.3 吸附与降解

**线性吸附**（平衡）：

$$S = K_d C$$

**阻滞因子**：

$$R = 1 + \frac{\rho_b K_d}{n}$$

**一阶降解**：

$$\frac{\partial C}{\partial t} = -\lambda C$$

**综合方程**：

$$R\frac{\partial C}{\partial t} + v\frac{\partial C}{\partial x} = D_L\frac{\partial^2 C}{\partial x^2} - \lambda C$$

## 三、污染羽模拟

### 3.1 高斯烟羽模型

**稳态连续点源**：

$$C(x,y,z) = \frac{Q_s}{2\pi v \sigma_y \sigma_z}\exp\left[-\frac{y^2}{2\sigma_y^2} - \frac{z^2}{2\sigma_z^2}\right]$$

**扩散参数**：

$$\sigma_y = \sqrt{2D_T x / v}$$

$$\sigma_z = \sqrt{2D_z x / v}$$

### 3.2 污染晕范围

**纵向扩散**：

$$\sigma_x = \sqrt{2D_L t}$$

**羽状污染晕**：
- 长度：$L \approx vt + 3\sigma_x$
- 宽度：$W \approx 6\sigma_y$
- 深度：$H \approx 6\sigma_z$

## 四、地下水防治技术

### 4.1 源头控制

**工程措施**：
- 防渗衬里
- 双层罐体
- 泄漏检测系统
- 雨污分流

**管理措施**：
- 污染源清单
- 定期检查
- 应急预案

### 4.2 抽出-处理技术

**水力控制**：

$$Q_{pump} = 2\pi K M(H - h_w)/\ln(R/r_w)$$

**捕获区宽度**：

$$W = \frac{Q_{pump}}{Ki}$$

**处理技术**：
- 活性炭吸附
- 空气吹脱
- 生物降解
- 化学氧化

### 4.3 原位修复技术

**(1) 空气注入**（Air Sparging）：

**通量**：

$$Q_{air} = K_{air} A \frac{\Delta P}{L}$$

**(2) 渗透反应墙**（PRB）：

**设计厚度**：

$$d_{PRB} = \frac{vC_0 t_{1/2}}{k_{obs}}$$

**(3) 生物修复**：

**降解速率**：

$$r = \frac{k_{max} C}{K_s + C}$$

## 五、风险评估

### 5.1 暴露途径

**直接接触**：
- 饮用水摄入
- 皮肤接触
- 吸入挥发物

**间接接触**：
- 食物链累积
- 农作物吸收

### 5.2 健康风险

**非致癌风险**（危害商）：

$$HQ = \frac{CDI}{RfD}$$

**致癌风险**：

$$CR = CDI \times SF$$

**参数**：
- CDI：慢性日摄入量（mg/kg·d）
- RfD：参考剂量（mg/kg·d）
- SF：致癌斜率因子（mg/kg·d）⁻¹

import numpy as np
import matplotlib.pyplot as plt
from scipy.special import erfc
from scipy.integrate import odeint

class GroundwaterContamination:
    """地下水污染与防治"""
    
    def __init__(self):
        pass
    
    def advection_dispersion_1D(self, x, t, C0, v, D_L):
        """
        一维对流-弥散方程解析解
        
        C(x,t) = (C0/2)·erfc[(x-vt)/(2√(D_L·t))]
        
        参数:
            x: 距离 (m)
            t: 时间 (d)
            C0: 初始浓度 (mg/L)
            v: 渗透速度 (m/d)
            D_L: 纵向弥散系数 (m²/d)
        """
        if t <= 0:
            return 0
        
        arg = (x - v * t) / (2 * np.sqrt(D_L * t))
        C = (C0 / 2) * erfc(arg)
        
        return C
    
    def dispersion_coefficient(self, v, alpha_L, D_m=1e-5):
        """
        弥散系数计算
        
        D_L = α_L·v + D_m
        
        参数:
            v: 渗透速度 (m/d)
            alpha_L: 弥散度 (m)
            D_m: 分子扩散系数 (m²/d)
        """
        D_L = alpha_L * v + D_m
        return D_L
    
    def dispersivity_estimate(self, L):
        """
        弥散度经验估算
        
        α_L = 0.1·L^1.2
        
        参数:
            L: 特征长度 (m)
        """
        alpha_L = 0.1 * L**1.2
        alpha_T = 0.1 * alpha_L  # 横向约为纵向的10%
        
        return alpha_L, alpha_T
    
    def retardation_factor(self, rho_b, K_d, n):
        """
        阻滞因子
        
        R = 1 + (ρ_b·K_d)/n
        
        参数:
            rho_b: 土壤干密度 (kg/L)
            K_d: 分配系数 (L/kg)
            n: 孔隙率
        """
        R = 1 + (rho_b * K_d) / n
        return R
    
    def gaussian_plume_2D(self, x, y, Q_s, v, D_T, b):
        """
        二维高斯烟羽模型
        
        C(x,y) = (Q_s/(2π·v·σ_y·b))·exp[-y²/(2σ_y²)]
        
        参数:
            x, y: 坐标 (m)
            Q_s: 源强 (mg/d)
            v: 渗透速度 (m/d)
            D_T: 横向弥散系数 (m²/d)
            b: 含水层厚度 (m)
        """
        if x <= 0:
            return 0
        
        # 横向扩散标准差
        sigma_y = np.sqrt(2 * D_T * x / v)
        
        # 浓度计算
        C = (Q_s / (2 * np.pi * v * sigma_y * b)) * \
            np.exp(-y**2 / (2 * sigma_y**2))
        
        return C
    
    def plume_dimensions(self, t, v, D_L, D_T):
        """
        污染羽尺寸估算
        
        返回: 长度、宽度、深度
        """
        sigma_x = np.sqrt(2 * D_L * t)
        sigma_y = np.sqrt(2 * D_T * t)
        
        # 3σ范围
        length = v * t + 3 * sigma_x
        width = 6 * sigma_y
        
        return {
            'length': length,
            'width': width,
            'sigma_x': sigma_x,
            'sigma_y': sigma_y
        }
    
    def capture_zone_width(self, Q_pump, K, i):
        """
        抽水井捕获区宽度
        
        W = Q_pump / (K·i)
        
        参数:
            Q_pump: 抽水量 (m³/d)
            K: 渗透系数 (m/d)
            i: 水力坡度
        """
        W = Q_pump / (K * i)
        return W
    
    def health_risk_assessment(self, C, IR, EF, ED, BW, AT, RfD=None, SF=None):
        """
        健康风险评估
        
        参数:
            C: 污染物浓度 (mg/L)
            IR: 摄入速率 (L/d)
            EF: 暴露频率 (d/yr)
            ED: 暴露年限 (yr)
            BW: 体重 (kg)
            AT: 平均时间 (d)
            RfD: 参考剂量 (mg/kg/d)，非致癌
            SF: 致癌斜率因子 (mg/kg/d)^-1，致癌
        """
        # 慢性日摄入量
        CDI = (C * IR * EF * ED) / (BW * AT)
        
        results = {'CDI': CDI}
        
        # 非致癌风险
        if RfD is not None:
            HQ = CDI / RfD
            results['HQ'] = HQ
            results['risk_type'] = 'non-cancer'
            results['acceptable'] = HQ < 1.0
        
        # 致癌风险
        if SF is not None:
            CR = CDI * SF
            results['CR'] = CR
            results['risk_type'] = 'cancer'
            results['acceptable'] = CR < 1e-6
        
        return results
    
    def plot_contamination_analysis(self):
        """绘制地下水污染分析图"""
        fig, axes = plt.subplots(2, 2, figsize=(16, 12))
        
        # 1. 一维对流-弥散浓度分布
        ax1 = axes[0, 0]
        
        C0 = 100  # mg/L
        v = 0.5   # m/d
        alpha_L = 10  # m
        D_L = self.dispersion_coefficient(v, alpha_L)
        
        x_range = np.linspace(0, 200, 500)
        t_values = [10, 30, 60, 100, 180]
        colors_t = plt.cm.viridis(np.linspace(0, 1, len(t_values)))
        
        for t, color in zip(t_values, colors_t):
            C_profile = [self.advection_dispersion_1D(x, t, C0, v, D_L) 
                        for x in x_range]
            ax1.plot(x_range, C_profile, color=color, linewidth=2.5,
                    label=f't={t}d')
        
        ax1.set_xlabel('距离 x (m)', fontsize=11)
        ax1.set_ylabel('浓度 C (mg/L)', fontsize=11)
        ax1.set_title(f'一维对流-弥散浓度分布\nv={v}m/d, D_L={D_L:.2f}m²/d',
                     fontsize=13, fontweight='bold')
        ax1.legend()
        ax1.grid(True, alpha=0.3)
        
        # 2. 二维污染羽扩散
        ax2 = axes[0, 1]
        
        Q_s = 1000  # mg/d
        b = 10      # m
        v_2d = 0.5  # m/d
        alpha_T = 1  # m
        D_T = self.dispersion_coefficient(v_2d, alpha_T)
        
        x_grid = np.linspace(0, 150, 100)
        y_grid = np.linspace(-40, 40, 100)
        X, Y = np.meshgrid(x_grid, y_grid)
        
        C_2d = np.zeros_like(X)
        for i in range(len(y_grid)):
            for j in range(len(x_grid)):
                C_2d[i, j] = self.gaussian_plume_2D(X[i, j], Y[i, j],
                                                    Q_s, v_2d, D_T, b)
        
        # 等值线图
        levels = [0.1, 0.5, 1, 2, 5, 10, 20]
        contour = ax2.contourf(X, Y, C_2d, levels=levels,
                              cmap='RdYlBu_r', alpha=0.8)
        
        contour_lines = ax2.contour(X, Y, C_2d, levels=levels,
                                    colors='black', linewidths=1, alpha=0.5)
        ax2.clabel(contour_lines, inline=True, fontsize=9)
        
        # 污染源
        ax2.plot(0, 0, 'r*', markersize=20, label='污染源')
        
        # 地下水流向
        ax2.arrow(10, -30, 30, 0, head_width=2, head_length=5,
                 fc='blue', ec='blue', linewidth=2, label='地下水流向')
        
        cbar = plt.colorbar(contour, ax=ax2)
        cbar.set_label('浓度 C (mg/L)', fontsize=11)
        
        ax2.set_xlabel('x (m)', fontsize=11)
        ax2.set_ylabel('y (m)', fontsize=11)
        ax2.set_title('二维污染羽扩散（高斯烟羽）', fontsize=13, fontweight='bold')
        ax2.legend(loc='upper right')
        ax2.axis('equal')
        
        # 3. 阻滞因子影响
        ax3 = axes[1, 0]
        
        K_d_range = np.logspace(-2, 2, 100)
        rho_b = 1.5  # kg/L
        n = 0.3
        
        R_values = [self.retardation_factor(rho_b, K_d, n) for K_d in K_d_range]
        
        ax3.semilogx(K_d_range, R_values, 'b-', linewidth=2.5)
        
        # 标注典型值
        materials = [
            ('砂土（有机污染物）', 0.1, 'green'),
            ('粘土（重金属）', 1.0, 'orange'),
            ('有机质土壤', 10, 'red')
        ]
        
        for name, K_d, color in materials:
            R = self.retardation_factor(rho_b, K_d, n)
            ax3.plot(K_d, R, 'o', color=color, markersize=12)
            ax3.text(K_d, R * 1.1, name, ha='center', fontsize=9,
                    bbox=dict(boxstyle='round', facecolor=color, alpha=0.3))
        
        ax3.set_xlabel('分配系数 K_d (L/kg)', fontsize=11)
        ax3.set_ylabel('阻滞因子 R', fontsize=11)
        ax3.set_title(f'阻滞因子与吸附作用\nρ_b={rho_b}kg/L, n={n}',
                     fontsize=13, fontweight='bold')
        ax3.grid(True, alpha=0.3, which='both')
        
        # 4. 捕获区宽度
        ax4 = axes[1, 1]
        
        Q_pump_range = np.linspace(10, 200, 100)
        K_aquifer = 20  # m/d
        i_gradient = 0.001
        
        W_values = [self.capture_zone_width(Q, K_aquifer, i_gradient)
                   for Q in Q_pump_range]
        
        ax4.plot(Q_pump_range, W_values, 'b-', linewidth=2.5,
                label='捕获区宽度')
        
        # 典型抽水量标注
        Q_typical = [50, 100, 150]
        colors_q = ['green', 'orange', 'red']
        
        for Q, color in zip(Q_typical, colors_q):
            W = self.capture_zone_width(Q, K_aquifer, i_gradient)
            ax4.plot(Q, W, 'o', color=color, markersize=12)
            ax4.text(Q, W * 1.1, f'Q={Q}m³/d\nW={W:.0f}m',
                    ha='center', fontsize=9,
                    bbox=dict(boxstyle='round', facecolor=color, alpha=0.3))
        
        ax4.set_xlabel('抽水量 Q (m³/d)', fontsize=11)
        ax4.set_ylabel('捕获区宽度 W (m)', fontsize=11)
        ax4.set_title(f'抽水井捕获区\nK={K_aquifer}m/d, i={i_gradient}',
                     fontsize=13, fontweight='bold')
        ax4.legend()
        ax4.grid(True, alpha=0.3)
        
        plt.tight_layout()
        plt.savefig('groundwater_contamination.png', dpi=300)

## 七、典型考题

### 【例题1】对流-弥散计算（15分）

**题目**：连续点源，$C_0=50$mg/L，$v=0.3$m/d，$D_L=3$m²/d。求$x=30$m，$t=100$d处浓度。

**【解答】**

$$C(x,t) = \frac{C_0}{2}\text{erfc}\left(\frac{x-vt}{2\sqrt{D_L t}}\right)$$

$$= \frac{50}{2}\text{erfc}\left(\frac{30-0.3 \times 100}{2\sqrt{3 \times 100}}\right)$$

$$= 25 \cdot \text{erfc}(0) = 25 \times 1 = 25 \text{ mg/L}$$

---
