## 一、地下水污染特征

### 1.1 污染特点

**隐蔽性**：地下，不易发现  
**滞后性**：污染显现时间长（数月至数年）  
**累积性**：污染物长期积累  
**难恢复性**：自净能力弱，治理周期长（数十年）

### 1.2 污染来源

**点源污染**：
- 工业废水渗坑
- 垃圾填埋场
- 加油站泄漏
- 化粪池渗漏

**面源污染**：
- 农业化肥、农药
- 大气沉降
- 不合理灌溉

**线源污染**：
- 污染河流补给
- 地下管道泄漏

## 二、污染物运移

### 2.1 对流-弥散方程

**一维稳态**：
$$D_x \frac{\partial^2 C}{\partial x^2} - v \frac{\partial C}{\partial x} = 0$$

**边界条件**：
- $x=0$: $C = C_0$
- $x \to \infty$: $C \to 0$

**解析解**：
$$C(x) = C_0 \exp\left(\frac{vx}{D_x}\right) \text{erfc}\left(\frac{x}{2\sqrt{D_x t}}\right)$$

简化（稳态）：
$$C(x) = C_0 \exp\left(-\frac{vx}{D_x}\right)$$

### 2.2 关键参数

**孔隙流速**：
$$v = \frac{K \cdot i}{n_e}$$

- $K$：渗透系数 (m/d)
- $i$：水力坡度
- $n_e$：有效孔隙度

**纵向弥散系数**：
$$D_x = \alpha_L \cdot v + D^*$$

- $\alpha_L$：纵向弥散度 (m)
- $D^*$：分子扩散系数 (m²/d)

### 2.3 滞后系数

**吸附作用**：
$$R_d = 1 + \frac{\rho_b K_d}{n_e}$$

- $\rho_b$：土壤容重 (kg/m³)
- $K_d$：分配系数 (L/kg)

**实际迁移速度**：
$$v_c = \frac{v}{R_d}$$

### Python实现

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.special import erfc

class GroundwaterContamination:
    """地下水污染模拟"""
    
    def __init__(self, K=10, i=0.005, ne=0.25):
        """
        参数:
            K: 渗透系数 (m/d)
            i: 水力坡度
            ne: 有效孔隙度
        """
        self.K = K
        self.i = i
        self.ne = ne
        
        # 孔隙流速
        self.v = K * i / ne
    
    def dispersion_coefficient(self, alpha_L=10, D_star=1e-5):
        """
        弥散系数
        
        D_x = α_L * v + D*
        
        参数:
            alpha_L: 纵向弥散度 (m)
            D_star: 分子扩散系数 (m²/d)
        """
        D_x = alpha_L * self.v + D_star
        
        return D_x
    
    def retardation_factor(self, rho_b=1600, K_d=0.5):
        """
        滞后系数
        
        R_d = 1 + (ρ_b * K_d) / n_e
        
        参数:
            rho_b: 土壤容重 (kg/m³)
            K_d: 分配系数 (L/kg)
        """
        R_d = 1 + (rho_b * K_d) / self.ne
        
        return R_d
    
    def concentration_1d_steady(self, x, C0, D_x):
        """
        一维稳态浓度分布
        
        C(x) = C₀ * exp(-vx/D_x)
        
        参数:
            x: 距离 (m)
            C0: 源浓度 (mg/L)
            D_x: 纵向弥散系数 (m²/d)
        """
        C = C0 * np.exp(-self.v * x / D_x)
        
        return C
    
    def concentration_1d_transient(self, x, t, C0, D_x, R_d=1):
        """
        一维非稳态浓度分布
        
        C(x,t) = (C₀/2) * [erfc((x-vt/R_d)/(2√(D_x*t/R_d))) + 
                           exp(vx/D_x)*erfc((x+vt/R_d)/(2√(D_x*t/R_d)))]
        
        简化为：
        C(x,t) = C₀ * erfc((R_d*x - v*t)/(2√(D_x*R_d*t)))
        
        参数:
            x: 距离 (m)
            t: 时间 (d)
            C0: 源浓度 (mg/L)
            D_x: 纵向弥散系数 (m²/d)
            R_d: 滞后系数
        """
        v_c = self.v / R_d  # 实际迁移速度
        
        arg = (x - v_c * t) / (2 * np.sqrt(D_x * t / R_d))
        
        C = (C0 / 2) * erfc(arg)
        
        return C
    
    def pollution_plume_area(self, x_max, y_max, C0, D_x, D_y, t):
        """
        二维污染羽
        
        C(x,y,t) = (C₀*A₀)/(4πt√(D_x*D_y)) * 
                   exp(-(x-vt)²/(4D_x*t) - y²/(4D_y*t))
        
        简化为高斯羽模型
        """
        x = np.linspace(0, x_max, 100)
        y = np.linspace(-y_max, y_max, 100)
        X, Y = np.meshgrid(x, y)
        
        # 高斯分布（简化）
        sigma_x = np.sqrt(2 * D_x * t)
        sigma_y = np.sqrt(2 * D_y * t)
        
        C = C0 * np.exp(-((X - self.v*t)**2 / (2*sigma_x**2) + 
                         Y**2 / (2*sigma_y**2)))
        
        return X, Y, C
    
    def plot_concentration_profile(self, C0, D_x, R_d_values, x_max=200):
        """绘制浓度剖面（不同滞后系数）"""
        x = np.linspace(0, x_max, 200)
        
        fig, axes = plt.subplots(1, 2, figsize=(16, 6))
        
        # (a) 稳态
        for R_d in R_d_values:
            v_actual = self.v / R_d
            D_x_actual = D_x / R_d
            
            C_steady = self.concentration_1d_steady(x, C0, D_x_actual)
            
            axes[0].plot(x, C_steady, linewidth=2,
                        label=f'R_d={R_d:.1f}')
        
        axes[0].set_xlabel('距离 x (m)', fontsize=12)
        axes[0].set_ylabel('浓度 C (mg/L)', fontsize=12)
        axes[0].set_title('稳态浓度分布', fontsize=14)
        axes[0].legend()
        axes[0].grid(True, alpha=0.3)
        axes[0].set_yscale('log')
        
        # (b) 非稳态（不同时间）
        t_values = [100, 200, 500, 1000]  # days
        
        for t in t_values:
            C_transient = self.concentration_1d_transient(x, t, C0, D_x)
            
            axes[1].plot(x, C_transient, linewidth=2,
                        label=f't={t}天')
        
        axes[1].set_xlabel('距离 x (m)', fontsize=12)
        axes[1].set_ylabel('浓度 C (mg/L)', fontsize=12)
        axes[1].set_title('非稳态浓度分布', fontsize=14)
        axes[1].legend()
        axes[1].grid(True, alpha=0.3)
        
        plt.tight_layout()
        plt.savefig('groundwater_contamination_profile.png', dpi=300)
        plt.show()
    
    def plot_pollution_plume(self, C0, D_x, D_y, t):
        """绘制污染羽平面图"""
        X, Y, C = self.pollution_plume_area(500, 100, C0, D_x, D_y, t)
        
        fig, ax = plt.subplots(figsize=(14, 6))
        
        # 等浓度线
        levels = [C0*0.01, C0*0.05, C0*0.1, C0*0.25, C0*0.5, C0*0.75, C0*0.9]
        
        contour = ax.contourf(X, Y, C, levels=20, cmap='RdYlBu_r', alpha=0.7)
        contour_lines = ax.contour(X, Y, C, levels=levels, colors='k',
                                   linewidths=1, alpha=0.5)
        
        ax.clabel(contour_lines, inline=True, fontsize=9, fmt='%.2f')
        
        # 污染源标记
        ax.plot(0, 0, 'r*', markersize=20, label='污染源')
        
        # 地下水流向
        ax.arrow(50, 80, 100, 0, head_width=8, head_length=20,
                fc='blue', ec='blue', alpha=0.5, label='地下水流向')
        
        cbar = plt.colorbar(contour, ax=ax)
        cbar.set_label('浓度 (mg/L)', fontsize=12)
        
        ax.set_xlabel('距离 x (m)', fontsize=12)
        ax.set_ylabel('距离 y (m)', fontsize=12)
        ax.set_title(f'地下水污染羽平面图 (t={t}天)', fontsize=14)
        ax.legend(loc='upper right')
        ax.set_aspect('equal')
        
        plt.tight_layout()
        plt.savefig('pollution_plume_2d.png', dpi=300)
        plt.show()

# 示例1：浓度剖面计算
print("="*60)
print("示例1：地下水污染浓度计算")

# 弥散系数

# 绘制浓度剖面
gc.plot_concentration_profile(C0, D_x, [1.0, 2.0, 5.0])

# 示例2：污染羽模拟
print("\n" + "="*60)
print("示例2：二维污染羽模拟")
print("="*60)

D_y = D_x / 10  # 横向弥散系数通常为纵向的1/10

print(f"弥散系数:")
```

## 三、污染防治

### 3.1 预防措施

1. **源头控制**：
   - 防渗措施（垃圾填埋场、化工园区）
   - 地下储罐双层设计
   - 定期检漏

2. **监测预警**：
   - 布设监测井网
   - 实时在线监测
   - 预警阈值设定

### 3.2 修复技术

**抽出-处理（Pump and Treat）**：
- 抽取污染地下水
- 地上处理后回灌或排放

**原位生物修复**：
- 投加营养物质，促进微生物降解
- 适用于石油烃、BTEX等

**化学氧化**：
- 注入氧化剂（过硫酸盐、芬顿试剂）
- 降解有机污染物

**渗透反应墙（PRB）**：
- 在地下水径流路径上设置反应墙
- 被动处理污染物

## 四、典型考题

### 【例题】污染物迁移计算

**题目**（20分）

某污染源C₀=200mg/L，含水层K=8m/d，i=0.008，n_e=0.3，α_L=5m。求距源100m处的稳态浓度。

**【解答】**

(1) **孔隙流速**（5分）
$$v = \frac{K \cdot i}{n_e} = \frac{8 \times 0.008}{0.3} = 0.213 \text{ m/d}$$

(2) **弥散系数**（5分）
$$D_x = \alpha_L \cdot v = 5 \times 0.213 = 1.065 \text{ m}^2\text{/d}$$

(3) **稳态浓度**（10分）
$$C(100) = C_0 \exp\left(-\frac{vx}{D_x}\right) = 200 \times \exp\left(-\frac{0.213 \times 100}{1.065}\right)$$

$$C(100) = 200 \times e^{-20} \approx 0 \text{ mg/L}$$

（实际很小，基本检测不到）
