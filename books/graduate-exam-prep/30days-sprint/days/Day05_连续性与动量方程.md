## 📚 核心考点

### 考点1：连续性方程 ⭐⭐⭐⭐⭐

#### 基本形式
```
Q = A₁v₁ = A₂v₂ = 常数
```

#### 微分形式（不可压缩）
```
∂u/∂x + ∂v/∂y + ∂w/∂z = 0
```

---

### 考点2：动量方程 ⭐⭐⭐⭐⭐

#### 基本形式
```
ΣF = ρQ(v₂ - v₁)
```

#### 分量形式
```
ΣF_x = ρQ(v₂ₓ - v₁ₓ)
ΣF_y = ρQ(v₂ᵧ - v₁ᵧ)
```

其中：
- `ΣF` - 作用于控制体的合外力（N）
- `ρQ` - 质量流量（kg/s）
- `v` - 流速（m/s）

---

## 📝 精选例题（5题）

### 题目1：渐变管流速 ⭐⭐⭐

**题目**：
水管直径从D₁=0.3m逐渐缩小到D₂=0.15m，v₁=2m/s。求v₂和Q。

**解答**：

**连续性方程**
```
Q = A₁v₁ = A₂v₂
v₂ = v₁(A₁/A₂) = v₁(D₁/D₂)² = 2×(0.3/0.15)² = 8 m/s
Q = A₁v₁ = π×0.3²/4 × 2 = 0.141 m³/s = 141 L/s
```

**答案**：v₂ = 8 m/s，Q = 141 L/s

---

### 题目2：弯管水流作用力 ⭐⭐⭐⭐⭐

**题目**：
水平90°弯管，进口d₁=0.2m，出口d₂=0.1m，进口压强p₁=200kPa，流量Q=0.05m³/s。求水流对弯管的作用力（忽略损失）。

**解答**：

**Step 1：求流速**
```
v₁ = Q/A₁ = 0.05/(π×0.2²/4) = 1.59 m/s
v₂ = Q/A₂ = 0.05/(π×0.1²/4) = 6.37 m/s
```

**Step 2：伯努利方程求p₂**
```
p₁/(ρg) + v₁²/(2g) = p₂/(ρg) + v₂²/(2g)
200,000/9800 + 1.59²/19.6 = p₂/9800 + 6.37²/19.6
20.41 + 0.13 = p₂/9800 + 2.07
p₂ = 180,620 Pa = 180.6 kPa
```

**Step 3：动量方程（x方向）**
```
F_x = p₁A₁ - ρQv₁ = 200,000×π×0.04 - 1000×0.05×1.59
F_x = 6,283 - 79.5 = 6,203 N
```

**Step 4：动量方程（y方向）**
```
F_y = -p₂A₂ - ρQ×(-v₂) = -180,600×π×0.01 + 1000×0.05×6.37
F_y = -1,420 + 318.5 = -1,101 N（向下）
```

**Step 5：合力**
```
F = √(F_x² + F_y²) = √(6,203² + 1,101²) = 6,300 N = 6.3 kN
```

**答案**：水流对弯管作用力6.3kN（方向需标明）

### 题目3：喷水推力 ⭐⭐⭐⭐

**题目**：
消防水枪，喷嘴直径d=0.05m，出口流速v=20m/s。求消防员需提供的握持力。

**解答**：

**流量**
```
Q = Av = π×0.05²/4 × 20 = 0.0393 m³/s
```

**动量方程（水平方向）**
```
F = ρQ(v₂ - v₁) = ρQ×v（v₁≈0）
F = 1000×0.0393×20 = 786 N
```

**答案**：握持力F = 786 N ≈ 0.79 kN

---

### 题目4：三通管分流 ⭐⭐⭐⭐

**题目**：
T型三通管，主管d₁=0.2m，支管d₂=d₃=0.1m，Q₁=0.1m³/s，Q₂=0.06m³/s。求v₁，v₂，v₃。

**解答**：

**连续性方程**
```
Q₁ = Q₂ + Q₃
Q₃ = Q₁ - Q₂ = 0.1 - 0.06 = 0.04 m³/s
```

**流速**
```
v₁ = Q₁/A₁ = 0.1/(π×0.2²/4) = 3.18 m/s
v₂ = Q₂/A₂ = 0.06/(π×0.1²/4) = 7.64 m/s
v₃ = Q₃/A₃ = 0.04/(π×0.1²/4) = 5.09 m/s
```

**答案**：v₁=3.18m/s，v₂=7.64m/s，v₃=5.09m/s

### 题目5：喷射泵 ⭐⭐⭐⭐⭐

**题目**：
喷射泵，工作水流速v₁=15m/s，面积A₁=0.01m²；被抽水流速v₂=3m/s，A₂=0.04m²；混合后A₃=0.05m²。求混合后流速v₃。

**解答**：

**连续性方程**
```
Q₁ + Q₂ = Q₃
A₁v₁ + A₂v₂ = A₃v₃
0.01×15 + 0.04×3 = 0.05×v₃
0.15 + 0.12 = 0.05v₃
v₃ = 5.4 m/s
```

**答案**：v₃ = 5.4 m/s

|------|------|
| Q = Av = 常数 | 连续性方程 |
| ΣF = ρQ(v₂-v₁) | 动量方程 |
| F = ρQv | 喷射推力 |
