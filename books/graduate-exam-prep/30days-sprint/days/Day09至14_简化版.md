## Day 11：临界水深与水跃 ⭐⭐⭐⭐⭐

### 核心公式
```
临界水深：Fr = v/√(gh) = 1
h_c = ∛(q²/g)（矩形）
水跃共轭水深：h₂ = h₁/2×(√(1+8Fr₁²) - 1)
```

## Day 14：第二周总结与测试 ⭐⭐⭐⭐⭐

### 核心公式总结

**管流**：
- 达西公式：h_f = λ(L/d)×v²/(2g)
- 长管：忽略流速水头，只考虑损失

**明渠均匀流**：
- 曼宁公式：Q = (1/n)AR^(2/3)i^(1/2)
- 最优断面：R = h/2

**临界流**：
- Fr = 1
- h_c = ∛(q²/g)

**水跃**：
- h₂/h₁ = 0.5×(√(1+8Fr₁²) - 1)
