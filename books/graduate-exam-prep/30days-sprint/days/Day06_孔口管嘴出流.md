## 📚 核心考点

### 考点1：孔口出流 ⭐⭐⭐⭐

#### 基本公式
```
Q = μA√(2gH)
```

其中：
- `μ` - 流量系数，μ = εφ
- `ε` - 收缩系数（0.60-0.64）
- `φ` - 流速系数（0.95-0.99）
- `A` - 孔口面积
- `H` - 作用水头

#### 典型值
- **薄壁孔口**：μ ≈ 0.60-0.62
- **圆角孔口**：μ ≈ 0.95-0.97

---

### 考点2：管嘴出流 ⭐⭐⭐⭐

#### 外管嘴
```
Q = μA√(2gH)
μ ≈ 0.82（比孔口大）
```

#### 内管嘴  
```
Q = μA√(2gH)
μ ≈ 0.71
```

## 📝 精选例题（5题）

### 题目1：薄壁孔口流量 ⭐⭐⭐

**题目**：水箱薄壁孔口d=0.06m，H=2.5m，μ=0.61。求Q。

**解答**：
```
A = πd²/4 = π×0.06²/4 = 0.00283 m²
Q = μA√(2gH) = 0.61×0.00283×√(2×9.8×2.5)
Q = 0.00283×0.61×7.0 = 0.0121 m³/s = 12.1 L/s
```

**答案**：Q = 12.1 L/s

---

### 题目2：孔口收缩断面 ⭐⭐⭐⭐

**题目**：薄壁孔口d=0.05m，H=1.8m，ε=0.62，φ=0.97。求：
1. 收缩断面直径d_c
2. 收缩断面流速v_c
3. 流量Q

**解答**：

**1. 收缩断面直径**
```
A_c = εA = 0.62×π×0.05²/4 = 0.00122 m²
d_c = √(4A_c/π) = √(4×0.00122/π) = 0.0394 m = 39.4 mm
```

**2. 收缩断面流速**
```
v_理论 = √(2gH) = √(2×9.8×1.8) = 5.94 m/s
v_c = φv_理论 = 0.97×5.94 = 5.76 m/s
```

**3. 流量**
```
Q = A_c×v_c = 0.00122×5.76 = 0.00703 m³/s = 7.03 L/s
或：Q = μA√(2gH)，μ=0.62×0.97=0.60
Q = 0.60×π×0.05²/4×5.94 = 7.03 L/s
```

**答案**：d_c=39.4mm，v_c=5.76m/s，Q=7.03L/s

---

### 题目3：外管嘴流量 ⭐⭐⭐⭐

**题目**：圆柱形外管嘴，d=0.08m，长l=0.24m（l=3d），H=3m，μ=0.82。求：
1. 流量Q
2. 与同直径孔口比较

**解答**：

**1. 管嘴流量**
```
Q_管嘴 = μA√(2gH) = 0.82×π×0.08²/4×√(2×9.8×3)
Q_管嘴 = 0.82×0.00503×7.67 = 0.0316 m³/s = 31.6 L/s
```

**2. 同直径孔口流量**
```
Q_孔口 = 0.62×0.00503×7.67 = 0.0239 m³/s = 23.9 L/s
```

**比较**：
```
Q_管嘴/Q_孔口 = 31.6/23.9 = 1.32（增加32%）
```

**答案**：Q_管嘴=31.6L/s，比孔口增加32%

---

### 题目4：孔口淹没出流 ⭐⭐⭐⭐

**题目**：水箱两侧水深H₁=3m，H₂=1m，孔口d=0.05m，μ=0.60。求流量Q（从深侧流向浅侧）。

**解答**：

**作用水头**
```
ΔH = H₁ - H₂ = 3 - 1 = 2 m
```

**流量**
```
Q = μA√(2gΔH) = 0.60×π×0.05²/4×√(2×9.8×2)
Q = 0.60×0.00196×6.26 = 0.00736 m³/s = 7.36 L/s
```

**答案**：Q = 7.36 L/s

---

### 题目5：变水头出流 ⭐⭐⭐⭐⭐

**题目**：圆柱形水箱，直径D=2m，初始水深H₀=2m，底部孔口d=0.1m，μ=0.60。求放空时间T。

**解答**：

**微分方程**
```
-A_箱dH/dt = Q_孔 = μA_孔√(2gH)
```

**分离变量**
```
-A_箱∫dH/√H = μA_孔√(2g)∫dt
-2A_箱√H |₀^H₀ = μA_孔√(2g)×T
2A_箱√H₀ = μA_孔√(2g)×T
```

**求解**
```
A_箱 = π×2²/4 = 3.14 m²
A_孔 = π×0.1²/4 = 0.00785 m²
T = 2A_箱√H₀/(μA_孔√(2g))
T = 2×3.14×√2/(0.60×0.00785×√19.6)
T = 8.88/(0.00471×4.43) = 426 s ≈ 7.1 分钟
```

**答案**：T ≈ 7.1 分钟
