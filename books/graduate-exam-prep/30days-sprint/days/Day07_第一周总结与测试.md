## 📚 核心公式总结（必背！）

### 1. 静水力学

```
【静水压强】 p = p₀ + γh
【平面总压力】 P = γh̄A
【压力中心】 hp = h̄ + Ic/(h̄A)
【浮力】 F_b = γV_排
【定倾中心】 h_M = I_c/V_排 - GC
```

### 2. 水动力学

```
【伯努利方程】 z + p/(ρg) + v²/(2g) = C
【连续性方程】 Q = Av = 常数
【动量方程】 ΣF = ρQ(v₂ - v₁)
【孔口出流】 Q = μA√(2gH)
```

### 3. 重要系数

```
矩形惯性矩：Ic = bh³/12
三角形惯性矩：Ic = bh³/36
圆形惯性矩：Ic = πr⁴/4
动能修正系数：α ≈ 1.05-1.1（紊流）
孔口流量系数：μ ≈ 0.60-0.62
管嘴流量系数：μ ≈ 0.82
```
