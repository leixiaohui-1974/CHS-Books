## 📚 核心考点

### 考点1：静水压强分布规律 ⭐⭐⭐⭐⭐

#### 基本公式
```
p = p₀ + γh
```

其中：
- `p` - 任意点的静水压强（Pa）
- `p₀` - 液面压强（Pa）
- `γ` - 液体容重（N/m³）
- `h` - 该点在液面下的深度（m）

#### 重要性质
1. **方向性**：静水压强垂直作用于受压面，并指向该面
2. **各向等性**：同一点各方向压强相等
3. **传递性**：液面压强变化量等值传递到液体内部各点（帕斯卡原理）

#### 常见错误
- ❌ 忘记加液面压强p₀
- ❌ 深度h计算错误（应从液面算起）
- ❌ 单位不统一（Pa vs kPa）

---

### 考点2：平面总压力计算 ⭐⭐⭐⭐⭐

#### 基本公式
```
P = γh̄A
```

其中：
- `P` - 平面总压力（N）
- `γ` - 液体容重（N/m³）
- `h̄` - 受压面形心深度（m）
- `A` - 受压面面积（m²）

#### 几何性质
- **矩形**：形心在对角线交点
- **三角形**：形心距底边1/3高度处
- **圆形**：形心在圆心
- **梯形**：形心位置需计算

#### 压力作用点计算
```
hp = h̄ + Ic/(h̄A)
```

其中：
- `hp` - 压力中心深度（m）
- `Ic` - 受压面对通过形心且平行于液面的轴的惯性矩（m⁴）
- `h̄` - 形心深度（m）
- `A` - 受压面面积（m²）

#### 常用惯性矩
1. **矩形**（宽b，高h）：
   - `Ic = bh³/12`
   
2. **三角形**（底边b，高h）：
   - `Ic = bh³/36`
   
3. **圆形**（半径r）：
   - `Ic = πr⁴/4`

## 📝 精选例题（5题）

### 题目1：矩形闸门总压力 ⭐⭐⭐

**题目描述**：
一矩形平板闸门，宽度b = 2m，高度h = 3m，顶部距水面深度d = 1m。求：
1. 作用在闸门上的总压力P
2. 压力中心距水面的深度hp

**已知条件**：
- 宽度：b = 2m
- 高度：h = 3m
- 顶部深度：d = 1m
- 水的容重：γ = 9800 N/m³
- 液面为大气压：p₀ = 0（表压）

**解答**：

**第1步：计算受压面面积**
```
A = b × h = 2 × 3 = 6 m²
```

**第2步：计算形心深度**
```
h̄ = d + h/2 = 1 + 3/2 = 2.5 m
```

**第3步：计算总压力**
```
P = γh̄A = 9800 × 2.5 × 6 = 147,000 N = 147 kN
```

**第4步：计算惯性矩**
```
Ic = bh³/12 = 2 × 3³/12 = 4.5 m⁴
```

**第5步：计算压力中心深度**
```
hp = h̄ + Ic/(h̄A) = 2.5 + 4.5/(2.5×6) = 2.5 + 0.3 = 2.8 m
```

**答案**：
1. 总压力 P = 147 kN
2. 压力中心深度 hp = 2.8 m

**踩分点**：
- ✅ 正确计算形心深度（2分）
- ✅ 正确计算总压力（3分）
- ✅ 正确计算惯性矩（2分）
- ✅ 正确计算压力中心（3分）
- ✅ 单位正确（必须）

**常见错误**：
- ❌ 形心深度忘记加d
- ❌ 压力中心公式错误
- ❌ 单位换算错误

### 题目2：倾斜矩形平板总压力 ⭐⭐⭐⭐

**题目描述**：
一倾斜矩形平板闸门，宽度b = 1.5m，沿斜面长度L = 2m，与水平面夹角θ = 60°。顶部距水面深度d = 0.5m。求总压力P和压力中心位置。

**已知条件**：
- 宽度：b = 1.5m
- 斜长：L = 2m
- 倾角：θ = 60°
- 顶部深度：d = 0.5m
- 水的容重：γ = 9800 N/m³

**解答**：

**第1步：计算受压面面积**
```
A = b × L = 1.5 × 2 = 3 m²
```

**第2步：计算形心深度**
```
形心距顶部的斜面距离：L/2 = 1 m
形心垂直深度增量：L/2 × sinθ = 1 × sin60° = 0.866 m
h̄ = d + 0.866 = 1.366 m
```

**第3步：计算总压力**
```
P = γh̄A = 9800 × 1.366 × 3 = 40,160 N ≈ 40.2 kN
```

**第4步：计算压力中心深度**
```
Ic = bL³/12 = 1.5 × 2³/12 = 1.0 m⁴

hp = h̄ + Ic×sinθ/(h̄A)
hp = 1.366 + 1.0×sin60°/(1.366×3)
hp = 1.366 + 0.211 = 1.577 m
```

**答案**：
1. 总压力 P = 40.2 kN
2. 压力中心深度 hp = 1.577 m

**踩分点**：
- ✅ 倾斜面形心深度计算（3分）
- ✅ 总压力计算（2分）
- ✅ 惯性矩计算（2分）
- ✅ 压力中心考虑倾角（3分）

**关键知识点**：
- 倾斜面要把斜长投影到垂直方向
- 压力中心公式要乘以sinθ

### 题目3：三角形闸门总压力 ⭐⭐⭐⭐

**题目描述**：
一三角形平板闸门，底边水平，底边长b = 2m，高h = 1.5m，顶点朝上。底边距水面深度d = 2m。求：
1. 总压力P
2. 压力中心距水面深度hp

**已知条件**：
- 底边长：b = 2m
- 高度：h = 1.5m
- 底边深度：d = 2m
- 水的容重：γ = 9800 N/m³

**解答**：

**第1步：计算受压面面积**
```
A = bh/2 = 2 × 1.5 / 2 = 1.5 m²
```

**第2步：计算形心深度**
```
三角形形心距底边：h/3 = 0.5 m
h̄ = d - h/3 = 2 - 0.5 = 1.5 m
```

**第3步：计算总压力**
```
P = γh̄A = 9800 × 1.5 × 1.5 = 22,050 N = 22.05 kN
```

**第4步：计算惯性矩**
```
Ic = bh³/36 = 2 × 1.5³/36 = 0.1875 m⁴
```

**第5步：计算压力中心深度**
```
hp = h̄ + Ic/(h̄A) = 1.5 + 0.1875/(1.5×1.5) = 1.5 + 0.0833 = 1.583 m
```

**答案**：
1. 总压力 P = 22.05 kN
2. 压力中心深度 hp = 1.583 m

**踩分点**：
- ✅ 三角形面积公式（1分）
- ✅ 形心位置（底边上1/3高度）（3分）
- ✅ 总压力计算（2分）
- ✅ 三角形惯性矩（3分）
- ✅ 压力中心计算（1分）

**易错点**：
- ❌ 形心距底边h/3容易记错
- ❌ 惯性矩公式bh³/36容易忘记

### 题目4：多层液体压强 ⭐⭐⭐⭐⭐

**题目描述**：
一水箱内分两层液体：上层为密度ρ₁ = 800 kg/m³的油，厚度h₁ = 1m；下层为水，厚度h₂ = 2m。求：
1. 油水交界面的压强
2. 水箱底部的压强

**已知条件**：
- 油的密度：ρ₁ = 800 kg/m³
- 油层厚度：h₁ = 1m
- 水的密度：ρ₂ = 1000 kg/m³
- 水层厚度：h₂ = 2m
- 重力加速度：g = 9.8 m/s²
- 液面为大气压：p₀ = 101.325 kPa

**解答**：

**第1步：计算油和水的容重**
```
γ₁ = ρ₁g = 800 × 9.8 = 7,840 N/m³
γ₂ = ρ₂g = 1000 × 9.8 = 9,800 N/m³
```

**第2步：计算油水交界面压强（绝对压）**
```
p_interface = p₀ + γ₁h₁
p_interface = 101,325 + 7,840 × 1
p_interface = 101,325 + 7,840 = 109,165 Pa = 109.165 kPa
```

或者用表压（相对大气压）：
```
p_interface(表压) = γ₁h₁ = 7,840 Pa = 7.84 kPa
```

**第3步：计算水箱底部压强（绝对压）**
```
p_bottom = p₀ + γ₁h₁ + γ₂h₂
p_bottom = 101,325 + 7,840 + 9,800 × 2
p_bottom = 101,325 + 7,840 + 19,600 = 128,765 Pa = 128.765 kPa
```

或者用表压：
```
p_bottom(表压) = γ₁h₁ + γ₂h₂ = 7,840 + 19,600 = 27,440 Pa = 27.44 kPa
```

**答案**：
1. 油水交界面压强：p = 109.165 kPa（绝对压）或 7.84 kPa（表压）
2. 水箱底部压强：p = 128.765 kPa（绝对压）或 27.44 kPa（表压）

**踩分点**：
- ✅ 分别计算两种液体容重（2分）
- ✅ 逐层累加压强（3分）
- ✅ 区分绝对压与表压（2分）
- ✅ 单位正确（3分）

**关键知识点**：
- 多层液体压强逐层累加
- 明确题目要求绝对压还是表压
- 注意单位换算（Pa ↔ kPa）

### 题目5：U型测压管 ⭐⭐⭐⭐

**题目描述**：
一U型测压管，左侧连接水管（A点），右侧开口通大气。测压管内装水银（密度ρ_Hg = 13,600 kg/m³）。已知：A点到水银面高度差h₁ = 0.5m（A点高），水银左右液面高差h₂ = 0.2m（右侧高）。求A点的压强（表压）。

**已知条件**：
- 水的密度：ρ_w = 1000 kg/m³
- 水银密度：ρ_Hg = 13,600 kg/m³
- A点到水银面高度：h₁ = 0.5m
- 水银液面高差：h₂ = 0.2m
- 重力加速度：g = 9.8 m/s²

**解答**：

**第1步：绘制压强关系图**
```
A点 → 向下h₁（水） → 水银左液面 → 同一水平面 → 水银右液面 → 向上h₂（水银） → 大气
```

**第2步：建立压强方程**
```
根据同一液体同一水平面压强相等原理：
p_A + ρ_w·g·h₁ = p_atm + ρ_Hg·g·h₂
```

**第3步：求解A点表压**
```
p_A(表压) = p_A - p_atm
p_A - p_atm = ρ_Hg·g·h₂ - ρ_w·g·h₁
p_A(表压) = 13,600 × 9.8 × 0.2 - 1000 × 9.8 × 0.5
p_A(表压) = 26,656 - 4,900 = 21,756 Pa ≈ 21.76 kPa
```

**答案**：
A点压强（表压）= 21.76 kPa

**踩分点**：
- ✅ 正确建立压强关系（3分）
- ✅ 同一液体同一水平面压强相等（2分）
- ✅ 正确计算容重（2分）
- ✅ 正确求解表压（3分）

**解题技巧**：
1. 画出示意图
2. 从已知点出发，逐步建立压强关系
3. 利用"同一液体同一水平面压强相等"
4. 注意正负号（向下为正，向上为负）

## 💻 Python代码示例

### 代码功能：矩形闸门总压力与压力中心计算

```python
import numpy as np
import matplotlib.pyplot as plt

# 设置中文字体
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

def rectangular_gate_pressure(b, h, d, gamma=9800):
    """
    计算矩形闸门总压力和压力中心
    
    参数:
        b: 宽度 (m)
        h: 高度 (m)
        d: 顶部距水面深度 (m)
        gamma: 液体容重 (N/m³)
    
    返回:
        P: 总压力 (N)
        hp: 压力中心深度 (m)
    """
    # 受压面面积
    A = b * h
    
    # 形心深度
    h_bar = d + h/2
    
    # 总压力
    P = gamma * h_bar * A
    
    # 惯性矩
    Ic = b * h**3 / 12
    
    # 压力中心深度
    hp = h_bar + Ic / (h_bar * A)
    
    return P, hp, h_bar

# 示例：计算题目1
b = 2.0    # 宽度 (m)
h = 3.0    # 高度 (m)
d = 1.0    # 顶部深度 (m)
gamma = 9800  # 水的容重 (N/m³)

P, hp, h_bar = rectangular_gate_pressure(b, h, d, gamma)

print("="*50)
print("矩形闸门总压力计算")
print("="*50)
print(f"闸门宽度 b = {b} m")
print(f"闸门高度 h = {h} m")
print(f"顶部深度 d = {d} m")
print(f"液体容重 γ = {gamma} N/m³")
print("-"*50)
print(f"形心深度 h̄ = {h_bar} m")
print(f"总压力 P = {P/1000:.2f} kN")
print(f"压力中心深度 hp = {hp:.3f} m")
print(f"压力中心距形心距离 = {hp-h_bar:.3f} m")
print("="*50)

# 可视化
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 6))

# 左图：压强分布
depth = np.linspace(d, d+h, 100)
pressure = gamma * depth / 1000  # kPa

ax1.plot(pressure, depth, 'b-', linewidth=2, label='压强分布')
ax1.axhline(y=h_bar, color='r', linestyle='--', linewidth=1, label=f'形心深度 h̄={h_bar}m')
ax1.axhline(y=hp, color='g', linestyle='--', linewidth=1, label=f'压力中心 hp={hp:.2f}m')
ax1.fill_betweenx(depth, 0, pressure, alpha=0.3)
ax1.set_xlabel('压强 (kPa)', fontsize=12)
ax1.set_ylabel('深度 (m)', fontsize=12)
ax1.set_title('静水压强分布', fontsize=14, fontweight='bold')
ax1.invert_yaxis()  # Y轴反向，深度向下
ax1.grid(True, alpha=0.3)
ax1.legend(fontsize=10)

# 右图：闸门示意图
ax2.add_patch(plt.Rectangle((0, d), 0.5, h, 
                            fill=False, edgecolor='blue', linewidth=2))
ax2.plot([0.25], [h_bar], 'ro', markersize=10, label=f'形心 h̄={h_bar}m')
ax2.plot([0.25], [hp], 'g^', markersize=10, label=f'压力中心 hp={hp:.2f}m')

# 画总压力箭头
ax2.arrow(0.7, hp, 0.3, 0, head_width=0.2, head_length=0.1, 
         fc='red', ec='red', linewidth=2)
ax2.text(1.1, hp, f'P={P/1000:.1f} kN', fontsize=12, va='center')

# 标注
ax2.axhline(y=0, color='cyan', linestyle='-', linewidth=3, label='水面')
ax2.text(-0.2, 0, '水面', fontsize=10, ha='right')
ax2.text(-0.2, d, f'd={d}m', fontsize=10, ha='right')
ax2.text(0.6, d+h/2, f'h={h}m', fontsize=10, ha='left')
ax2.text(0.25, d+h+0.3, f'b={b}m', fontsize=10, ha='center')

ax2.set_xlim(-0.5, 1.5)
ax2.set_ylim(-0.5, d+h+0.5)
ax2.set_xlabel('水平距离 (m)', fontsize=12)
ax2.set_ylabel('深度 (m)', fontsize=12)
ax2.set_title('矩形闸门示意图', fontsize=14, fontweight='bold')
ax2.invert_yaxis()
ax2.grid(True, alpha=0.3)
ax2.legend(fontsize=10)
ax2.set_aspect('equal')

plt.tight_layout()
plt.savefig('day01_rectangular_gate.png', dpi=300, bbox_inches='tight')
plt.show()

print("\n可视化图表已保存为 'day01_rectangular_gate.png'")
```

**代码说明**：
1. 定义了矩形闸门压力计算函数
2. 计算总压力和压力中心
3. 可视化压强分布和闸门示意图

## 📊 今日学习总结

### 核心公式速记

| 公式名称 | 公式 | 说明 |
|---------|------|------|
| 静水压强 | p = p₀ + γh | h为深度 |
| 平面总压力 | P = γh̄A | h̄为形心深度 |
| 压力中心 | hp = h̄ + Ic/(h̄A) | Ic为惯性矩 |
| 矩形惯性矩 | Ic = bh³/12 | - |
| 三角形惯性矩 | Ic = bh³/36 | - |

### 重点知识回顾

✅ **静水压强三大特性**：
1. 垂直作用于受压面
2. 同一点各方向相等
3. 随深度线性增加

✅ **平面总压力计算**：
1. 计算形心深度
2. 用h̄计算总压力
3. 用惯性矩计算压力中心

✅ **常见图形惯性矩**：
- 矩形：bh³/12
- 三角形：bh³/36
- 圆形：πr⁴/4

### 易错点提示

❌ **常见错误**：
1. 形心深度计算错误（忘记加顶部深度d）
2. 惯性矩公式记混（矩形12 vs 三角形36）
3. 压力中心公式漏项
4. 单位换算错误（N ↔ kN，Pa ↔ kPa）

✅ **避免方法**：
1. 画示意图，标清各个深度
2. 记住惯性矩公式口诀："矩形十二，三角三六"
3. 压力中心必定在形心下方
4. 统一单位后再计算
