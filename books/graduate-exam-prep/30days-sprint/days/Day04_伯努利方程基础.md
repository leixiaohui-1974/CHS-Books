## 📚 核心考点

### 伯努利方程 ⭐⭐⭐⭐⭐

#### 理想液体伯努利方程
```
z₁ + p₁/(ρg) + v₁²/(2g) = z₂ + p₂/(ρg) + v₂²/(2g)
```

#### 实际液体伯努利方程
```
z₁ + p₁/(ρg) + α₁v₁²/(2g) = z₂ + p₂/(ρg) + α₂v₂²/(2g) + h_w
```

其中：
- `z` - 位置水头（m）
- `p/(ρg)` - 压强水头（m）
- `v²/(2g)` - 流速水头（m）
- `α` - 动能修正系数（层流α=2，紊流α≈1.05-1.1）
- `h_w` - 水头损失（m）

#### 适用条件
1. 同一流线或同一微小流束
2. 不可压缩流体
3. 定常流动
4. 质量力只有重力

## 📝 精选例题（5题）

### 题目1：虹吸管 ⭐⭐⭐⭐⭐

**题目**：
虹吸管从水库A引水至B，管径d=0.1m，最高点C距A水面h=3m，A-B高差H=5m。忽略损失，求：
1. 流量Q
2. C点压强p_C

**解答**：

**1. 流量（A→B伯努利方程）**
```
取A水面为基准：z_A = 0
p_A = p_B = 0（表压）
v_A ≈ 0
v_B = √(2gH) = √(2×9.8×5) = 9.9 m/s
Q = Av_B = π×0.1²/4 × 9.9 = 0.0778 m³/s = 77.8 L/s
```

**2. C点压强（A→C）**
```
0 + 0 + 0 = 3 + p_C/(ρg) + v_C²/(2g)
v_C = v_B = 9.9 m/s（连续性）
p_C/(ρg) = -3 - 9.9²/(2×9.8) = -3 - 5 = -8 m
p_C = -8×9800 = -78,400 Pa = -78.4 kPa（负压）
```

**答案**：Q = 77.8 L/s，p_C = -78.4 kPa

### 题目2：文丘里流量计 ⭐⭐⭐⭐

**题目**：
文丘里流量计，管径D₁=0.2m，喉部D₂=0.1m，水银测压计读数Δh=0.15m。求流量Q（忽略损失）。

**解答**：

**伯努利方程（1-2断面，水平）**
```
p₁/(ρg) + v₁²/(2g) = p₂/(ρg) + v₂²/(2g)
```

**连续性方程**
```
v₁A₁ = v₂A₂
v₂ = v₁(D₁/D₂)² = v₁×4 = 4v₁
```

**测压计关系**
```
p₁/(ρ_w g) - p₂/(ρ_w g) = Δh×(ρ_Hg/ρ_w - 1)
= 0.15×(13.6-1) = 1.89 m
```

**求解**
```
1.89 = (v₂² - v₁²)/(2g) = (16v₁² - v₁²)/(2×9.8)
1.89 = 15v₁²/19.6
v₁ = 1.24 m/s
Q = A₁v₁ = π×0.2²/4 × 1.24 = 0.039 m³/s = 39 L/s
```

**答案**：Q = 39 L/s

### 题目3：水泵扬程 ⭐⭐⭐⭐⭐

**题目**：
水泵从低水池抽水至高水池，高差H=20m，吸水管d₁=0.15m，压水管d₂=0.1m，流量Q=0.03m³/s，总损失h_w=3m。求水泵扬程H_p。

**解答**：

**伯努利方程（低池→高池，加水泵）**
```
z₁ + p₁/(ρg) + v₁²/(2g) + H_p = z₂ + p₂/(ρg) + v₂²/(2g) + h_w
```

**简化（两水池液面，v≈0，p=0）**
```
0 + 0 + 0 + H_p = H + 0 + 0 + h_w
H_p = H + h_w = 20 + 3 = 23 m
```

**答案**：H_p = 23 m

### 题目4：管道出流 ⭐⭐⭐⭐

**题目**：
水箱水深h=2m，底部接管道L=10m，d=0.05m，沿程损失系数λ=0.03，局部损失可忽略。求流量Q。

**解答**：

**伯努利方程（水面→出口）**
```
h + 0 + 0 = 0 + 0 + v²/(2g) + h_w
h_w = λ(L/d)×v²/(2g) = 0.03×(10/0.05)×v²/(2g) = 6v²/(2g)
```

**求解**
```
h = v²/(2g) + 6v²/(2g) = 7v²/(2g)
v = √(2gh/7) = √(2×9.8×2/7) = 2.37 m/s
Q = Av = π×0.05²/4 × 2.37 = 0.00466 m³/s = 4.66 L/s
```

**答案**：Q = 4.66 L/s

### 题目5：孔口出流 ⭐⭐⭐

**题目**：
薄壁孔口，直径d=0.05m，水头H=1.5m，收缩系数ε=0.62，流速系数φ=0.97。求流量Q。

**解答**：

**理论流速**
```
v_理论 = √(2gH) = √(2×9.8×1.5) = 5.42 m/s
```

**实际流速**
```
v = φv_理论 = 0.97×5.42 = 5.26 m/s
```

**实际流量**
```
A_收缩 = εA = 0.62×π×0.05²/4 = 0.00122 m²
Q = A_收缩×v = 0.00122×5.26 = 0.00642 m³/s = 6.42 L/s
```

或直接：
```
Q = μA√(2gH)
其中μ = εφ = 0.62×0.97 = 0.60
Q = 0.60×π×0.05²/4×√(2×9.8×1.5) = 6.42 L/s
```

**答案**：Q = 6.42 L/s
