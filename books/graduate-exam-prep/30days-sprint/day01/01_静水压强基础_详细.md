## 一、静水压强基本概念

### 1.1 定义

**静水压强**：单位面积上的静水压力
$$p = \lim_{\Delta A \to 0} \frac{\Delta P}{\Delta A}$$

单位：Pa (N/m²)

### 1.2 性质

1. **方向**：垂直指向作用面
2. **大小**：与方向无关（各向等值）
3. **分布**：随深度线性增加

---

## 二、静水压强分布

### 2.1 基本公式

$$p = p_0 + \gamma h$$

- $p_0$：液面压强
- $\gamma$：容重（水：9810 N/m³）
- $h$：深度

### 2.2 等压面

**定义**：压强相等的面

**性质**：
- 静止液体中的等压面为水平面
- 连通容器等压面相同

## 三、压强计算

### Python实现

```python
import numpy as np
import matplotlib.pyplot as plt

class HydrostaticPressure:
    """静水压强计算"""
    
    def __init__(self, gamma=9810):
        self.gamma = gamma  # N/m³
    
    def pressure(self, h, p0=101325):
        """
        压强计算: p = p0 + γh
        
        参数:
            h: 深度 (m)
            p0: 液面压强 (Pa)
        """
        return p0 + self.gamma * h
    
    def pressure_head(self, p, p0=101325):
        """压强水头: h = (p - p0) / γ"""
        return (p - p0) / self.gamma
    
    def plot_pressure_distribution(self, H=10):
        """绘制压强分布"""
        h = np.linspace(0, H, 100)
        p = self.pressure(h, p0=0)  # 表压
        
        fig, axes = plt.subplots(1, 2, figsize=(12, 6))
        
        # 压强分布
        axes[0].plot(p/1000, h, 'b-', linewidth=2)
        axes[0].invert_yaxis()
        axes[0].set_xlabel('压强 (kPa)')
        axes[0].set_ylabel('深度 (m)')
        axes[0].set_title('静水压强分布')
        axes[0].grid(True, alpha=0.3)
        
        # 压强图示
        axes[1].barh(h[::10], p[::10]/1000, height=H/10, alpha=0.6)
        axes[1].invert_yaxis()
        axes[1].set_xlabel('压强 (kPa)')
        axes[1].set_ylabel('深度 (m)')
        axes[1].set_title('压强柱状图')
        axes[1].grid(True, alpha=0.3)
        
        plt.tight_layout()
        plt.savefig('pressure_distribution.png', dpi=300)
        plt.show()

# 示例
calc = HydrostaticPressure()

# 水深5m处的压强
h = 5
p = calc.pressure(h, p0=0)
print(f"水深{h}m处压强: {p/1000:.1f} kPa")

# 可视化
calc.plot_pressure_distribution(H=10)
```

## 四、典型考题

### 【例题1】压强计算

**题目**：水深10m，求底部压强（表压）

**解**：
$$p = \gamma h = 9810 \times 10 = 98100 \text{ Pa} = 98.1 \text{ kPa}$$

### 【例题2】U型管压差计

**题目**：U型管压差计，测压液为水银（γ=133400 N/m³），高差Δh=0.2m，求压差

**解**：
$$\Delta p = \gamma_{Hg} \Delta h = 133400 \times 0.2 = 26680 \text{ Pa} = 26.68 \text{ kPa}$$
