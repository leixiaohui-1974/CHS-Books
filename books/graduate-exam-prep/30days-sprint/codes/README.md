1. 阅读当天主题（30分钟）
2. 运行Python代码（30分钟）

**下午（60分钟）**
1. 手工计算例题1（20分钟）
2. 手工计算例题2（20分钟）
3. 对比代码结果（20分钟）

**晚上（30分钟）**
1. 总结笔记（20分钟）
2. 背诵公式（10分钟）

✅ **DO（推荐）**：
- ✓ 按天数顺序学习

### Q1：代码运行报错怎么办？
**A**：检查Python版本和依赖包：
```bash
python --version  # 需要>=3.8
pip list | grep numpy
pip list | grep matplotlib
```

### Q2：图表中文显示方框？
**A**：安装中文字体：
```bash
# Mac
# Linux: sudo apt install fonts-wqy-microhei
```
