## 静水力学（15个）

### 1. 静水压强
```
p = p₀ + γh
```
- p: 压强(Pa)
- p₀: 液面压强(Pa)
- γ: 容重(N/m³)，水γ=9800
- h: 深度(m)

### 2. 平面总压力
```
P = γh̄A
```
- h̄: 形心深度(m)
- A: 受压面积(m²)

### 3. 压力中心深度
```
hp = h̄ + Ic/(h̄A)
```
- Ic: 惯性矩(m⁴)

### 4. 矩形惯性矩
```
Ic = bh³/12
```

### 5. 三角形惯性矩
```
Ic = bh³/36
```

### 6. 圆形惯性矩
```
Ic = πr⁴/4
```

### 7. 浮力
```
F_b = γV_排 = ρgV_排
```

### 8. 定倾中心高度
```
h_M = I_c/V_排 - GC
```
- h_M > 0: 稳定
- h_M < 0: 不稳定

### 9. 曲面水平分力
```
P_x = γh̄A_x
```
- A_x: 垂直投影面积

### 10. 曲面垂直分力
```
P_z = γV
```
- V: 压力体体积

### 11. 曲面总压力
```
P = √(P_x² + P_z²)
```

### 12. 突然扩大阻力系数
```
ζ = (1 - A₁/A₂)²
```

### 13. 闸门启闭力
```
P × e = T × l
```
- e: 压力作用点到铰链距离
- l: 启闭力到铰链距离

### 14. 多层液体压强
```
p = p₀ + Σ(γᵢhᵢ)
```

### 15. U型测压管
```
p_A = p_atm + ρ_2gh_2 - ρ_1gh_1
```

## 水动力学（20个）

### 1. 伯努利方程（理想）
```
z₁ + p₁/(ρg) + v₁²/(2g) = z₂ + p₂/(ρg) + v₂²/(2g)
```

### 2. 伯努利方程（实际）
```
z₁ + p₁/(ρg) + α₁v₁²/(2g) = z₂ + p₂/(ρg) + α₂v₂²/(2g) + h_w
```
- α: 动能修正系数（紊流≈1.05-1.1）

### 3. 连续性方程
```
Q = A₁v₁ = A₂v₂ = 常数
```

### 4. 动量方程
```
ΣF = ρQ(v₂ - v₁)
```

### 5. 水泵扬程
```
H_p = (z₂-z₁) + (p₂-p₁)/(ρg) + h_w + (v₂²-v₁²)/(2g)
```
简化：
```
H_p = H_几何 + h_w
```

### 6. 虹吸管最高点压强
```
p_C = p_atm - γh - ρv²/2
```

### 7. 文丘里流量计
```
Q = CA₁A₂√(2gΔh/(A₁²-A₂²))
```

### 8. 孔口出流流速
```
v = φ√(2gH)
```
- φ: 流速系数(≈0.95-0.99)

### 9. 孔口出流流量
```
Q = μA√(2gH)
```
- μ = εφ (≈0.60-0.62)
- ε: 收缩系数

### 10. 管嘴出流
```
Q = μA√(2gH)
```
- μ ≈ 0.82（外管嘴）

### 11. 喷射推力
```
F = ρQv
```

### 12. 弯管水流作用力（x方向）
```
F_x = p₁A₁ - p₂A₂cosθ - ρQ(v₂cosθ - v₁)
```

### 13. 能量线斜率
```
J = h_w/L = λv²/(2gd)
```

### 14. 水力坡度
```
I = J（均匀流）
```

### 15. 总水头
```
H = z + p/(ρg) + v²/(2g)
```

### 16. 测压管水头
```
H_p = z + p/(ρg)
```

### 17. 流速水头
```
h_v = v²/(2g)
```

### 18. 位置水头
```
h_z = z
```

### 19. 压强水头
```
h_p = p/(ρg)
```

### 20. 变水头出流时间
```
T = 2A_容器√H₀/(μA_孔口√(2g))
```

## 管流（15个）

### 1. 雷诺数
```
Re = vd/ν = ρvd/μ
```
- ν: 运动粘滞系数(m²/s)
- 水(20℃): ν = 1.0×10⁻⁶ m²/s

### 2. 流态判别
```
Re < 2000: 层流
2000 < Re < 4000: 过渡流
Re > 4000: 紊流
```

### 3. 层流摩阻系数
```
λ = 64/Re
```

### 4. 紊流摩阻系数（Blasius）
```
λ = 0.3164/Re^0.25  (Re<10⁵)
```

### 5. 达西公式（沿程损失）
```
h_f = λ(L/d) × v²/(2g)
```

### 6. 局部损失
```
h_j = ζ × v²/(2g)
```

### 7. 总水头损失
```
h_w = h_f + Σh_j
```

### 8. 长管流量
```
Q = A√(2gH/[λL/d + Σζ])
```

### 9. 串联管道等效长度
```
L_e = Σ(λᵢLᵢ/dᵢ⁵)
```

### 10. 并联管道流量分配
```
Q₁/Q₂ = (d₁/d₂)^(5/2) × (L₂/L₁)^(1/2)
```

### 11. Hardy-Cross修正流量
```
ΔQ = -Σ(rQ|Q|)/[Σ(2r|Q|)]
```
- r = λL/(2gd⁵A²)

### 12. 经济管径
```
d_经济 = ∛(4Q/πv_经济)
```
- v_经济 ≈ 1-2 m/s

### 13. 水锤压强
```
Δp = ρcΔv
```
- c: 波速(m/s)

### 14. 管道阻力特性曲线
```
H = H_0 + SQ²
```
- S: 阻力系数

### 15. 管网节点流量平衡
```
ΣQ_in = ΣQ_out
```

## 明渠流（20个）

### 1. 曼宁公式
```
v = (1/n)R^(2/3)i^(1/2)
```
或
```
Q = (1/n)AR^(2/3)i^(1/2)
```
- n: 糙率(水泥≈0.014，土渠≈0.025)
- R: 水力半径 R=A/χ
- i: 水力坡度

### 2. 谢才公式
```
v = C√(RI)
```
- C: 谢才系数

### 3. 水力半径
```
R = A/χ
```
- χ: 湿周

### 4. 矩形断面（b×h）
```
A = bh
χ = b + 2h
R = bh/(b+2h)
```

### 5. 梯形断面
```
A = (b + mh)h
χ = b + 2h√(1+m²)
```
- m: 边坡系数

### 6. 最优断面（矩形）
```
R = h/2
b = 2h
```

### 7. 最优断面（梯形）
```
R = h/2
b = 2h(√(1+m²) - m)
```

### 8. Froude数
```
Fr = v/√(gh̄)
```
- h̄: 平均水深

### 9. 临界水深（矩形）
```
h_c = ∛(q²/g)
```
- q = Q/b: 单宽流量

### 10. 临界流判别
```
Fr = 1: 临界流
Fr < 1: 缓流
Fr > 1: 急流
```

### 11. 临界坡度
```
i_c = n²q²/(h_c^(10/3))
```

### 12. 水跃共轭水深（矩形）
```
h₂ = (h₁/2) × (√(1+8Fr₁²) - 1)
```

### 13. 水跃长度
```
L_j ≈ (6-7)h₂
```

### 14. 水跃消能率
```
η = (E₁-E₂)/E₁ × 100%
```
- E: 断面比能

### 15. 断面比能
```
E = h + v²/(2g) = h + q²/(2gh²)
```

### 16. 宽顶堰流量
```
Q = mbH^(3/2)√(2g)
```
- m: 流量系数(≈0.32-0.42)

### 17. 薄壁堰流量
```
Q = m₀(2/3)√(2g) × bH^(3/2)
```
- m₀ ≈ 0.40-0.45

### 18. 闸孔自由出流
```
Q = ε₀bh₀√(2gH₀)
```
- ε₀ ≈ 0.60

### 19. 闸孔淹没出流
```
Q = ε_s be√(2g(H₁-H₂))
```
- ε_s: 淹没系数

### 20. 水面线微分方程
```
dh/dx = (i - i₀)/(1 - Fr²)
```
- i: 底坡
- i₀: 摩阻坡度

## 渗流与其他（30个）

### 1. Darcy定律
```
v = kI
```
- k: 渗透系数(m/s)
- I: 水力坡度

### 2. 渗流量
```
Q = kIA
```

### 3. 承压含水层单井流量
```
Q = 2πkm(H-h)/ln(R/r)
```
- m: 承压水厚度

### 4. 潜水层单井流量
```
Q = πk(H²-h²)/ln(R/r)
```

### 5. 干扰井流
```
s = s₁ + s₂
```
- s: 降深

### 6. 渗透系数测定（抽水试验）
```
k = Q×ln(r₂/r₁)/[πm(h₂-h₁)]
```

### 7. 水泵扬程
```
H = (z₂-z₁) + h_w
```

### 8. 水泵功率
```
N = ρgQH/η
```
- η: 效率

### 9. 水泵相似定律（流量）
```
Q₁/Q₂ = (n₁/n₂)(D₁/D₂)³
```

### 10. 水泵相似定律（扬程）
```
H₁/H₂ = (n₁/n₂)²(D₁/D₂)²
```

### 11. 水泵相似定律（功率）
```
N₁/N₂ = (n₁/n₂)³(D₁/D₂)⁵
```

### 12. 比转速
```
n_s = nQ^(1/2)/H^(3/4)
```

### 13. 泥沙沉降速度（Stokes）
```
ω = gd²(ρ_s-ρ)/(18μ)  (d<0.1mm)
```

### 14. 泥沙起动流速
```
v_c = f(d, ρ_s, h)
```

### 15. 泥沙输沙率
```
Q_s = K × v^m × h^n
```

### 16. 溢洪道泄流能力
```
Q = mbH^(3/2)√(2g)
```

### 17. 水跃消能
```
ΔE = (h₂-h₁)³/(4h₁h₂)
```

### 18. 挑流冲刷坑深度
```
t = K × q^(0.5) × H^(0.25)
```

### 19. 掺气减蚀起始点
```
x = 0.3Re^(0.75)δ
```

### 20. 闸门启闭力
```
T = P×e/l
```
