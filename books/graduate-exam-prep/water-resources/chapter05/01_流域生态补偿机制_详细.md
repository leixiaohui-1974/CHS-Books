## 一、生态补偿基本概念

### 1.1 定义与内涵

**生态补偿（Ecological Compensation）**：为保护和可持续利用生态系统服务功能，通过经济手段调节相关者利益关系的制度安排。

**核心要素**：
- **补偿主体**：受益者、政府、社会
- **补偿客体**：生态保护者、资源所有者
- **补偿标准**：补偿金额的确定依据
- **补偿方式**：资金、实物、政策、技术

### 1.2 理论基础

**外部性理论**：生态保护产生正外部性，应由受益者支付成本

**公共物品理论**：生态服务具有非排他性和非竞争性

**产权理论**：明晰生态资源产权，实现市场化补偿

---

## 二、流域生态补偿模式

### 2.1 纵向补偿（政府主导）

**中央-地方补偿**：
- 中央财政向重点生态功能区转移支付
- 例：新安江流域生态补偿试点（浙皖两省）

**上下级政府补偿**：
- 省级政府对市县级的生态补偿
- 例：广东省对东江源区的补偿

### 2.2 横向补偿（市场化）

**上下游补偿**：
- 下游受益地区向上游保护地区支付补偿
- 例：密云水库流域（河北-北京）

**跨区域补偿**：
- 不同行政区域间的协商补偿
- 例：东江流域（江西-广东）

### 2.3 市场化补偿

**排污权交易**：总量控制下的排放权买卖

**水权交易**：节水地区向用水地区出售水权

**生态服务付费（PES）**：用户直接为生态服务付费

## 三、补偿标准测算

### 3.1 机会成本法

**基本思路**：补偿生态保护区放弃发展机会的经济损失

$$EC_{机会成本} = \sum_{i=1}^{n} (R_i^{不保护} - R_i^{保护})$$

**参数说明**：
- $R_i^{不保护}$：不实施保护情景下第$i$项收益
- $R_i^{保护}$：实施保护后的收益
- $n$：受影响的产业数量

**常见损失**：
- 农业收入减少（退耕还林）
- 工业发展受限（禁止高污染产业）
- 旅游开发受限

### 3.2 生态服务价值法

**基本思路**：基于生态系统服务功能价值确定补偿

$$EC_{生态服务} = \sum_{j=1}^{m} V_j \times A_j$$

**参数说明**：
- $V_j$：第$j$类生态服务单位面积价值（元/ha/年）
- $A_j$：第$j$类生态系统面积（ha）
- $m$：生态系统类型数

**常见服务**：
- 水源涵养
- 水质净化
- 洪水调蓄
- 生物多样性保护
- 固碳释氧

### 3.3 恢复成本法

**基本思路**：以恢复退化生态系统的成本作为补偿标准

$$EC_{恢复成本} = C_{治理} + C_{维护}$$

**适用场景**：生态破坏已经发生，需要修复

### 3.4 支付意愿法（WTP）

**基本思路**：通过调查受益者的支付意愿确定补偿

$$EC_{WTP} = \sum_{k=1}^{N} WTP_k$$

**方法**：
- 条件价值评估法（CVM）
- 选择实验法（CE）

---

## 四、补偿资金分配

### 4.1 按贡献分配

**水质改善贡献法**：

$$S_i = EC \times \frac{Q_i \times \Delta C_i}{\sum_{j=1}^{n} Q_j \times \Delta C_j}$$

**参数说明**：
- $S_i$：第$i$个区域获得的补偿（元）
- $EC$：总补偿资金（元）
- $Q_i$：第$i$个区域出境流量（m³）
- $\Delta C_i$：第$i$个区域污染物削减量（mg/L）

### 4.2 按需求分配

**基于保护成本**：
$$S_i = EC \times \frac{C_i}{\sum_{j=1}^{n} C_j}$$

- $C_i$：第$i$个区域的保护成本

### 4.3 按面积分配

**生态功能区面积权重**：
$$S_i = EC \times \frac{A_i \times W_i}{\sum_{j=1}^{n} A_j \times W_j}$$

- $A_i$：第$i$个区域生态功能区面积（km²）
- $W_i$：权重系数（根据重要性）

import numpy as np
import matplotlib.pyplot as plt
import pandas as pd

class EcologicalCompensation:
    """流域生态补偿机制"""
    
    @staticmethod
    def opportunity_cost(income_baseline, income_protected, duration_years):
        """
        机会成本法
        
        参数:
            income_baseline: 基准情景年收入 (元/年)
            income_protected: 保护情景年收入 (元/年)
            duration_years: 补偿年限
        
        返回:
            total_compensation: 总补偿金额 (元)
            annual_compensation: 年补偿金额 (元/年)
        """
        annual_loss = income_baseline - income_protected
        annual_compensation = annual_loss
        total_compensation = annual_compensation * duration_years
        
        return total_compensation, annual_compensation
    
    @staticmethod
    def ecosystem_service_value(ecosystem_data):
        """
        生态服务价值法
        
        参数:
            ecosystem_data: 列表，每项为(ecosystem_type, area_ha, value_per_ha)
        
        返回:
            total_value: 总生态服务价值 (元/年)
            breakdown: 各类生态系统贡献
        """
        breakdown = {}
        total_value = 0
        
        for eco_type, area, value in ecosystem_data:
            value_total = area * value
            breakdown[eco_type] = value_total
            total_value += value_total
        
        return total_value, breakdown
    
    @staticmethod
    def wtp_survey(wtp_data, population):
        """
        支付意愿法
        
        参数:
            wtp_data: 不同群体的支付意愿 [(group_size, wtp_per_capita)]
            population: 总人口
        
        返回:
            average_wtp: 平均支付意愿 (元/人/年)
            total_wtp: 总支付意愿 (元/年)
        """
        total_wtp_sample = sum(size * wtp for size, wtp in wtp_data)
        total_sample_size = sum(size for size, wtp in wtp_data)
        
        average_wtp = total_wtp_sample / total_sample_size
        total_wtp = average_wtp * population
        
        return average_wtp, total_wtp
    
    @staticmethod
    def distribute_by_contribution(total_fund, regions_data):
        """
        按贡献分配补偿资金
        
        参数:
            total_fund: 总补偿资金 (元)
            regions_data: [(region_name, outflow_m3, pollutant_reduction_mg_L)]
        
        返回:
            distribution: 各区域补偿金额 (字典)
        """
        # 计算各区域贡献
        contributions = []
        for name, Q, delta_C in regions_data:
            contribution = Q * delta_C
            contributions.append((name, contribution))
        
        total_contribution = sum(c for _, c in contributions)
        
        # 按贡献比例分配
        distribution = {}
        for name, contribution in contributions:
            if total_contribution > 0:
                allocation = total_fund * (contribution / total_contribution)
            else:
                allocation = 0
            distribution[name] = allocation
        
        return distribution
    
    @staticmethod
    def distribute_by_area(total_fund, regions_area, weights=None):
        """
        按面积分配补偿资金
        
        参数:
            total_fund: 总补偿资金 (元)
            regions_area: [(region_name, area_km2)]
            weights: 权重系数 (可选)
        
        返回:
            distribution: 各区域补偿金额 (字典)
        """
        if weights is None:
            weights = {name: 1.0 for name, _ in regions_area}
        
        # 计算加权面积
        weighted_areas = []
        for name, area in regions_area:
            weighted_area = area * weights.get(name, 1.0)
            weighted_areas.append((name, weighted_area))
        
        total_weighted_area = sum(wa for _, wa in weighted_areas)
        
        # 按加权面积分配
        distribution = {}
        for name, weighted_area in weighted_areas:
            allocation = total_fund * (weighted_area / total_weighted_area)
            distribution[name] = allocation
        
        return distribution
    
    @staticmethod
    def plot_compensation_analysis(regions_data, total_fund):
        """绘制补偿分析图"""
        fig, axes = plt.subplots(2, 2, figsize=(16, 12))
        
        # 1. 不同方法的补偿标准对比
        ax1 = axes[0, 0]
        
        methods = ['机会成本法', '生态服务价值法', '恢复成本法', '支付意愿法']
        values = [500, 850, 650, 720]  # 示例数据（万元/年）
        colors = ['skyblue', 'lightgreen', 'salmon', 'orange']
        
        bars = ax1.barh(methods, values, color=colors, alpha=0.8, edgecolor='black')
        
        for bar, value in zip(bars, values):
            width = bar.get_width()
            ax1.text(width + 20, bar.get_y() + bar.get_height()/2,
                    f'{value}万元/年', va='center', fontsize=10)
        
        ax1.set_xlabel('补偿标准 (万元/年)', fontsize=11)
        ax1.set_title('不同测算方法的补偿标准对比', fontsize=13, fontweight='bold')
        ax1.grid(True, alpha=0.3, axis='x')
        
        # 2. 按贡献分配
        ax2 = axes[0, 1]
        
        ec = EcologicalCompensation()
        distribution_contrib = ec.distribute_by_contribution(total_fund, regions_data)
        
        regions = list(distribution_contrib.keys())
        allocations = list(distribution_contrib.values())
        
        wedges, texts, autotexts = ax2.pie(allocations, labels=regions, autopct='%1.1f%%',
                                            startangle=90, colors=plt.cm.Set3.colors)
        
        for autotext in autotexts:
            autotext.set_color('white')
            autotext.set_fontsize(10)
            autotext.set_fontweight('bold')
        
        ax2.set_title(f'按贡献分配补偿资金\n总额={total_fund/10000:.0f}万元',
                     fontsize=13, fontweight='bold')
        
        # 3. 各区域贡献与补偿对比
        ax3 = axes[1, 0]
        
        x_pos = np.arange(len(regions))
        contributions = [Q * dC for _, Q, dC in regions_data]
        contributions_norm = np.array(contributions) / max(contributions) * 100
        allocations_norm = np.array(allocations) / max(allocations) * 100
        
        width = 0.35
        ax3.bar(x_pos - width/2, contributions_norm, width, label='贡献度',
               color='steelblue', alpha=0.8)
        ax3.bar(x_pos + width/2, allocations_norm, width, label='补偿额',
               color='coral', alpha=0.8)
        
        ax3.set_xlabel('区域', fontsize=11)
        ax3.set_ylabel('归一化值 (%)', fontsize=11)
        ax3.set_title('各区域贡献度与补偿额对比', fontsize=13, fontweight='bold')
        ax3.set_xticks(x_pos)
        ax3.set_xticklabels(regions)
        ax3.legend()
        ax3.grid(True, alpha=0.3, axis='y')
        
        # 4. 生态服务价值构成
        ax4 = axes[1, 1]
        
        services = ['水源涵养', '水质净化', '洪水调蓄', '固碳释氧', '生物多样性']
        service_values = [320, 180, 150, 110, 90]  # 万元/年
        
        colors_service = plt.cm.Greens(np.linspace(0.4, 0.9, len(services)))
        
        ax4.barh(services, service_values, color=colors_service, alpha=0.9, edgecolor='black')
        
        for i, (service, value) in enumerate(zip(services, service_values)):
            ax4.text(value + 10, i, f'{value}万元', va='center', fontsize=10)
        
        ax4.set_xlabel('价值 (万元/年)', fontsize=11)
        ax4.set_title('生态服务价值构成', fontsize=13, fontweight='bold')
        ax4.grid(True, alpha=0.3, axis='x')
        
        plt.tight_layout()
        plt.savefig('ecological_compensation_analysis.png', dpi=300)
        plt.show()

## 七、典型考题

### 【例题1】补偿标准计算（12分）

**题目**：某流域生态功能区，森林面积3000ha，单位价值6000元/ha/年；湿地800ha，单位价值10000元/ha/年。求年生态服务价值。

**【解答】**

$$V_{total} = 3000 \times 6000 + 800 \times 10000$$

$$= 18,000,000 + 8,000,000 = 26,000,000 \text{ 元/年} = 2600 \text{ 万元/年}$$

---

### 【例题2】补偿资金分配（15分）

**题目**：总补偿1000万元，三个区域：A区（流量5000万m³，削减2mg/L），B区（3000万m³，3mg/L），C区（2000万m³，1.5mg/L）。按贡献分配。

**【解答】**

贡献度：
- A区：$5000 \times 2 = 10000$
- B区：$3000 \times 3 = 9000$
- C区：$2000 \times 1.5 = 3000$
- 总贡献：$22000$

分配：
- A区：$1000 \times \frac{10000}{22000} = 454.5$ 万元
- B区：$1000 \times \frac{9000}{22000} = 409.1$ 万元
- C区：$1000 \times \frac{3000}{22000} = 136.4$ 万元
