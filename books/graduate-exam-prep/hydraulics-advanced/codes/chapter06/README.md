### 1. 水泵基本性能参数计算
**文件**: `ch06_problem01_pump_performance.py` (约750行)  
**难度**: ⭐⭐⭐⭐⭐ 基础  
**核心知识**:
- 欧拉方程：Ht = u₂·cu₂/g
- 圆周速度：u = π·D·n/60
- 速度三角形分析
- 有效功率与效率
- 相似定律：Q∝n, H∝n², N∝n³
- 比转速：ns = n·Q^0.5/H^0.75

**主要功能**:
- 圆周速度与径向速度计算
- 速度三角形精确绘制
- 理论扬程与实际扬程
- 有效功率与效率分析

### 2. 水泵工况点分析
**文件**: `ch06_problem03_operating_point.py` (约700行)  
**难度**: ⭐⭐⭐⭐⭐ 基础  
**核心知识**:
- 工况点：H泵 = H管
- 水泵特性：H = H0 - kQ²
- 管路特性：H = Hst + rQ²
- 并联运行：Q并 = Q1 + Q2

**主要功能**:
- 工况点自动求解（解析法）
