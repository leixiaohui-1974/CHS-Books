### 2. 水泵工况点分析
**文件**: `ch06_problem03_operating_point.py` (约700行)  
**难度**: ⭐⭐⭐⭐⭐ 基础  
**核心知识**:
- 工况点：H泵 = H管
- 水泵特性：H = H0 - kQ²
- 管路特性：H = Hst + rQ²
- 并联运行：Q并 = Q1 + Q2

**主要功能**:

### 3. 水泵并联和串联运行
**文件**: `ch06_problem06_series_parallel.py` (约780行)  
**难度**: ⭐⭐⭐⭐⭐ 强化  
**核心知识**:
- 并联特性：Q并 = Q1 + Q2（相同H）
- 串联特性：H串 = H1 + H2（相同Q）
- 并联曲线：H = H0 - k(Q/n)²
- 串联曲线：H = n(H0 - kQ²)
- 功率计算与对比
- 适用场合判断

**主要功能**:
- 单泵工况点计算
- 并联工况点分析（流量增大）

### 4. 水泵汽蚀分析与防护
**文件**: `ch06_problem09_cavitation.py` (约850行)  
**难度**: ⭐⭐⭐⭐⭐ 强化  
**核心知识**:
- 有效汽蚀余量：NPSHa = pa/(ρg) - Hs - hws - pv/(ρg)
- 必需汽蚀余量：(NPSH)r（泵特性）
- 安全系数：K = NPSHa/(NPSH)r ≥ 1.3
- 允许安装高度：[Hs] = pa/(ρg) - (NPSH)r - hws - pv/(ρg)
- 温度影响分析
- 防汽蚀措施

**主要功能**:
- 吸水管损失计算（沿程+局部）
- NPSHa精确计算
- 汽蚀校核（安全系数）
- 允许安装高度计算
- 安装高度影响分析
- 温度影响分析（饱和蒸汽压）
- 防汽蚀措施建议
