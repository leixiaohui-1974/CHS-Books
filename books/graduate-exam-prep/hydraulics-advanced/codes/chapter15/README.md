## 高频考点

### 必考公式（背诵）

**静水力学**：
```
p = p0 + γh                    # 静水压强
P = γhc·A                      # 总压力
```

**能量方程**：
```
z1 + p1/γ + v1²/(2g) = z2 + p2/γ + v2²/(2g) + hw
```

**连续方程**：
```
Q = A1v1 = A2v2
```

**雷诺数**：
```
Re = vd/ν                      # 判别流态
Re_c = 2320（圆管）
```

**沿程损失**：
```
hf = λ·L/d·v²/(2g)            # 管流
层流：λ = 64/Re
紊流：λ = f(Re, Δ/d)
```

**曼宁公式**：
```
Q = A·R^(2/3)·i^(1/2)/n       # 明渠流
```

**临界水深**：
```
hc = (q²/g)^(1/3)             # 矩形断面
```

**弗劳德数**：
```
Fr = v/√(gh)
Fr<1: 缓流
Fr=1: 临界流
Fr>1: 急流
```

**达西定律**：
```
v = Ki                         # 渗流
```

**重力坝稳定**：
```
Kc = f·ΣV/ΣH ≥ 3.0           # 抗滑
σ = ΣV/B ± 6ΣM/B²             # 应力
```

**泵的相似定律**：
```
Q1/Q2 = (n1/n2)(D1/D2)³
H1/H2 = (n1/n2)²(D1/D2)²
```

**水电站出力**：
```
N = 9.81ηQH/1000 (MW)
```

## 常见陷阱

### 1. 单位陷阱
- 流量：m³/s 和 L/s
- 压强：Pa、kPa、MPa
- 长度：m、cm、mm
- **对策**：统一单位后再计算

### 2. 公式选择陷阱
- 层流 vs 紊流（Re判别）
- 自由出流 vs 淹没出流
- 缓流 vs 急流（Fr判别）
- **对策**：先判别条件，再选公式

### 3. 符号含义陷阱
- h：水深 or 水头损失
- i：底坡 or 水力坡度
- n：糙率 or 转速
- **对策**：根据上下文确定

### 4. 计算精度陷阱
- 中间结果保留位数
- 最终结果有效数字
- **对策**：中间多保留，最终3-4位

### 5. 题目条件陷阱
- 遗漏隐含条件
- 误解题目要求
- **对策**：仔细审题，标注关键词
