### 2. 承压水井流
**文件**: `ch07_problem03_confined_well.py` (约850行)  
**难度**: ⭐⭐⭐⭐⭐ 重点  
**核心知识**:
- Dupuit公式：Q = 2πKM(H₀-hw)/ln(R/r₀)
- 水位分布：h = hw + (H₀-hw)ln(r/r₀)/ln(R/r₀)
- 降深分布：s = s₀[1 - ln(r/r₀)/ln(R/r₀)]
- 渗流速度：v = Q/(2πrM) ∝ 1/r

**主要功能**:
- 井流量计算（Dupuit公式）
- 任意点水位与降深
- 水位降深曲线（对数分布）
- 渗流速度分布
- 影响半径R的影响分析
- 井半径r₀的影响分析
- 参数敏感性分析
- 9个专业图表
