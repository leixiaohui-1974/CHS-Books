## 题目16：多层液体压力计算【基础强化】⭐⭐⭐

### 题目描述

**背景**：在石油储罐、化工容器中常遇到多层不相溶液体共存的情况。

**题目**：如图所示，容器内有三层液体：
- 上层：油（密度ρ₁=850 kg/m³，厚度h₁=1.0 m）
- 中层：水（密度ρ₂=1000 kg/m³，厚度h₂=1.5 m）
- 下层：水银（密度ρ₃=13600 kg/m³，厚度h₃=0.3 m）

容器顶部为大气压。求：
1. 油水界面处的压强
2. 水银面上的压强
3. 容器底部的压强
4. 如果在容器顶部加压至0.2 MPa（表压），各处压强如何变化？

**分值**：15分  
**考频**：★★★★（多层液体是考试重点）

---

### 解答

#### (1) 油水界面压强

**位置**：从液面向下h₁=1.0 m处

**压强计算**：
$$p_1 = p_0 + \rho_1 g h_1$$

其中p₀为大气压强（表压为0）

$$p_1 = 0 + 850 \times 9.81 \times 1.0 = 8338.5 \text{ Pa}$$

$$p_1 = 8.34 \text{ kPa (表压)}$$

**踩分点**：
- ✓ 认识到是表压计算（2分）
- ✓ 正确使用静水压强公式（2分）
- ✓ 计算正确（1分）

---

#### (2) 水银面上的压强

**位置**：从液面向下h₁+h₂=2.5 m处

**压强计算**：
$$p_2 = p_0 + \rho_1 g h_1 + \rho_2 g h_2$$

$$p_2 = 0 + 850 \times 9.81 \times 1.0 + 1000 \times 9.81 \times 1.5$$

$$p_2 = 8338.5 + 14715 = 23053.5 \text{ Pa}$$

$$p_2 = 23.05 \text{ kPa (表压)}$$

**踩分点**：
- ✓ 分层累加各层压强（3分）
- ✓ 计算正确（2分）

---

#### (3) 容器底部压强

**位置**：从液面向下h₁+h₂+h₃=2.8 m处

**压强计算**：
$$p_3 = p_0 + \rho_1 g h_1 + \rho_2 g h_2 + \rho_3 g h_3$$

$$p_3 = 0 + 8338.5 + 14715 + 13600 \times 9.81 \times 0.3$$

$$p_3 = 23053.5 + 40003.2 = 63056.7 \text{ Pa}$$

$$p_3 = 63.06 \text{ kPa (表压)}$$

**踩分点**：
- ✓ 正确累加三层压强（2分）
- ✓ 计算准确（1分）

---

#### (4) 顶部加压后的压强变化

**原理**：根据帕斯卡定律，密闭容器内液体的压强变化处处相等。

**加压值**：Δp = 0.2 MPa = 200 kPa

**各处压强**：

**油水界面**：
$$p_1' = p_1 + \Delta p = 8.34 + 200 = 208.34 \text{ kPa}$$

**水银面**：
$$p_2' = p_2 + \Delta p = 23.05 + 200 = 223.05 \text{ kPa}$$

**容器底部**：
$$p_3' = p_3 + \Delta p = 63.06 + 200 = 263.06 \text{ kPa}$$

**压强增量**：
- 所有位置的压强增量都是200 kPa
- 这体现了帕斯卡定律

**踩分点**：
- ✓ 理解帕斯卡定律（2分）
- ✓ 正确计算三处压强（2分）

---

### 知识点总结

**核心原理**：
1. **多层液体静压强**：$p = p_0 + \sum_{i=1}^{n} \rho_i g h_i$
2. **帕斯卡定律**：密闭液体中压强变化处处相等
3. **表压与绝对压**：本题用表压更方便

**易错点**：
1. ❌ 忘记累加各层压强
2. ❌ 各层密度记错或混淆
3. ❌ 帕斯卡定律理解不深

**工程应用**：
- 石油储罐设计
- 化工容器强度计算
- 分层液体储存
