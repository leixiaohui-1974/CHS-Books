## 📐 核心公式速查

### 静水力学
```python
p = ρgh              # 压强
P = ρghcA           # 总压力
F = ρgV             # 浮力
```

### 水动力学
```python
Q = Av              # 连续性
z + p/(ρg) + v²/(2g) = H   # Bernoulli
F = ρQ(v₂ - v₁)     # 动量
```

### 管流
```python
Re = vd/ν           # Reynolds数
λ = 64/Re           # 层流阻力
h_f = λ(L/d)(v²/2g) # 沿程损失
ΔH = cv/g           # 水击
```

### 明渠流
```python
Fr = v/√(gh)        # Froude数
Q = (1/n)AR^(2/3)i^(1/2)  # Manning
h₂ = (h₁/2)(√(1+8Fr₁²)-1) # 水跃
```

### 渗流
```python
v = ki              # Darcy定律
H(r) = H₀ + Q/(2πkM)ln(r/R)  # 承压井
```

### 水泵
```python
H = H₀ - aQ²        # 泵特性
H_pump = H_pipe     # 工况点
```
