## 📊 题型分析与策略

### 选择题（30分，30题×1分）

**特点**：概念为主，计算为辅

**常考**：
- 流态判别（Re、Fr）
- 公式适用条件
- 概念辨析
- 简单计算

**策略**：
- 时间：30分钟
- 方法：排除法
- 陷阱：单位换算、符号正负

**必会题型**：
1. Re > 2320是什么流态？（湍流）
2. 伯努利方程适用条件？
3. 达西公式中d是什么？（直径）
4. 曼宁n值，混凝土多少？（0.013）
5. 临界Fr等于几？（1）

### 填空题（20分，10题×2分）

**特点**：公式填空、数值计算

**常考**：
- 核心公式填空
- 常用系数
- 简单计算结果

**策略**：
- 时间：20分钟
- 方法：直接套公式
- 注意：单位、有效数字

**必会题型**：
1. 伯努利方程：p/γ + v²/2g + z = ___
2. 达西公式：h_f = λ(___)(___)
3. 层流λ = ___/Re
4. 水的重度γ = ___ N/m³
5. 临界Re = ___

### 计算题（60分，4题×15分）

**特点**：综合计算，步骤多

**常考**：
- 伯努利+连续性（必考1题）
- 达西公式+管道计算（必考1题）
- 曼宁+明渠计算（可能1题）
- 动量方程（可能1题）

**策略**：
- 时间：60分钟
- 步骤：画图→列式→计算→验证
- 得分：步骤分很重要！

**标准答题格式**：
```
解：
(1) 已知：xxx
    求：xxx

(2) 画图：[简图]

(3) 连续性方程：
    v₁A₁ = v₂A₂
    ...

(4) 伯努利方程：
    p₁/γ + v₁²/2g + z₁ = ...
    ...

(5) 代入数值：
    ...

(6) 答：xxx
```

### 综合题（40分，2题×20分）

**特点**：多知识点融合，难度大

**常考**：
- 水泵+管道+伯努利
- 明渠+水跃+堰流
- 复杂管网
- 水工建筑物

**策略**：
- 时间：60分钟
- 方法：分步求解
- 关键：不要慌，一步一步来

**得分技巧**：
1. 即使不会，也要写公式（有步骤分）
2. 已知条件都要用到
3. 答案要写单位
4. 检查数量级是否合理

## 💡 快速提分技巧

### 技巧1：公式变形要熟练

**伯努利**：
- 求压力：$p_2 = p_1 + \gamma[(v_1^2 - v_2^2)/2g + z_1 - z_2 - h_f]$
- 求流速：$v_2 = \sqrt{2g[p_1/\gamma + v_1^2/2g + z_1 - p_2/\gamma - z_2 - h_f]}$

**达西**：
- 求损失：$h_f = \lambda(L/d)(v^2/2g)$
- 求流速：$v = \sqrt{2gh_f d/(\lambda L)}$
- 求管径：$d = \lambda L v^2/(2g h_f)$

### 技巧2：画图很重要

**必须画图的题型**：
- 伯努利方程（标出1、2断面）
- 动量方程（画控制体、受力）
- 平面/曲面压力（画压力方向）
- 明渠流（画水面线）

**画图要点**：
- 清晰、规范
- 标注关键参数
- 箭头表示方向

### 技巧3：单位统一

**常见错误**：
- ❌ 压强用kPa，γ用N/m³（不统一）
- ❌ 流速用cm/s（应该m/s）
- ❌ 管径用mm（应该m）

**正确做法**：
- ✅ 全部用SI基本单位
- ✅ m、kg、s、N、Pa
- ✅ 最后换算成题目要求单位

### 技巧4：数量级检查

**经验数值**：
- 管道流速：0.5-5 m/s
- 明渠流速：0.3-2 m/s
- 水头损失：一般不超过管长的10%
- 水泵扬程：一般几十米
- 压强：不超过1 MPa（一般）

**如果算出**：
- 流速100 m/s → 肯定错了
- 损失1000 m → 肯定错了
- 扬程10000 m → 肯定错了

## ⚠️ 常见错误与陷阱

### 错误1：伯努利方程选错断面

❌ **错误**：选不在同一流线的两个点

✅ **正确**：
- 选同一流线
- 或选过流断面（平均）
- 大容器水面流速≈0

### 错误2：达西公式d搞错

❌ **错误**：d是半径

✅ **正确**：d是直径！

### 错误3：曼宁公式R搞错

❌ **错误**：R是管道半径

✅ **正确**：R是水力半径 = A/P

### 错误4：压力中心就是形心

❌ **错误**：$y_p = h_c$

✅ **正确**：$y_p = h_c + I_c/(h_c A)$，总在形心下方

### 错误5：动量方程符号错

❌ **错误**：$F = \rho Q(v_1 - v_2)$

✅ **正确**：$F = \rho Q(v_2 - v_1)$，出口减入口

### 错误6：水泵扬程漏算损失

❌ **错误**：$H = z_2 - z_1$

✅ **正确**：$H = (z_2 - z_1) + h_f$

### 错误7：临界判别搞混

❌ **错误**：Re > 2320是层流

✅ **正确**：Re > 2320是湍流！

❌ **错误**：Fr > 1是缓流

✅ **正确**：Fr > 1是急流！

## 📋 考场策略

### 答题顺序

1. **先做会做的**（保证基础分）
2. **再做有思路的**（争取多得分）
3. **最后攻难题**（不留遗憾）
4. **留10分钟检查**（避免低级错误）

### 时间分配（150分，180分钟）

| 题型 | 分值 | 建议时间 | 时间分配 |
|------|------|----------|----------|
| 选择题 | 30分 | 30分钟 | 1分/题 |
| 填空题 | 20分 | 20分钟 | 2分/题 |
| 计算题 | 60分 | 60分钟 | 15分/题 |
| 综合题 | 40分 | 60分钟 | 30分/题 |
| 检查 | - | 10分钟 | - |

### 应急措施

**遇到不会的题**：
1. 不要慌张
2. 写出相关公式（有步骤分）
3. 尝试部分求解
4. 标记跳过，最后再看

**时间不够**：
1. 优先保证会做的全对
2. 难题写公式（捞步骤分）
3. 不要在一题上死磕

**算错了怎么办**：
1. 如果发现早，立即改
2. 如果临近交卷，检查关键步骤
3. 不要大面积涂改
