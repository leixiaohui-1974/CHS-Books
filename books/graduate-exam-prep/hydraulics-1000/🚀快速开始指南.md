## ⚙️ 修改参数示例

### 示例1：修改流量和管径

**创建新文件 `my_calculation.py`**：
```python
from problem_351_pipe_calculation import PipeCalculation

# 创建计算对象
calc = PipeCalculation()

# 修改参数为你的实际值
calc.Q = 0.08      # 流量 0.08 m³/s
calc.d = 0.4       # 管径 0.4 m
calc.L = 1000      # 长度 1000 m

# 执行计算
calc.calculate_all()

# 查看结果
print(f"流速: {calc.v:.3f} m/s")
print(f"水头损失: {calc.h_f:.3f} m")

# 可视化
calc.visualize()
```

**运行**：
```bash
python3 my_calculation.py
```

---

### 示例2：批量计算不同参数

**创建文件 `batch_calc.py`**：
```python
import numpy as np
from problem_351_pipe_calculation import PipeCalculation

# 测试不同管径
diameters = [0.3, 0.4, 0.5, 0.6]

print("管径(m)  流速(m/s)  损失(m)")
print("-" * 35)

for d in diameters:
    calc = PipeCalculation()
    calc.Q = 0.05
    calc.d = d
    calc.L = 500
    calc.calculate_all()
    
    print(f"{d:6.2f}  {calc.v:8.3f}  {calc.h_f:8.3f}")
```

