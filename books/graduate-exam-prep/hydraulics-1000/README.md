## 📖 核心知识点

### 最重要的10个公式

1. **伯努利方程** ⭐⭐⭐⭐⭐
   ```
   p₁/γ + v₁²/2g + z₁ = p₂/γ + v₂²/2g + z₂ + h_f
   ```

2. **达西公式** ⭐⭐⭐⭐⭐
   ```
   h_f = λ(L/d)(v²/2g)
   ```

3. **曼宁公式** ⭐⭐⭐⭐⭐
   ```
   Q = (1/n)AR^(2/3)i^(1/2)
   ```

4. **连续性方程** ⭐⭐⭐⭐
   ```
   v₁A₁ = v₂A₂
   ```

5. **动量方程** ⭐⭐⭐⭐
   ```
   ΣF = ρQ(v₂ - v₁)
   ```

6-10. 详见 `公式速查卡.md`

```python

    d1=0.2,    # 入口直径
```
