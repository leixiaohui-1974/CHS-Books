## 题目031：物体浮沉判断 ⭐

### 题目描述

有三个物体：
- 物体A：铁块，密度\(\rho_A = 7800\,\text{kg/m}^3\)，体积\(V_A = 0.01\,\text{m}^3\)
- 物体B：木块，密度\(\rho_B = 600\,\text{kg/m}^3\)，体积\(V_B = 0.05\,\text{m}^3\)
- 物体C：冰块，密度\(\rho_C = 900\,\text{kg/m}^3\)，体积\(V_C = 0.02\,\text{m}^3\)

将它们放入水中（\(\rho_水 = 1000\,\text{kg/m}^3\)），判断它们的浮沉状态。

### 解答

**判断原则**：
- \(\rho_物 > \rho_液\) → 下沉
- \(\rho_物 = \rho_液\) → 悬浮
- \(\rho_物 < \rho_液\) → 上浮（漂浮）

#### 物体A（铁块）

$$\rho_A = 7800\,\text{kg/m}^3 > \rho_水 = 1000\,\text{kg/m}^3$$

**结论**：**下沉**

#### 物体B（木块）

$$\rho_B = 600\,\text{kg/m}^3 < \rho_水 = 1000\,\text{kg/m}^3$$

**结论**：**上浮，最终漂浮**

**浸没深度**（设木块为长方体，高度\(h\)）：
$$\frac{V_{浸}}{V_B} = \frac{\rho_B}{\rho_水} = \frac{600}{1000} = 0.6 = 60\%$$

木块有60%体积在水下，40%在水面以上。

#### 物体C（冰块）

$$\rho_C = 900\,\text{kg/m}^3 < \rho_水 = 1000\,\text{kg/m}^3$$

**结论**：**上浮，最终漂浮**

**浸没深度**：
$$\frac{V_{浸}}{V_C} = \frac{\rho_C}{\rho_水} = \frac{900}{1000} = 0.9 = 90\%$$

冰块有90%体积在水下，10%露出水面。

### 关键知识点

1. **浮沉条件**：比较物体密度与液体密度
2. **漂浮状态**：\(\rho_物 < \rho_液\)，部分浸没
3. **浸没比例**：\(V_{浸}/V = \rho_物/\rho_液\)

## 题目032：完全浸没物体浮力 ⭐

### 题目描述

球形物体，半径\(R = 0.5\,\text{m}\)，密度\(\rho_{球} = 2500\,\text{kg/m}^3\)，完全浸没在水中（\(\rho_水 = 1000\,\text{kg/m}^3\)）。

**求**：
1. 物体受到的浮力
2. 物体的重力
3. 物体的净重（视重）
4. 绳索拉力（若用绳索悬挂）

### 解答

#### （1）浮力

**阿基米德原理**：\(F_浮 = \rho_液 g V_排\)

球体体积：
$$V = \frac{4}{3}\pi R^3 = \frac{4}{3}\pi \times 0.5^3 = 0.5236\,\text{m}^3$$

浮力：
$$F_浮 = \rho_水 \times g \times V = 1000 \times 9.8 \times 0.5236 = 5131\,\text{N} = 5.13\,\text{kN}$$

#### （2）重力

$$G = \rho_{球} \times g \times V = 2500 \times 9.8 \times 0.5236 = 12,829\,\text{N} = 12.83\,\text{kN}$$

#### （3）净重（视重）

$$G_{净} = G - F_浮 = 12,829 - 5,131 = 7,698\,\text{N} = 7.70\,\text{kN}$$

物体在水中的"感觉重量"减轻了\(F_浮 = 5.13\,\text{kN}\)。

#### （4）绳索拉力

若用绳索从上方悬挂，绳索需要提供的拉力等于净重：
$$T = G_{净} = 7.70\,\text{kN}$$

**比较**：
- 空气中拉力：\(T_空 = G = 12.83\,\text{kN}\)
- 水中拉力：\(T_水 = 7.70\,\text{kN}\)
- 减轻比例：\((12.83 - 7.70)/12.83 = 40\%\)

### 关键知识点

1. **阿基米德原理**：\(F_浮 = \rho_液 g V_排\)
2. **净重**：\(G_{净} = G - F_浮\)
3. **视重效应**：物体在液体中"变轻"

## 题目033：漂浮物体浮力 ⭐

### 题目描述

长方体木块，长\(a = 2\,\text{m}\)，宽\(b = 1\,\text{m}\)，高\(h = 0.5\,\text{m}\)，密度\(\rho_木 = 700\,\text{kg/m}^3\)，漂浮在水面上。

**求**：
1. 木块受到的浮力
2. 木块的吃水深度\(d\)（浸入水中的深度）
3. 若在木块上放置重物，使木块刚好完全浸没，重物重量为多少？

### 解答

#### （1）浮力

**漂浮条件**：\(F_浮 = G\)

木块重量：
$$G = \rho_木 \times g \times V = 700 \times 9.8 \times (2 \times 1 \times 0.5) = 6,860\,\text{N} = 6.86\,\text{kN}$$

浮力：
$$F_浮 = G = 6.86\,\text{kN}$$

#### （2）吃水深度

$$F_浮 = \rho_水 \times g \times V_排 = \rho_水 \times g \times (a \times b \times d)$$

$$d = \frac{F_浮}{\rho_水 \times g \times a \times b} = \frac{6860}{1000 \times 9.8 \times 2 \times 1} = 0.35\,\text{m}$$

或用密度比：
$$d = h \times \frac{\rho_木}{\rho_水} = 0.5 \times \frac{700}{1000} = 0.35\,\text{m}$$

木块浸入水中0.35m，露出水面0.15m。

#### （3）加载重物

木块完全浸没时，排开水的体积为总体积\(V\)：
$$F_{浮,满} = \rho_水 \times g \times V = 1000 \times 9.8 \times 1.0 = 9,800\,\text{N}$$

增加的浮力：
$$\Delta F_浮 = F_{浮,满} - F_浮 = 9,800 - 6,860 = 2,940\,\text{N}$$

重物重量：
$$G_{重物} = \Delta F_浮 = 2,940\,\text{N} = 2.94\,\text{kN}$$

约300kg重物。

### 关键知识点

1. **漂浮平衡**：\(F_浮 = G\)
2. **吃水深度**：\(d = h \times \rho_物/\rho_液\)
3. **加载能力**：额外浮力空间

## 题目034：潜艇上浮下潜 ⭐⭐

### 题目描述

潜艇体积\(V = 2000\,\text{m}^3\)，海水密度\(\rho = 1025\,\text{kg/m}^3\)。

**状态1**：悬浮状态，潜艇总重量\(W_1\)
**状态2**：下潜，向水舱注水\(m = 50\)吨
**状态3**：上浮，排出水舱水

**求**：
1. 悬浮时潜艇总重量
2. 下潜时的净下沉力
3. 上浮时需排出多少水才能浮出水面

### 解答

#### （1）悬浮状态

$$W_1 = F_浮 = \rho \times g \times V = 1025 \times 9.8 \times 2000 = 20,090,000\,\text{N} \approx 20.09\,\text{MN}$$

潜艇总重量约**2049吨**。

#### （2）下潜

注水后总重量：
$$W_2 = W_1 + m \times g = 20,090,000 + 50,000 \times 9.8 = 20,580,000\,\text{N}$$

浮力不变（体积不变）：
$$F_浮 = 20,090,000\,\text{N}$$

净下沉力：
$$F_{下} = W_2 - F_浮 = 20,580,000 - 20,090,000 = 490,000\,\text{N} = 490\,\text{kN}$$

#### （3）上浮

要浮出水面，需要\(W < F_浮\)，至少排出：
$$\Delta m = m = 50\,\text{吨}$$

排出50吨水后恢复悬浮，要上浮需排出更多。

实际上浮：排出\(> 50\)吨水。

### 关键知识点

1. **潜艇悬浮**：\(W = F_浮\)
2. **调节重量**：通过水舱进排水
3. **浮沉控制**：\(\Delta W = \Delta m \times g\)

## 题目035：密度计原理 ⭐⭐

### 题目描述

密度计是测量液体密度的仪器。设密度计为圆柱形，底部有铅块，总质量\(m = 0.05\,\text{kg}\)，圆柱直径\(d = 0.01\,\text{m}\)。

在纯水（\(\rho_1 = 1000\,\text{kg/m}^3\)）中，吃水深度\(h_1 = 0.15\,\text{m}\)。

**求**：
1. 在密度为\(\rho_2 = 1200\,\text{kg/m}^3\)的盐水中，吃水深度\(h_2\)
2. 密度计刻度原理

### 解答

#### （1）纯水中

$$F_浮1 = G = mg = 0.05 \times 9.8 = 0.49\,\text{N}$$

$$F_浮1 = \rho_1 g A h_1 = 1000 \times 9.8 \times \frac{\pi \times 0.01^2}{4} \times 0.15 = 0.1155\,\text{N}$$

**矛盾！** 说明题目参数需调整。假设\(m = 0.012\,\text{kg}\)：

$$G = 0.012 \times 9.8 = 0.1176\,\text{N} \approx F_浮1$$

#### （2）盐水中

$$F_浮2 = G = 0.1176\,\text{N}$$

$$\rho_2 g A h_2 = 0.1176$$

$$h_2 = \frac{0.1176}{\rho_2 g A} = \frac{0.1176}{1200 \times 9.8 \times \frac{\pi \times 0.01^2}{4}} = 0.1275\,\text{m} = 12.75\,\text{cm}$$

或用比例：
$$\frac{h_1}{h_2} = \frac{\rho_2}{\rho_1}$$

$$h_2 = h_1 \times \frac{\rho_1}{\rho_2} = 15 \times \frac{1000}{1200} = 12.5\,\text{cm}$$

#### （3）刻度原理

- 液体密度**越大**，吃水深度**越小**
- 刻度从上到下，密度值**递增**
- 刻度不均匀，呈双曲线分布

### 关键知识点

1. **漂浮平衡**：\(F_浮 = G\)（不变）
2. **吃水深度与密度反比**：\(h \propto 1/\rho\)
3. **密度计应用**：酒精度、盐度测量

## 题目036：冰块融化水位变化 ⭐⭐

### 题目描述

容器中有水，水面上漂浮一块冰，冰的密度\(\rho_{冰} = 900\,\text{kg/m}^3\)，体积\(V_{冰} = 0.001\,\text{m}^3\)。

**求**：
1. 冰块漂浮时排开的水的体积
2. 冰块完全融化后，水位如何变化？（上升、下降、不变）

### 解答

#### （1）漂浮时排开水的体积

$$G_{冰} = \rho_{冰} g V_{冰} = 900 \times 9.8 \times 0.001 = 8.82\,\text{N}$$

$$F_浮 = G_{冰} = 8.82\,\text{N}$$

$$V_排 = \frac{F_浮}}{\rho_水 g} = \frac{8.82}{1000 \times 9.8} = 0.0009\,\text{m}^3 = 0.9\,\text{升}$$

#### （2）融化后水位变化

冰块融化成水的体积：
$$V_水 = \frac{m_{冰}}{\rho_水} = \frac{\rho_{冰} V_{冰}}{\rho_水} = \frac{900 \times 0.001}{1000} = 0.0009\,\text{m}^3$$

**发现**：\(V_水 = V_排 = 0.0009\,\text{m}^3\)

**结论**：水位**不变**！

#### 原理分析

漂浮时：冰排开水的体积 = 冰的质量 / 水的密度
融化后：冰变成水的体积 = 冰的质量 / 水的密度

两者相等，故水位不变。

**推广**：
- 纯冰块融化，水位不变
- 冰块内有气泡，融化后水位下降
- 冰块上有重物，融化后水位下降

### 关键知识点

1. **质量守恒**：\(m_{冰} = m_水\)
2. **体积关系**：\(V_排 = V_水\)
3. **水位不变**：经典物理问题

## 题目037：浮体稳定性基础 ⭐⭐

### 题目描述

长方体浮体，长\(a = 3\,\text{m}\)，宽\(b = 2\,\text{m}\)，高\(h = 1\,\text{m}\)，密度\(\rho = 800\,\text{kg/m}^3\)，漂浮在水面。

**求**：
1. 重心G位置（从底部算起）
2. 浮心B位置（从底部算起）
3. 判断稳定性（定性）

### 解答

#### （1）重心位置

均质长方体，重心在几何中心：
$$z_G = \frac{h}{2} = 0.5\,\text{m}$$（从底部算起）

#### （2）浮心位置

吃水深度：
$$d = h \times \frac{\rho}{\rho_水} = 1 \times \frac{800}{1000} = 0.8\,\text{m}$$

浮心在浸没部分的几何中心：
$$z_B = \frac{d}{2} = 0.4\,\text{m}$$（从底部算起）

#### （3）稳定性判断

**比较**：\(z_G = 0.5\,\text{m} > z_B = 0.4\,\text{m}\)

重心在浮心**上方**。

**初步判断**：可能不稳定（需进一步计算稳心）

**完整判断**需要计算**稳心M**：
$$BM = \frac{I_L}{V_排}$$

其中\(I_L\)为水线面对其中心轴的惯性矩。

对于长方体：
$$I_L = \frac{a b^3}{12} = \frac{3 \times 2^3}{12} = 2\,\text{m}^4$$

$$V_排 = a \times b \times d = 3 \times 2 \times 0.8 = 4.8\,\text{m}^3$$

$$BM = \frac{2}{4.8} = 0.417\,\text{m}$$

$$z_M = z_B + BM = 0.4 + 0.417 = 0.817\,\text{m}$$

**最终判断**：\(z_M = 0.817\,\text{m} > z_G = 0.5\,\text{m}\)

稳心M在重心G上方，**稳定**！

### 关键知识点

1. **重心G**：物体的质心
2. **浮心B**：浸没部分的几何中心
3. **稳心M**：浮体倾斜时浮力作用线与中心轴的交点
4. **稳定性判据**：M在G上方→稳定

## 题目038：船舶稳定性 ⭐⭐⭐

### 题目描述

船舶简化为长方体，长\(L = 50\,\text{m}\)，宽\(B = 10\,\text{m}\)，高\(H = 6\,\text{m}\)，吃水深度\(T = 3\,\text{m}\)，重心高度\(KG = 4\,\text{m}\)（从基线算起）。

**求**：
1. 浮心高度\(KB\)
2. 稳心高度\(KM\)
3. 稳心半径\(GM\)
4. 稳定性判断

### 解答

#### （1）浮心高度

浮心在吃水的一半：
$$KB = \frac{T}{2} = \frac{3}{2} = 1.5\,\text{m}$$

#### （2）稳心高度

水线面惯性矩（矩形）：
$$I_L = \frac{L B^3}{12} = \frac{50 \times 10^3}{12} = 4166.67\,\text{m}^4$$

排水体积：
$$\nabla = L \times B \times T = 50 \times 10 \times 3 = 1500\,\text{m}^3$$

稳心半径：
$$BM = \frac{I_L}{\nabla} = \frac{4166.67}{1500} = 2.78\,\text{m}$$

稳心高度：
$$KM = KB + BM = 1.5 + 2.78 = 4.28\,\text{m}$$

#### （3）稳心半径（初稳性高）

$$GM = KM - KG = 4.28 - 4.0 = 0.28\,\text{m}$$

#### （4）稳定性判断

\(GM = 0.28\,\text{m} > 0\) → **稳定**

但GM偏小，稳定性较弱。

**稳定性标准**（一般货船）：
- \(GM < 0\)：不稳定
- \(0 < GM < 0.3\,\text{m}\)：稳定性差
- \(0.3\,\text{m} < GM < 1.0\,\text{m}\)：稳定性良好
- \(GM > 1.0\,\text{m}\)：稳定性过强（摇摆剧烈）

**建议**：降低重心或加宽船体。

### 关键知识点

1. **KB**：浮心高度 = T/2
2. **BM**：稳心半径 = I_L/∇
3. **GM**：初稳性高 = KM - KG
4. **稳定性判据**：GM > 0

## 题目039：气球升空 ⭐⭐

### 题目描述

热气球，球囊体积\(V = 2000\,\text{m}^3\)，充入热空气密度\(\rho_{热} = 0.9\,\text{kg/m}^3\)，外部冷空气密度\(\rho_{冷} = 1.2\,\text{kg/m}^3\)。气球总重（含球囊、吊篮、载荷）\(W = 5000\,\text{N}\)。

**求**：
1. 浮力
2. 净升力
3. 最大载重量

### 解答

#### （1）浮力

$$F_浮 = \rho_{冷} g V = 1.2 \times 9.8 \times 2000 = 23,520\,\text{N} = 23.52\,\text{kN}$$

#### （2）净升力

热空气重量：
$$W_{热} = \rho_{热} g V = 0.9 \times 9.8 \times 2000 = 17,640\,\text{N}$$

总重：
$$W_{总} = W + W_{热} = 5,000 + 17,640 = 22,640\,\text{N}$$

净升力：
$$F_{净} = F_浮 - W_{总} = 23,520 - 22,640 = 880\,\text{N}$$

气球可以升空！

#### （3）最大载重量

$$W_{max} = F_浮 - W_{热} = 23,520 - 17,640 = 5,880\,\text{N}$$

约600kg。

**现有载荷**：\(W = 5000\,\text{N} \approx 510\,\text{kg}\)

**剩余载重**：\(880\,\text{N} \approx 90\,\text{kg}\)

### 关键知识点

1. **气球浮力**：排开外部空气的重量
2. **净升力**：浮力 - 总重
3. **载重能力**：密度差决定

## 题目040：浮筒稳定性 ⭐⭐⭐

### 题目描述

圆柱形浮筒，直径\(D = 2\,\text{m}\)，高度\(H = 5\,\text{m}\)，密度\(\rho = 500\,\text{kg/m}^3\)，竖直漂浮。

**求**：
1. 吃水深度\(T\)
2. 稳心半径\(BM\)
3. 初稳性高\(GM\)
4. 若横向放置（轴线水平），稳定性如何？

### 解答

#### （1）吃水深度

$$T = H \times \frac{\rho}{\rho_水} = 5 \times \frac{500}{1000} = 2.5\,\text{m}$$

#### （2）稳心半径

水线面为圆形，惯性矩：
$$I_L = \frac{\pi D^4}{64} = \frac{\pi \times 2^4}{64} = 0.7854\,\text{m}^4$$

排水体积：
$$\nabla = \frac{\pi D^2}{4} \times T = \frac{\pi \times 2^2}{4} \times 2.5 = 7.854\,\text{m}^3$$

$$BM = \frac{I_L}{\nabla} = \frac{0.7854}{7.854} = 0.1\,\text{m}$$

#### （3）初稳性高

重心：
$$KG = \frac{H}{2} = 2.5\,\text{m}$$

浮心：
$$KB = \frac{T}{2} = 1.25\,\text{m}$$

稳心：
$$KM = KB + BM = 1.25 + 0.1 = 1.35\,\text{m}$$

初稳性高：
$$GM = KM - KG = 1.35 - 2.5 = -1.15\,\text{m}$$

**结论**：\(GM < 0\)，**不稳定**！

竖直放置的细长浮筒容易翻倒。

#### （4）横向放置

水线面变为矩形\(L \times D\)，惯性矩大幅增加，\(BM\)增大，稳定性改善。

### 关键知识点

1. **细长浮体**：竖直放置不稳定
2. **稳定性与形状**：宽扁体稳定，细长体不稳定
3. **工程应用**：浮筒需横放或加稳定装置
