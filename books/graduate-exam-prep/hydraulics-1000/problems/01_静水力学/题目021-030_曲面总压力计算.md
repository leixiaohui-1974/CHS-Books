## 题目021：圆柱面曲面总压力 ⭐⭐

### 题目描述

半圆柱形闸门AB，半径\(R = 2\,\text{m}\)，宽度\(b = 3\,\text{m}\)，圆心O位于水面。闸门凸面向上。水的容重\(\gamma = 9800\,\text{N/m}^3\)。

**求**：
1. 曲面上水平分力\(P_x\)
2. 曲面上垂直分力\(P_z\)
3. 总压力\(P\)的大小和方向

### 解答

#### （1）水平分力\(P_x\)

**原理**：曲面水平分力等于曲面在铅垂投影面上的压力。

**投影面**：矩形，宽度\(b = 3\,\text{m}\)，高度\(h = R = 2\,\text{m}\)

**计算**：
$$P_x = \gamma h_c A_x = \gamma \times \frac{R}{2} \times (b \times R)$$

$$P_x = 9800 \times 1 \times (3 \times 2) = 9800 \times 6 = 58,800\,\text{N} = 58.8\,\text{kN}$$

**作用点高度**（从水面算起）：
$$y_{px} = \frac{2R}{3} = \frac{2 \times 2}{3} = 1.333\,\text{m}$$

#### （2）垂直分力\(P_z\)

**原理**：曲面垂直分力等于压力体的重量。

**压力体**：半圆柱体（水体重量）
- 体积：\(V = \frac{\pi R^2}{2} \times b = \frac{\pi \times 2^2}{2} \times 3 = 6\pi = 18.85\,\text{m}^3\)

**计算**：
$$P_z = \gamma V = 9800 \times 18.85 = 184,730\,\text{N} = 184.7\,\text{kN}$$

**方向**：竖直向下（凸面向上，压力体为实）

**作用点**（半圆形心）：距圆心水平距离
$$x_{pz} = \frac{4R}{3\pi} = \frac{4 \times 2}{3\pi} = 0.849\,\text{m}$$

#### （3）总压力\(P\)

**大小**：
$$P = \sqrt{P_x^2 + P_z^2} = \sqrt{58.8^2 + 184.7^2} = \sqrt{3457 + 34134} = 194.0\,\text{kN}$$

**方向**（与水平面夹角）：
$$\alpha = \arctan\frac{P_z}{P_x} = \arctan\frac{184.7}{58.8} = \arctan(3.142) = 72.3°$$

总压力方向：向右下，与水平面成72.3°角。

### 关键知识点

1. **曲面总压力计算**：分解为水平分力和垂直分力
2. **水平分力**：\(P_x = \gamma h_c A_x\)（投影面积法）
3. **垂直分力**：\(P_z = \gamma V\)（压力体法）
4. **压力体判断**：凸面向上→实体，凹面向上→虚体
5. **总压力合成**：\(P = \sqrt{P_x^2 + P_z^2}\)

## 题目022：球面曲面总压力 ⭐⭐

### 题目描述

半球形容器，半径\(R = 1.5\,\text{m}\)，开口向上，装满水。

**求**：
1. 球面上水平分力（任意径向）
2. 球面上垂直分力
3. 总压力分布特征

### 解答

#### （1）水平分力

**特点**：球面具有轴对称性，各径向水平分力相互抵消。

**结论**：净水平分力\(P_x = 0\)

#### （2）垂直分力

**压力体**：半球体水体重量

**体积**：
$$V = \frac{2}{3}\pi R^3 = \frac{2}{3}\pi \times 1.5^3 = 7.069\,\text{m}^3$$

**垂直分力**：
$$P_z = \gamma V = 9800 \times 7.069 = 69,276\,\text{N} = 69.3\,\text{kN}$$

**方向**：竖直向下

**作用点**（半球形心）：
$$z_c = \frac{3R}{8} = \frac{3 \times 1.5}{8} = 0.5625\,\text{m}$$（从球心向上）

#### （3）总压力特征

- 总压力\(P = P_z = 69.3\,\text{kN}\)（竖直向下）
- 各点压强不同，但合力只有垂直分量
- 压力分布呈轴对称

### 关键知识点

1. **轴对称曲面**：水平分力相互抵消
2. **半球形心**：距球心\(3R/8\)
3. **球面总压力**：只有垂直分力

## 题目023：圆弧闸门总压力 ⭐⭐

### 题目描述

圆弧形闸门AB，圆心角\(\theta = 60°\)，半径\(R = 3\,\text{m}\)，宽度\(b = 2\,\text{m}\)。A点位于水面，B点在A点下方，圆弧凸面向水。

**求**：
1. 水平分力\(P_x\)及作用点
2. 垂直分力\(P_z\)及作用点
3. 总压力及其对圆心O的力矩

### 解答

#### （1）水平分力

**投影面**：矩形
- 高度：\(h = R\sin\theta = 3 \times \sin 60° = 2.598\,\text{m}\)
- 宽度：\(b = 2\,\text{m}\)
- 面积：\(A_x = 2.598 \times 2 = 5.196\,\text{m}^2\)

**形心深度**：
$$h_c = \frac{h}{2} = 1.299\,\text{m}$$

**水平分力**：
$$P_x = \gamma h_c A_x = 9800 \times 1.299 \times 5.196 = 66,128\,\text{N} = 66.1\,\text{kN}$$

**作用点**：
$$y_{px} = \frac{2h}{3} = \frac{2 \times 2.598}{3} = 1.732\,\text{m}$$（从水面算起）

#### （2）垂直分力

**压力体**：扇形水体
- 扇形面积：\(A_{扇} = \frac{\theta}{360°} \times \pi R^2 = \frac{60°}{360°} \times \pi \times 3^2 = 4.712\,\text{m}^2\)
- 体积：\(V = A_{扇} \times b = 4.712 \times 2 = 9.424\,\text{m}^3\)

**垂直分力**：
$$P_z = \gamma V = 9800 \times 9.424 = 92,355\,\text{N} = 92.4\,\text{kN}$$

**方向**：竖直向下（凸面向水，压力体为实）

#### （3）总压力及力矩

**总压力大小**：
$$P = \sqrt{P_x^2 + P_z^2} = \sqrt{66.1^2 + 92.4^2} = 113.6\,\text{kN}$$

**方向角**：
$$\alpha = \arctan\frac{P_z}{P_x} = \arctan\frac{92.4}{66.1} = 54.4°$$

**特性**：圆弧面总压力必过圆心O！
因此，**对圆心O的力矩为零**：\(M_O = 0\)

### 关键知识点

1. **圆弧面总压力**：必过圆心
2. **扇形面积**：\(A = \frac{\theta}{360°} \times \pi R^2\)
3. **力矩平衡**：圆弧闸门可绕圆心自由转动

## 题目024：抛物线形曲面 ⭐⭐⭐

### 题目描述

抛物线形曲面，方程\(y = x^2/2\)（单位：米），\(0 \le x \le 2\,\text{m}\)，宽度\(b = 1\,\text{m}\)。左侧有水，水面与\(x = 0\)齐平，曲面凸向水。

**求**：
1. 水平分力\(P_x\)
2. 垂直分力\(P_z\)
3. 总压力\(P\)

### 解答

#### （1）水平分力

**投影面**：矩形
- 在\(x = 2\)处，\(y = 2^2/2 = 2\,\text{m}\)
- 投影高度：\(h = 2\,\text{m}\)
- 面积：\(A_x = b \times h = 1 \times 2 = 2\,\text{m}^2\)

**水平分力**：
$$P_x = \gamma h_c A_x = 9800 \times 1 \times 2 = 19,600\,\text{N} = 19.6\,\text{kN}$$

#### （2）垂直分力

**压力体体积**（抛物线下方面积×宽度）：

抛物线下方面积：
$$A = \int_0^2 x^2/2 \, dx = \frac{1}{2} \times \frac{x^3}{3}\Big|_0^2 = \frac{1}{2} \times \frac{8}{3} = \frac{4}{3}\,\text{m}^2$$

体积：
$$V = A \times b = \frac{4}{3} \times 1 = 1.333\,\text{m}^3$$

**垂直分力**：
$$P_z = \gamma V = 9800 \times 1.333 = 13,063\,\text{N} = 13.1\,\text{kN}$$

#### （3）总压力

$$P = \sqrt{P_x^2 + P_z^2} = \sqrt{19.6^2 + 13.1^2} = 23.5\,\text{kN}$$

### 关键知识点

1. **任意曲面压力**：积分法求压力体体积
2. **抛物线积分**：\(\int x^2 dx = x^3/3\)
3. **复杂曲面**：可用数值积分

## 题目025：柱面闸门启闭力 ⭐⭐⭐

### 题目描述

圆柱形闸门，半径\(R = 2\,\text{m}\)，宽度\(b = 4\,\text{m}\)，圆心O位于水面。闸门可绕圆心O转动。闸门弧长对应圆心角\(\theta = 90°\)，凸面向水，A端在水面，B端在水下。

**求**：
1. 水平和垂直分力
2. 闸门刚好开启需要的力矩\(M\)
3. 若在B端施加垂直向上的力\(F\)，求\(F\)的大小

### 解答

#### （1）水平和垂直分力

**水平分力**（投影矩形：\(R \times b\)）：
$$P_x = \gamma \times \frac{R}{2} \times (R \times b) = 9800 \times 1 \times (2 \times 4) = 78,400\,\text{N} = 78.4\,\text{kN}$$

**垂直分力**（\(1/4\)圆面积×宽度）：
$$V = \frac{\pi R^2}{4} \times b = \frac{\pi \times 2^2}{4} \times 4 = 4\pi = 12.566\,\text{m}^3$$

$$P_z = \gamma V = 9800 \times 12.566 = 123,147\,\text{N} = 123.1\,\text{kN}$$

#### （2）对圆心O的力矩

**特性**：圆弧面总压力过圆心，对圆心力矩为零！

**结论**：\(M_O = 0\)

闸门处于**自平衡状态**，无需额外力矩即可保持平衡。

#### （3）B端施力

若要开启闸门（逆时针转动），需要克服摩擦和其他阻力。

**理想情况**（无摩擦）：\(F = 0\)

**实际情况**（考虑摩擦）：需要实验测定摩擦力矩。

### 关键知识点

1. **圆弧闸门自平衡**：总压力过圆心
2. **力矩为零**：可自由转动
3. **工程应用**：弧形闸门设计原理

## 题目026：双曲面压力 ⭐⭐⭐

### 题目描述

水箱隔板为双曲面，左侧水深\(h_1 = 3\,\text{m}\)，右侧水深\(h_2 = 1\,\text{m}\)。曲面为半圆柱形，半径\(R = 1\,\text{m}\)，宽度\(b = 2\,\text{m}\)，圆心在两侧水面之间。

**求**：
1. 左侧水压力\(P_左\)
2. 右侧水压力\(P_右\)
3. 净压力\(P_净\)及方向

### 解答

#### （1）左侧水压力

**水平分力**（投影高度\(h_1 = 3\,\text{m}\)）：
$$P_{x,左} = 9800 \times 1.5 \times (3 \times 2) = 88,200\,\text{N}$$（向右）

**垂直分力**（压力体）：
- 若圆心在左侧水面下1m，则压力体复杂，需分段计算
- 简化：假设圆心在左侧水面，则\(P_{z,左} = 9800 \times \frac{\pi R^2}{2} \times b = 30,790\,\text{N}\)（向下）

#### （2）右侧水压力

**水平分力**（投影高度\(h_2 = 1\,\text{m}\)）：
$$P_{x,右} = 9800 \times 0.5 \times (1 \times 2) = 9,800\,\text{N}$$（向左）

**垂直分力**：
$$P_{z,右} = 9800 \times \frac{\pi R^2}{2} \times b = 30,790\,\text{N}$$（向上，虚压力体）

#### （3）净压力

**水平净力**：
$$P_{x,净} = P_{x,左} - P_{x,右} = 88,200 - 9,800 = 78,400\,\text{N} = 78.4\,\text{kN}$$（向右）

**垂直净力**：
$$P_{z,净} = P_{z,左} - P_{z,右} = 30,790 - 30,790 = 0\,\text{N}$$

**净压力**：\(P_净 = 78.4\,\text{kN}\)（水平向右）

### 关键知识点

1. **双侧受压曲面**：分别计算，求矢量差
2. **虚压力体**：凹面向上，垂直分力向上
3. **对称抵消**：垂直分力可能抵消

## 题目027：坝面总压力 ⭐⭐

### 题目描述

重力坝上游面由两段组成：
- 上段：竖直面，高\(h_1 = 10\,\text{m}\)
- 下段：圆弧面，半径\(R = 5\,\text{m}\)，圆心角\(\theta = 60°\)，凸向水

坝宽（垂直纸面）\(b = 10\,\text{m}\)。

**求**：
1. 上段竖直面压力
2. 下段圆弧面压力
3. 总压力及作用点

### 解答

#### （1）上段竖直面压力

**平面总压力**：
$$P_1 = \gamma h_{c1} A_1 = 9800 \times 5 \times (10 \times 10) = 4,900,000\,\text{N} = 4900\,\text{kN}$$

**作用点**（从水面算）：
$$y_{p1} = \frac{2h_1}{3} = 6.67\,\text{m}$$

#### （2）下段圆弧面压力

**水平分力**（投影高度\(h = R\sin 60° = 4.33\,\text{m}\)）：
$$P_{x2} = 9800 \times (10 + 2.165) \times (4.33 \times 10) = 5,168,000\,\text{N} = 5168\,\text{kN}$$

**垂直分力**（扇形压力体）：
$$V_2 = \frac{60°}{360°} \times \pi \times 5^2 \times 10 = 130.9\,\text{m}^3$$

$$P_{z2} = 9800 \times 130.9 = 1,282,820\,\text{N} = 1283\,\text{kN}$$

#### （3）总压力

**水平总力**：
$$P_x = P_1 + P_{x2} = 4900 + 5168 = 10,068\,\text{kN}$$

**垂直总力**：
$$P_z = P_{z2} = 1283\,\text{kN}$$

**合力**：
$$P = \sqrt{10068^2 + 1283^2} = 10,150\,\text{kN}$$

**方向角**：
$$\alpha = \arctan\frac{1283}{10068} = 7.3°$$（接近水平）

### 关键知识点

1. **组合曲面**：分段计算后合成
2. **重力坝设计**：总压力是关键荷载
3. **安全稳定**：需验算抗滑、抗倾覆

## 题目028：球面容器 ⭐⭐

### 题目描述

球形容器，半径\(R = 2\,\text{m}\)，容器内充满水。

**求**：
1. 上半球面受到的水压力
2. 下半球面受到的水压力
3. 水平中截面受到的压力

### 解答

#### （1）上半球面压力

**压力体**：上半球体（虚体，因为凹面向上）

**垂直分力**（向上）：
$$P_{上} = \gamma \times \frac{2\pi R^3}{3} = 9800 \times \frac{2\pi \times 2^3}{3} = 164,500\,\text{N} = 164.5\,\text{kN}$$

**方向**：竖直向上

#### （2）下半球面压力

**压力体**：下半球体+圆柱体（实体）

**圆柱体体积**：
$$V_{圆柱} = \pi R^2 \times 2R = 2\pi R^3 = 50.27\,\text{m}^3$$

**压力体总体积**：
$$V_{总} = V_{圆柱} + V_{半球} = 2\pi R^3 + \frac{2\pi R^3}{3} = \frac{8\pi R^3}{3} = 67.02\,\text{m}^3$$

**垂直分力**（向下）：
$$P_{下} = 9800 \times 67.02 = 656,796\,\text{N} = 656.8\,\text{kN}$$

#### （3）水平中截面压力

**圆盘面积**：
$$A = \pi R^2 = 12.566\,\text{m}^2$$

**形心深度**（球心）：
$$h_c = R = 2\,\text{m}$$

**总压力**：
$$P_中 = \gamma h_c A = 9800 \times 2 \times 12.566 = 246,294\,\text{N} = 246.3\,\text{kN}$$

**方向**：竖直向下

**验证**：\(P_中 = P_{下} - P_{上} - \gamma V_{下半球} = ...\)（力平衡）

### 关键知识点

1. **封闭容器**：内外压力体分析
2. **球形容器**：压力分布均匀
3. **力平衡**：上下球面+中截面力平衡

## 题目029：椭圆形曲面 ⭐⭐⭐

### 题目描述

椭圆形曲面，方程\(\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1\)，\(a = 2\,\text{m}\)，\(b = 1\,\text{m}\)，宽度\(w = 1\,\text{m}\)。右半椭圆，凸向左侧水，水深\(h = 2\,\text{m}\)（至椭圆最高点）。

**求**：
1. 水平分力\(P_x\)
2. 垂直分力\(P_z\)（需用积分）
3. 总压力\(P\)

### 解答

#### （1）水平分力

**投影面**：矩形，\(2b \times w = 2 \times 1 = 2\,\text{m}^2\)

$$P_x = \gamma \times 1 \times 2 = 19,600\,\text{N} = 19.6\,\text{kN}$$

#### （2）垂直分力

**压力体体积**（半椭圆面积×宽度）：

半椭圆面积：
$$A_{半椭圆} = \frac{\pi ab}{2} = \frac{\pi \times 2 \times 1}{2} = \pi = 3.1416\,\text{m}^2$$

体积：
$$V = A \times w = 3.1416 \times 1 = 3.1416\,\text{m}^3$$

**垂直分力**：
$$P_z = \gamma V = 9800 \times 3.1416 = 30,788\,\text{N} = 30.8\,\text{kN}$$

#### （3）总压力

$$P = \sqrt{19.6^2 + 30.8^2} = 36.5\,\text{kN}$$

### 关键知识点

1. **椭圆面积**：\(A = \pi ab\)
2. **复杂曲面**：利用几何公式
3. **积分应用**：必要时数值积分

## 题目030：水轮机蜗壳压力 ⭐⭐⭐

### 题目描述

水轮机蜗壳简化为圆环形曲面，内半径\(r = 1\,\text{m}\)，外半径\(R = 2\,\text{m}\)，环面垂直放置，圆心在水下\(h = 10\,\text{m}\)处。内外两侧都有水。

**求**：
1. 外侧圆环面压力
2. 内侧圆环面压力
3. 净压力及工程意义

### 解答

#### （1）外侧圆环面压力

**投影面积**（圆环）：
$$A_{外} = \pi R^2 = \pi \times 2^2 = 12.566\,\text{m}^2$$

**形心深度**：
$$h_c = 10\,\text{m}$$

**压力**：
$$P_{外} = \gamma h_c A_{外} = 9800 \times 10 \times 12.566 = 1,231,468\,\text{N} = 1231.5\,\text{kN}$$

#### （2）内侧圆环面压力

**投影面积**：
$$A_{内} = \pi r^2 = \pi \times 1^2 = 3.1416\,\text{m}^2$$

**压力**：
$$P_{内} = 9800 \times 10 \times 3.1416 = 307,876\,\text{N} = 307.9\,\text{kN}$$

#### （3）净压力

**环面净压力**：
$$P_{净} = P_{外} - P_{内} = 1231.5 - 307.9 = 923.6\,\text{kN}$$

**工程意义**：
- 蜗壳承受巨大径向压力
- 需要加强环筋结构
- 厚壁设计，材料强度校核
- 焊缝质量要求高

### 关键知识点

1. **圆环压力**：外圈-内圈
2. **深水高压**：10m水深约1个大气压
3. **水轮机设计**：蜗壳是关键部件

### 常见错误

1. ❌ 压力体判断错误（实体/虚体）
2. ❌ 垂直分力方向搞反
3. ❌ 忘记圆弧面压力过圆心
4. ❌ 积分计算错误
5. ❌ 双侧受压忘记取矢量差
6. ❌ 投影面积计算错误

### 解题技巧

1. **画压力体**：判断实/虚，确定方向
2. **找投影**：水平分力用投影面
3. **算体积**：垂直分力用压力体体积
4. **矢量合成**：\(P = \sqrt{P_x^2 + P_z^2}\)
5. **特殊性质**：圆弧面压力过圆心
6. **对称分析**：球面等轴对称体

### 重点公式

| 类型 | 公式 | 说明 |
|------|------|------|
| 水平分力 | \(P_x = \gamma h_c A_x\) | 投影面积法 |
| 垂直分力 | \(P_z = \gamma V\) | 压力体法 |
| 总压力 | \(P = \sqrt{P_x^2 + P_z^2}\) | 矢量合成 |
| 方向角 | \(\alpha = \arctan(P_z/P_x)\) | 与水平面夹角 |
| 圆弧特性 | 总压力过圆心 | \(M_O = 0\) |

### 压力体判断口诀

```
凸面向上，压力体实，垂直向下；
凹面向上，压力体虚，垂直向上；
凸面向水，压力体实，垂直向内；
凹面向水，压力体虚，垂直向外。
```
