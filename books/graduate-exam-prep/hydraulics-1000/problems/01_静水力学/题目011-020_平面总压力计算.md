## 题目011：矩形竖直闸门总压力 ⭐

### 题目描述

矩形竖直闸门，宽度\(b = 3\,\text{m}\)，高度\(h = 4\,\text{m}\)，闸门顶部与水面齐平。水的容重\(\gamma = 9800\,\text{N/m}^3\)。

**求**：
1. 闸门所受的总压力\(P\)
2. 压力中心位置\(y_p\)（从水面算起）
3. 绘制压强分布图

### 解答

#### （1）总压力计算

**基本公式**：
$$P = \gamma h_c A$$

其中：
- \(\gamma = 9800\,\text{N/m}^3\)（水的容重）
- \(h_c = h/2 = 2\,\text{m}\)（形心深度）
- \(A = b \times h = 3 \times 4 = 12\,\text{m}^2\)（面积）

**计算**：
$$P = 9800 \times 2 \times 12 = 235,200\,\text{N} = 235.2\,\text{kN}$$

#### （2）压力中心位置

**基本公式**：
$$y_p = h_c + \frac{I_c}{h_c A}$$

对于矩形：
$$I_c = \frac{bh^3}{12} = \frac{3 \times 4^3}{12} = 16\,\text{m}^4$$

**计算**：
$$y_p = 2 + \frac{16}{2 \times 12} = 2 + 0.667 = 2.667\,\text{m}$$

**结论**：压力中心位于水面下2.667m，即距闸门顶部2.667m，距闸门底部1.333m。

#### （3）压强分布

- 闸门顶部（水面）：\(p = 0\)
- 闸门底部（水深4m）：\(p = \gamma h = 39.2\,\text{kPa}\)
- 分布形式：线性三角形分布
- 合力作用点：位于2.667m深度（三角形形心高度2h/3）

### 关键知识点

1. **总压力公式**：\(P = \gamma h_c A\)
2. **压力中心公式**：\(y_p = h_c + I_c/(h_c A)\)
3. **矩形惯性矩**：\(I_c = bh^3/12\)
4. **压力中心特点**：总在形心下方

## 题目012：倾斜矩形平板总压力 ⭐⭐

### 题目描述

矩形平板，宽度\(b = 2\,\text{m}\)，长度\(l = 3\,\text{m}\)，与水平面成\(\alpha = 60°\)角放置，平板顶边位于水面下\(d = 1\,\text{m}\)。

**求**：
1. 平板所受的总压力
2. 压力中心位置（沿平板方向）

### 解答

#### （1）总压力计算

**形心深度**：
- 平板顶边深度：\(h_1 = d = 1\,\text{m}\)
- 平板底边深度：\(h_2 = d + l \sin\alpha = 1 + 3 \times \sin 60° = 1 + 2.598 = 3.598\,\text{m}\)
- 形心深度：\(h_c = (h_1 + h_2)/2 = (1 + 3.598)/2 = 2.299\,\text{m}\)

**总压力**：
$$P = \gamma h_c A = 9800 \times 2.299 \times (2 \times 3) = 9800 \times 2.299 \times 6 = 135,162\,\text{N} = 135.2\,\text{kN}$$

#### （2）压力中心位置

**沿竖直方向的压力中心深度**：
$$y_p = h_c + \frac{I_c}{h_c A}$$

对于矩形（关于形心轴）：
$$I_c = \frac{bl^3 \sin^2\alpha}{12} = \frac{2 \times 3^3 \times (\sin 60°)^2}{12} = \frac{2 \times 27 \times 0.75}{12} = 3.375\,\text{m}^4$$

$$y_p = 2.299 + \frac{3.375}{2.299 \times 6} = 2.299 + 0.245 = 2.544\,\text{m}$$

**沿平板方向的距离（从顶边算起）**：
$$s_p = \frac{y_p - d}{\sin\alpha} = \frac{2.544 - 1}{\sin 60°} = \frac{1.544}{0.866} = 1.783\,\text{m}$$

### 关键知识点

1. **倾斜平板形心深度**：\(h_c = h_0 + (l/2)\sin\alpha\)
2. **惯性矩需考虑倾角**：\(I_c = (bl^3/12)\sin^2\alpha\)
3. **坐标转换**：竖直深度 ↔ 沿平板距离

## 题目013：圆形闸门总压力 ⭐⭐

### 题目描述

圆形闸门，直径\(D = 2\,\text{m}\)，闸门中心位于水面下\(h_c = 5\,\text{m}\)。

**求**：
1. 闸门所受的总压力
2. 压力中心位置（从圆心算起）

### 解答

#### （1）总压力

**面积**：
$$A = \frac{\pi D^2}{4} = \frac{\pi \times 2^2}{4} = \pi\,\text{m}^2 = 3.1416\,\text{m}^2$$

**总压力**：
$$P = \gamma h_c A = 9800 \times 5 \times 3.1416 = 153,938\,\text{N} = 153.9\,\text{kN}$$

#### （2）压力中心位置

**圆形对形心轴的惯性矩**：
$$I_c = \frac{\pi D^4}{64} = \frac{\pi \times 2^4}{64} = \frac{\pi \times 16}{64} = 0.7854\,\text{m}^4$$

**压力中心深度**：
$$y_p = h_c + \frac{I_c}{h_c A} = 5 + \frac{0.7854}{5 \times 3.1416} = 5 + 0.05 = 5.05\,\text{m}$$

**从圆心算起的距离**：
$$e = y_p - h_c = 0.05\,\text{m} = 5\,\text{cm}$$

压力中心位于圆心下方5cm。

### 关键知识点

1. **圆形面积**：\(A = \pi D^2/4\)
2. **圆形惯性矩**：\(I_c = \pi D^4/64\)
3. **压力中心偏移小**：当\(h_c\)较大时，\(e\)很小

## 题目014：三角形闸门总压力 ⭐⭐

### 题目描述

等腰三角形闸门，底边\(b = 2\,\text{m}\)（水平），高\(h = 3\,\text{m}\)，三角形顶点向下，底边与水面齐平。

**求**：
1. 闸门所受的总压力
2. 压力中心位置

### 解答

#### （1）总压力

**面积**：
$$A = \frac{1}{2}bh = \frac{1}{2} \times 2 \times 3 = 3\,\text{m}^2$$

**形心深度**（三角形形心在高的1/3处）：
$$h_c = \frac{h}{3} = \frac{3}{3} = 1\,\text{m}$$

**总压力**：
$$P = \gamma h_c A = 9800 \times 1 \times 3 = 29,400\,\text{N} = 29.4\,\text{kN}$$

#### （2）压力中心位置

**三角形对形心轴的惯性矩**：
$$I_c = \frac{bh^3}{36} = \frac{2 \times 3^3}{36} = \frac{54}{36} = 1.5\,\text{m}^4$$

**压力中心深度**：
$$y_p = h_c + \frac{I_c}{h_c A} = 1 + \frac{1.5}{1 \times 3} = 1 + 0.5 = 1.5\,\text{m}$$

从水面算起，压力中心位于1.5m深度，即闸门高度的1/2处（中点）。

### 关键知识点

1. **三角形形心**：在高的1/3处（从底边算起）
2. **三角形惯性矩**：\(I_c = bh^3/36\)
3. **压力中心在形心下方**：\(y_p = 3h_c/2\)（对于此三角形）

## 题目015：组合平板总压力 ⭐⭐⭐

### 题目描述

闸门由两部分组成：
- 上部：矩形，宽3m，高2m，顶边与水面齐平
- 下部：等腰三角形，底边3m（与矩形底边重合），高1m，顶点向下

**求**：
1. 各部分所受的总压力
2. 整体闸门的总压力和压力中心

### 解答

#### （1）上部矩形

- 面积：\(A_1 = 3 \times 2 = 6\,\text{m}^2\)
- 形心深度：\(h_{c1} = 1\,\text{m}\)
- 总压力：\(P_1 = 9800 \times 1 \times 6 = 58,800\,\text{N}\)
- 压力中心：\(y_{p1} = 1 + 16/(1 \times 6) = 1.33\,\text{m}\)（惯性矩\(I_{c1} = 3 \times 2^3/12 = 2\,\text{m}^4\)...修正为4m⁴）

实际计算：
$$I_{c1} = \frac{3 \times 2^3}{12} = 2\,\text{m}^4$$
$$y_{p1} = 1 + \frac{2}{1 \times 6} = 1.333\,\text{m}$$

#### （2）下部三角形

- 面积：\(A_2 = 0.5 \times 3 \times 1 = 1.5\,\text{m}^2\)
- 形心深度（从水面算）：\(h_{c2} = 2 + 1/3 = 2.333\,\text{m}\)
- 总压力：\(P_2 = 9800 \times 2.333 \times 1.5 = 34,296\,\text{N}\)
- 惯性矩：\(I_{c2} = 3 \times 1^3/36 = 0.0833\,\text{m}^4\)
- 压力中心：\(y_{p2} = 2.333 + 0.0833/(2.333 \times 1.5) = 2.357\,\text{m}\)

#### （3）整体闸门

**总压力**：
$$P = P_1 + P_2 = 58,800 + 34,296 = 93,096\,\text{N} = 93.1\,\text{kN}$$

**整体压力中心**（力矩平衡）：
$$y_p = \frac{P_1 y_{p1} + P_2 y_{p2}}{P} = \frac{58,800 \times 1.333 + 34,296 \times 2.357}{93,096} = \frac{78,370 + 80,831}{93,096} = 1.710\,\text{m}$$

### 关键知识点

1. **组合图形分部计算**：各部分单独计算后求和
2. **整体压力中心**：用力矩平衡法
3. **公式**：\(y_p = \sum P_i y_{pi} / \sum P_i\)

## 题目016：水箱侧壁总压力 ⭐

### 题目描述

矩形水箱，底面积5m×4m，水深3m，求四个侧壁各自受到的总压力。

### 解答

**长边侧壁（5m宽）**：
- 面积：\(A_1 = 5 \times 3 = 15\,\text{m}^2\)
- 形心深度：\(h_c = 1.5\,\text{m}\)
- 总压力：\(P_1 = 9800 \times 1.5 \times 15 = 220,500\,\text{N} = 220.5\,\text{kN}\)
- 两面共：\(2P_1 = 441\,\text{kN}\)

**短边侧壁（4m宽）**：
- 面积：\(A_2 = 4 \times 3 = 12\,\text{m}^2\)
- 形心深度：\(h_c = 1.5\,\text{m}\)
- 总压力：\(P_2 = 9800 \times 1.5 \times 12 = 176,400\,\text{N} = 176.4\,\text{kN}\)
- 两面共：\(2P_2 = 352.8\,\text{kN}\)

**总侧压力**：\(2P_1 + 2P_2 = 793.8\,\text{kN}\)

**注**：各侧壁压力中心均在水深2/3处，即水面下2m。

### 关键知识点

1. **对称性**：对边压力相等
2. **压力方向**：垂直于侧壁向内
3. **工程应用**：水箱设计、结构强度校核

## 题目017：闸门启闭力计算 ⭐⭐⭐

### 题目描述

矩形闸门，宽2m，高3m，顶边与水面齐平，底部铰接，顶部用绳索拉起。绳索与闸门垂直。

**求**：
1. 水压总力和压力中心
2. 绳索拉力\(T\)（闸门自重不计）

### 解答

#### （1）水压总力

- 形心深度：\(h_c = 1.5\,\text{m}\)
- 面积：\(A = 2 \times 3 = 6\,\text{m}^2\)
- 总压力：\(P = 9800 \times 1.5 \times 6 = 88,200\,\text{N} = 88.2\,\text{kN}\)
- 压力中心：\(y_p = 1.5 + 4.5/(1.5 \times 6) = 1.5 + 0.5 = 2\,\text{m}\)

#### （2）绳索拉力（力矩平衡）

以铰链点O（闸门底部）为力矩中心：
- 水压力矩（顺时针）：\(M_P = P \times (3 - y_p) = 88,200 \times (3 - 2) = 88,200\,\text{N·m}\)
- 绳索力矩（逆时针）：\(M_T = T \times 3\)

**力矩平衡**：
$$M_T = M_P$$
$$T \times 3 = 88,200$$
$$T = 29,400\,\text{N} = 29.4\,\text{kN}$$

### 关键知识点

1. **力矩平衡法**：\(\sum M = 0\)
2. **压力作用点**：关键在于确定\(y_p\)
3. **工程应用**：闸门启闭机设计

## 题目018：双侧受压平板 ⭐⭐

### 题目描述

竖直隔板分隔两个水箱，左侧水深4m，右侧水深2m，隔板宽3m。

**求**：
1. 左侧总压力
2. 右侧总压力
3. 隔板净压力及作用点

### 解答

#### （1）左侧总压力

- 形心深度：\(h_{c1} = 2\,\text{m}\)
- 面积：\(A = 3 \times 4 = 12\,\text{m}^2\)
- 总压力：\(P_1 = 9800 \times 2 \times 12 = 235,200\,\text{N} = 235.2\,\text{kN}\)
- 压力中心：\(y_{p1} = 2 + 16/(2 \times 12) = 2.667\,\text{m}\)

#### （2）右侧总压力

- 形心深度：\(h_{c2} = 1\,\text{m}\)
- 面积：\(A_2 = 3 \times 2 = 6\,\text{m}^2\)
- 总压力：\(P_2 = 9800 \times 1 \times 6 = 58,800\,\text{N} = 58.8\,\text{kN}\)
- 压力中心：\(y_{p2} = 1 + 2/(1 \times 6) = 1.333\,\text{m}\)

#### （3）净压力

**净压力大小**：
$$P_{净} = P_1 - P_2 = 235.2 - 58.8 = 176.4\,\text{kN}$$（方向：由左向右）

**净压力作用点**（力矩平衡）：
$$y_{p,净} = \frac{P_1 y_{p1} - P_2 y_{p2}}{P_{净}} = \frac{235.2 \times 2.667 - 58.8 \times 1.333}{176.4} = \frac{627.3 - 78.4}{176.4} = 3.11\,\text{m}$$

### 关键知识点

1. **双侧受压**：分别计算，求矢量差
2. **净压力中心**：用力矩法求
3. **注意**：净压力中心不在几何形心

## 题目019：闸门开启角度 ⭐⭐⭐

### 题目描述

矩形闸门，宽2m，高1.5m，底部铰接，可绕底边转动。上游水深2m，下游无水。闸门自重忽略。

**求**：
1. 闸门刚好开启时，需要多大的向上作用力\(F\)（作用在闸门顶部）？
2. 若闸门开启角度\(\theta = 30°\)（从竖直位置向下游倾斜），水压力和作用力的变化？

### 解答

#### （1）闸门竖直时的开启力

**水压总力**：
- 浸没部分：闸门高1.5m，水深2m，闸门全部浸没
- 形心深度：\(h_c = 2 - 1.5/2 = 1.25\,\text{m}\)（从水面算）
- 面积：\(A = 2 \times 1.5 = 3\,\text{m}^2\)
- 总压力：\(P = 9800 \times 1.25 \times 3 = 36,750\,\text{N}\)
- 压力中心（从铰链点算）：高度为\(h' = 1.5 - (2 - 1.75) = 1.25\,\text{m}\)（需重新计算）

实际：
- 压力中心深度：\(y_p = 1.25 + 0.5625/(1.25 \times 3) = 1.4\,\text{m}\)
- 从铰链点算：\(l_p = 2 - 1.4 = 0.6\,\text{m}\)

**力矩平衡**：
$$F \times 1.5 = P \times 0.6$$
$$F = \frac{36,750 \times 0.6}{1.5} = 14,700\,\text{N} = 14.7\,\text{kN}$$

#### （2）倾斜\(\theta = 30°\)时

当闸门倾斜，浸没深度和压力分布改变，需重新计算，此处略。

### 关键知识点

1. **力矩法**：开启力计算的关键
2. **压力作用点**：决定力臂长度
3. **动态分析**：闸门角度变化影响压力

## 题目020：潜水器舱门压力 ⭐⭐

### 题目描述

潜水器圆形舱门，直径\(D = 0.8\,\text{m}\)，舱门中心位于水下100m深度。海水密度\(\rho = 1025\,\text{kg/m}^3\)。

**求**：
1. 舱门受到的总压力
2. 若舱门材料为钛合金，截面积\(A_s = 0.01\,\text{m}^2\)，舱门框架受到的平均应力
3. 工程安全性分析

### 解答

#### （1）总压力

**面积**：
$$A = \frac{\pi D^2}{4} = \frac{\pi \times 0.8^2}{4} = 0.5027\,\text{m}^2$$

**形心深度**：\(h_c = 100\,\text{m}\)

**总压力**：
$$P = \rho g h_c A = 1025 \times 9.8 \times 100 \times 0.5027 = 505,218\,\text{N} = 505.2\,\text{kN}$$

约**50.5吨力**！

#### （2）框架应力

假设压力全部由舱门框架承担：
$$\sigma = \frac{P}{A_s} = \frac{505,218}{0.01} = 50,521,800\,\text{Pa} = 50.5\,\text{MPa}$$

#### （3）安全性分析

**钛合金性能**：
- 屈服强度：\(\sigma_s \approx 800-1000\,\text{MPa}\)
- 许用应力（安全系数2）：\(\sigma_{允} = 400-500\,\text{MPa}\)

**安全系数**：
$$n = \frac{\sigma_{允}}{\sigma} = \frac{400}{50.5} \approx 7.9$$

**结论**：设计安全，但需考虑：
- 密封圈性能
- 疲劳载荷
- 极端工况

### 关键知识点

1. **深海压强巨大**：100m深度约10个大气压
2. **材料选择关键**：钛合金高强度重量比
3. **工程安全系数**：一般取2-4
