## 📚 明渠流基础理论

### 基本概念

**明渠流**：有自由表面的流动
- 重力驱动（不同于管流的压力驱动）
- 自由水面（受大气压）
- 水深变化（影响流态）

**与管流对比**：
| 特性 | 明渠流 | 管流 |
|------|--------|------|
| 驱动力 | 重力 | 压力差 |
| 自由面 | 有 | 无 |
| 流速分布 | 不对称 | 对称 |
| 主要参数 | 水深h | 压强p |

### 水力要素

**几何要素**：
- \(A\)：过流断面面积（m²）
- \(P\)：湿周（与水接触的固体周界）（m）
- \(B\)：水面宽度（m）
- \(R\)：水力半径 \(R = A/P\)（m）
- \(\chi\)：水力半径（某些书记为\(R_h\)）

**矩形断面**：
- 面积：\(A = bh\)
- 湿周：\(P = b + 2h\)
- 水力半径：\(R = \frac{bh}{b + 2h}\)

**梯形断面**：
- 面积：\(A = (b + mh)h\)
- 湿周：\(P = b + 2h\sqrt{1 + m^2}\)
- 水力半径：\(R = \frac{(b + mh)h}{b + 2h\sqrt{1 + m^2}}\)

**圆形断面**（非满流）：
- 面积：\(A = \frac{D^2}{8}(\theta - \sin\theta)\)
- 湿周：\(P = \frac{D\theta}{2}\)

### 流量公式

**基本公式**：
$$Q = vA = v \cdot bh$$

**谢才公式**（Chezy）：
$$v = C\sqrt{Ri}$$

其中：
- \(C\)：谢才系数（m^{1/2}/s）
- \(i\)：水力坡度（渠底坡度，均匀流时）

**曼宁公式**（Manning）：
$$v = \frac{1}{n}R^{2/3}i^{1/2}$$

或：
$$Q = \frac{1}{n}AR^{2/3}i^{1/2}$$

其中：
- \(n\)：曼宁糙率系数
- 混凝土：\(n = 0.012-0.015\)
- 土渠：\(n = 0.020-0.025\)
- 天然河道：\(n = 0.025-0.040\)

**谢才系数与曼宁的关系**：
$$C = \frac{1}{n}R^{1/6}$$

### 断面比能（单位重量能量）

**定义**：
$$E = h + \frac{v^2}{2g} = h + \frac{Q^2}{2gA^2}$$

其中：
- \(h\)：水深（势能）
- \(\frac{v^2}{2g}\)：流速水头（动能）

**物理意义**：
以渠底为基准的总水头（不含位置水头）

## 第一部分：基础应用题（题451-465）

### 题目451：矩形明渠水力要素 ⭐

**题目**：
矩形明渠，底宽\(b = 3\,\text{m}\)，水深\(h = 2\,\text{m}\)。求：
1. 过流断面积\(A\)
2. 湿周\(P\)
3. 水力半径\(R\)
4. 水面宽度\(B\)

**解答**：

**步骤1**：过流断面积
$$A = bh = 3 \times 2 = 6\,\text{m}^2$$

**步骤2**：湿周
$$P = b + 2h = 3 + 2 \times 2 = 7\,\text{m}$$

**步骤3**：水力半径
$$R = \frac{A}{P} = \frac{6}{7} = 0.857\,\text{m}$$

**步骤4**：水面宽度
$$B = b = 3\,\text{m}\quad (\text{矩形断面水面宽度等于底宽})$$

**答案**：
- \(A = 6\,\text{m}^2\)
- \(P = 7\,\text{m}\)
- \(R = 0.857\,\text{m}\)
- \(B = 3\,\text{m}\)

### 题目452：梯形明渠水力要素 ⭐⭐

**题目**：
梯形断面明渠，底宽\(b = 2\,\text{m}\)，水深\(h = 1.5\,\text{m}\)，边坡系数\(m = 1.5\)（1:1.5）。求水力要素\(A, P, R, B\)。

**解答**：

**步骤1**：过流断面积
$$A = (b + mh)h = (2 + 1.5 \times 1.5) \times 1.5 = 6.5625\,\text{m}^2$$

**步骤2**：湿周
$$P = b + 2h\sqrt{1 + m^2} = 2 + 2 \times 1.5 \times \sqrt{1 + 1.5^2}$$
$$= 2 + 3 \times \sqrt{3.25} = 2 + 5.41 = 7.41\,\text{m}$$

**步骤3**：水力半径
$$R = \frac{A}{P} = \frac{6.5625}{7.41} = 0.886\,\text{m}$$

**步骤4**：水面宽度
$$B = b + 2mh = 2 + 2 \times 1.5 \times 1.5 = 6.5\,\text{m}$$

**答案**：
- \(A = 6.56\,\text{m}^2\)
- \(P = 7.41\,\text{m}\)
- \(R = 0.886\,\text{m}\)
- \(B = 6.5\,\text{m}\)

### 题目453：曼宁公式计算流速 ⭐⭐

**题目**：
矩形混凝土渠道，底宽\(b = 4\,\text{m}\)，水深\(h = 2\,\text{m}\)，渠底坡度\(i = 0.001\)，曼宁系数\(n = 0.014\)。求流速\(v\)和流量\(Q\)（均匀流）。

**解答**：

**步骤1**：水力要素
$$A = 4 \times 2 = 8\,\text{m}^2$$
$$P = 4 + 2 \times 2 = 8\,\text{m}$$
$$R = \frac{8}{8} = 1.0\,\text{m}$$

**步骤2**：曼宁公式求流速
$$v = \frac{1}{n}R^{2/3}i^{1/2} = \frac{1}{0.014} \times 1.0^{2/3} \times 0.001^{1/2}$$
$$= \frac{1}{0.014} \times 1.0 \times 0.03162 = 2.26\,\text{m/s}$$

**步骤3**：流量
$$Q = vA = 2.26 \times 8 = 18.08\,\text{m}^3/\text{s}$$

**答案**：
- 流速：\(v = 2.26\,\text{m/s}\)
- 流量：\(Q = 18.08\,\text{m}^3/\text{s}\)

### 题目454：断面比能计算 ⭐⭐

**题目**：
矩形明渠，底宽\(b = 3\,\text{m}\)，水深\(h = 2\,\text{m}\)，流速\(v = 1.5\,\text{m/s}\)。求断面比能\(E\)。

**解答**：

$$E = h + \frac{v^2}{2g} = 2 + \frac{1.5^2}{2 \times 9.8} = 2 + 0.115 = 2.115\,\text{m}$$

**答案**：\(E = 2.115\,\text{m}\)

**物理意义**：以渠底为基准的总能量（单位重量）。

### 题目455：谢才系数与曼宁关系 ⭐⭐

**题目**：
已知水力半径\(R = 1.2\,\text{m}\)，曼宁系数\(n = 0.020\)。求谢才系数\(C\)。

**解答**：

$$C = \frac{1}{n}R^{1/6} = \frac{1}{0.020} \times 1.2^{1/6}$$
$$= 50 \times 1.031 = 51.5\,\text{m}^{1/2}/\text{s}$$

**答案**：\(C = 51.5\,\text{m}^{1/2}/\text{s}\)

### 题目466：给定流量求水深 ⭐⭐⭐

**题目**：
矩形混凝土渠道，底宽\(b = 3\,\text{m}\)，渠底坡度\(i = 0.0008\)，\(n = 0.013\)，设计流量\(Q = 12\,\text{m}^3/\text{s}\)。求均匀流水深\(h\)（正常水深）。

**解答**（需迭代）：

**曼宁公式**：
$$Q = \frac{1}{n}AR^{2/3}i^{1/2}$$

**矩形断面**：
$$A = 3h, \quad P = 3 + 2h, \quad R = \frac{3h}{3 + 2h}$$

**代入**：
$$12 = \frac{1}{0.013} \times 3h \times \left(\frac{3h}{3 + 2h}\right)^{2/3} \times 0.0008^{1/2}$$

**简化**：
$$12 = \frac{0.02828}{0.013} \times 3h \times \left(\frac{3h}{3 + 2h}\right)^{2/3}$$

$$12 = 6.53 \times h \times \left(\frac{3h}{3 + 2h}\right)^{2/3}$$

**迭代求解**（或试算法）：
- 试\(h = 2.0\,\text{m}\)：\(Q \approx 11.5\,\text{m}^3/\text{s}\)（偏小）
- 试\(h = 2.1\,\text{m}\)：\(Q \approx 12.3\,\text{m}^3/\text{s}\)（偏大）
- 试\(h = 2.05\,\text{m}\)：\(Q \approx 11.9\,\text{m}^3/\text{s}\)
- 试\(h = 2.08\,\text{m}\)：\(Q \approx 12.05\,\text{m}^3/\text{s}\) ✓

**答案**：\(h \approx 2.08\,\text{m}\)

### 题目467：最优水力断面 ⭐⭐⭐

**题目**：
矩形断面，给定过流面积\(A = 10\,\text{m}^2\)，求湿周最小的底宽\(b\)和水深\(h\)。

**解答**：

**约束条件**：\(A = bh = 10\)

**目标**：最小化 \(P = b + 2h\)

**拉格朗日法**或**代入法**：
$$b = \frac{10}{h}$$
$$P(h) = \frac{10}{h} + 2h$$

**求导**：
$$\frac{dP}{dh} = -\frac{10}{h^2} + 2 = 0$$

**解得**：
$$h^2 = 5 \Rightarrow h = \sqrt{5} = 2.236\,\text{m}$$
$$b = \frac{10}{2.236} = 4.472\,\text{m}$$

**验证**：\(b = 2h\)（矩形最优断面的特性）

**答案**：
- 底宽：\(b = 4.47\,\text{m}\)
- 水深：\(h = 2.24\,\text{m}\)
- 湿周：\(P = 8.94\,\text{m}\)

**最优条件**：\(b = 2h\)

## 📝 明渠流基础总结（题451-480）

### 核心公式

**水力要素**：
- 面积：\(A = bh\)（矩形）
- 湿周：\(P = b + 2h\)
- 水力半径：\(R = A/P\)

**流量公式**：
- 曼宁：\(Q = \frac{1}{n}AR^{2/3}i^{1/2}\)
- 谢才：\(v = C\sqrt{Ri}\)

**断面比能**：
$$E = h + \frac{v^2}{2g}$$

### 常用断面

| 断面形状 | 面积\(A\) | 湿周\(P\) | 最优条件 |
|----------|-----------|-----------|----------|
| 矩形 | \(bh\) | \(b + 2h\) | \(b = 2h\) |
| 梯形 | \((b + mh)h\) | \(b + 2h\sqrt{1 + m^2}\) | 半六边形 |
| 圆形 | \(\frac{\pi D^2}{4}\) | \(\pi D\) | 满流 |

### 曼宁糙率\(n\)

| 渠道类型 | \(n\)值 |
|----------|---------|
| 玻璃、塑料 | 0.009-0.011 |
| 混凝土 | 0.012-0.015 |
| 浆砌石 | 0.015-0.020 |
| 土渠（光滑） | 0.020-0.025 |
| 土渠（粗糙） | 0.025-0.030 |
| 天然河道 | 0.030-0.050 |
