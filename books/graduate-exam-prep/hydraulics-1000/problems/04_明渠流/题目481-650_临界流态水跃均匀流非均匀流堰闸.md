## 第二部分：临界流态（题目481-510，30题）⭐⭐⭐⭐⭐

### 理论基础

**临界水深\(h_c\)**：
$$h_c = \sqrt[3]{\frac{q^2}{g}}\quad (\text{矩形断面，单宽流量}q = Q/b)$$

**弗劳德数Fr**：
$$Fr = \frac{v}{\sqrt{gh}}\quad (\text{矩形断面})$$

**流态判别**：
- \(Fr < 1\)：缓流（h > h_c）
- \(Fr = 1\)：临界流（h = h_c）
- \(Fr > 1\)：急流（h < h_c）

**断面比能E与水深h关系**：
$$E = h + \frac{v^2}{2g} = h + \frac{q^2}{2gh^2}$$

**临界水深时，E最小**：
$$E_{min} = \frac{3}{2}h_c$$

### 题目481：临界水深计算 ⭐⭐

矩形明渠，底宽\(b = 4\,\text{m}\)，流量\(Q = 20\,\text{m}^3/\text{s}\)。求临界水深\(h_c\)。

**解答**：
$$q = \frac{Q}{b} = \frac{20}{4} = 5\,\text{m}^2/\text{s}$$
$$h_c = \sqrt[3]{\frac{q^2}{g}} = \sqrt[3]{\frac{25}{9.8}} = 1.36\,\text{m}$$

### 题目482：弗劳德数与流态 ⭐⭐

矩形明渠，\(b = 3\,\text{m}\)，\(h = 2\,\text{m}\)，\(v = 2.5\,\text{m/s}\)。判断流态。

**解答**：
$$Fr = \frac{v}{\sqrt{gh}} = \frac{2.5}{\sqrt{9.8 \times 2}} = 0.566 < 1$$

**答案**：缓流

## 第三部分：水跃（题目511-535，25题）

### 理论基础

**水跃**：急流向缓流过渡时发生的局部急剧水位抬升现象。

**共轭水深公式**（矩形断面）：
$$\frac{h_2}{h_1} = \frac{1}{2}\left(\sqrt{1 + 8Fr_1^2} - 1\right)$$

其中：
- \(h_1\)：跃前水深（急流）
- \(h_2\)：跃后水深（缓流）
- \(Fr_1 = v_1/\sqrt{gh_1}\)

**水跃长度**：
$$L_j \approx (5-7)h_2$$

**水跃损失**：
$$\Delta E = E_1 - E_2 = \frac{(h_2 - h_1)^3}{4h_1 h_2}$$

### 题目511：水跃计算 ⭐⭐⭐

矩形明渠，跃前：\(h_1 = 0.5\,\text{m}\)，\(v_1 = 8\,\text{m/s}\)。求跃后水深\(h_2\)和水跃损失。

**解答**：
$$Fr_1 = \frac{8}{\sqrt{9.8 \times 0.5}} = 3.62$$
$$\frac{h_2}{h_1} = \frac{1}{2}(\sqrt{1 + 8 \times 3.62^2} - 1) = 4.33$$
$$h_2 = 4.33 \times 0.5 = 2.17\,\text{m}$$

## 第四部分：明渠均匀流（题目536-575，40题）⭐⭐⭐⭐⭐

### 理论基础

**均匀流定义**：
- 水深、流速沿程不变
- 水面平行于渠底
- 水力坡度 = 渠底坡度 = 能量坡度

**正常水深\(h_0\)**：
均匀流水深，满足曼宁公式：
$$Q = \frac{1}{n}A(h_0)R^{2/3}(h_0)i^{1/2}$$

**临界底坡\(i_c\)**：
使正常水深等于临界水深的坡度：
$$i_c = \frac{n^2 v_c^2}{R_c^{4/3}}$$

**底坡分类**：
- \(i > i_c\)：陡坡（\(h_0 < h_c\)，急流）
- \(i = i_c\)：临界坡（\(h_0 = h_c\)）
- \(i < i_c\)：缓坡（\(h_0 > h_c\)，缓流）
- \(i = 0\)：平坡
- \(i < 0\)：逆坡

### 题目536：正常水深计算 ⭐⭐⭐

矩形渠道，\(b = 5\,\text{m}\)，\(i = 0.001\)，\(n = 0.015\)，\(Q = 15\,\text{m}^3/\text{s}\)。求正常水深\(h_0\)。

**解答**（迭代）：
$$15 = \frac{1}{0.015} \times 5h_0 \times \left(\frac{5h_0}{5 + 2h_0}\right)^{2/3} \times 0.001^{1/2}$$

## 第五部分：明渠非均匀流（题目576-615，40题）

### 理论基础

**非均匀流**：水深沿程变化

**水面曲线分类**（12种）：

**缓坡（M类）**：
- M1：壅水曲线（下游水位抬高）
- M2：降水曲线（接近临界）
- M3：跌水曲线（接近陡坡）

**陡坡（S类）**：
- S1：壅水曲线
- S2：降水曲线
- S3：跌水曲线

**临界坡（C类）**：
- C1、C3

**平坡（H类）**：
- H2、H3

**逆坡（A类）**：
- A2、A3

**水面曲线方程**：
$$\frac{dh}{dx} = \frac{i - i_f}{1 - Fr^2}$$

### 题目576：水面曲线类型判别 ⭐⭐⭐

缓坡渠道，正常水深\(h_0 = 3\,\text{m}\)，临界水深\(h_c = 2\,\text{m}\)，下游水位抬高至\(h = 4\,\text{m}\)。判断水面曲线类型。

**解答**：
\(h = 4 > h_0 = 3 > h_c = 2\)，缓坡，\(h > h_0\)

**答案**：M1壅水曲线

## 第六部分：堰闸出流（题目616-650，35题）

### 理论基础

**薄壁堰**：
$$Q = m b \sqrt{2g} H^{3/2}$$

其中：
- \(m\)：流量系数（0.40-0.49）
- \(b\)：堰宽
- \(H\)：堰上水头

**实用堰**：
$$Q = m b H^{n}$$

**宽顶堰**（临界流）：
$$Q = m b \sqrt{2g} H_0^{3/2}$$

其中\(H_0\)为包括流速水头的总水头。

**闸孔出流**：

**自由出流**：
$$Q = \mu b a \sqrt{2g H_0}$$

**淹没出流**：
$$Q = \mu_s b a \sqrt{2g(H_1 - H_2)}$$

其中：
- \(a\)：闸门开度
- \(H_1, H_2\)：上下游水头

### 题目616：薄壁堰流量 ⭐⭐

矩形薄壁堰，堰宽\(b = 2\,\text{m}\)，堰上水头\(H = 0.5\,\text{m}\)，\(m = 0.42\)。求流量。

**解答**：
$$Q = 0.42 \times 2 \times \sqrt{2 \times 9.8} \times 0.5^{1.5} = 0.42 \times 2 \times 4.43 \times 0.354 = 1.32\,\text{m}^3/\text{s}$$
