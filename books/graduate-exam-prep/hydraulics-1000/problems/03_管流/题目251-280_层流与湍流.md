## 📚 层流与湍流理论基础

### 雷诺数（Reynolds Number）

**定义**：
$$Re = \frac{\rho vd}{\mu} = \frac{vd}{\nu}$$

其中：
- \(v\)：平均流速（m/s）
- \(d\)：管道直径（m）
- \(\rho\)：密度（kg/m³）
- \(\mu\)：动力粘度（Pa·s）
- \(\nu = \mu/\rho\)：运动粘度（m²/s）

**物理意义**：
$$Re = \frac{\text{惯性力}}{\text{粘性力}}$$

### 流态判别

**临界雷诺数**（圆管）：
$$Re_{临界} = 2320$$

**流态分类**：
- \(Re < 2000\)：层流
- \(2000 < Re < 4000\)：过渡区
- \(Re > 4000\)：湍流

**实际工程**：
- 一般认为 \(Re > 2320\) 为湍流
- 水管：通常为湍流（\(Re > 10^5\)）
- 油管：可能为层流（粘度大）

### 层流特性

**速度分布**（圆管）：
$$v(r) = v_{max}\left(1 - \frac{r^2}{R^2}\right)$$

其中：
- \(v_{max}\)：轴心最大流速
- \(r\)：距管轴距离
- \(R\)：管道半径

**抛物线分布**！

**平均流速与最大流速关系**：
$$\bar{v} = \frac{1}{2}v_{max}$$

**压降计算**（哈根-泊肃叶公式）：
$$\Delta p = \frac{32\mu L v}{d^2} = \frac{128\mu L Q}{\pi d^4}$$

**沿程损失**：
$$h_f = \frac{32\mu L v}{\rho g d^2} = \frac{64}{\rho vd/\mu} \cdot \frac{L}{d} \cdot \frac{v^2}{2g} = \frac{64}{Re} \cdot \frac{L}{d} \cdot \frac{v^2}{2g}$$

**达西公式形式**：
$$h_f = \lambda \frac{L}{d} \frac{v^2}{2g}$$

其中层流阻力系数：
$$\lambda = \frac{64}{Re}$$

### 湍流特性

**速度分布**（时均）：
$$\frac{v}{v_*} = 5.75\log\left(\frac{y}{\delta}\right) + 8.5$$

（对数分布，复杂）

**简化指数公式**（1/7次方律）：
$$\frac{v}{v_{max}} = \left(\frac{y}{R}\right)^{1/7}$$

**平均流速与最大流速关系**：
$$\bar{v} \approx 0.82 v_{max}$$

**阻力系数**（光滑管）：
$$\lambda = \frac{0.3164}{Re^{0.25}}\quad (Re < 10^5)$$

$$\lambda = \frac{0.0032 + 0.221}{Re^{0.237}}\quad (\text{更精确})$$

**尼古拉兹公式**（粗糙管）：
$$\frac{1}{\sqrt{\lambda}} = -2\log\left(\frac{\Delta/d}{3.7} + \frac{2.51}{Re\sqrt{\lambda}}\right)$$

（科尔布鲁克公式，隐式，需迭代）

## 第一部分：基础应用题（题251-265）

### 题目251：雷诺数计算与流态判别 ⭐

**题目**：
圆管，直径\(d = 0.1\,\text{m}\)，水（20°C，\(\nu = 1.0 \times 10^{-6}\,\text{m}^2/\text{s}\)）流速\(v = 0.5\,\text{m/s}\)。判断流态。

**解答**：

**步骤1**：计算雷诺数
$$Re = \frac{vd}{\nu} = \frac{0.5 \times 0.1}{1.0 \times 10^{-6}} = 50,000$$

**步骤2**：判别流态
$$Re = 50,000 > 2320\quad \Rightarrow\quad \text{湍流}$$

**答案**：湍流

---

### 题目252：层流转湍流临界流速 ⭐⭐

**题目**：
圆管，直径\(d = 0.05\,\text{m}\)，水（20°C）。求：
1. 层流最大流速\(v_{层,max}\)
2. 湍流最小流速\(v_{湍,min}\)

**解答**：

**步骤1**：层流最大雷诺数
$$Re_{层,max} = 2000$$

**步骤2**：层流最大流速
$$v_{层,max} = \frac{Re_{层,max} \times \nu}{d} = \frac{2000 \times 1.0 \times 10^{-6}}{0.05} = 0.04\,\text{m/s} = 4\,\text{cm/s}$$

**步骤3**：湍流最小雷诺数
$$Re_{湍,min} = 4000$$

**步骤4**：湍流最小流速
$$v_{湍,min} = \frac{4000 \times 1.0 \times 10^{-6}}{0.05} = 0.08\,\text{m/s} = 8\,\text{cm/s}$$

**答案**：
- 层流最大流速：4 cm/s
- 湍流最小流速：8 cm/s
- 过渡区：4-8 cm/s

### 题目253：层流速度分布 ⭐⭐

**题目**：
圆管层流，直径\(d = 0.1\,\text{m}\)，轴心最大流速\(v_{max} = 2\,\text{m/s}\)。求：
1. 平均流速\(\bar{v}\)
2. 管壁处(\(r = R\))流速
3. \(r = R/2\)处流速

**解答**：

**步骤1**：平均流速
$$\bar{v} = \frac{v_{max}}{2} = \frac{2}{2} = 1\,\text{m/s}$$

**步骤2**：管壁流速
$$v(R) = v_{max}\left(1 - \frac{R^2}{R^2}\right) = 0$$

（无滑移条件）

**步骤3**：\(r = R/2\)处流速
$$v(R/2) = v_{max}\left(1 - \frac{(R/2)^2}{R^2}\right) = 2 \times \left(1 - \frac{1}{4}\right) = 1.5\,\text{m/s}$$

**答案**：
- 平均流速：1 m/s
- 管壁流速：0
- \(r = R/2\)流速：1.5 m/s

## 第二部分：中等难度题（题266-275）

### 题目266：层流压降计算 ⭐⭐⭐

**题目**：
水平圆管层流，直径\(d = 0.02\,\text{m}\)，长度\(L = 10\,\text{m}\)，油（\(\mu = 0.1\,\text{Pa·s}\)，\(\rho = 900\,\text{kg/m}^3\)），流速\(v = 0.05\,\text{m/s}\)。求：
1. 雷诺数Re
2. 压降\(\Delta p\)
3. 沿程损失\(h_f\)

**解答**：

**步骤1**：雷诺数
$$Re = \frac{\rho vd}{\mu} = \frac{900 \times 0.05 \times 0.02}{0.1} = 9$$

（\(Re = 9 < 2320\)，确认为层流）

**步骤2**：压降（哈根-泊肃叶）
$$\Delta p = \frac{32\mu L v}{d^2} = \frac{32 \times 0.1 \times 10 \times 0.05}{0.02^2} = 4000\,\text{Pa} = 4\,\text{kPa}$$

**步骤3**：沿程损失
$$h_f = \frac{\Delta p}{\rho g} = \frac{4000}{900 \times 9.8} = 0.453\,\text{m}$$

**或者用达西公式**：
$$\lambda = \frac{64}{Re} = \frac{64}{9} = 7.11$$
$$h_f = \lambda \frac{L}{d} \frac{v^2}{2g} = 7.11 \times \frac{10}{0.02} \times \frac{0.05^2}{2 \times 9.8} = 0.453\,\text{m}$$

**答案**：
- \(Re = 9\)（层流）
- \(\Delta p = 4\,\text{kPa}\)
- \(h_f = 0.453\,\text{m}\)

### 题目267：湍流阻力系数 ⭐⭐⭐

**题目**：
光滑圆管，直径\(d = 0.2\,\text{m}\)，水（20°C），流速\(v = 2\,\text{m/s}\)。求：
1. 雷诺数Re
2. 阻力系数\(\lambda\)（用布拉修斯公式）
3. 若流速增加到\(v = 4\,\text{m/s}\)，\(\lambda\)如何变化？

**解答**：

**步骤1**：雷诺数
$$Re = \frac{vd}{\nu} = \frac{2 \times 0.2}{1.0 \times 10^{-6}} = 400,000$$

（湍流）

**步骤2**：阻力系数（布拉修斯公式，\(Re < 10^5\)不太适用，用经验公式）
$$\lambda = \frac{0.3164}{Re^{0.25}} = \frac{0.3164}{400,000^{0.25}} = \frac{0.3164}{25.12} = 0.0126$$

**步骤3**：流速增加
$$Re' = 2 \times Re = 800,000$$
$$\lambda' = \frac{0.3164}{800,000^{0.25}} = \frac{0.3164}{29.86} = 0.0106$$

**变化**：
$$\frac{\lambda'}{\lambda} = \frac{0.0106}{0.0126} = 0.84$$

阻力系数减小约16%。

**规律**：湍流中，Re增大，\(\lambda\)减小（但减小速度慢于层流）。

## 第三部分：困难题（题276-280）

### 题目276：流态转换的稳定性分析 ⭐⭐⭐⭐

**题目**：
解释为什么层流转湍流的雷诺数（2320）远小于湍流转层流的雷诺数（可达1000以下）？这种滞后现象的物理机制是什么？

**解答**（定性分析）：

**层流→湍流**：
- 小扰动→放大→失稳
- 临界Re较低（2320）
- 容易发生

**湍流→层流**：
- 需要湍流涡旋完全耗散
- 需要外部保持极端稳定
- 临界Re可低至1000以下
- 难以实现（实验室条件）

**滞后现象**：流态具有"历史记忆"。

## 📝 层流与湍流总结（题251-280）

### 核心对比

| 特性 | 层流 | 湍流 |
|------|------|------|
| **雷诺数** | \(Re < 2000\) | \(Re > 4000\) |
| **速度分布** | 抛物线 | 对数/1/7次方 |
| **\(\bar{v}/v_{max}\)** | 0.5 | 0.82 |
| **阻力系数** | \(\lambda = 64/Re\) | \(\lambda = 0.3164/Re^{0.25}\) |
| **压降随流速** | \(\Delta p \propto v\) | \(\Delta p \propto v^{1.75\sim 2}\) |
| **流动特征** | 有序、分层 | 混乱、涡旋 |
| **能量损失** | 小 | 大 |
| **应用** | 润滑油、血液 | 水管、风管 |

### 重要公式

**雷诺数**：
$$Re = \frac{vd}{\nu} = \frac{\rho vd}{\mu}$$

**层流阻力系数**：
$$\lambda = \frac{64}{Re}$$

**湍流阻力系数**（光滑管）：
$$\lambda = \frac{0.3164}{Re^{0.25}}$$

**达西-魏斯巴赫公式**：
$$h_f = \lambda \frac{L}{d} \frac{v^2}{2g}$$

**层流速度分布**：
$$v(r) = v_{max}\left(1 - \frac{r^2}{R^2}\right), \quad \bar{v} = \frac{v_{max}}{2}$$

**湍流速度分布**（1/7次方）：
$$\frac{v}{v_{max}} = \left(\frac{y}{R}\right)^{1/7}, \quad \bar{v} \approx 0.82 v_{max}$$
