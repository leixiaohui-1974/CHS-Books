## 📚 沿程阻力理论基础

### 达西-魏斯巴赫公式

**基本形式**：
$$h_f = \lambda \frac{L}{d} \frac{v^2}{2g}$$

其中：
- \(h_f\)：沿程损失水头（m）
- \(\lambda\)：沿程阻力系数（无量纲）
- \(L\)：管道长度（m）
- \(d\)：管道直径（m）
- \(v\)：平均流速（m/s）

**物理意义**：
$$h_f \propto \frac{L}{d} \times \frac{v^2}{2g}$$

长度越长、直径越小、流速越大 → 损失越大

### 阻力系数\(\lambda\)的确定

**1. 层流**（\(Re < 2320\)）：
$$\lambda = \frac{64}{Re}$$

**2. 湍流光滑管**：

**(a) 布拉修斯公式**（\(Re < 10^5\)）：
$$\lambda = \frac{0.3164}{Re^{0.25}}$$

**(b) 柯尔莫果洛夫公式**（\(10^4 < Re < 3 \times 10^6\)）：
$$\lambda = \frac{1}{(1.8\log Re - 1.5)^2}$$

**3. 湍流粗糙管**：

**(a) 尼古拉兹公式**（均匀粗糙）：
$$\frac{1}{\sqrt{\lambda}} = 2\log\left(\frac{d}{\Delta}\right) + 1.14$$

（\(\Delta\)：绝对粗糙度）

**(b) 科尔布鲁克公式**（工程常用）：
$$\frac{1}{\sqrt{\lambda}} = -2\log\left(\frac{\Delta/d}{3.7} + \frac{2.51}{Re\sqrt{\lambda}}\right)$$

（隐式方程，需迭代求解）

**(c) 舍维列夫公式**（中国水管）：
$$\lambda = 0.021 \times 10^{-\frac{\Delta}{d}} + \frac{0.0075}{Re^{0.3}}$$

### 莫迪图（Moody Diagram）

**横轴**：\(Re\)  
**纵轴**：\(\lambda\)  
**参数**：\(\Delta/d\)（相对粗糙度）

**使用方法**：
1. 计算\(Re\)
2. 确定\(\Delta/d\)
3. 在莫迪图上查\(\lambda\)

### 管道粗糙度

| 管材 | 绝对粗糙度\(\Delta\) (mm) |
|------|--------------------------|
| 铜管、玻璃管 | 0.0015-0.01 |
| 拉制钢管 | 0.01-0.05 |
| 新铸铁管 | 0.25-1.0 |
| 旧铸铁管 | 1.0-3.0 |
| 混凝土管 | 0.3-3.0 |
| 镀锌钢管 | 0.1-0.3 |

## 第一部分：基础应用题（题281-295）

### 题目281：达西公式基本计算 ⭐

**题目**：
水平圆管，直径\(d = 0.2\,\text{m}\)，长度\(L = 100\,\text{m}\)，流速\(v = 1.5\,\text{m/s}\)，阻力系数\(\lambda = 0.02\)。求沿程损失\(h_f\)。

**解答**：

$$h_f = \lambda \frac{L}{d} \frac{v^2}{2g} = 0.02 \times \frac{100}{0.2} \times \frac{1.5^2}{2 \times 9.8}$$
$$h_f = 0.02 \times 500 \times 0.1148 = 1.148\,\text{m}$$

**答案**：\(h_f = 1.148\,\text{m}\)

### 题目282：层流阻力系数与损失 ⭐⭐

**题目**：
油管，直径\(d = 0.05\,\text{m}\)，长度\(L = 20\,\text{m}\)，油（\(\nu = 2 \times 10^{-4}\,\text{m}^2/\text{s}\)），流速\(v = 0.2\,\text{m/s}\)。求：
1. 雷诺数Re
2. 阻力系数\(\lambda\)
3. 沿程损失\(h_f\)

**解答**：

**步骤1**：雷诺数
$$Re = \frac{vd}{\nu} = \frac{0.2 \times 0.05}{2 \times 10^{-4}} = 50$$

（层流）

**步骤2**：阻力系数
$$\lambda = \frac{64}{Re} = \frac{64}{50} = 1.28$$

**步骤3**：沿程损失
$$h_f = 1.28 \times \frac{20}{0.05} \times \frac{0.2^2}{2 \times 9.8} = 1.28 \times 400 \times 0.00204 = 1.044\,\text{m}$$

**答案**：
- \(Re = 50\)（层流）
- \(\lambda = 1.28\)
- \(h_f = 1.044\,\text{m}\)

### 题目283：湍流光滑管阻力系数 ⭐⭐

**题目**：
水管（光滑），直径\(d = 0.3\,\text{m}\)，流速\(v = 2\,\text{m/s}\)，水（20°C，\(\nu = 1.0 \times 10^{-6}\,\text{m}^2/\text{s}\)）。用布拉修斯公式求\(\lambda\)。

**解答**：

**步骤1**：雷诺数
$$Re = \frac{vd}{\nu} = \frac{2 \times 0.3}{1.0 \times 10^{-6}} = 600,000$$

**步骤2**：布拉修斯公式（虽然\(Re > 10^5\)，用作近似）
$$\lambda = \frac{0.3164}{Re^{0.25}} = \frac{0.3164}{600,000^{0.25}} = \frac{0.3164}{27.8} = 0.0114$$

**答案**：\(\lambda = 0.0114\)

## 第二部分：中等难度题（题296-310）

### 题目296：粗糙管阻力系数 ⭐⭐⭐

**题目**：
旧铸铁管，直径\(d = 0.4\,\text{m}\)，绝对粗糙度\(\Delta = 2\,\text{mm}\)，水（\(\nu = 1.0 \times 10^{-6}\,\text{m}^2/\text{s}\)），流速\(v = 1.5\,\text{m/s}\)。用科尔布鲁克公式求\(\lambda\)（迭代1次）。

**解答**：

**步骤1**：雷诺数
$$Re = \frac{1.5 \times 0.4}{1.0 \times 10^{-6}} = 600,000$$

**步骤2**：相对粗糙度
$$\frac{\Delta}{d} = \frac{2}{400} = 0.005$$

**步骤3**：科尔布鲁克公式
$$\frac{1}{\sqrt{\lambda}} = -2\log\left(\frac{0.005}{3.7} + \frac{2.51}{600,000\sqrt{\lambda}}\right)$$

**初值**：假设\(\lambda_0 = 0.025\)

$$\frac{1}{\sqrt{0.025}} = 6.32$$

$$\frac{1}{\sqrt{\lambda_1}} = -2\log\left(\frac{0.005}{3.7} + \frac{2.51}{600,000 \times 6.32}\right)$$
$$= -2\log(0.00135 + 0.0000066) = -2\log(0.001357) = 5.69$$

$$\lambda_1 = \frac{1}{5.69^2} = 0.0309$$

**答案**：\(\lambda \approx 0.031\)（粗糙管阻力系数明显大于光滑管）

### 题目297：管道压降与功率损失 ⭐⭐⭐

**题目**：
输水管道，直径\(d = 0.5\,\text{m}\)，长度\(L = 1000\,\text{m}\)，流量\(Q = 0.3\,\text{m}^3/\text{s}\)，\(\lambda = 0.02\)。求：
1. 沿程损失\(h_f\)
2. 压降\(\Delta p\)
3. 功率损失\(P_损\)

**解答**：

**步骤1**：流速
$$v = \frac{Q}{A} = \frac{0.3}{\pi \times 0.25^2} = 1.53\,\text{m/s}$$

**步骤2**：沿程损失
$$h_f = 0.02 \times \frac{1000}{0.5} \times \frac{1.53^2}{19.6} = 0.02 \times 2000 \times 0.119 = 4.76\,\text{m}$$

**步骤3**：压降
$$\Delta p = \rho g h_f = 1000 \times 9.8 \times 4.76 = 46,648\,\text{Pa} = 46.6\,\text{kPa}$$

**步骤4**：功率损失
$$P_损 = \gamma Q h_f = 9800 \times 0.3 \times 4.76 = 14,006\,\text{W} = 14\,\text{kW}$$

**答案**：
- \(h_f = 4.76\,\text{m}\)
- \(\Delta p = 46.6\,\text{kPa}\)
- \(P_损 = 14\,\text{kW}\)

## 📝 沿程阻力总结（题281-320）

### 核心公式

**达西公式**：
$$h_f = \lambda \frac{L}{d} \frac{v^2}{2g}$$

**阻力系数**：
- 层流：\(\lambda = 64/Re\)
- 湍流光滑管：\(\lambda = 0.3164/Re^{0.25}\)
- 湍流粗糙管：科尔布鲁克公式（隐式）

**压降**：
$$\Delta p = \rho g h_f = \lambda \frac{L}{d} \frac{\rho v^2}{2}$$

**功率损失**：
$$P = \gamma Q h_f$$

### 工程常用值

| 管材 | \(\Delta\) (mm) | 典型\(\lambda\) |
|------|-----------------|----------------|
| 新钢管 | 0.05 | 0.015-0.020 |
| 新铸铁 | 0.3 | 0.020-0.025 |
| 旧铸铁 | 2.0 | 0.030-0.040 |
| 混凝土 | 1.0 | 0.020-0.030 |
| 塑料管 | 0.01 | 0.010-0.015 |

### 影响因素

**阻力系数\(\lambda\)**：
1. ✅ 雷诺数Re（层流和湍流光滑区）
2. ✅ 相对粗糙度\(\Delta/d\)（湍流粗糙区）
3. ✅ 管材、使用年限

**沿程损失\(h_f\)**：
1. ✅ 管长\(L\)（正比）
2. ✅ 管径\(d\)（反比）
3. ✅ 流速\(v\)（湍流约为\(v^{1.75\sim 2}\)）
4. ✅ 阻力系数\(\lambda\)
