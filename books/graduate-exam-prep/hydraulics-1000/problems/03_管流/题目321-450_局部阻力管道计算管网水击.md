## 第四部分：局部阻力（题目321-350，30题）

### 理论基础

**局部阻力公式**：
$$h_m = \zeta \frac{v^2}{2g}$$

其中\(\zeta\)为局部阻力系数

**常见局部阻力**：
- 突然扩大：\(\zeta = (1 - A_1/A_2)^2\)
- 突然缩小：\(\zeta = 0.5(1 - A_2/A_1)\)
- 90°弯头：\(\zeta = 0.9-1.1\)
- 闸阀全开：\(\zeta = 0.2\)
- 闸阀半开：\(\zeta = 2.1\)
- 入口：\(\zeta = 0.5\)
- 出口：\(\zeta = 1.0\)

### 题目321：突然扩大损失 ⭐⭐

管道突然扩大，\(d_1 = 0.1\,\text{m}\)，\(d_2 = 0.2\,\text{m}\)，\(v_1 = 2\,\text{m/s}\)。求局部损失\(h_m\)。

**解答**：
$$\zeta = \left(1 - \frac{A_1}{A_2}\right)^2 = \left(1 - \frac{0.1^2}{0.2^2}\right)^2 = 0.5625$$
$$h_m = 0.5625 \times \frac{2^2}{19.6} = 0.1148\,\text{m}$$

## 第五部分：管道计算（题目351-400，50题）⭐⭐⭐⭐⭐

### 三类基本问题

**I类问题**：已知\(d, L, \lambda, Q\) → 求\(h_f\)  
**II类问题**：已知\(d, L, \lambda, h_f\) → 求\(Q\)  
**III类问题**：已知\(Q, L, h_f, \lambda\) → 求\(d\)

### 题目351：I类问题 ⭐⭐

已知\(d = 0.3\,\text{m}\)，\(L = 500\,\text{m}\)，\(\lambda = 0.025\)，\(Q = 0.2\,\text{m}^3/\text{s}\)。求\(h_f\)。

**解答**：
$$v = \frac{Q}{A} = \frac{0.2}{\pi \times 0.15^2} = 2.83\,\text{m/s}$$
$$h_f = 0.025 \times \frac{500}{0.3} \times \frac{2.83^2}{19.6} = 17.0\,\text{m}$$

### 题目352：II类问题 ⭐⭐⭐

已知\(d = 0.2\,\text{m}\)，\(L = 300\,\text{m}\)，\(\lambda = 0.02\)，\(h_f = 5\,\text{m}\)。求\(Q\)。

**解答**：
$$v = \sqrt{\frac{h_f \times 2g \times d}{\lambda \times L}} = \sqrt{\frac{5 \times 19.6 \times 0.2}{0.02 \times 300}} = 1.81\,\text{m/s}$$
$$Q = vA = 1.81 \times \pi \times 0.1^2 = 0.0569\,\text{m}^3/\text{s}$$

### 题目353：III类问题 ⭐⭐⭐⭐

已知\(Q = 0.1\,\text{m}^3/\text{s}\)，\(L = 1000\,\text{m}\)，\(h_f = 10\,\text{m}\)，\(\lambda = 0.02\)。求\(d\)。

**解答**（需迭代）：
$$d = \left(\frac{8\lambda L Q^2}{\pi^2 g h_f}\right)^{1/5}$$

（经济管径设计中的核心问题）

## 第六部分：管网分析（题目401-430，30题）

### 基本原理

**节点连续性**：
$$\sum Q_{入} = \sum Q_{出}$$

**回路能量守恒**：
$$\sum h_f = \sum \Delta H$$

### 哈代-克罗斯法（Hardy-Cross Method）

**迭代公式**：
$$\Delta Q = -\frac{\sum h_f}{\sum(2h_f/Q)}$$

### 题目401：简单管网 ⭐⭐⭐

两管并联：管1（\(d_1 = 0.2\,\text{m}\)，\(L_1 = 500\,\text{m}\)，\(\lambda_1 = 0.02\)），管2（\(d_2 = 0.15\,\text{m}\)，\(L_2 = 500\,\text{m}\)，\(\lambda_2 = 0.025\)）。总流量\(Q = 0.2\,\text{m}^3/\text{s}\)，求流量分配。

**解答**：
并联管道：\(h_{f1} = h_{f2}\)

## 第七部分：水击（题目431-450，20题）

### 基本理论

**水击压强**（刚性水击）：
$$\Delta p = \rho c \Delta v$$

其中\(c\)为水击波速：
$$c = \frac{1}{\sqrt{\rho\left(\frac{1}{K} + \frac{d}{E\delta}\right)}}$$

- \(K\)：水的体积弹性模量（约\(2.1 \times 10^9\,\text{Pa}\)）
- \(E\)：管壁弹性模量（钢：\(2.1 \times 10^{11}\,\text{Pa}\)）
- \(\delta\)：管壁厚度

### 题目431：水击波速 ⭐⭐⭐

钢管，\(d = 0.5\,\text{m}\)，\(\delta = 10\,\text{mm}\)，\(E = 2.1 \times 10^{11}\,\text{Pa}\)。求水击波速\(c\)。

**解答**：
$$c = \frac{1}{\sqrt{1000\left(\frac{1}{2.1 \times 10^9} + \frac{0.5}{2.1 \times 10^{11} \times 0.01}\right)}} \approx 1000\,\text{m/s}$$

### 题目432：水击压强增量 ⭐⭐⭐

管道流速\(v_0 = 2\,\text{m/s}\)，阀门突然关闭（\(\Delta v = 2\,\text{m/s}\)），水击波速\(c = 1000\,\text{m/s}\)。求压强增量\(\Delta p\)。

**解答**：
$$\Delta p = \rho c \Delta v = 1000 \times 1000 \times 2 = 2 \times 10^6\,\text{Pa} = 2\,\text{MPa}$$
