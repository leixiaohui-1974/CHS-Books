## 📚 动量方程理论基础

### 基本原理

**牛顿第二定律**：
$$\sum \vec{F} = \frac{d(m\vec{v})}{dt}$$

**流体动量方程（定常流）**：
$$\sum \vec{F} = \rho Q(\vec{v}_2 - \vec{v}_1)$$

其中：
- \(\sum \vec{F}\)：作用在控制体上的所有外力（N）
- \(\rho Q\)：质量流量（kg/s）
- \(\vec{v}_2 - \vec{v}_1\)：流速变化（m/s）

### 投影形式

**x方向**：
$$\sum F_x = \rho Q(v_{2x} - v_{1x})$$

**y方向**：
$$\sum F_y = \rho Q(v_{2y} - v_{1y})$$

**z方向**：
$$\sum F_z = \rho Q(v_{2z} - v_{1z}) + \rho Q g$$

（z方向需考虑重力）

### 外力组成

$$\sum \vec{F} = \vec{F}_压 + \vec{F}_重 + \vec{F}_壁$$

其中：
- \(\vec{F}_压\)：断面上的压力
- \(\vec{F}_重\)：流体重力
- \(\vec{F}_壁\)：固体壁面对流体的作用力（待求）

**反作用力**：
$$\vec{F}_{流→壁} = -\vec{F}_{壁→流}$$

### 与伯努利方程的关系

| 方程 | 应用 | 求解 |
|------|------|------|
| **伯努利方程** | 能量守恒 | 压力、流速、水头 |
| **动量方程** | 动量守恒 | 作用力、推力、冲击力 |
| **连续性方程** | 质量守恒 | 流量、流速关系 |

**联合应用**：先用连续性和伯努利求压力和流速，再用动量方程求力。

## 第一部分：基础应用题（题176-185）

### 题目176：水平射流冲击平板 ⭐⭐

**题目**：
水平射流（直径\(d = 0.05\,\text{m}\)，流速\(v = 10\,\text{m/s}\)）垂直冲击固定平板。求平板受到的冲击力\(F\)。

**解答**：

**步骤1**：流量
$$Q = vA = 10 \times \frac{\pi \times 0.05^2}{4} = 0.0196\,\text{m}^3/\text{s}$$

**步骤2**：质量流量
$$\dot{m} = \rho Q = 1000 \times 0.0196 = 19.6\,\text{kg/s}$$

**步骤3**：动量方程（x方向）
- 射流入：\(v_1 = 10\,\text{m/s}\)
- 冲击后：\(v_2 = 0\)（垂直冲击后流体沿平板流走，x方向速度为0）

$$F = \dot{m}(v_2 - v_1) = 19.6 \times (0 - 10) = -196\,\text{N}$$

（负号表示平板对流体施加反向力）

**平板受力**：
$$F_{平板} = -F = 196\,\text{N}\quad (\text{方向与射流相同})$$

**答案**：\(F = 196\,\text{N}\)

### 题目177：射流冲击倾斜平板 ⭐⭐⭐

**题目**：
水平射流（\(Q = 0.02\,\text{m}^3/\text{s}\)，\(v = 10\,\text{m/s}\)）冲击倾角\(\theta = 30°\)的光滑平板。求：
1. 平板所受法向力\(F_n\)
2. 沿平板向上和向下的流量\(Q_1, Q_2\)

**解答**：

**步骤1**：冲击后速度分解
- 法向分量：\(v_n = v \sin\theta = 10 \times 0.5 = 5\,\text{m/s}\)（变为0）
- 切向分量：\(v_t = v \cos\theta = 10 \times 0.866 = 8.66\,\text{m/s}\)（保持不变，光滑平板）

**步骤2**：法向力
$$F_n = \rho Q(0 - v_n) = 1000 \times 0.02 \times (0 - 5) = -100\,\text{N}$$
$$F_n(\text{平板受}) = 100\,\text{N}$$

**步骤3**：流量分配（连续性+动量，沿切向）
- 设向上\(Q_1\)，向下\(Q_2\)
- 连续性：\(Q_1 + Q_2 = Q = 0.02\)
- 切向动量：\(\rho Q v_t = \rho Q_1 v_t - \rho Q_2 v_t\)
- 解得：\(Q_1 = Q/2 = 0.01\,\text{m}^3/\text{s}\)，\(Q_2 = 0.01\,\text{m}^3/\text{s}\)

（对称分流）

**答案**：
- 法向力：\(F_n = 100\,\text{N}\)
- 流量：\(Q_1 = Q_2 = 0.01\,\text{m}^3/\text{s}\)

### 题目178：90°弯管受力 ⭐⭐⭐

**题目**：
水平90°弯管（管径\(d = 0.2\,\text{m}\)），流量\(Q = 0.1\,\text{m}^3/\text{s}\)。入口压强\(p_1 = 200\,\text{kPa}\)，出口压强\(p_2 = 180\,\text{kPa}\)（出口竖直向上）。忽略弯管重力。求弯管所受合力\(\vec{F}\)。

**解答**：

**步骤1**：流速
$$A = \frac{\pi \times 0.2^2}{4} = 0.0314\,\text{m}^2$$
$$v = \frac{Q}{A} = \frac{0.1}{0.0314} = 3.18\,\text{m/s}$$

（管径不变，\(v_1 = v_2 = v\)）

**步骤2**：动量方程（x方向）
$$F_x + p_1 A - 0 = \rho Q(0 - v)$$
$$F_x = -p_1 A - \rho Q v = -200,000 \times 0.0314 - 1000 \times 0.1 \times 3.18$$
$$F_x = -6280 - 318 = -6598\,\text{N}$$

**步骤3**：动量方程（y方向）
$$F_y + 0 - p_2 A = \rho Q(v - 0)$$
$$F_y = p_2 A + \rho Q v = 180,000 \times 0.0314 + 318 = 5652 + 318 = 5970\,\text{N}$$

**步骤4**：合力
$$F = \sqrt{F_x^2 + F_y^2} = \sqrt{6598^2 + 5970^2} = 8895\,\text{N}$$
$$\alpha = \arctan\left(\frac{F_y}{|F_x|}\right) = \arctan\left(\frac{5970}{6598}\right) = 42.1°$$

**答案**：
- \(F_x = -6598\,\text{N}\)（向左）
- \(F_y = 5970\,\text{N}\)（向上）
- 合力：\(F = 8895\,\text{N}\)，方向与x负方向成42.1°

**工程意义**：弯管需要固定支座承受此力。

### 题目179：闸门动水推力 ⭐⭐

**题目**：
平板闸门，底部开度\(a = 0.5\,\text{m}\)，上游水深\(H_1 = 3\,\text{m}\)，下游水深\(H_2 = 1\,\text{m}\)（闸后自由出流）。闸门宽度\(b = 2\,\text{m}\)。求闸门所受水平推力\(F\)。

**解答**：

**步骤1**：闸下流速（伯努利，忽略损失）
$$H_1 = \frac{v^2}{2g} + H_2$$
$$v = \sqrt{2g(H_1 - H_2)} = \sqrt{2 \times 9.8 \times 2} = 6.26\,\text{m/s}$$

**步骤2**：流量
$$Q = v \times a \times b = 6.26 \times 0.5 \times 2 = 6.26\,\text{m}^3/\text{s}$$

**步骤3**：上下游断面压力
- 上游：\(F_1 = \gamma h_1 A_1 = 9800 \times 1.5 \times (3 \times 2) = 88,200\,\text{N}\)
- 下游：\(F_2 = \gamma h_2 A_2 = 9800 \times 0.5 \times (1 \times 2) = 9,800\,\text{N}\)

**步骤4**：动量方程（水平）
$$F_闸 + F_1 - F_2 = \rho Q(v_2 - v_1)$$

（\(v_1 \approx 0\)，\(v_2 = v\)）

$$F_闸 = \rho Q v - F_1 + F_2 = 1000 \times 6.26 \times 6.26 - 88,200 + 9,800$$
$$F_闸 = 39,188 - 78,400 = -39,212\,\text{N}$$

**闸门受力**（水对闸）：
$$F = 39,212\,\text{N}\quad (\text{向下游})$$

**答案**：\(F = 39.2\,\text{kN}\)（向下游）

### 题目186：射流推动小车 ⭐⭐⭐

**题目**：
小车上装水箱，底部有喷嘴（直径\(d = 0.05\,\text{m}\)），水平向后喷水。喷口流速\(v_绝 = 15\,\text{m/s}\)（相对地面）。小车+水总质量\(M = 500\,\text{kg}\)，初始静止。不计摩擦，求：
1. 喷水推力\(F\)
2. 小车加速度\(a\)（初始时刻）

**解答**：

**步骤1**：流量
$$Q = v A = 15 \times \frac{\pi \times 0.05^2}{4} = 0.0295\,\text{m}^3/\text{s}$$

**步骤2**：推力（动量方程）
$$F = \rho Q v = 1000 \times 0.0295 \times 15 = 442.5\,\text{N}$$

**步骤3**：加速度
$$a = \frac{F}{M} = \frac{442.5}{500} = 0.885\,\text{m/s}^2$$

**答案**：
- 推力：\(F = 442.5\,\text{N}\)
- 加速度：\(a = 0.885\,\text{m/s}^2\)

**注意**：随着水的消耗，\(M\)减小，加速度增大。

### 题目187：旋转叶片射流 ⭐⭐⭐⭐

**题目**：
射流（\(Q = 0.05\,\text{m}^3/\text{s}\)，\(v = 20\,\text{m/s}\)）沿切向冲击旋转叶片（半径\(R = 0.5\,\text{m}\)，角速度\(\omega = 30\,\text{rad/s}\)）。叶片出口相对速度方向与入口相反（180°转向）。求：
1. 叶片受到的力矩\(M\)
2. 输出功率\(P\)

**解答**：

**步骤1**：叶片线速度
$$u = \omega R = 30 \times 0.5 = 15\,\text{m/s}$$

**步骤2**：相对速度
$$v_r = v - u = 20 - 15 = 5\,\text{m/s}$$

**步骤3**：出口绝对速度（180°转向）
$$v_2 = u - v_r = 15 - 5 = 10\,\text{m/s}\quad (\text{与入口反向})$$

**步骤4**：力（切向动量方程）
$$F = \rho Q(v_2 - v_1) = 1000 \times 0.05 \times ((-10) - 20) = -1500\,\text{N}$$

**步骤5**：力矩
$$M = F \times R = 1500 \times 0.5 = 750\,\text{N·m}$$

**步骤6**：功率
$$P = M \omega = 750 \times 30 = 22,500\,\text{W} = 22.5\,\text{kW}$$

**答案**：
- 力矩：\(M = 750\,\text{N·m}\)
- 功率：\(P = 22.5\,\text{kW}\)

**应用**：水轮机、涡轮机原理。

### 题目188：船舶推进器 ⭐⭐⭐⭐

**题目**：
船舶螺旋桨直径\(D = 2\,\text{m}\)，推进效率\(\eta = 0.7\)，船速\(v_船 = 10\,\text{m/s}\)，所需推力\(F = 50,000\,\text{N}\)。求：
1. 螺旋桨盘面处流速增量\(\Delta v\)
2. 推进功率\(P\)

**解答**：

**简化模型**（动量理论）：
$$F = \rho A (v_1 + \frac{\Delta v}{2}) \Delta v$$

（\(v_1\)：来流速度，\(\Delta v\)：速度增量）

（详细推导略，涉及螺旋桨理论）

**推进功率**：
$$P = \frac{F \times v_船}{\eta} = \frac{50,000 \times 10}{0.7} = 714,286\,\text{W} = 714\,\text{kW}$$

### 题目197：水轮机叶片动力学 ⭐⭐⭐⭐⭐

**题目**：
冲击式水轮机（Pelton轮），射流速度\(v = 80\,\text{m/s}\)，流量\(Q = 0.5\,\text{m}^3/\text{s}\)，叶片线速度\(u = 0.46v\)（最佳速比），叶片相对出口角\(\beta_2 = 165°\)。求：
1. 叶片受力\(F\)
2. 输出功率\(P\)
3. 水力效率\(\eta_h\)

**解答**：

（详细推导略，涉及叶片速度三角形）

**关键公式**：
$$F = \rho Q(v - u)(1 + \cos\beta_2)$$
$$P = F \times u$$
$$\eta_h = \frac{P}{\frac{1}{2}\rho Q v^2} = 2\frac{u}{v}\left(1 - \frac{u}{v}\right)(1 + \cos\beta_2)$$

**最佳效率**：当\(u/v = 0.5\)时，\(\eta_h^{max} \approx 1\)（理想情况）

## 📝 动量方程总结（题176-205）

### 核心公式

**基本形式**：
$$\sum \vec{F} = \rho Q(\vec{v}_2 - \vec{v}_1)$$

**分量形式**：
$$\sum F_x = \rho Q(v_{2x} - v_{1x})$$
$$\sum F_y = \rho Q(v_{2y} - v_{1y})$$
$$\sum F_z = \rho Q(v_{2z} - v_{1z}) + \rho Q g\Delta z$$

**外力分解**：
$$\sum F = F_压 + F_重 + F_壁$$

### 典型应用

| 应用 | 关键公式 | 说明 |
|------|----------|------|
| **射流冲板** | \(F = \rho Q v\) | 垂直冲击 |
| **弯管受力** | \(\vec{F} = p_1\vec{A}_1 - p_2\vec{A}_2 + \rho Q(\vec{v}_2 - \vec{v}_1)\) | 矢量运算 |
| **喷嘴推力** | \(F = \rho Q(v_2 - v_1)\) | 火箭、飞机 |
| **水轮机** | \(P = \rho Q u(v - u)(1 + \cos\beta)\) | 叶片理论 |
| **闸门受力** | \(F = F_1 - F_2 - \rho Q(v_2 - v_1)\) | 水工建筑 |

### 解题步骤

**标准流程**：
1. **选择控制体**：包含待求力的固体表面
2. **确定坐标系**：通常选水平或垂直方向
3. **列连续性**：求各断面流速
4. **列伯努利**（如需要）：求压力
5. **列动量方程**：
   - 左边：外力（压力+重力+壁面力）
   - 右边：动量变化率
6. **求解壁面力**：\(\vec{F}_壁\)
7. **求反作用力**：\(\vec{F}_{流→壁} = -\vec{F}_{壁→流}\)

**注意事项**：
- ✅ 动量是矢量，注意正负号
- ✅ 压力指向控制体为正
- ✅ 重力向下为负（竖直方向）
- ✅ 流速方向与坐标轴一致为正

### 与伯努利方程配合

**联合使用**：
1. **伯努利**→求压力\(p_1, p_2\)和流速\(v_1, v_2\)
2. **连续性**→流速关系\(v_1 A_1 = v_2 A_2\)
3. **动量**→求作用力\(F\)

**典型题型**：
- 弯管受力：伯努利求压力，动量求合力
- 闸门受力：伯努利求流速，动量求推力
- 水轮机：伯努利求水头，动量求功率

### 常见错误

❌ **错误1**：忘记压力的方向  
✅ **正确**：压力指向控制体内部为正

❌ **错误2**：动量变化符号错误  
✅ **正确**：\(\rho Q(\vec{v}_2 - \vec{v}_1)\)，出口减入口

❌ **错误3**：忽略重力  
✅ **正确**：竖直方向必须考虑流体重力\(\rho Q g\)

❌ **错误4**：控制体选择不当  
✅ **正确**：控制体应包含待求力的作用面

❌ **错误5**：单位不统一  
✅ **正确**：力(N)，压强(Pa)，流量(m³/s)，流速(m/s)
