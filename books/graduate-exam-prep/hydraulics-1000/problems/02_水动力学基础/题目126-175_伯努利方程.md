## 📚 伯努利方程理论基础

### 基本形式

**伯努利方程（理想流体，沿流线）**：
$$\frac{p}{\gamma} + \frac{v^2}{2g} + z = C$$

**实际流体（考虑能量损失）**：
$$\frac{p_1}{\gamma} + \frac{v_1^2}{2g} + z_1 = \frac{p_2}{\gamma} + \frac{v_2^2}{2g} + z_2 + h_f$$

其中：
- \(\frac{p}{\gamma}\)：压强水头（压力能，m）
- \(\frac{v^2}{2g}\)：流速水头（动能，m）
- \(z\)：位置水头（势能，m）
- \(h_f\)：水头损失（能量损失，m）

### 物理意义

**三个水头**：
1. **压强水头**：压力能转换为水柱高度
2. **流速水头**：动能转换为水柱高度
3. **位置水头**：势能（相对于基准面的高度）

**总水头线**：\(H = \frac{p}{\gamma} + \frac{v^2}{2g} + z\)  
**测压管水头线**：\(h = \frac{p}{\gamma} + z\)

### 应用条件

✅ **满足条件**：
1. 沿同一流线
2. 不可压缩流体
3. 定常流动
4. 忽略粘性（或考虑损失）

❌ **不满足条件**：
- 不同流线之间（除非整个断面）
- 可压缩流体（需气体动力学）
- 非定常流动（需修正）

### 方程形式变换

**能量形式（单位重量）**：
$$E = \frac{p}{\gamma} + \frac{v^2}{2g} + z\quad (\text{m})$$

**压力形式（单位面积）**：
$$p + \frac{1}{2}\rho v^2 + \rho g z = C\quad (\text{Pa})$$

**功率形式（单位时间）**：
$$P = \gamma Q H\quad (\text{W})$$

### 题目126：水平管道压力变化 ⭐

**题目**：
水平管道（忽略损失），断面1：\(d_1 = 0.3\,\text{m}\)，\(p_1 = 200\,\text{kPa}\)，\(v_1 = 2\,\text{m/s}\)；断面2：\(d_2 = 0.15\,\text{m}\)。求断面2的压力\(p_2\)。

**解答**：

**步骤1**：连续性方程求\(v_2\)
$$v_2 = v_1 \times \frac{A_1}{A_2} = 2 \times \frac{0.3^2}{0.15^2} = 8\,\text{m/s}$$

**步骤2**：伯努利方程（水平，\(z_1 = z_2\)）
$$\frac{p_1}{\gamma} + \frac{v_1^2}{2g} = \frac{p_2}{\gamma} + \frac{v_2^2}{2g}$$

**步骤3**：求\(p_2\)
$$\frac{p_2}{\gamma} = \frac{p_1}{\gamma} + \frac{v_1^2 - v_2^2}{2g}$$
$$p_2 = p_1 + \frac{\rho(v_1^2 - v_2^2)}{2}$$
$$p_2 = 200,000 + \frac{1000(4 - 64)}{2} = 200,000 - 30,000 = 170\,\text{kPa}$$

**答案**：\(p_2 = 170\,\text{kPa}\)

**物理解释**：断面缩小，流速增大（2→8 m/s），根据伯努利方程，压力必然降低（200→170 kPa）。这就是**文丘里效应**！

### 题目127：毕托管测速 ⭐⭐

**题目**：
毕托管测量管道中心流速，测压管液面高度差\(\Delta h = 0.5\,\text{m}\)（水），求流速\(v\)。

**解答**：

**原理**：毕托管前端（滞止点）流速为0，压力最大。

**伯努利方程**（滞止点0，来流点1）：
$$\frac{p_1}{\gamma} + \frac{v_1^2}{2g} = \frac{p_0}{\gamma} + 0$$

**压力差**：
$$\frac{p_0 - p_1}{\gamma} = \frac{v^2}{2g} = \Delta h$$

**流速**：
$$v = \sqrt{2g\Delta h} = \sqrt{2 \times 9.8 \times 0.5} = 3.13\,\text{m/s}$$

**答案**：\(v = 3.13\,\text{m/s}\)

**应用**：毕托管广泛用于飞机测速、管道测速。

### 题目128：水箱出流 ⭐

**题目**：
大水箱，水面高度\(H = 5\,\text{m}\)（相对于出口），侧壁开孔出流（大气中），忽略损失。求出流速度\(v\)。

**解答**：

**伯努利方程**（水面→出口）：
$$\frac{p_a}{\gamma} + 0 + H = \frac{p_a}{\gamma} + \frac{v^2}{2g} + 0$$

（水面流速≈0，压力为大气压；出口压力也为大气压）

**求流速**：
$$H = \frac{v^2}{2g}$$
$$v = \sqrt{2gH} = \sqrt{2 \times 9.8 \times 5} = 9.90\,\text{m/s}$$

**答案**：\(v = 9.90\,\text{m/s}\)

**托里拆利定理**：\(v = \sqrt{2gH}\)

### 题目129：虹吸管 ⭐⭐

**题目**：
虹吸管从水箱抽水，水箱水面至虹吸管最高点高度\(h_1 = 2\,\text{m}\)，最高点至出口高度\(h_2 = 3\,\text{m}\)。求：
1. 出口流速\(v\)
2. 最高点压强\(p_顶\)

**解答**：

**步骤1**：出口流速（托里拆利）
$$v = \sqrt{2g(h_1 + h_2)} = \sqrt{2 \times 9.8 \times 5} = 9.90\,\text{m/s}$$

**步骤2**：最高点压强（伯努利：水面→顶点）
$$\frac{p_a}{\gamma} + 0 + 0 = \frac{p_顶}{\gamma} + \frac{v^2}{2g} + h_1$$

（顶点流速等于出口流速\(v\)，因为管径不变）

$$\frac{p_顶}{\gamma} = -h_1 - \frac{v^2}{2g} = -2 - 5 = -7\,\text{m}$$
$$p_顶 = -7 \times 9800 = -68.6\,\text{kPa}\quad (\text{相对压强})$$

**绝对压强**：
$$p_顶(\text{绝}) = 101.325 - 68.6 = 32.7\,\text{kPa}$$

**答案**：
- 出口流速：\(v = 9.90\,\text{m/s}\)
- 顶点压强：\(p_顶 = -68.6\,\text{kPa}\)（相对），\(32.7\,\text{kPa}\)（绝对）

**工程意义**：顶点压强不能低于水的汽化压强（约2.3 kPa），否则会气穴。

### 题目130：文丘里流量计 ⭐⭐

**题目**：
文丘里管测流量，管道直径\(D = 0.2\,\text{m}\)，喉部直径\(d = 0.1\,\text{m}\)，压差计读数\(\Delta h = 0.8\,\text{m}\)（水柱）。求流量\(Q\)（忽略损失）。

**解答**：

**步骤1**：设管道流速\(v_1\)，喉部流速\(v_2\)

**连续性方程**：
$$v_1 A_1 = v_2 A_2$$
$$v_2 = v_1 \times \frac{D^2}{d^2} = 4v_1$$

**步骤2**：伯努利方程（水平）
$$\frac{p_1}{\gamma} + \frac{v_1^2}{2g} = \frac{p_2}{\gamma} + \frac{v_2^2}{2g}$$

**压差**：
$$\frac{p_1 - p_2}{\gamma} = \frac{v_2^2 - v_1^2}{2g} = \frac{16v_1^2 - v_1^2}{2g} = \frac{15v_1^2}{2g} = \Delta h$$

**步骤3**：求\(v_1\)
$$v_1 = \sqrt{\frac{2g\Delta h}{15}} = \sqrt{\frac{2 \times 9.8 \times 0.8}{15}} = 1.02\,\text{m/s}$$

**步骤4**：流量
$$Q = v_1 A_1 = 1.02 \times \frac{\pi \times 0.2^2}{4} = 0.032\,\text{m}^3/\text{s} = 32\,\text{L/s}$$

**答案**：\(Q = 32\,\text{L/s}\)

### 题目141：虹吸管最大吸程 ⭐⭐⭐

**题目**：
虹吸管，已知水温20°C时饱和蒸汽压\(p_v = 2.34\,\text{kPa}\)（绝对），大气压\(p_a = 101.325\,\text{kPa}\)。为防止气穴，虹吸管最高点绝对压强不低于\(1.2 p_v = 2.8\,\text{kPa}\)。若出口相对于水面下降\(h_2 = 3\,\text{m}\)，求虹吸管最高点相对于水面的最大高度\(h_1^{max}\)。（忽略损失）

**解答**：

**步骤1**：出口流速
$$v = \sqrt{2gh_2} = \sqrt{2 \times 9.8 \times 3} = 7.67\,\text{m/s}$$

**步骤2**：伯努利方程（水面→最高点）
$$\frac{p_a}{\gamma} = \frac{p_顶}{\gamma} + \frac{v^2}{2g} + h_1$$

**步骤3**：最高点压强限制
$$p_顶(\min) = 2.8\,\text{kPa} = 2800\,\text{Pa}$$

**步骤4**：求\(h_1^{max}\)
$$\frac{p_a - p_顶}{\gamma} = h_1 + \frac{v^2}{2g}$$
$$h_1^{max} = \frac{101,325 - 2,800}{9800} - \frac{7.67^2}{2 \times 9.8}$$
$$h_1^{max} = 10.05 - 3.0 = 7.05\,\text{m}$$

**答案**：\(h_1^{max} = 7.05\,\text{m}\)

**工程意义**：虹吸管吸程一般不超过7-8m，理论极限约10m（不考虑流速）。

### 题目142：水轮机伯努利方程 ⭐⭐⭐

**题目**：
水轮机，进水口（断面1）：\(p_1 = 500\,\text{kPa}\)，\(v_1 = 3\,\text{m/s}\)，\(z_1 = 10\,\text{m}\)；出水口（断面2）：\(p_2 = 120\,\text{kPa}\)，\(v_2 = 5\,\text{m/s}\)，\(z_2 = 2\,\text{m}\)。流量\(Q = 2\,\text{m}^3/\text{s}\)。求：
1. 水轮机提取的水头\(H_T\)
2. 水轮机理论功率\(P_T\)

**解答**：

**步骤1**：总水头（1和2）
$$H_1 = \frac{p_1}{\gamma} + \frac{v_1^2}{2g} + z_1 = \frac{500,000}{9800} + \frac{9}{19.6} + 10 = 51.02 + 0.46 + 10 = 61.48\,\text{m}$$

$$H_2 = \frac{p_2}{\gamma} + \frac{v_2^2}{2g} + z_2 = \frac{120,000}{9800} + \frac{25}{19.6} + 2 = 12.24 + 1.28 + 2 = 15.52\,\text{m}$$

**步骤2**：水轮机提取水头
$$H_T = H_1 - H_2 = 61.48 - 15.52 = 45.96\,\text{m}$$

**步骤3**：理论功率
$$P_T = \gamma Q H_T = 9800 \times 2 \times 45.96 = 901,008\,\text{W} = 901\,\text{kW}$$

**答案**：
- 提取水头：\(H_T = 45.96\,\text{m}\)
- 理论功率：\(P_T = 901\,\text{kW}\)

**实际功率**：考虑效率\(\eta\)（一般80%-90%），\(P = \eta P_T\)

### 题目143：水泵伯努利方程 ⭐⭐⭐

**题目**：
水泵从低水池抽水至高水池，两水面高差\(H = 30\,\text{m}\)，流量\(Q = 0.1\,\text{m}^3/\text{s}\)，管路总损失\(h_f = 5\,\text{m}\)。求：
1. 水泵需要提供的扬程\(H_P\)
2. 水泵功率\(P\)（效率\(\eta = 0.75\)）

**解答**：

**步骤1**：伯努利方程（下池水面→上池水面）
$$0 + 0 + 0 + H_P = 0 + 0 + H + h_f$$

（两水面流速≈0，压力均为大气压）

**步骤2**：水泵扬程
$$H_P = H + h_f = 30 + 5 = 35\,\text{m}$$

**步骤3**：水泵功率
$$P = \frac{\gamma Q H_P}{\eta} = \frac{9800 \times 0.1 \times 35}{0.75} = 45,733\,\text{W} = 45.7\,\text{kW}$$

**答案**：
- 水泵扬程：\(H_P = 35\,\text{m}\)
- 水泵功率：\(P = 45.7\,\text{kW}\)

### 题目144：文丘里管考虑损失 ⭐⭐⭐

**题目**：
重做题目130，但考虑喉部至管道断面的损失\(h_f = 0.1 \frac{v_2^2}{2g}\)。求实际流量\(Q\)。

**解答**：

**伯努利方程（考虑损失）**：
$$\frac{p_1}{\gamma} + \frac{v_1^2}{2g} = \frac{p_2}{\gamma} + \frac{v_2^2}{2g} + h_f$$

（注意损失项在右边）

**压差**：
$$\Delta h = \frac{v_2^2 - v_1^2}{2g} + h_f = \frac{v_2^2 - v_1^2}{2g} + 0.1 \frac{v_2^2}{2g}$$

（代入\(v_2 = 4v_1\)，求解）

结果：流量略小于理想值（约减少5%）。

### 题目161：复杂虹吸系统 ⭐⭐⭐⭐

### 题目171：清华大学2019真题改编 ⭐⭐⭐⭐

**题目**：
水平文丘里管，管道直径\(D = 0.2\,\text{m}\)，喉部直径\(d = 0.1\,\text{m}\)。管道压强\(p_1 = 200\,\text{kPa}\)，喉部压强\(p_2 = 80\,\text{kPa}\)。水中溶解空气，当压强降至\(p_v = 2\,\text{kPa}\)时开始气穴。问：
1. 当前流量\(Q\)
2. 是否发生气穴？
3. 若发生气穴,如何改进设计？

**解答**：

**步骤1**：当前流量（伯努利+连续性）
$$\frac{p_1}{\gamma} + \frac{v_1^2}{2g} = \frac{p_2}{\gamma} + \frac{v_2^2}{2g}$$
$$v_2 = 4v_1$$
$$\frac{200,000 - 80,000}{9800} = \frac{16v_1^2 - v_1^2}{2g}$$
$$v_1 = 3.95\,\text{m/s}$$
$$Q = 0.124\,\text{m}^3/\text{s}$$

**步骤2**：气穴判断
$$p_2 = 80\,\text{kPa} >> p_v = 2\,\text{kPa}$$
未发生气穴 ✓

**步骤3**：临界流量（气穴发生时）
$$p_2 = 2\,\text{kPa}$$
$$Q_{crit} = 0.18\,\text{m}^3/\text{s}$$

**改进措施**：
1. 增大喉部直径
2. 降低流量
3. 提高管道压强
4. 降低喉部位置（增加静压）

### 题目172：河海大学2020真题改编 ⭐⭐⭐

**题目**：
虹吸排水管，水箱水位\(H = 4\,\text{m}\)（相对于出口），管道直径均匀\(d = 0.1\,\text{m}\)，最高点高出水面\(h_1 = 1.5\,\text{m}\)。管道沿程损失系数\(K = 6\)（即\(h_f = K \frac{v^2}{2g}\)）。求：
1. 出口流速
2. 最高点压强
3. 流量

**解答**：

**步骤1**：伯努利方程（水面→出口）
$$H = \frac{v^2}{2g} + h_f = \frac{v^2}{2g}(1 + K)$$
$$v = \sqrt{\frac{2gH}{1 + K}} = \sqrt{\frac{2 \times 9.8 \times 4}{7}} = 3.35\,\text{m/s}$$

**步骤2**：流量
$$Q = vA = 3.35 \times \frac{\pi \times 0.1^2}{4} = 0.0263\,\text{m}^3/\text{s} = 26.3\,\text{L/s}$$

**步骤3**：最高点压强（伯努利：水面→顶点）
- 水面→顶点的损失：\(h_f' = K' \frac{v^2}{2g}\)（\(K'\)需根据管长比例）
- 假设\(K' = 2\)（示例）

$$\frac{p_a}{\gamma} = \frac{p_顶}{\gamma} + \frac{v^2}{2g} + h_1 + h_f'$$
$$\frac{p_顶}{\gamma} = -h_1 - \frac{v^2}{2g}(1 + K') = -1.5 - \frac{3.35^2}{19.6} \times 3 = -3.22\,\text{m}$$
$$p_顶 = -31.6\,\text{kPa}$$

## 📝 伯努利方程总结（题126-175）

### 核心公式体系

| 方程形式 | 公式 | 单位 | 说明 |
|----------|------|------|------|
| **水头形式** | \(\frac{p}{\gamma} + \frac{v^2}{2g} + z = H\) | m | 最常用 |
| **压力形式** | \(p + \frac{1}{2}\rho v^2 + \rho gz = p_总\) | Pa | 力学分析 |
| **能量形式** | \(E = \frac{p}{\rho g} + \frac{v^2}{2g} + z\) | m | 能量守恒 |
| **功率形式** | \(P = \gamma Q H\) | W | 水力机械 |

### 三种水头

```
总水头线 H = p/γ + v²/2g + z ───────────────
                        ↓ 流速水头 v²/2g
测压管水头线 h = p/γ + z ─────────────────
                        ↓ 压强水头 p/γ
基准面 z = 0 ────────────────────────────
```

### 实际流体修正

**考虑损失**：
$$H_1 = H_2 + h_f$$

**损失分类**：
1. **沿程损失**：\(h_f = \lambda \frac{L}{d} \frac{v^2}{2g}\)
2. **局部损失**：\(h_m = \zeta \frac{v^2}{2g}\)

**水力机械**：
- 水泵：\(H_1 + H_P = H_2 + h_f\)（加能量）
- 水轮机：\(H_1 = H_2 + H_T + h_f\)（取能量）

### 典型应用

| 应用 | 关键公式 | 说明 |
|------|----------|------|
| **托里拆利** | \(v = \sqrt{2gH}\) | 水箱出流 |
| **毕托管** | \(v = \sqrt{2g\Delta h}\) | 测速装置 |
| **文丘里** | \(Q = C A \sqrt{2g\Delta h}\) | 测流量 |
| **虹吸** | \(p_顶 = p_a - \rho g(h_1 + \frac{v^2}{2g})\) | 最大吸程 |
| **气穴** | \(p_{min} > p_v\) | 防气穴条件 |

### 解题策略

**标准步骤**：
1. **选取断面**：能量方程两端（已知或待求）
2. **列连续性**：\(v_1 A_1 = v_2 A_2\)（消去一个未知数）
3. **列伯努利**：\(\frac{p_1}{\gamma} + \frac{v_1^2}{2g} + z_1 = \frac{p_2}{\gamma} + \frac{v_2^2}{2g} + z_2 + h_f\)
4. **求解未知**：联立方程组
5. **检查合理性**：压强、流速、功率等是否合理

**选取基准面原则**：
- 通常选最低点为\(z = 0\)
- 选择能简化计算的位置
- 保持一致性

**单位注意**：
- 压强：Pa或kPa
- 流速：m/s
- 水头：m
- 流量：m³/s或L/s
- 功率：W或kW

### 常见错误

❌ **错误1**：忘记选同一流线  
✅ **正确**：伯努利方程只在同一流线或整个断面适用

❌ **错误2**：基准面选取不一致  
✅ **正确**：两断面必须用同一基准面

❌ **错误3**：忽略大气压  
✅ **正确**：相对压强计算时，大气压可以抵消；绝对压强计算时必须考虑

❌ **错误4**：损失项位置错误  
✅ **正确**：\(h_f\)总是在方程右边（下游），表示能量损失

❌ **错误5**：水泵/水轮机项位置  
✅ **正确**：水泵\(H_P\)加在左边（上游），水轮机\(H_T\)加在右边（下游）

### 物理意义深度理解

**能量转换**：
- 断面缩小：压力能→动能（\(p\)↓，\(v\)↑）
- 断面扩大：动能→压力能（\(v\)↓，\(p\)↑）
- 水泵：电能→机械能→水流能量
- 水轮机：水流能量→机械能→电能

**总水头线特点**：
- 理想流体：水平线
- 实际流体：沿流动方向下降（损失）
- 斜率= \(J = -\frac{dH}{ds}\)（水力坡度）

**测压管水头线特点**：
- 与总水头线相差流速水头\(\frac{v^2}{2g}\)
- 可升可降
- 流速增大处下降（压力降低）
