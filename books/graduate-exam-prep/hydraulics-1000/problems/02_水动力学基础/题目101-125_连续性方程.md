## 📚 连续性方程理论基础

### 基本原理

**质量守恒定律**：流体在流动过程中，质量既不能增加也不能减少。

**连续性方程（不可压缩流体）**：
$$Q_1 = Q_2$$
$$v_1 A_1 = v_2 A_2$$

其中：
- \(Q\)：体积流量（m³/s）
- \(v\)：平均流速（m/s）
- \(A\)：过流断面面积（m²）

**物理意义**：
> 当管道截面突然变小时，流速必然增大；  
> 当管道截面突然变大时，流速必然减小。

### 推广形式

**可压缩流体**：
$$\rho_1 v_1 A_1 = \rho_2 v_2 A_2$$

**微分形式**（三维）：
$$\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{v}) = 0$$

**定常流动**（不可压缩）：
$$\nabla \cdot \mathbf{v} = 0$$

## 第一部分：基础应用题（题101-110）

### 题目101：等直径管道 ⭐

**题目**：
圆管直径\(d = 0.2\,\text{m}\)，某断面流速\(v_1 = 2\,\text{m/s}\)，求体积流量\(Q\)。

**解答**：

**步骤1**：计算过流断面面积
$$A = \frac{\pi d^2}{4} = \frac{\pi \times 0.2^2}{4} = 0.0314\,\text{m}^2$$

**步骤2**：计算流量
$$Q = vA = 2 \times 0.0314 = 0.0628\,\text{m}^3/\text{s} = 62.8\,\text{L/s}$$

**答案**：\(Q = 0.0628\,\text{m}^3/\text{s} = 62.8\,\text{L/s}\)

### 题目102：变径管道 ⭐

**题目**：
渐缩管，入口直径\(d_1 = 0.3\,\text{m}\)，出口直径\(d_2 = 0.15\,\text{m}\)，入口流速\(v_1 = 1.5\,\text{m/s}\)，求出口流速\(v_2\)。

**解答**：

**步骤1**：根据连续性方程
$$v_1 A_1 = v_2 A_2$$

**步骤2**：面积比
$$\frac{A_1}{A_2} = \frac{d_1^2}{d_2^2} = \frac{0.3^2}{0.15^2} = 4$$

**步骤3**：出口流速
$$v_2 = v_1 \times \frac{A_1}{A_2} = 1.5 \times 4 = 6\,\text{m/s}$$

**答案**：\(v_2 = 6\,\text{m/s}\)（速度增加4倍）

**物理解释**：断面缩小为原来的1/4，流速增加为原来的4倍。

### 题目103：矩形明渠 ⭐

**题目**：
矩形明渠，宽\(b = 2\,\text{m}\)，水深\(h_1 = 1.5\,\text{m}\)，流速\(v_1 = 0.8\,\text{m/s}\)。下游水深变为\(h_2 = 1.0\,\text{m}\)（渠宽不变），求下游流速\(v_2\)。

**解答**：

**步骤1**：过流断面面积
$$A_1 = b \times h_1 = 2 \times 1.5 = 3.0\,\text{m}^2$$
$$A_2 = b \times h_2 = 2 \times 1.0 = 2.0\,\text{m}^2$$

**步骤2**：连续性方程
$$Q = v_1 A_1 = v_2 A_2$$

**步骤3**：下游流速
$$v_2 = v_1 \times \frac{A_1}{A_2} = 0.8 \times \frac{3.0}{2.0} = 1.2\,\text{m/s}$$

**答案**：\(v_2 = 1.2\,\text{m/s}\)

### 题目104：流量单位换算 ⭐

**题目**：
某管道流量\(Q = 180\,\text{m}^3/\text{h}\)，换算为：
1. L/s
2. m³/s
3. m³/day

**解答**：

1. **转换为 L/s**：
$$Q = 180\,\text{m}^3/\text{h} = \frac{180 \times 1000}{3600}\,\text{L/s} = 50\,\text{L/s}$$

2. **转换为 m³/s**：
$$Q = \frac{180}{3600}\,\text{m}^3/\text{s} = 0.05\,\text{m}^3/\text{s}$$

3. **转换为 m³/day**：
$$Q = 180 \times 24 = 4320\,\text{m}^3/\text{day}$$

**答案**：50 L/s, 0.05 m³/s, 4320 m³/day

### 题目105：支管分流 ⭐⭐

**题目**：
主管直径\(D = 0.4\,\text{m}\)，流速\(v_0 = 2.5\,\text{m/s}\)，分为两个支管（直径均为\(d = 0.2\,\text{m}\)），若流速相等，求支管流速\(v\)。

**解答**：

**步骤1**：主管流量
$$Q_0 = v_0 A_0 = 2.5 \times \frac{\pi \times 0.4^2}{4} = 0.314\,\text{m}^3/\text{s}$$

**步骤2**：每个支管流量（均分）
$$Q = \frac{Q_0}{2} = 0.157\,\text{m}^3/\text{s}$$

**步骤3**：支管流速
$$A = \frac{\pi \times 0.2^2}{4} = 0.0314\,\text{m}^2$$
$$v = \frac{Q}{A} = \frac{0.157}{0.0314} = 5.0\,\text{m/s}$$

**答案**：\(v = 5.0\,\text{m/s}\)（支管流速是主管的2倍）

**规律**：分流时，断面积减半，流速加倍（流量减半）。

### 题目106：三通管汇流 ⭐⭐

**题目**：
两支管汇入主管：
- 支管1：直径\(d_1 = 0.15\,\text{m}\)，流速\(v_1 = 3\,\text{m/s}\)
- 支管2：直径\(d_2 = 0.10\,\text{m}\)，流速\(v_2 = 4\,\text{m/s}\)
- 主管：直径\(D = 0.25\,\text{m}\)

求主管流速\(v_0\)。

**解答**：

**步骤1**：各管流量
$$Q_1 = v_1 A_1 = 3 \times \frac{\pi \times 0.15^2}{4} = 0.0530\,\text{m}^3/\text{s}$$
$$Q_2 = v_2 A_2 = 4 \times \frac{\pi \times 0.10^2}{4} = 0.0314\,\text{m}^3/\text{s}$$

**步骤2**：主管流量（汇流）
$$Q_0 = Q_1 + Q_2 = 0.0530 + 0.0314 = 0.0844\,\text{m}^3/\text{s}$$

**步骤3**：主管流速
$$A_0 = \frac{\pi \times 0.25^2}{4} = 0.0491\,\text{m}^2$$
$$v_0 = \frac{Q_0}{A_0} = \frac{0.0844}{0.0491} = 1.72\,\text{m/s}$$

**答案**：\(v_0 = 1.72\,\text{m/s}\)

### 题目107：环形断面 ⭐⭐

**题目**：
环形管道，外径\(D = 0.3\,\text{m}\)，内径\(d = 0.2\,\text{m}\)，流速\(v = 2.0\,\text{m/s}\)，求流量\(Q\)。

**解答**：

**步骤1**：环形面积
$$A = \frac{\pi}{4}(D^2 - d^2) = \frac{\pi}{4}(0.3^2 - 0.2^2) = 0.0393\,\text{m}^2$$

**步骤2**：流量
$$Q = vA = 2.0 \times 0.0393 = 0.0786\,\text{m}^3/\text{s} = 78.6\,\text{L/s}$$

**答案**：\(Q = 78.6\,\text{L/s}\)

### 题目108：梯形明渠 ⭐⭐

**题目**：
梯形断面明渠，底宽\(b = 3\,\text{m}\)，水深\(h = 2\,\text{m}\)，边坡系数\(m = 1.5\)（即1:1.5），流速\(v = 1.2\,\text{m/s}\)，求流量\(Q\)。

**解答**：

**步骤1**：梯形过流面积公式
$$A = (b + mh) \times h = (3 + 1.5 \times 2) \times 2 = 12\,\text{m}^2$$

**步骤2**：流量
$$Q = vA = 1.2 \times 12 = 14.4\,\text{m}^3/\text{s}$$

**答案**：\(Q = 14.4\,\text{m}^3/\text{s}\)

### 题目109：圆管非满流 ⭐⭐

**题目**：
圆管（直径\(D = 1.0\,\text{m}\)）非满流，水深\(h = 0.6\,\text{m}\)，流速\(v = 1.5\,\text{m/s}\)，求流量\(Q\)。

**解答**：

**步骤1**：非满流过流面积（几何法）
- 水深与半径关系：\(h = 0.6\,\text{m}\)，\(R = 0.5\,\text{m}\)
- 圆心角\(\theta\)：通过\(\cos(\theta/2) = (R-h)/R = -0.2\)，得\(\theta = 2.498\,\text{rad} = 143.1°\)
- 面积：\(A = \frac{R^2}{2}(\theta - \sin\theta) = 0.574\,\text{m}^2\)（查表或计算）

**步骤2**：流量
$$Q = vA = 1.5 \times 0.574 = 0.861\,\text{m}^3/\text{s}$$

**答案**：\(Q \approx 0.86\,\text{m}^3/\text{s}\)

**提示**：圆管非满流面积计算复杂，实际中常查水力计算表。

### 题目110：平均流速定义 ⭐

**题目**：
管道过流断面上各点流速不同，如何定义平均流速\(\bar{v}\)？若断面面积\(A = 0.5\,\text{m}^2\)，流量\(Q = 0.75\,\text{m}^3/\text{s}\)，求\(\bar{v}\)。

**解答**：

**定义**：平均流速是过流断面上的断面平均流速：
$$\bar{v} = \frac{Q}{A}$$

**物理意义**：以平均流速\(\bar{v}\)通过断面\(A\)产生的流量，等于实际流量\(Q\)。

**计算**：
$$\bar{v} = \frac{0.75}{0.5} = 1.5\,\text{m/s}$$

**答案**：\(\bar{v} = 1.5\,\text{m/s}\)

## 第二部分：中等难度题（题111-120）

### 题目111：渐变管段流速分布 ⭐⭐

**题目**：
渐缩管，长度\(L = 10\,\text{m}\)，入口直径\(d_1 = 0.4\,\text{m}\)，出口直径\(d_2 = 0.2\,\text{m}\)，流量\(Q = 0.1\,\text{m}^3/\text{s}\)。求：
1. 入口流速\(v_1\)
2. 出口流速\(v_2\)
3. 中间位置（\(x = 5\,\text{m}\)）处的直径\(d(x)\)和流速\(v(x)\)

**解答**：

**步骤1**：入口流速
$$A_1 = \frac{\pi \times 0.4^2}{4} = 0.1257\,\text{m}^2$$
$$v_1 = \frac{Q}{A_1} = \frac{0.1}{0.1257} = 0.795\,\text{m/s}$$

**步骤2**：出口流速
$$A_2 = \frac{\pi \times 0.2^2}{4} = 0.0314\,\text{m}^2$$
$$v_2 = \frac{Q}{A_2} = \frac{0.1}{0.0314} = 3.18\,\text{m/s}$$

**步骤3**：中间位置（线性渐变）
$$d(x) = d_1 - \frac{d_1 - d_2}{L} \times x = 0.4 - \frac{0.2}{10} \times 5 = 0.3\,\text{m}$$
$$A(x) = \frac{\pi \times 0.3^2}{4} = 0.0707\,\text{m}^2$$
$$v(x) = \frac{Q}{A(x)} = \frac{0.1}{0.0707} = 1.41\,\text{m/s}$$

**答案**：
- \(v_1 = 0.795\,\text{m/s}\)
- \(v_2 = 3.18\,\text{m/s}\)
- \(d(5) = 0.3\,\text{m}\)，\(v(5) = 1.41\,\text{m/s}\)

### 题目112：多管并联分流 ⭐⭐⭐

**题目**：
一根主管（直径\(D = 0.5\,\text{m}\)，流速\(v_0 = 2\,\text{m/s}\)）分为3根并联支管（直径均为\(d\)），若要求各支管流速相同且为\(v = 3\,\text{m/s}\)，求支管直径\(d\)。

**解答**：

**步骤1**：主管流量
$$Q_0 = v_0 A_0 = 2 \times \frac{\pi \times 0.5^2}{4} = 0.393\,\text{m}^3/\text{s}$$

**步骤2**：每根支管流量
$$Q = \frac{Q_0}{3} = 0.131\,\text{m}^3/\text{s}$$

**步骤3**：支管面积
$$A = \frac{Q}{v} = \frac{0.131}{3} = 0.0437\,\text{m}^2$$

**步骤4**：支管直径
$$A = \frac{\pi d^2}{4} \Rightarrow d = \sqrt{\frac{4A}{\pi}} = \sqrt{\frac{4 \times 0.0437}{\pi}} = 0.236\,\text{m}$$

**答案**：\(d = 0.236\,\text{m} \approx 236\,\text{mm}\)

### 题目113：管网节点流量平衡 ⭐⭐⭐

**题目**：
管网节点A，4根管道汇聚：
- 管1：流入，\(Q_1 = 50\,\text{L/s}\)
- 管2：流入，\(Q_2 = 30\,\text{L/s}\)
- 管3：流出，\(Q_3 = ?\)
- 管4：流出，\(d_4 = 200\,\text{mm}\)，\(v_4 = 1.2\,\text{m/s}\)

求\(Q_3\)。

**解答**：

**步骤1**：管4流量
$$Q_4 = v_4 A_4 = 1.2 \times \frac{\pi \times 0.2^2}{4} = 0.0377\,\text{m}^3/\text{s} = 37.7\,\text{L/s}$$

**步骤2**：节点连续性
$$\sum Q_{入} = \sum Q_{出}$$
$$Q_1 + Q_2 = Q_3 + Q_4$$
$$50 + 30 = Q_3 + 37.7$$

**步骤3**：求\(Q_3\)
$$Q_3 = 80 - 37.7 = 42.3\,\text{L/s}$$

**答案**：\(Q_3 = 42.3\,\text{L/s}\)

### 题目114：渠道变宽流速变化 ⭐⭐

**题目**：
矩形明渠，上游：宽\(b_1 = 2\,\text{m}\)，水深\(h_1 = 1.0\,\text{m}\)，流速\(v_1 = 2.0\,\text{m/s}\)。下游渠道加宽至\(b_2 = 3\,\text{m}\)，水深不变，求下游流速\(v_2\)。

**解答**：

**步骤1**：流量
$$Q = v_1 A_1 = 2.0 \times (2 \times 1) = 4.0\,\text{m}^3/\text{s}$$

**步骤2**：下游流速
$$A_2 = b_2 \times h_2 = 3 \times 1 = 3.0\,\text{m}^2$$
$$v_2 = \frac{Q}{A_2} = \frac{4.0}{3.0} = 1.33\,\text{m/s}$$

**答案**：\(v_2 = 1.33\,\text{m/s}\)（渠宽增加50%，流速降低33%）

### 题目115：变截面管道压力计算前奏 ⭐⭐

**题目**：
水平放置的变径管，已知：
- 断面1：\(d_1 = 0.3\,\text{m}\)，\(v_1 = 1.5\,\text{m/s}\)
- 断面2：\(d_2 = 0.15\,\text{m}\)

求断面2的流速\(v_2\)。（注意：本题只用连续性方程，不涉及压力）

**解答**：

$$v_2 = v_1 \times \frac{A_1}{A_2} = v_1 \times \frac{d_1^2}{d_2^2} = 1.5 \times \frac{0.3^2}{0.15^2} = 6.0\,\text{m/s}$$

**答案**：\(v_2 = 6.0\,\text{m/s}\)

**说明**：这是伯努利方程的前置题，先用连续性求流速，再用伯努利求压力。

### 题目121：锥形管段流速分布 ⭐⭐⭐

**题目**：
锥形管，入口半径\(r_1 = 0.2\,\text{m}\)，出口半径\(r_2 = 0.1\,\text{m}\)，长度\(L = 5\,\text{m}\)，流量\(Q = 0.05\,\text{m}^3/\text{s}\)。假设半径随\(x\)线性变化：\(r(x) = r_1 - \frac{r_1 - r_2}{L} x\)。求：
1. 流速\(v(x)\)的表达式
2. 加速度\(a(x) = v \frac{dv}{dx}\)的表达式
3. 最大加速度位置

**解答**：

**步骤1**：半径和面积
$$r(x) = 0.2 - 0.02x$$
$$A(x) = \pi r^2(x) = \pi (0.2 - 0.02x)^2$$

**步骤2**：流速
$$v(x) = \frac{Q}{A(x)} = \frac{0.05}{\pi (0.2 - 0.02x)^2}$$

**步骤3**：加速度
$$a(x) = v \frac{dv}{dx} = \frac{Q}{A} \times \frac{d}{dx}\left(\frac{Q}{A}\right)$$

（详细推导略，涉及链式法则）

结果：加速度在出口处最大。

## 📝 连续性方程总结（题101-125）

### 核心公式

| 公式 | 适用条件 | 说明 |
|------|----------|------|
| \(Q = vA\) | 通用 | 流量定义 |
| \(v_1 A_1 = v_2 A_2\) | 不可压缩定常流 | 连续性方程 |
| \(\sum Q_{入} = \sum Q_{出}\) | 节点平衡 | 管网节点 |
| \(\rho_1 v_1 A_1 = \rho_2 v_2 A_2\) | 可压缩流体 | 推广形式 |

### 常用断面面积

| 形状 | 面积公式 | 说明 |
|------|----------|------|
| 圆形 | \(A = \frac{\pi d^2}{4}\) | 管道最常见 |
| 矩形 | \(A = bh\) | 明渠 |
| 梯形 | \(A = (b + mh)h\) | 土质明渠 |
| 环形 | \(A = \frac{\pi}{4}(D^2 - d^2)\) | 同心管 |

### 解题步骤

1. **识别已知量**：\(d, v, Q, A\) 等
2. **选择公式**：\(Q = vA\) 或 \(v_1 A_1 = v_2 A_2\)
3. **计算面积**：根据几何形状
4. **求解未知量**：流速、流量或直径
5. **检查合理性**：流速通常 0.5-5 m/s（管道），0.3-2 m/s（明渠）

### 物理规律

**缩管**：断面减小 → 流速增大  
**扩管**：断面增大 → 流速减小  
**分流**：总流量守恒 → 各支流流量之和等于主流  
**汇流**：各支流汇合 → 主流流量等于各支流之和

### 实际应用

1. **管道设计**：根据流量和流速选择管径
2. **流量测量**：通过测速计算流量
3. **管网分析**：节点流量平衡
4. **渠道设计**：根据流量确定断面尺寸
