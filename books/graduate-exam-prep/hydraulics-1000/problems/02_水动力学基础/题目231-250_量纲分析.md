## 📚 量纲分析理论基础

### 基本量纲

**力学基本量纲**（MLT系统）：
- M：质量 [kg]
- L：长度 [m]
- T：时间 [s]

**导出量纲**：
- 速度：\([v] = LT^{-1}\)
- 加速度：\([a] = LT^{-2}\)
- 力：\([F] = MLT^{-2}\)
- 压强：\([p] = ML^{-1}T^{-2}\)
- 功率：\([P] = ML^2T^{-3}\)

### 白金汉π定理

若物理现象涉及\(n\)个物理量，\(m\)个基本量纲，则可组成\(n-m\)个无量纲准则数：
$$f(\pi_1, \pi_2, ..., \pi_{n-m}) = 0$$

### 相似准则数

**雷诺数**（Re）：
$$Re = \frac{\rho vL}{\mu} = \frac{vL}{\nu}$$
- 表征惯性力/粘性力比
- 层流/湍流判据

**弗劳德数**（Fr）：
$$Fr = \frac{v}{\sqrt{gL}}$$
- 表征惯性力/重力比
- 明渠流、自由面流动

**欧拉数**（Eu）：
$$Eu = \frac{p}{\rho v^2}$$
- 表征压力/惯性力比

**韦伯数**（We）：
$$We = \frac{\rho v^2 L}{\sigma}$$
- 表征惯性力/表面张力比

### 模型相似

**几何相似**：
$$\lambda_L = \frac{L_p}{L_m} = \text{常数}$$

**运动相似**：
$$\lambda_v = \frac{v_p}{v_m} = \text{常数}$$

**动力相似**：
$$\frac{F_{惯p}}{F_{粘p}} = \frac{F_{惯m}}{F_{粘m}}\quad (Re_p = Re_m)$$
