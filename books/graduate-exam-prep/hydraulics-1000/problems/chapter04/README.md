## 🎯 核心知识点

### 1. 流态判别（弗劳德数）

$$Fr = \frac{v}{\sqrt{gA/B}}$$

- Fr < 1：缓流（亚临界流）
- Fr = 1：临界流
- Fr > 1：急流（超临界流）

### 2. 谢才-曼宁公式

$$Q = AC\sqrt{Ri}$$
$$C = \frac{1}{n}R^{1/6}$$（曼宁公式）

- n：糙率系数
- R：水力半径
- i：水力坡度

### 3. 临界水深

$$h_c = \sqrt[3]{\frac{\alpha Q^2}{gb^2}}$$（矩形断面）

### 4. 水跃方程

$$\frac{h_2}{h_1} = \frac{1}{2}\left(\sqrt{1 + 8Fr_1^2} - 1\right)$$

### 5. 渐变流方程

$$\frac{dh}{dx} = \frac{i - i_f}{1 - Fr^2}$$
