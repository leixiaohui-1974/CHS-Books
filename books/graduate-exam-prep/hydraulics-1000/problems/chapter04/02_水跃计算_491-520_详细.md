## 题491：矩形断面水跃基本计算

**【题目】**（20分，考频⭐⭐⭐⭐⭐）

矩形明渠，宽度b=2m，水跃前水深h₁=0.5m，流速v₁=6m/s。求：
1. 弗劳德数Fr₁
2. 水跃后水深h₂
3. 水跃后流速v₂
4. 水跃高度Δh
5. 水跃水头损失Δh_f
6. 能量相对损失率η

**【解答】**

### 1. 弗劳德数

$$Fr_1 = \frac{v_1}{\sqrt{gh_1}} = \frac{6}{\sqrt{9.8 \times 0.5}}$$
$$= \frac{6}{2.214} = 2.710$$

Fr₁ > 1，确认为急流，可发生水跃 ✓

### 2. 水跃后水深（矩形断面）

$$h_2 = \frac{h_1}{2}\left(\sqrt{1 + 8Fr_1^2} - 1\right)$$

$$= \frac{0.5}{2}\left(\sqrt{1 + 8 \times 2.710^2} - 1\right)$$

$$= 0.25 \times \left(\sqrt{1 + 58.72} - 1\right)$$

$$= 0.25 \times (7.728 - 1) = 1.682 \text{ m}$$

### 3. 水跃后流速

连续性方程：
$$v_2 = \frac{h_1 v_1}{h_2} = \frac{0.5 \times 6}{1.682} = 1.783 \text{ m/s}$$

验证：$Fr_2 = \frac{1.783}{\sqrt{9.8 \times 1.682}} = 0.438 < 1$ ✓（缓流）

### 4. 水跃高度

$$\Delta h = h_2 - h_1 = 1.682 - 0.5 = 1.182 \text{ m}$$

### 5. 水头损失

$$\Delta h_f = E_1 - E_2 = \left(h_1 + \frac{v_1^2}{2g}\right) - \left(h_2 + \frac{v_2^2}{2g}\right)$$

$$E_1 = 0.5 + \frac{6^2}{2 \times 9.8} = 0.5 + 1.837 = 2.337 \text{ m}$$

$$E_2 = 1.682 + \frac{1.783^2}{2 \times 9.8} = 1.682 + 0.162 = 1.844 \text{ m}$$

$$\Delta h_f = 2.337 - 1.844 = 0.493 \text{ m}$$

### 6. 能量相对损失率

$$\eta = \frac{\Delta h_f}{E_1 - h_1} \times 100\%$$

$$= \frac{0.493}{2.337 - 0.5} \times 100\% = \frac{0.493}{1.837} \times 100\%$$

$$= 26.8\%$$

### Python代码

```python
import numpy as np

def hydraulic_jump_rectangular(h1, v1, b, g=9.8):
    """
    矩形断面水跃计算
    
    参数:
        h1: 水跃前水深 (m)
        v1: 水跃前流速 (m/s)
        b: 渠宽 (m)
        g: 重力加速度 (m/s²)
    
    返回:
        Fr1, h2, v2, delta_h, delta_hf, eta
    """
    # 弗劳德数
    Fr1 = v1 / np.sqrt(g * h1)
    
    # 水跃后水深
    h2 = h1 / 2 * (np.sqrt(1 + 8 * Fr1**2) - 1)
    
    # 水跃后流速（连续性方程）
    v2 = h1 * v1 / h2
    
    # 水跃高度
    delta_h = h2 - h1
    
    # 断面比能
    E1 = h1 + v1**2 / (2*g)
    E2 = h2 + v2**2 / (2*g)
    
    # 水头损失
    delta_hf = E1 - E2
    
    # 能量相对损失率
    eta = delta_hf / (E1 - h1) * 100
    
    # 验证Fr2
    Fr2 = v2 / np.sqrt(g * h2)
    
    return {
        'Fr1': Fr1,
        'h2': h2,
        'v2': v2,
        'delta_h': delta_h,
        'delta_hf': delta_hf,
        'eta': eta,
        'Fr2': Fr2,
        'E1': E1,
        'E2': E2
    }

# 计算
result = hydraulic_jump_rectangular(0.5, 6, 2)

print(f"弗劳德数 Fr₁ = {result['Fr1']:.3f}")
print(f"水跃后水深 h₂ = {result['h2']:.3f} m")
print(f"水跃后流速 v₂ = {result['v2']:.3f} m/s")
print(f"水跃高度 Δh = {result['delta_h']:.3f} m")
print(f"水头损失 Δh_f = {result['delta_hf']:.3f} m")
print(f"能量损失率 η = {result['eta']:.1f}%")
print(f"验证 Fr₂ = {result['Fr2']:.3f} (应<1)")
```

### 答案

1. 弗劳德数：**Fr₁ = 2.710**
2. 水跃后水深：**h₂ = 1.682 m**
3. 水跃后流速：**v₂ = 1.783 m/s**
4. 水跃高度：**Δh = 1.182 m**
5. 水头损失：**Δh_f = 0.493 m**
6. 能量损失率：**η = 26.8%**

## 题492：水跃类型判断与消能率

**【题目】**（20分，考频⭐⭐⭐⭐）

已知矩形明渠水跃，跃前Fr₁分别为：
1. Fr₁ = 1.2
2. Fr₁ = 2.5
3. Fr₁ = 4.5
4. Fr₁ = 9.0

求各种情况下的：
- 水跃类型
- 共轭水深比h₂/h₁
- 能量损失率η

**【解答】**

### 水跃分类标准

| Fr₁范围 | 水跃类型 | 特征 |
|---------|---------|------|
| 1.0-1.7 | 波状水跃 | 表面波动，消能率<5% |
| 1.7-2.5 | 弱水跃 | 水面稍有波动，5%<η<15% |
| 2.5-4.5 | 稳定水跃 | 表面稳定，15%<η<45% |
| 4.5-9.0 | 强水跃 | 跃后强烈波动，45%<η<70% |
| >9.0 | 激烈水跃 | 剧烈波动，η>70% |

### 1. Fr₁ = 1.2

**水跃类型**：波状水跃

$$h_2/h_1 = \frac{1}{2}(\sqrt{1 + 8 \times 1.2^2} - 1) = 0.689$$
$$h_2 = 1.689h_1$$

能量损失率：
$$\eta = \frac{(Fr_1^2 - 1)^3}{4Fr_1^2(1 + \sqrt{1+8Fr_1^2})^2} \times 100\%$$
$$= \frac{(1.44-1)^3}{4 \times 1.44 \times (1+3.378)^2} \times 100\% = 0.37\%$$

### 2. Fr₁ = 2.5

**水跃类型**：弱水跃（临界稳定水跃）

$$h_2/h_1 = 0.5(\sqrt{1+50} - 1) = 3.071$$

$$\eta = \frac{(6.25-1)^3}{4 \times 6.25 \times (1+7.14)^2} \times 100\% = 17.1\%$$

### 3. Fr₁ = 4.5

**水跃类型**：稳定水跃（临界强水跃）

$$h_2/h_1 = 0.5(\sqrt{1+162} - 1) = 5.872$$

$$\eta = \frac{(20.25-1)^3}{4 \times 20.25 \times (1+12.74)^2} \times 100\% = 45.2\%$$

### 4. Fr₁ = 9.0

**水跃类型**：强水跃

$$h_2/h_1 = 0.5(\sqrt{1+648} - 1) = 12.26$$

$$\eta = \frac{(81-1)^3}{4 \times 81 \times (1+25.48)^2} \times 100\% = 69.7\%$$

### Python代码

```python
def classify_jump(Fr1):
    """
    水跃分类与计算
    """
    # 共轭水深比
    ratio = 0.5 * (np.sqrt(1 + 8*Fr1**2) - 1)
    
    # 能量损失率（精确公式）
    eta = (Fr1**2 - 1)**3 / (4 * Fr1**2 * (1 + np.sqrt(1+8*Fr1**2))**2) * 100
    
    # 分类
    if Fr1 < 1.7:
        jump_type = "波状水跃"
    elif Fr1 < 2.5:
        jump_type = "弱水跃"
    elif Fr1 < 4.5:
        jump_type = "稳定水跃"
    elif Fr1 < 9.0:
        jump_type = "强水跃"
    else:
        jump_type = "激烈水跃"
    
    return jump_type, ratio, eta

# 批量计算
Fr1_list = [1.2, 2.5, 4.5, 9.0]
for Fr1 in Fr1_list:
    jtype, ratio, eta = classify_jump(Fr1)
    print(f"\nFr₁ = {Fr1:.1f}:")
    print(f"  类型: {jtype}")
    print(f"  h₂/h₁ = {ratio:.3f}")
    print(f"  η = {eta:.1f}%")
```

| 2.5 | 弱水跃 | 3.071 | 17.1 |
| 4.5 | 稳定水跃 | 5.872 | 45.2 |
| 9.0 | 强水跃 | 12.26 | 69.7 |
