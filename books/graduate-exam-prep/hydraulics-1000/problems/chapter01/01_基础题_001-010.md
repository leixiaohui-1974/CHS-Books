## 题001：静水压强计算

**【题目】**（10分，考频⭐⭐⭐⭐⭐）

某水库水面以下10m深处有一点A，求该点的静水压强。已知：
- 水的密度 ρ = 1000 kg/m³
- 重力加速度 g = 9.8 m/s²
- 当地大气压 p₀ = 101325 Pa

求：
1. 相对压强（kPa）
2. 绝对压强（kPa）

**【解答】**

### 基本公式

静水压强公式：

$$p = \rho g h$$

其中：
- $p$ = 相对压强（Pa）
- $\rho$ = 液体密度（kg/m³）
- $g$ = 重力加速度（m/s²）
- $h$ = 水深（m）

### 计算步骤

**步骤1：计算相对压强**

$$p = \rho g h = 1000 \times 9.8 \times 10 = 98000 \text{ Pa} = 98 \text{ kPa}$$

**步骤2：计算绝对压强**

$$p_{abs} = p_0 + p = 101.325 + 98 = 199.325 \text{ kPa}$$

### 答案

1. 相对压强：**98 kPa**
2. 绝对压强：**199.325 kPa**

### 踩分点（10分）

- 公式正确（2分）：$p = \rho g h$
- 相对压强计算（4分）：98 kPa
- 绝对压强概念（2分）：$p_{abs} = p_0 + p$
- 绝对压强计算（2分）：199.325 kPa

### 易错点

⚠️ **易错1**：忘记区分相对压强和绝对压强
- 相对压强：以大气压为零点
- 绝对压强：以绝对真空为零点

⚠️ **易错2**：单位换算错误
- 注意 Pa 与 kPa 的换算：1 kPa = 1000 Pa

⚠️ **易错3**：g值使用错误
- 标准重力加速度：9.8 m/s² 或 9.81 m/s²
- 有些题目给出10 m/s²（简化计算）

### Python代码

```python
# 文件：problem_001_hydrostatic_pressure.py
def calculate_pressure(rho=1000, g=9.8, h=10, p0=101325):
    """
    计算静水压强
    
    参数:
        rho: 液体密度 (kg/m³)
        g: 重力加速度 (m/s²)
        h: 水深 (m)
        p0: 大气压 (Pa)
    
    返回:
        p_relative: 相对压强 (Pa)
        p_absolute: 绝对压强 (Pa)
    """
    p_relative = rho * g * h
    p_absolute = p0 + p_relative
    
    return p_relative, p_absolute

# 计算
p_rel, p_abs = calculate_pressure()
print(f"相对压强: {p_rel/1000:.2f} kPa")
print(f"绝对压强: {p_abs/1000:.2f} kPa")
```

## 题002：压强分布规律

**【题目】**（15分，考频⭐⭐⭐⭐⭐）

某蓄水池深度为5m，水面以上为大气压。绘制静水压强分布图，并计算以下深度处的压强：
1. 水面（h=0）
2. 2m深处
3. 池底（h=5m）

**【解答】**

### 压强分布规律

静水压强的特点：
1. **沿铅直方向线性增加**：$p = \rho g h$
2. **同一水平面压强相等**：等压面
3. **压强大小与深度成正比**

### 计算

**1. 水面处（h=0）**

$$p_0 = \rho g \times 0 = 0 \text{ Pa（相对压强）}$$

**2. 2m深处（h=2m）**

$$p_2 = 1000 \times 9.8 \times 2 = 19600 \text{ Pa} = 19.6 \text{ kPa}$$

**3. 池底处（h=5m）**

$$p_5 = 1000 \times 9.8 \times 5 = 49000 \text{ Pa} = 49 \text{ kPa}$$

### 压强分布图

```
水面 ────────────── p = 0 kPa
  │
  │ h=2m           p = 19.6 kPa
  │
池底 ────────────── p = 49 kPa

压强分布：呈三角形，线性增长
```

### 答案

| 深度 | 压强（相对） |
|------|-------------|
| 0m   | 0 kPa       |
| 2m   | 19.6 kPa    |
| 5m   | 49 kPa      |

### 踩分点（15分）

- 压强分布规律描述（3分）
- 水面压强（3分）：0 kPa
- 2m处压强（3分）：19.6 kPa
- 池底压强（3分）：49 kPa
- 分布图绘制（3分）

## 题003：不同液体的压强

**【题目】**（15分，考频⭐⭐⭐⭐）

某容器内分别装有：
1. 水（ρ₁ = 1000 kg/m³）
2. 煤油（ρ₂ = 800 kg/m³）
3. 水银（ρ₃ = 13600 kg/m³）

液面以下均为3m深，求各液体在该深度处的压强。

**【解答】**

### 不同液体压强特点

$$p = \rho g h$$

在相同深度下：
- **密度越大，压强越大**
- 压强与密度成正比

### 计算

取 g = 9.8 m/s²，h = 3 m

**1. 水中压强**

$$p_1 = \rho_1 g h = 1000 \times 9.8 \times 3 = 29400 \text{ Pa} = 29.4 \text{ kPa}$$

**2. 煤油中压强**

$$p_2 = \rho_2 g h = 800 \times 9.8 \times 3 = 23520 \text{ Pa} = 23.52 \text{ kPa}$$

**3. 水银中压强**

$$p_3 = \rho_3 g h = 13600 \times 9.8 \times 3 = 399840 \text{ Pa} = 399.84 \text{ kPa}$$

### 比较

$$p_3 : p_1 : p_2 = 13600 : 1000 : 800 = 17 : 1.25 : 1$$

水银压强约为水的13.6倍！

### 答案

| 液体 | 密度(kg/m³) | 压强(kPa) |
|------|------------|-----------|
| 水   | 1000       | 29.4      |
| 煤油 | 800        | 23.52     |
| 水银 | 13600      | 399.84    |

### 踩分点（15分）

- 公式应用（3分）
- 水中压强（4分）
- 煤油压强（4分）
- 水银压强（4分）

## 题004：U型测压管

**【题目】**（15分，考频⭐⭐⭐⭐⭐）

U型测压管如图所示，左侧接水箱（水深2m），右侧开口与大气相通。求测压管中液面高差。

**【解答】**

### U型测压管原理

**核心原理：等压面**
- 同一连通液体的同一水平面压强相等
- U型管两侧液面高差反映压强差

### 分析

设：
- 水箱水深 h₁ = 2 m
- 测压管液面高于水箱底面 h₂ = ?

**等压面分析**：
在U型管最低点建立等压面：

左侧压强：
$$p_L = \rho g h_1 = 1000 \times 9.8 \times 2 = 19600 \text{ Pa}$$

右侧压强：
$$p_R = \rho g h_2$$

等压面原理：
$$p_L = p_R$$

### 计算

$$\rho g h_1 = \rho g h_2$$
$$h_2 = h_1 = 2 \text{ m}$$

**结论**：测压管液面高度等于水箱水深

### 液面高差

如果U型管右侧初始液面在管底：
$$\Delta h = h_2 - 0 = 2 \text{ m}$$

### 答案

测压管液面高差：**2 m**

### 踩分点（15分）

- 等压面原理（4分）
- 左侧压强分析（3分）
- 右侧压强分析（3分）
- 高差计算（3分）
- 结论（2分）

### 易错点

⚠️ **易错**：混淆测压管液面与水箱水面
- 测压管反映的是**底部压强**
- 与水箱水深相等（连通器原理）

## 题005：多层液体压强

**【题目】**（20分，考频⭐⭐⭐⭐）

某容器内自上而下依次装有：
1. 煤油层：厚度0.5m，密度800 kg/m³
2. 水层：厚度1.5m，密度1000 kg/m³  
3. 水银层：厚度0.2m，密度13600 kg/m³

求：
1. 煤油与水分界面处的压强
2. 水与水银分界面处的压强
3. 容器底部压强

**【解答】**

### 多层液体压强叠加

对于n层液体：

$$p = \sum_{i=1}^{n} \rho_i g h_i = \rho_1 g h_1 + \rho_2 g h_2 + ... + \rho_n g h_n$$

### 计算

**1. 煤油-水分界面（深度0.5m）**

$$p_1 = \rho_{煤油} g h_1 = 800 \times 9.8 \times 0.5 = 3920 \text{ Pa} = 3.92 \text{ kPa}$$

**2. 水-水银分界面（深度2m）**

$$p_2 = \rho_{煤油} g h_1 + \rho_{水} g h_2$$
$$= 800 \times 9.8 \times 0.5 + 1000 \times 9.8 \times 1.5$$
$$= 3920 + 14700 = 18620 \text{ Pa} = 18.62 \text{ kPa}$$

**3. 容器底部（深度2.2m）**

$$p_3 = p_2 + \rho_{水银} g h_3$$
$$= 18620 + 13600 \times 9.8 \times 0.2$$
$$= 18620 + 26656 = 45276 \text{ Pa} = 45.28 \text{ kPa}$$

### 压强分布图

```
煤油表面 ──────── p = 0 kPa
    │ 0.5m (煤油)
界面1 ──────────── p = 3.92 kPa
    │ 1.5m (水)
界面2 ──────────── p = 18.62 kPa
    │ 0.2m (水银)
底部 ────────────── p = 45.28 kPa
```

### 答案

| 位置 | 压强（kPa） |
|------|------------|
| 煤油-水分界面 | 3.92 |
| 水-水银分界面 | 18.62 |
| 容器底部 | 45.28 |

### 踩分点（20分）

- 多层液体原理（4分）
- 界面1压强（5分）
- 界面2压强（5分）
- 底部压强（5分）
- 分析完整性（1分）

## 题006：真空压强

**【题目】**（15分，考频⭐⭐⭐）

某密闭容器内液面上方为真空（绝对压强接近0），液体为水（ρ=1000 kg/m³），液面下3m处有一点A。求：
1. A点的相对压强
2. A点的绝对压强
3. 与敞口容器的压强对比

**【解答】**

### 真空容器压强特点

**关键区别**：
- 敞口容器：液面上方为大气压 p₀
- 真空容器：液面上方压强≈0

### 计算

**1. A点相对压强**

相对压强以大气压为基准：

$$p_{相对} = \rho g h - p_0 = 1000 \times 9.8 \times 3 - 101325$$
$$= 29400 - 101325 = -71925 \text{ Pa} = -71.925 \text{ kPa}$$

负值表示**真空度**（低于大气压）

**2. A点绝对压强**

$$p_{绝对} = \rho g h = 1000 \times 9.8 \times 3 = 29400 \text{ Pa} = 29.4 \text{ kPa}$$

**3. 对比敞口容器**

敞口容器A点：

$$p_{敞口,绝对} = p_0 + \rho g h = 101325 + 29400 = 130725 \text{ Pa}$$

### 比较

| 类型 | 绝对压强(kPa) | 相对压强(kPa) |
|------|--------------|--------------|
| 真空容器 | 29.4 | -71.925 |
| 敞口容器 | 130.725 | +29.4 |

**差值**：101.325 kPa（恰好等于大气压）

### 答案

1. 相对压强：**-71.925 kPa**（真空度）
2. 绝对压强：**29.4 kPa**
3. 比敞口容器低 **101.325 kPa**

### 踩分点（15分）

- 真空概念理解（3分）
- 相对压强计算（4分）
- 绝对压强计算（4分）
- 对比分析（4分）

### 易错点

⚠️ **易错**：混淆真空度与绝对压强
- 真空度 = 大气压 - 绝对压强
- 相对压强 = 绝对压强 - 大气压（可为负）

## 题007：压强表与真空表读数

**【题目】**（15分，考频⭐⭐⭐⭐）

某管道上安装压强表和真空表。已知：
- 压强表读数：150 kPa
- 真空表读数：40 kPa  
- 当地大气压：100 kPa

求：
1. 压强表测点的绝对压强
2. 真空表测点的绝对压强
3. 两测点的压强差

**【解答】**

### 仪表读数含义

**压强表（Pressure Gauge）**：
- 测量**高于大气压**的压强
- 读数 = 相对压强（正值）
- $p_{绝对} = p_0 + p_{表}$

**真空表（Vacuum Gauge）**：
- 测量**低于大气压**的压强
- 读数 = 真空度（正值表示真空程度）
- $p_{绝对} = p_0 - p_{真空}$

### 计算

**1. 压强表测点绝对压强**

$$p_1 = p_0 + p_{表} = 100 + 150 = 250 \text{ kPa}$$

**2. 真空表测点绝对压强**

$$p_2 = p_0 - p_{真空} = 100 - 40 = 60 \text{ kPa}$$

**3. 压强差**

$$\Delta p = p_1 - p_2 = 250 - 60 = 190 \text{ kPa}$$

### 关系图

```
绝对压强        相对压强
────────────────────────
250 kPa  ←─→  +150 kPa (压强表)
100 kPa  ←─→    0 kPa  (大气压基准)
 60 kPa  ←─→  -40 kPa (真空表,表示为真空度40kPa)
  0 kPa  ←─→  -100 kPa (绝对真空)
```

### 答案

1. 压强表测点：**250 kPa**
2. 真空表测点：**60 kPa**
3. 压强差：**190 kPa**

### 踩分点（15分）

- 仪表读数理解（3分）
- 压强表测点（4分）
- 真空表测点（4分）
- 压强差计算（4分）

## 题008：水银气压计

**【题目】**（15分，考频⭐⭐⭐）

某地用水银气压计测量大气压，水银柱高度为750 mmHg。求：
1. 当地大气压（Pa）
2. 换算成标准大气压（atm）
3. 换算成水柱高度（mH₂O）

已知：水银密度 ρHg = 13600 kg/m³，水密度 ρ = 1000 kg/m³

**【解答】**

### 气压计原理

水银柱压强 = 大气压强

$$p_0 = \rho_{Hg} g h_{Hg}$$

### 计算

**1. 大气压（Pa）**

$$p_0 = 13600 \times 9.8 \times 0.75 = 99960 \text{ Pa} ≈ 100 \text{ kPa}$$

**2. 标准大气压（atm）**

$$p_0 = \frac{99960}{101325} = 0.987 \text{ atm} ≈ 0.99 \text{ atm}$$

**3. 等效水柱高度**

设等效水柱高度为 h：

$$\rho g h = \rho_{Hg} g h_{Hg}$$
$$h = \frac{\rho_{Hg}}{\rho} \times h_{Hg} = \frac{13600}{1000} \times 0.75 = 10.2 \text{ m}$$

### 单位换算关系

```
1 标准大气压 = 101325 Pa
            = 760 mmHg
            = 10.33 mH₂O
            = 1.0333 kgf/cm²
            = 1.0133 bar
```

### 答案

1. 大气压：**99960 Pa ≈ 100 kPa**
2. 标准大气压：**0.987 atm**
3. 水柱高度：**10.2 m**

## 题009：液封原理

**【题目】**（20分，考频⭐⭐⭐⭐）

某化工设备采用水封，水封高度为2m。设备内部为负压（真空度20 kPa）。求：
1. 水封底部相对压强
2. 水封底部绝对压强
3. 判断水封是否有效（大气压100 kPa）

**【解答】**

### 水封原理

水封作用：
- 隔离内外气体
- 保持设备真空度
- 防止外界空气进入

**有效条件**：
$$\rho g h_{水封} > p_{真空度}$$

### 分析

设备内部：
- 绝对压强：$p_{内} = p_0 - p_{真空} = 100 - 20 = 80$ kPa
- 相对压强：-20 kPa

### 计算

**1. 水封底部相对压强（以大气压为基准）**

水封外侧底部：大气压 + 水柱压强
$$p_{外,相对} = 0 + \rho g h = 0 + 1000 \times 9.8 \times 2 = 19.6 \text{ kPa}$$

水封内侧底部：真空 + 水柱压强
$$p_{内,相对} = -20 + 19.6 = -0.4 \text{ kPa}$$

**2. 水封底部绝对压强**

外侧：
$$p_{外,绝对} = 100 + 19.6 = 119.6 \text{ kPa}$$

内侧：
$$p_{内,绝对} = 80 + 19.6 = 99.6 \text{ kPa}$$

**3. 有效性判断**

水封提供的压差：
$$\Delta p = \rho g h = 19.6 \text{ kPa}$$

设备真空度：
$$p_{真空} = 20 \text{ kPa}$$

判断：
$$19.6 < 20$$

**结论：水封不够！** 需要增加水封高度。

### 最小水封高度

$$h_{min} = \frac{p_{真空}}{\rho g} = \frac{20000}{1000 \times 9.8} = 2.04 \text{ m}$$

需要至少 **2.04 m** 水封高度。

### 答案

1. 水封底部相对压强：
   - 外侧：19.6 kPa
   - 内侧：-0.4 kPa

2. 水封底部绝对压强：
   - 外侧：119.6 kPa
   - 内侧：99.6 kPa

3. **水封无效**，建议高度≥2.04 m

### 踩分点（20分）

- 水封原理（4分）
- 相对压强（5分）
- 绝对压强（5分）
- 有效性判断（4分）
- 最小高度（2分）

## 题010：混合液体分层

**【题目】**（20分，考频⭐⭐⭐）

容器内有两种不互溶液体：
- 上层：油（密度900 kg/m³，厚度1.5m）
- 下层：水（密度1000 kg/m³，厚度2.0m）

求：
1. 油水界面处压强
2. 容器底部压强
3. 若在底部开孔（孔径很小），哪种液体先流出？

**【解答】**

### 分层液体特点

**关键概念**：
1. 密度小的液体在上层
2. 界面处压强连续
3. 底部压强最大

### 计算

**1. 油水界面压强**

$$p_1 = \rho_{油} g h_{油} = 900 \times 9.8 \times 1.5 = 13230 \text{ Pa} = 13.23 \text{ kPa}$$

**2. 容器底部压强**

$$p_2 = p_1 + \rho_{水} g h_{水}$$
$$= 13230 + 1000 \times 9.8 \times 2.0$$
$$= 13230 + 19600 = 32830 \text{ Pa} = 32.83 \text{ kPa}$$

**3. 流出顺序判断**

底部开孔时：
- 底层水直接接触孔口
- 底层压强最大
- **水先流出**

### 流出过程分析

**阶段1**：水流出
- 水面下降
- 油水界面下移
- 底部压强减小

**阶段2**：油开始流出
- 当水全部流完
- 油接触孔口
- 油开始流出

### 压强分布

```
油表面 ──────── p = 0 kPa
    │ 1.5m (油)
    │ p = 13.23 kPa
界面 ────────── 
    │ 2.0m (水)
    │ p = 32.83 kPa
底部 ──────────
```

### 答案

1. 油水界面压强：**13.23 kPa**
2. 容器底部压强：**32.83 kPa**
3. **水先流出**（底层液体优先）

### 踩分点（20分）

- 界面压强计算（5分）
- 底部压强计算（5分）
- 流出顺序判断（6分）
- 流出过程分析（4分）
