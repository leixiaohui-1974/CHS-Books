## 🎯 核心知识点

### 1. 层流与湍流

**判别准则**：
$$Re = \frac{vd}{\nu}$$

- Re < 2300：层流
- Re > 4000：湍流
- 2300 < Re < 4000：过渡区

**层流速度分布**：
$$u = u_{max}\left(1 - \frac{r^2}{R^2}\right)$$

### 2. 沿程阻力

**达西-魏斯巴赫公式**：
$$h_f = \lambda \frac{L}{d} \frac{v^2}{2g}$$

**层流**：$\lambda = \frac{64}{Re}$

**湍流**：查莫迪图或用公式

### 3. 管道系统

**串联**：
$$h_w = \sum h_{fi} + \sum h_{ji}$$

**并联**：
$$h_{w1} = h_{w2} = ... = h_{wn}$$
$$Q = \sum Q_i$$

### 4. 管网计算

**节点法**：
$$\sum Q_{in} = \sum Q_{out}$$

**环路法**：
$$\sum h_w = 0$$（闭合环路）

### 5. 水锤

**水锤压强**：
$$\Delta p = \rho c \Delta v$$

c：水锤波速
$$c = \frac{K/\rho}{\sqrt{1 + \frac{Kd}{E\delta}}}$$
