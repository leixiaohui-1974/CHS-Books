## 题751：离心泵工作点确定

**【题目】**（25分，考频⭐⭐⭐⭐⭐）

某离心泵特性曲线为：
$$H = 30 - 0.002Q^2$$

管路系统特性：
$$H_s = 5 + 0.003Q^2$$

式中：H(m)，Q(m³/h)

求：
1. 泵的工作点（Q₀, H₀）
2. 轴功率（效率η=75%）
3. 若并联两台同型泵，新工作点
4. 若串联两台，新工作点

**【解答】**

### 1. 工作点计算

工作点处：$H = H_s$

$$30 - 0.002Q^2 = 5 + 0.003Q^2$$

$$25 = 0.005Q^2$$

$$Q_0 = \sqrt{5000} = 70.71 \text{ m}^3/\text{h}$$

$$H_0 = 30 - 0.002 \times 70.71^2 = 30 - 10 = 20 \text{ m}$$

### 2. 轴功率

$$P = \frac{\rho g Q H}{\eta}$$

$$= \frac{1000 \times 9.8 \times (70.71/3600) \times 20}{0.75}$$

$$= \frac{1000 \times 9.8 \times 0.01964 \times 20}{0.75}$$

$$= \frac{3851.7}{0.75} = 5135.6 \text{ W} = 5.14 \text{ kW}$$

### 3. 并联运行

并联特性：
$$H_{并} = 30 - 0.002\left(\frac{Q}{2}\right)^2 = 30 - 0.0005Q^2$$

新工作点：
$$30 - 0.0005Q^2 = 5 + 0.003Q^2$$

$$25 = 0.0035Q^2$$

$$Q_{并} = \sqrt{7142.9} = 84.52 \text{ m}^3/\text{h}$$

$$H_{并} = 5 + 0.003 \times 84.52^2 = 26.4 \text{ m}$$

### 4. 串联运行

串联特性：
$$H_{串} = 2 \times (30 - 0.002Q^2) = 60 - 0.004Q^2$$

新工作点：
$$60 - 0.004Q^2 = 5 + 0.003Q^2$$

$$55 = 0.007Q^2$$

$$Q_{串} = \sqrt{7857.1} = 88.64 \text{ m}^3/\text{h}$$

$$H_{串} = 5 + 0.003 \times 88.64^2 = 28.6 \text{ m}$$

### Python代码

```python
import numpy as np
import matplotlib.pyplot as plt

def pump_curve(Q):
    """泵特性曲线"""
    return 30 - 0.002 * Q**2

def system_curve(Q):
    """管路特性曲线"""
    return 5 + 0.003 * Q**2

def find_working_point(pump_func, system_func, Q_range=(0, 200)):
    """
    求工作点
    """
    from scipy.optimize import fsolve
    
    def equation(Q):
        return pump_func(Q) - system_func(Q)
    
    Q0 = fsolve(equation, 50)[0]
    H0 = pump_func(Q0)
    
    return Q0, H0

def shaft_power(Q, H, rho=1000, g=9.8, eta=0.75):
    """
    轴功率计算
    
    参数:
        Q: 流量 (m³/h)
        H: 扬程 (m)
        eta: 效率
    """
    Q_si = Q / 3600  # 转换为 m³/s
    P = rho * g * Q_si * H / eta  # W
    return P / 1000  # kW

# 单泵工作点
Q0, H0 = find_working_point(pump_curve, system_curve)
P0 = shaft_power(Q0, H0)

print(f"单泵工作点:")
print(f"  流量 Q₀ = {Q0:.2f} m³/h")
print(f"  扬程 H₀ = {H0:.2f} m")
print(f"  功率 P = {P0:.2f} kW")

# 并联
def pump_parallel(Q):
    return 30 - 0.0005 * Q**2

Q_par, H_par = find_working_point(pump_parallel, system_curve)
print(f"\n并联工作点:")
print(f"  流量 Q = {Q_par:.2f} m³/h")
print(f"  扬程 H = {H_par:.2f} m")

# 串联
def pump_series(Q):
    return 60 - 0.004 * Q**2

Q_ser, H_ser = find_working_point(pump_series, system_curve)
print(f"\n串联工作点:")
print(f"  流量 Q = {Q_ser:.2f} m³/h")
print(f"  扬程 H = {H_ser:.2f} m")

# 绘图
Q = np.linspace(0, 150, 300)
plt.figure(figsize=(10, 6))
plt.plot(Q, pump_curve(Q), label='单泵特性', linewidth=2)
plt.plot(Q, pump_parallel(Q), label='并联特性', linewidth=2)
plt.plot(Q, pump_series(Q), label='串联特性', linewidth=2)
plt.plot(Q, system_curve(Q), label='管路特性', linewidth=2, linestyle='--')

plt.scatter([Q0], [H0], s=100, c='red', marker='o', label='单泵工作点')
plt.scatter([Q_par], [H_par], s=100, c='green', marker='s', label='并联工作点')
plt.scatter([Q_ser], [H_ser], s=100, c='blue', marker='^', label='串联工作点')

plt.xlabel('流量 Q (m³/h)', fontsize=12)
plt.ylabel('扬程 H (m)', fontsize=12)
plt.title('泵站工作点分析', fontsize=14)
plt.legend()
plt.grid(True, alpha=0.3)
plt.savefig('pump_working_points.png', dpi=300, bbox_inches='tight')
plt.show()
```

### 答案

1. 单泵工作点：**Q₀ = 70.71 m³/h，H₀ = 20 m**
2. 轴功率：**P = 5.14 kW**
3. 并联工作点：**Q = 84.52 m³/h，H = 26.4 m**
4. 串联工作点：**Q = 88.64 m³/h，H = 28.6 m**

### 工程意义

1. **并联**：增加流量（+19.5%），扬程略增（+32%）
2. **串联**：增加扬程（+43%），流量增加较少（+25.3%）
3. **选型**：根据系统需求选择并联或串联

## 题752：泵站汽蚀余量计算

**【题目】**（20分，考频⭐⭐⭐⭐⭐）

某离心泵安装在海拔500m处，水温20℃。泵参数：
- 额定流量：Q = 100 m³/h
- 必需汽蚀余量：NPSHr = 3.5 m
- 吸水管路损失：Σh_s = 0.5 m
- 吸水高度：H_s = 2.5 m

已知：
- 大气压：p_a = 101325 Pa（海平面）
- 水温20℃饱和蒸汽压：p_v = 2337 Pa
- 海拔每升高100m，大气压降低约1200 Pa

求：
1. 当地大气压
2. 有效汽蚀余量NPSHa
3. 是否会发生汽蚀
4. 最大允许安装高度

**【解答】**

### 1. 当地大气压

$$p_a' = p_a - \frac{500}{100} \times 1200$$
$$= 101325 - 6000 = 95325 \text{ Pa}$$

### 2. 有效汽蚀余量

$$NPSH_a = \frac{p_a'}{\rho g} - H_s - \Sigma h_s - \frac{p_v}{\rho g}$$

$$= \frac{95325}{1000 \times 9.8} - 2.5 - 0.5 - \frac{2337}{1000 \times 9.8}$$

$$= 9.726 - 2.5 - 0.5 - 0.238 = 6.488 \text{ m}$$

### 3. 汽蚀判断

$$NPSH_a = 6.488 \text{ m} > NPSH_r = 3.5 \text{ m}$$

**安全裕量**：
$$\Delta = NPSH_a - NPSH_r = 6.488 - 3.5 = 2.988 \text{ m}$$

**不会发生汽蚀** ✓

### 4. 最大允许安装高度

$$[H_s]_{max} = \frac{p_a'}{\rho g} - \Sigma h_s - \frac{p_v}{\rho g} - NPSH_r$$

$$= 9.726 - 0.5 - 0.238 - 3.5 = 5.488 \text{ m}$$

实际安装高度2.5m < 5.488m ✓

### Python代码

```python
def cavitation_analysis(
    elevation, 
    water_temp,
    Q,
    NPSHr,
    Hs,
    sum_hs,
    rho=1000,
    g=9.8
):
    """
    泵站汽蚀分析
    
    参数:
        elevation: 海拔 (m)
        water_temp: 水温 (°C)
        Q: 流量 (m³/h)
        NPSHr: 必需汽蚀余量 (m)
        Hs: 吸水高度 (m)
        sum_hs: 吸水管路损失 (m)
    """
    # 大气压修正（海拔效应）
    pa_sea = 101325  # Pa
    pa_local = pa_sea - (elevation / 100) * 1200
    
    # 饱和蒸汽压（简化）
    pv_dict = {
        0: 611,
        10: 1228,
        20: 2337,
        30: 4242,
        40: 7375,
        50: 12335,
        60: 19920,
        70: 31160,
        80: 47360,
        90: 70100,
        100: 101325
    }
    pv = pv_dict.get(water_temp, 2337)
    
    # 有效汽蚀余量
    NPSHa = (pa_local / (rho * g)) - Hs - sum_hs - (pv / (rho * g))
    
    # 安全裕量
    delta_NPSH = NPSHa - NPSHr
    
    # 最大允许安装高度
    Hs_max = (pa_local / (rho * g)) - sum_hs - (pv / (rho * g)) - NPSHr
    
    # 判断
    is_safe = NPSHa > NPSHr
    
    return {
        'pa_local': pa_local,
        'pv': pv,
        'NPSHa': NPSHa,
        'NPSHr': NPSHr,
        'delta_NPSH': delta_NPSH,
        'Hs_max': Hs_max,
        'is_safe': is_safe
    }

# 计算
result = cavitation_analysis(
    elevation=500,
    water_temp=20,
    Q=100,
    NPSHr=3.5,
    Hs=2.5,
    sum_hs=0.5
)

print("汽蚀分析结果:")
print(f"当地大气压: {result['pa_local']:.0f} Pa")
print(f"饱和蒸汽压: {result['pv']:.0f} Pa")
print(f"有效汽蚀余量 NPSHa: {result['NPSHa']:.3f} m")
print(f"必需汽蚀余量 NPSHr: {result['NPSHr']:.1f} m")
print(f"安全裕量: {result['delta_NPSH']:.3f} m")
print(f"最大安装高度: {result['Hs_max']:.3f} m")
print(f"是否安全: {'是' if result['is_safe'] else '否'}")
```

### 答案

1. 当地大气压：**95325 Pa**
2. 有效汽蚀余量：**NPSHa = 6.488 m**
3. 不会发生汽蚀（**安全裕量2.988m**）
4. 最大允许安装高度：**5.488 m**
