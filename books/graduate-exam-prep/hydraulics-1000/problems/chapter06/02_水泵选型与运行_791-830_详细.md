## 题791：水泵选型计算

**【题目】**（30分，考频⭐⭐⭐⭐⭐）

某泵站设计参数：
- 设计流量：Q = 2.5 m³/s
- 静扬程：H_st = 8 m
- 吸水管：L_s = 20 m，d_s = 500 mm，局部损失系数Σζ_s = 3.0
- 压水管：L_d = 150 m，d_d = 450 mm，Σζ_d = 5.0
- 管道糙率：n = 0.012

求：
1. 沿程损失（吸水管、压水管）
2. 局部损失
3. 总扬程H
4. 轴功率（η = 80%）
5. 选择合适水泵

**【解答】**

### 1. 流速计算

**吸水管**：
$$v_s = \frac{4Q}{\pi d_s^2} = \frac{4 \times 2.5}{\pi \times 0.5^2} = 12.73 \text{ m/s}$$

**压水管**：
$$v_d = \frac{4Q}{\pi d_d^2} = \frac{4 \times 2.5}{\pi \times 0.45^2} = 15.71 \text{ m/s}$$

### 2. 沿程损失

**吸水管**（曼宁公式）：
$$h_{fs} = \frac{n^2 v_s^2 L_s}{d_s^{4/3}} = \frac{0.012^2 \times 12.73^2 \times 20}{0.5^{4/3}}$$
$$= \frac{0.465}{0.420} = 1.11 \text{ m}$$

**压水管**：
$$h_{fd} = \frac{0.012^2 \times 15.71^2 \times 150}{0.45^{4/3}} = 6.89 \text{ m}$$

总沿程损失：$h_f = 1.11 + 6.89 = 8.00$ m

### 3. 局部损失

**吸水管**：
$$h_{js} = \zeta_s \frac{v_s^2}{2g} = 3.0 \times \frac{12.73^2}{2 \times 9.8} = 24.86 \text{ m}$$

**压水管**：
$$h_{jd} = 5.0 \times \frac{15.71^2}{2 \times 9.8} = 62.85 \text{ m}$$

总局部损失：$h_j = 24.86 + 62.85 = 87.71$ m

### 4. 总扬程

$$H = H_{st} + h_f + h_j = 8 + 8.00 + 87.71 = 103.71 \text{ m}$$

**注**：局部损失过大，需优化设计！

### 5. 轴功率

$$P = \frac{\rho g Q H}{\eta} = \frac{1000 \times 9.8 \times 2.5 \times 103.71}{0.80}$$
$$= 3,177,694 \text{ W} = 3177.7 \text{ kW}$$

### Python代码

```python
import numpy as np
import matplotlib.pyplot as plt

class PumpSelection:
    """
    水泵选型计算
    """
    
    def __init__(self, Q, H_st, rho=1000, g=9.8):
        """
        参数:
            Q: 设计流量 (m³/s)
            H_st: 静扬程 (m)
            rho: 流体密度 (kg/m³)
            g: 重力加速度 (m/s²)
        """
        self.Q = Q
        self.H_st = H_st
        self.rho = rho
        self.g = g
    
    def velocity(self, d):
        """
        管道流速
        
        参数:
            d: 管径 (m)
        """
        A = np.pi * d**2 / 4
        v = self.Q / A
        return v
    
    def friction_loss(self, v, L, d, n=0.012):
        """
        沿程损失（曼宁公式）
        
        参数:
            v: 流速 (m/s)
            L: 管长 (m)
            d: 管径 (m)
            n: 糙率
        """
        h_f = (n**2 * v**2 * L) / (d**(4/3))
        return h_f
    
    def local_loss(self, v, zeta):
        """
        局部损失
        
        参数:
            v: 流速 (m/s)
            zeta: 局部损失系数之和
        """
        h_j = zeta * (v**2) / (2 * self.g)
        return h_j
    
    def total_head(self, pipe_config):
        """
        总扬程计算
        
        参数:
            pipe_config: 管路配置字典列表
                [{'L': 长度, 'd': 管径, 'zeta': 局部损失系数}, ...]
        """
        h_f_total = 0
        h_j_total = 0
        
        for pipe in pipe_config:
            v = self.velocity(pipe['d'])
            h_f = self.friction_loss(v, pipe['L'], pipe['d'])
            h_j = self.local_loss(v, pipe['zeta'])
            
            h_f_total += h_f
            h_j_total += h_j
            
            print(f"  管段: d={pipe['d']}m, L={pipe['L']}m")
            print(f"    流速: {v:.2f} m/s")
            print(f"    沿程损失: {h_f:.2f} m")
            print(f"    局部损失: {h_j:.2f} m")
        
        H_total = self.H_st + h_f_total + h_j_total
        
        return {
            'H_st': self.H_st,
            'h_f': h_f_total,
            'h_j': h_j_total,
            'H_total': H_total
        }
    
    def shaft_power(self, H, eta):
        """
        轴功率
        
        参数:
            H: 扬程 (m)
            eta: 效率
        """
        P = (self.rho * self.g * self.Q * H) / eta  # W
        return P / 1000  # kW
    
    def pump_matching(self, H_required, pump_database):
        """
        水泵选型匹配
        
        参数:
            H_required: 所需扬程 (m)
            pump_database: 水泵数据库 [(型号, Q额, H额, P额, η), ...]
        
        返回:
            推荐水泵列表
        """
        candidates = []
        
        for pump in pump_database:
            model, Q_rated, H_rated, P_rated, eta = pump
            
            # 匹配条件：
            # 1. 扬程满足要求（±10%）
            # 2. 流量接近（80%-120%）
            H_match = (H_rated >= H_required * 0.9 and 
                      H_rated <= H_required * 1.15)
            Q_match = (Q_rated >= self.Q * 0.8 and 
                      Q_rated <= self.Q * 1.2)
            
            if H_match and Q_match:
                # 计算运行点效率（简化）
                eta_operating = eta * 0.95  # 考虑偏离最优工况
                score = eta_operating - abs(Q_rated - self.Q) * 0.01
                
                candidates.append({
                    'model': model,
                    'Q_rated': Q_rated,
                    'H_rated': H_rated,
                    'P_rated': P_rated,
                    'eta': eta,
                    'score': score
                })
        
        # 按评分排序
        candidates.sort(key=lambda x: x['score'], reverse=True)
        
        return candidates


# 使用示例
print("="*60)
print("水泵选型计算")
print("="*60)

pump_sel = PumpSelection(Q=2.5, H_st=8)

# 管路配置
pipes = [
    {'L': 20, 'd': 0.5, 'zeta': 3.0},   # 吸水管
    {'L': 150, 'd': 0.45, 'zeta': 5.0}  # 压水管
]

# 计算扬程
result = pump_sel.total_head(pipes)

print(f"\n扬程计算结果:")
print(f"  静扬程: {result['H_st']:.2f} m")
print(f"  沿程损失: {result['h_f']:.2f} m")
print(f"  局部损失: {result['h_j']:.2f} m")
print(f"  总扬程: {result['H_total']:.2f} m")

# 轴功率
eta = 0.80
P = pump_sel.shaft_power(result['H_total'], eta)
print(f"\n轴功率: {P:.1f} kW (η={eta*100}%)")

# 水泵选型（示例数据库）
pump_db = [
    ('200S58', 2.8, 105, 3500, 0.82),
    ('200S52', 2.6, 98, 3200, 0.81),
    ('250S60', 3.0, 110, 3800, 0.83),
    ('200S65', 2.5, 112, 3600, 0.80),
]

print(f"\n推荐水泵:")
candidates = pump_sel.pump_matching(result['H_total'], pump_db)

for i, pump in enumerate(candidates[:3], 1):
    print(f"{i}. {pump['model']}: "
          f"Q={pump['Q_rated']}m³/s, H={pump['H_rated']}m, "
          f"P={pump['P_rated']}kW, η={pump['eta']*100:.0f}%")
```

### 答案

## 题792：水泵调节方式比较

**【题目】**（25分，考频⭐⭐⭐⭐）

某泵站，水泵额定工况：Q₀=300 L/s，H₀=50m，P₀=200kW，η₀=85%。

需要调节到：Q₁=240 L/s。

比较三种调节方式：
1. 节流调节（关小阀门）
2. 变速调节（降低转速）
3. 切削调节（切削叶轮）

分析各自的功率消耗和效率。

**【解答】**

### 1. 节流调节

增加阀门阻力，管路特性曲线上移：

新工况：Q₁=240 L/s，H₁≈52m（需增加扬程克服阀门阻力）

功率：
$$P_1 = \frac{\rho g Q_1 H_1}{\eta_1} \approx \frac{1000 \times 9.8 \times 0.24 \times 52}{0.83}$$
$$\approx 148 \text{ kW}$$

效率下降至约83%

**缺点**：能量浪费在阀门上

### 2. 变速调节

相似定律：
$$\frac{Q_1}{Q_0} = \frac{n_1}{n_0}, \quad \frac{H_1}{H_0} = \left(\frac{n_1}{n_0}\right)^2$$

$$\frac{n_1}{n_0} = \frac{240}{300} = 0.8$$

$$H_1 = 50 \times 0.8^2 = 32 \text{ m}$$

功率：
$$P_1 = P_0 \times \left(\frac{n_1}{n_0}\right)^3 = 200 \times 0.8^3 = 102.4 \text{ kW}$$

效率基本不变：η₁≈85%

**优点**：节能效果最好

### 3. 切削调节

切削直径：
$$\frac{D_1}{D_0} = \frac{Q_1}{Q_0} = 0.8$$

效率略有下降：η₁≈82%

功率：约110 kW

**缺点**：不可逆
