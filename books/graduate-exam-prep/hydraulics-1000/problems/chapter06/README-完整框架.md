## 🎯 核心公式

### 泵的扬程
$$H = \frac{p_2-p_1}{\rho g} + \frac{v_2^2-v_1^2}{2g} + (z_2-z_1)$$

### 泵的功率
$$P = \frac{\rho gQH}{\eta}$$

### 比转速
$$n_s = 3.65\frac{n\sqrt{Q}}{H^{3/4}}$$
