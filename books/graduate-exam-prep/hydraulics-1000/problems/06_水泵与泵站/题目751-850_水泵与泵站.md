## 📚 水泵基础理论

### 基本概念

**水泵扬程H**：
$$H = H_实 + h_f$$

其中：
- \(H_实\)：实际扬程（出水池-进水池水位差）
- \(h_f\)：管路总损失

**水泵功率**：

**有效功率**：
$$P_e = \gamma QH$$

**轴功率**：
$$P_轴 = \frac{P_e}{\eta}$$

**电机功率**：
$$P_电 = \frac{P_轴}{\eta_电}$$

### 水泵特性曲线

**Q-H曲线**：流量-扬程曲线  
**Q-η曲线**：流量-效率曲线  
**Q-P曲线**：流量-功率曲线

### 相似定律

**比转速**：
$$n_s = \frac{nQ^{0.5}}{H^{0.75}}$$

**相似工况**：
$$\frac{Q_1}{Q_2} = \frac{n_1}{n_2}$$
$$\frac{H_1}{H_2} = \left(\frac{n_1}{n_2}\right)^2$$
$$\frac{P_1}{P_2} = \left(\frac{n_1}{n_2}\right)^3$$

## 第一部分：水泵特性（题751-790，40题）⭐⭐⭐⭐⭐

### 题目751：水泵扬程计算 ⭐⭐

水泵提水，进水池水位0m，出水池水位30m，管路总损失5m。求水泵扬程。

**解答**：
$$H = (30 - 0) + 5 = 35\,\text{m}$$

### 题目752：水泵功率 ⭐⭐

流量\(Q = 0.1\,\text{m}^3/\text{s}\)，扬程\(H = 40\,\text{m}\)，效率\(\eta = 0.75\)。求轴功率。

**解答**：
$$P_轴 = \frac{\gamma QH}{\eta} = \frac{9800 \times 0.1 \times 40}{0.75} = 52,267\,\text{W} = 52.3\,\text{kW}$$

### 题目753：比转速 ⭐⭐⭐

水泵转速\(n = 1450\,\text{r/min}\)，流量\(Q = 0.05\,\text{m}^3/\text{s} = 180\,\text{m}^3/\text{h}\)，扬程\(H = 50\,\text{m}\)。求比转速\(n_s\)。

**解答**：
$$n_s = \frac{1450 \times 180^{0.5}}{50^{0.75}} = \frac{1450 \times 13.42}{18.8} = 1034$$

（高速泵）

## 第二部分：泵站设计（题791-830，40题）

### 题目791：泵站设计流量 ⭐⭐

灌溉面积1000亩，灌水定额\(m = 600\,\text{m}^3/\text{亩}\)，灌水周期10天，每天工作20小时。求泵站设计流量。

**解答**：
$$Q = \frac{1000 \times 600}{10 \times 20 \times 3600} = 0.833\,\text{m}^3/\text{s}$$
