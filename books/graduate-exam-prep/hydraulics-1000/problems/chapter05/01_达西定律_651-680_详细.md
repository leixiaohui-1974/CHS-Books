## 题651：达西定律基础

**【题目】**（15分，考频⭐⭐⭐⭐⭐）

某含水层，渗透系数k=5m/d，断面积A=100m²，水力梯度i=0.01。求：
1. 渗透流速v
2. 实际流速u（孔隙率n=0.3）
3. 渗流量Q

**【解答】**

### 达西定律

$$v = ki$$

### 渗透流速

$$v = 5 \times 0.01 = 0.05 \text{ m/d}$$

### 实际流速

$$u = \frac{v}{n} = \frac{0.05}{0.3} = 0.167 \text{ m/d}$$

### 渗流量

$$Q = vA = 0.05 \times 100 = 5 \text{ m}^3/\text{d}$$

### Python代码

```python
def darcy_law(k, i, A, n=None):
    """
    达西定律计算
    
    参数:
        k: 渗透系数 (m/d)
        i: 水力梯度
        A: 断面积 (m²)
        n: 孔隙率
    """
    v = k * i  # 渗透流速
    Q = v * A  # 渗流量
    u = v / n if n else None  # 实际流速
    
    return v, u, Q

v, u, Q = darcy_law(5, 0.01, 100, 0.3)
print(f"渗透流速: {v:.3f} m/d")
print(f"实际流速: {u:.3f} m/d")
print(f"渗流量: {Q:.3f} m³/d")
```

### 答案

1. 渗透流速：**v = 0.05 m/d**
2. 实际流速：**u = 0.167 m/d**
3. 渗流量：**Q = 5 m³/d**

## 题652：承压水完整井流量

**【题目】**（25分，考频⭐⭐⭐⭐⭐）

承压含水层，厚度M=20m，渗透系数k=10m/d，井半径r₀=0.1m，影响半径R=100m，井内水位降深s=5m。求：
1. 稳定井流量Q
2. 距井r=10m处的水位降深
3. 如果k增大到15m/d，流量增大多少？

**【解答】**

### 裘布依公式

$$Q = \frac{2\pi kM(H-h_0)}{\ln(R/r_0)}$$

其中：
- M：含水层厚度
- H：初始水头
- h₀：井内水头
- H-h₀ = s：水位降深

$$Q = \frac{2\pi \times 10 \times 20 \times 5}{\ln(100/0.1)}$$
$$= \frac{6283.2}{\ln(1000)} = \frac{6283.2}{6.908} = 909.5 \text{ m}^3/\text{d}$$

### 距离井10m处水位降深

$$s_r = \frac{Q}{2\pi kM}\ln\frac{R}{r}$$
$$= \frac{909.5}{2\pi \times 10 \times 20}\ln\frac{100}{10}$$
$$= \frac{909.5}{1256.6}\times 2.303 = 1.667 \text{ m}$$

### k增大到15m/d时

$$Q' = \frac{2\pi \times 15 \times 20 \times 5}{\ln(100/0.1)}$$
$$= \frac{9424.8}{6.908} = 1364.3 \text{ m}^3/\text{d}$$

流量增大：
$$\frac{Q'}{Q} = \frac{15}{10} = 1.5$$

**流量增大50%** ✓

### Python代码

```python
import numpy as np

def confined_well_flow(k, M, s, R, r0):
    """
    承压水完整井流量
    
    参数:
        k: 渗透系数 (m/d)
        M: 含水层厚度 (m)
        s: 水位降深 (m)
        R: 影响半径 (m)
        r0: 井半径 (m)
    """
    Q = 2 * np.pi * k * M * s / np.log(R / r0)
    return Q

def drawdown_at_r(Q, k, M, R, r):
    """
    距井r处的水位降深
    """
    s_r = Q / (2 * np.pi * k * M) * np.log(R / r)
    return s_r

# 计算
Q = confined_well_flow(10, 20, 5, 100, 0.1)
s_10 = drawdown_at_r(Q, 10, 20, 100, 10)
Q_15 = confined_well_flow(15, 20, 5, 100, 0.1)

print(f"井流量: {Q:.1f} m³/d")
print(f"10m处降深: {s_10:.3f} m")
print(f"k=15时流量: {Q_15:.1f} m³/d")
print(f"流量增大: {(Q_15/Q-1)*100:.1f}%")
```

### 答案

1. 井流量：**Q = 909.5 m³/d**
2. 10m处降深：**s = 1.667 m**
3. k增大到15m/d时，流量增大**50%**→1364.3 m³/d
