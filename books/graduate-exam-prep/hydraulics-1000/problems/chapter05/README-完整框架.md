## 🎯 核心公式

### 达西定律
$$v = ki$$
$$Q = kiA$$

### 承压水完整井
$$Q = \frac{2\pi kM(H-h_0)}{\ln(R/r_0)}$$

### 潜水完整井
$$Q = \frac{\pi k(H^2-h_0^2)}{\ln(R/r_0)}$$
