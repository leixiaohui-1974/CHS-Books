## 题681：双井干扰计算

**【题目】**（25分，考频⭐⭐⭐⭐⭐）

两口承压完整井，距离d=100m，同时抽水。已知：
- 含水层厚度：M = 30 m
- 渗透系数：k = 15 m/d
- 两井半径：r₀ = 0.15 m
- 两井降深：s₁ = s₂ = 6 m
- 影响半径：R = 150 m

求：
1. 单井独立抽水流量Q₀
2. 两井中点处的水位降深
3. 考虑干扰后，每口井的实际流量
4. 干扰系数
5. 两井最优间距建议

**【解答】**

### 1. 单井独立抽水流量

裘布依公式：
$$Q_0 = \frac{2\pi kMs}{\ln(R/r_0)}$$

$$Q_0 = \frac{2\pi \times 15 \times 30 \times 6}{\ln(150/0.15)}$$

$$= \frac{16,965}{\ln(1000)} = \frac{16,965}{6.908} = 2456 \text{ m}^3/\text{d}$$

### 2. 两井中点处水位降深

距离两井中点：$r_c = d/2 = 50$ m

单井在中点产生的降深：
$$s_{c1} = \frac{Q_0}{2\pi kM}\ln\frac{R}{r_c}$$

$$= \frac{2456}{2\pi \times 15 \times 30}\ln\frac{150}{50}$$

$$= \frac{2456}{2827}\times 1.099 = 0.955 \text{ m}$$

两井叠加（干扰）：
$$s_c = s_{c1} + s_{c2} = 2 \times 0.955 = 1.910 \text{ m}$$

### 3. 考虑干扰后的实际流量

井1在井2处产生的附加降深：
$$\Delta s_{12} = \frac{Q_0}{2\pi kM}\ln\frac{R}{d}$$

$$= \frac{2456}{2827}\ln\frac{150}{100} = 0.352 \text{ m}$$

井1的总降深（含干扰）：
$$s_1' = s_1 + \Delta s_{12} = 6 + 0.352 = 6.352 \text{ m}$$

实际流量（修正）：
$$Q_1' = \frac{2\pi kM(s_1' - \Delta s_{12}/2)}{\ln(R/r_0)}$$

$$\approx Q_0 \times \frac{s_1'}{s_1} = 2456 \times \frac{6.352}{6} = 2600 \text{ m}^3/\text{d}$$

### 4. 干扰系数

$$\eta = \frac{\Delta s_{12}}{s_1} = \frac{0.352}{6} = 0.059 = 5.9\%$$

干扰较小（<10%），间距基本合理 ✓

### 5. 最优间距建议

经验公式：
$$d_{opt} = (3 \sim 5)R_0$$

其中$R_0$为单井影响半径。

若要干扰<5%：
$$d \geq 2R_0 = 2 \times 150 = 300 \text{ m}$$

当前间距100m < 300m，建议**增大到至少200m**

### Python代码

```python
import numpy as np
import matplotlib.pyplot as plt

class WellInterference:
    """
    井群干扰计算
    """
    
    def __init__(self, k, M, R, r0):
        self.k = k      # 渗透系数 (m/d)
        self.M = M      # 含水层厚度 (m)
        self.R = R      # 影响半径 (m)
        self.r0 = r0    # 井半径 (m)
    
    def single_well_flow(self, s):
        """
        单井流量（裘布依公式）
        """
        Q = 2 * np.pi * self.k * self.M * s / np.log(self.R / self.r0)
        return Q
    
    def drawdown_at_distance(self, Q, r):
        """
        距井r处的水位降深
        """
        s = Q / (2 * np.pi * self.k * self.M) * np.log(self.R / r)
        return s
    
    def two_wells_interference(self, s1, s2, d):
        """
        双井干扰分析
        
        参数:
            s1, s2: 两井的降深 (m)
            d: 井距 (m)
        """
        # 单井流量
        Q1 = self.single_well_flow(s1)
        Q2 = self.single_well_flow(s2)
        
        # 中点处降深
        r_mid = d / 2
        s_mid1 = self.drawdown_at_distance(Q1, r_mid)
        s_mid2 = self.drawdown_at_distance(Q2, r_mid)
        s_mid_total = s_mid1 + s_mid2
        
        # 井1在井2处的附加降深
        delta_s12 = self.drawdown_at_distance(Q1, d)
        delta_s21 = self.drawdown_at_distance(Q2, d)
        
        # 干扰系数
        eta1 = delta_s12 / s1
        eta2 = delta_s21 / s2
        
        # 修正流量
        Q1_corr = Q1 * (1 + eta1)
        Q2_corr = Q2 * (1 + eta2)
        
        return {
            'Q1': Q1,
            'Q2': Q2,
            's_mid': s_mid_total,
            'delta_s12': delta_s12,
            'delta_s21': delta_s21,
            'eta1': eta1,
            'eta2': eta2,
            'Q1_corr': Q1_corr,
            'Q2_corr': Q2_corr
        }
    
    def optimal_spacing(self, s, interference_limit=0.05):
        """
        最优井距（基于干扰限制）
        
        参数:
            s: 设计降深 (m)
            interference_limit: 允许干扰系数
        """
        Q = self.single_well_flow(s)
        
        # 根据干扰系数反推井距
        # eta = delta_s / s = (Q/(2πkM) * ln(R/d)) / s
        # 解得: d = R * exp(-eta * s * 2πkM / Q)
        
        d_opt = self.R * np.exp(-interference_limit * s * 2*np.pi*self.k*self.M / Q)
        
        return d_opt
    
    def plot_interference_field(self, s1, s2, d, x_range=(-200, 200), y_range=(-200, 200)):
        """
        绘制干扰水位场
        """
        Q1 = self.single_well_flow(s1)
        Q2 = self.single_well_flow(s2)
        
        # 网格
        x = np.linspace(x_range[0], x_range[1], 200)
        y = np.linspace(y_range[0], y_range[1], 200)
        X, Y = np.meshgrid(x, y)
        
        # 井1位置: (-d/2, 0)
        # 井2位置: (d/2, 0)
        x1, y1 = -d/2, 0
        x2, y2 = d/2, 0
        
        # 距离
        R1 = np.sqrt((X - x1)**2 + (Y - y1)**2)
        R2 = np.sqrt((X - x2)**2 + (Y - y2)**2)
        
        # 避免除零
        R1[R1 < self.r0] = self.r0
        R2[R2 < self.r0] = self.r0
        
        # 降深场
        S1 = Q1 / (2*np.pi*self.k*self.M) * np.log(self.R / R1)
        S2 = Q2 / (2*np.pi*self.k*self.M) * np.log(self.R / R2)
        S_total = S1 + S2
        
        # 绘图
        fig, axes = plt.subplots(1, 2, figsize=(14, 6))
        
        # 等降深线
        levels = np.linspace(0, max(s1, s2), 20)
        cs1 = axes[0].contour(X, Y, S1, levels=levels, cmap='Blues')
        axes[0].clabel(cs1, inline=True, fontsize=8)
        axes[0].plot(x1, y1, 'ro', markersize=10, label='井1')
        axes[0].set_xlabel('X (m)')
        axes[0].set_ylabel('Y (m)')
        axes[0].set_title('井1单独抽水')
        axes[0].legend()
        axes[0].grid(True, alpha=0.3)
        axes[0].axis('equal')
        
        # 干扰场
        cs2 = axes[1].contourf(X, Y, S_total, levels=levels, cmap='RdYlBu_r')
        axes[1].plot(x1, y1, 'ro', markersize=10, label='井1')
        axes[1].plot(x2, y2, 'bs', markersize=10, label='井2')
        axes[1].set_xlabel('X (m)')
        axes[1].set_ylabel('Y (m)')
        axes[1].set_title('双井干扰水位场')
        axes[1].legend()
        axes[1].grid(True, alpha=0.3)
        axes[1].axis('equal')
        plt.colorbar(cs2, ax=axes[1], label='降深 (m)')
        
        plt.tight_layout()
        plt.savefig('well_interference_field.png', dpi=300, bbox_inches='tight')
        plt.show()

# 计算实例
well_sys = WellInterference(k=15, M=30, R=150, r0=0.15)

# 单井流量
Q0 = well_sys.single_well_flow(s=6)
print(f"单井流量: Q₀ = {Q0:.0f} m³/d")

# 双井干扰
result = well_sys.two_wells_interference(s1=6, s2=6, d=100)

print(f"\n双井干扰分析:")
print(f"  井1流量: {result['Q1']:.0f} m³/d")
print(f"  井2流量: {result['Q2']:.0f} m³/d")
print(f"  中点降深: {result['s_mid']:.3f} m")
print(f"  井1在井2处附加降深: {result['delta_s12']:.3f} m")
print(f"  干扰系数: η₁ = {result['eta1']*100:.1f}%")
print(f"  修正流量: Q₁' = {result['Q1_corr']:.0f} m³/d")

# 最优井距
d_opt = well_sys.optimal_spacing(s=6, interference_limit=0.05)
print(f"\n最优井距建议:")
print(f"  当前井距: 100 m")
print(f"  最优井距（η<5%）: {d_opt:.0f} m")

# 绘图
well_sys.plot_interference_field(s1=6, s2=6, d=100)
```

### 答案

1. 单井流量：**Q₀ = 2456 m³/d**
2. 中点降深：**s_c = 1.910 m**
3. 修正流量：**Q' ≈ 2600 m³/d**
4. 干扰系数：**η = 5.9%**
5. 最优井距：**建议≥200m**（当前100m干扰偏大）
