## 🎯 核心知识点

### 1. 连续性方程

**基本形式**：
$$\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \vec{v}) = 0$$

**不可压缩流体**：
$$\nabla \cdot \vec{v} = 0$$

**一维稳定流**：
$$Q = v_1 A_1 = v_2 A_2 = \text{常数}$$

### 2. 伯努利方程

**理想流体**：
$$\frac{p}{\rho g} + \frac{v^2}{2g} + z = C$$

**实际流体**：
$$\frac{p_1}{\rho g} + \frac{v_1^2}{2g} + z_1 = \frac{p_2}{\rho g} + \frac{v_2^2}{2g} + z_2 + h_f$$

### 3. 动量方程

**微分形式**（N-S方程）：
$$\rho \frac{D\vec{v}}{Dt} = -\nabla p + \mu \nabla^2 \vec{v} + \rho \vec{g}$$

**积分形式**：
$$\sum \vec{F} = \frac{\partial}{\partial t}\int_V \rho \vec{v} dV + \int_S \rho \vec{v}(\vec{v} \cdot \vec{n}) dS$$

### 4. 量纲分析

**白金汉π定理**：
$$f(q_1, q_2, ..., q_n) = 0$$
$$\Rightarrow F(\pi_1, \pi_2, ..., \pi_{n-m}) = 0$$

### 5. 阻力系数

**达西-魏斯巴赫公式**：
$$h_f = \lambda \frac{L}{D} \frac{v^2}{2g}$$

**雷诺数**：
$$Re = \frac{vD}{\nu}$$
