## 核心知识点

### 1. 量纲的基本概念

**基本量纲**（SI制）：
- [L]：长度
- [M]：质量
- [T]：时间

**导出量纲**：
- 速度：[LT⁻¹]
- 加速度：[LT⁻²]
- 力：[MLT⁻²]
- 压强：[ML⁻¹T⁻²]

### 2. 白金汉π定理

若物理现象涉及n个物理量，包含m个基本量纲：
$$f(q_1, q_2, ..., q_n) = 0$$

则可转化为(n-m)个无量纲π项：
$$F(\pi_1, \pi_2, ..., \pi_{n-m}) = 0$$

### 3. 相似准则

#### 几何相似
$$\lambda_L = \frac{L_m}{L_p} = \text{常数}$$

#### 运动相似
$$\lambda_v = \frac{v_m}{v_p} = \text{常数}$$

#### 动力相似
各种力的比例相同

### 4. 重要无量纲数

| 名称 | 符号 | 定义 | 物理意义 |
|------|------|------|---------|
| 雷诺数 | Re | $\frac{vL}{\nu}$ | 惯性力/粘性力 |
| 弗劳德数 | Fr | $\frac{v}{\sqrt{gL}}$ | 惯性力/重力 |
| 欧拉数 | Eu | $\frac{\Delta p}{\rho v^2}$ | 压力/惯性力 |
| 马赫数 | Ma | $\frac{v}{c}$ | 流速/声速 |
| 韦伯数 | We | $\frac{\rho v^2 L}{\sigma}$ | 惯性力/表面张力 |

## 题171：物理量量纲判断

**【题目】**（15分，考频⭐⭐⭐⭐）

检验下列公式的量纲一致性：
1. $p = \rho g h$
2. $F = \frac{1}{2} C_D \rho A v^2$
3. $Q = C A \sqrt{2gH}$

**【解答】**

### 公式1：$p = \rho g h$

**左边**：压强
$$[p] = [ML^{-1}T^{-2}]$$

**右边**：
$$[\rho] = [ML^{-3}]$$
$$[g] = [LT^{-2}]$$
$$[h] = [L]$$

$$[\rho g h] = [ML^{-3}] \cdot [LT^{-2}] \cdot [L]$$
$$= [ML^{-1}T^{-2}]$$

**结论**：✅ 量纲一致

### 公式2：$F = \frac{1}{2} C_D \rho A v^2$

**左边**：力
$$[F] = [MLT^{-2}]$$

**右边**（$C_D$无量纲）：
$$[C_D] = [1]$$
$$[\rho A v^2] = [ML^{-3}] \cdot [L^2] \cdot [L^2T^{-2}]$$
$$= [ML^{-3+2+2}T^{-2}] = [MLT^{-2}]$$

**结论**：✅ 量纲一致

### 公式3：$Q = C A \sqrt{2gH}$

**左边**：流量
$$[Q] = [L^3T^{-1}]$$

**右边**（$C$无量纲）：
$$[C] = [1]$$
$$[A] = [L^2]$$
$$[\sqrt{2gH}] = \sqrt{[LT^{-2}] \cdot [L]} = [LT^{-1}]$$

$$[C A \sqrt{2gH}] = [L^2] \cdot [LT^{-1}] = [L^3T^{-1}]$$

**结论**：✅ 量纲一致

### 答案

三个公式均**量纲一致**✓

## 题172：雷诺数计算

**【题目】**（18分，考频⭐⭐⭐⭐⭐）

圆管流动，直径d=0.1m，流速v=2m/s，水温20°C（运动粘度ν=1.0×10⁻⁶m²/s）。求：
1. 雷诺数Re
2. 判断流态
3. 若改为油（ν=5×10⁻⁵m²/s），流态如何？

**【解答】**

### 雷诺数定义

$$Re = \frac{vd}{\nu}$$

### 水的情况

$$Re_水 = \frac{2 \times 0.1}{1.0 \times 10^{-6}} = \frac{0.2}{10^{-6}} = 2 \times 10^5 = 200000$$

### 流态判断

圆管流动临界雷诺数：
$$Re_c = 2300$$

$$Re_水 = 200000 \gg 2300$$

**流态**：**湍流** ✓

### 油的情况

$$Re_油 = \frac{2 \times 0.1}{5 \times 10^{-5}} = \frac{0.2}{5 \times 10^{-5}} = 4000$$

$$Re_油 = 4000 > 2300$$

仍是**湍流**，但接近过渡区。

### 答案

1. 水的雷诺数：**Re = 2×10⁵**
2. 流态：**湍流**（Re >> 2300）
3. 油的流态：**湍流**（Re = 4000）

## 题176：泵的相似定律

**【题目】**（25分，考频⭐⭐⭐⭐⭐）

某离心泵，转速n₁=1450 r/min，流量Q₁=100 m³/h，扬程H₁=20m，功率P₁=7.5kW。若转速提高到n₂=1750 r/min，求新的Q₂、H₂、P₂。

**【解答】**

### 泵的相似定律

几何相似、运动相似的泵：

**流量比**：
$$\frac{Q_2}{Q_1} = \frac{n_2}{n_1}$$

**扬程比**：
$$\frac{H_2}{H_1} = \left(\frac{n_2}{n_1}\right)^2$$

**功率比**：
$$\frac{P_2}{P_1} = \left(\frac{n_2}{n_1}\right)^3$$

### 转速比

$$\frac{n_2}{n_1} = \frac{1750}{1450} = 1.207$$

### 新的流量Q₂

$$Q_2 = Q_1 \times \frac{n_2}{n_1} = 100 \times 1.207 = 120.7 \text{ m}^3/\text{h}$$

### 新的扬程H₂

$$H_2 = H_1 \times \left(\frac{n_2}{n_1}\right)^2 = 20 \times 1.207^2$$
$$= 20 \times 1.457 = 29.14 \text{ m}$$

### 新的功率P₂

$$P_2 = P_1 \times \left(\frac{n_2}{n_1}\right)^3 = 7.5 \times 1.207^3$$
$$= 7.5 \times 1.758 = 13.19 \text{ kW}$$

### 相似定律的应用

**变速调节**是泵的重要调节方式：
- 流量正比于转速
- 扬程正比于转速平方
- 功率正比于转速立方

**能耗比较**：
节能显著！降低转速可大幅降低功耗。

### 答案

- 新流量：**Q₂ = 120.7 m³/h**（增加20.7%）
- 新扬程：**H₂ = 29.14 m**（增加45.7%）
- 新功率：**P₂ = 13.19 kW**（增加75.9%）

### 踩分点（25分）

- 相似定律（8分）
- 流量计算（5分）
- 扬程计算（6分）
- 功率计算（6分）

### 题178：重力相似

$$Fr_m = Fr_p \Rightarrow \frac{v_m}{\sqrt{g L_m}} = \frac{v_p}{\sqrt{g L_p}}$$

**比尺关系**：
$$\lambda_v = \sqrt{\lambda_L}$$

2. **运动相似**：速度场相似
3. **动力相似**：力场相似
