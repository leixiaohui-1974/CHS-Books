## 核心知识点

### 1. 伯努利方程物理意义

**能量守恒定律**：理想流体沿流线运动时，单位重量流体的总水头保持不变。

### 2. 理想流体伯努利方程

$$\frac{p}{\rho g} + \frac{v^2}{2g} + z = C$$

- $\frac{p}{\rho g}$：压强水头
- $\frac{v^2}{2g}$：速度水头
- $z$：位置水头

### 3. 实际流体能量方程

$$\frac{p_1}{\rho g} + \frac{v_1^2}{2g} + z_1 = \frac{p_2}{\rho g} + \frac{v_2^2}{2g} + z_2 + h_f$$

$h_f$：水头损失

### 4. 应用条件

✅ 不可压缩流体  
✅ 沿同一流线  
✅ 稳定流动  
✅ 理想流体（无粘性）

## 题121：水平管道压强变化

**【题目】**（15分，考频⭐⭐⭐⭐⭐）

水平管道，断面1：d₁=0.2m，p₁=200kPa；断面2：d₂=0.1m。流量Q=0.06m³/s。求断面2处的压强p₂。（忽略水头损失）

**【解答】**

### 连续性方程

$$v_1 A_1 = v_2 A_2 = Q$$

断面1流速：
$$v_1 = \frac{Q}{A_1} = \frac{0.06}{\pi \times 0.2^2/4} = \frac{0.06}{0.0314} = 1.91 \text{ m/s}$$

断面2流速：
$$v_2 = \frac{Q}{A_2} = \frac{0.06}{\pi \times 0.1^2/4} = \frac{0.06}{0.00785} = 7.64 \text{ m/s}$$

### 伯努利方程

水平管道（z₁=z₂），理想流体（h_f=0）：

$$\frac{p_1}{\rho g} + \frac{v_1^2}{2g} = \frac{p_2}{\rho g} + \frac{v_2^2}{2g}$$

解出p₂：
$$p_2 = p_1 + \rho g \left(\frac{v_1^2 - v_2^2}{2g}\right)$$
$$= p_1 + \frac{\rho(v_1^2 - v_2^2)}{2}$$

代入数值：
$$p_2 = 200000 + \frac{1000 \times (1.91^2 - 7.64^2)}{2}$$
$$= 200000 + \frac{1000 \times (3.65 - 58.37)}{2}$$
$$= 200000 - 27360 = 172640 \text{ Pa} = 172.6 \text{ kPa}$$

### 物理解释

**管径减小→流速增大→压强降低！**

这就是**伯努利定理**的直观体现：
- 速度水头↑ → 压强水头↓
- 总水头不变

### 答案

断面2压强：**p₂ = 172.6 kPa**（降低27.4kPa）

### 踩分点（15分）

- 连续性方程（4分）
- 两断面流速（4分）
- 伯努利方程（4分）
- 压强计算（3分）

## 题122：水塔供水压强

**【题目】**（18分，考频⭐⭐⭐⭐⭐）

某住宅区水塔，水面高程H=50m，用户水龙头高程h=10m。求：
1. 用户处的理论压强
2. 考虑管道损失h_f=8m时的实际压强
3. 要保证用户压强≥200kPa，水塔最低高度

**【解答】**

### 理论压强（无损失）

取水塔水面为断面1（p₁=p_0，v₁≈0）  
取用户水龙头为断面2

伯努利方程：
$$\frac{p_0}{\rho g} + 0 + H = \frac{p_2}{\rho g} + \frac{v_2^2}{2g} + h$$

忽略v₂（水龙头关闭时）：
$$p_2 = p_0 + \rho g (H - h)$$
$$= 101325 + 1000 \times 9.8 \times (50 - 10)$$
$$= 101325 + 392000 = 493325 \text{ Pa} = 493.3 \text{ kPa}$$

### 实际压强（有损失）

能量方程：
$$\frac{p_0}{\rho g} + 0 + H = \frac{p_2}{\rho g} + \frac{v_2^2}{2g} + h + h_f$$

$$p_2 = p_0 + \rho g (H - h - h_f)$$
$$= 101325 + 1000 \times 9.8 \times (50 - 10 - 8)$$
$$= 101325 + 313600 = 414925 \text{ Pa} = 414.9 \text{ kPa}$$

### 最低高度（p₂≥200kPa）

$$p_2 \geq 200000 \text{ Pa}$$
$$p_0 + \rho g(H_{min} - h - h_f) \geq 200000$$
$$H_{min} \geq h + h_f + \frac{p_2 - p_0}{\rho g}$$
$$= 10 + 8 + \frac{200000 - 101325}{1000 \times 9.8}$$
$$= 18 + 10.07 = 28.07 \text{ m}$$

### 答案

1. 理论压强：**493.3 kPa**
2. 实际压强：**414.9 kPa**（损失78.4kPa）
3. 最低高度：**H_min = 28.07 m**

### 踩分点（18分）

- 理论压强（6分）
- 实际压强（6分）
- 最低高度（6分）

## 题123：文丘里流量计

**【题目】**（20分，考频⭐⭐⭐⭐⭐）

文丘里流量计：入口d₁=0.2m，喉部d₂=0.1m，U型压差计读数Δh=0.5m（汞柱）。求管道流量Q。

**【解答】**

### 文丘里原理

**收缩段**：流速↑，压强↓  
**测压差**：推算流量

### 伯努利方程

$$\frac{p_1}{\rho g} + \frac{v_1^2}{2g} = \frac{p_2}{\rho g} + \frac{v_2^2}{2g}$$

压差：
$$\Delta p = p_1 - p_2 = \frac{\rho(v_2^2 - v_1^2)}{2}$$

### 连续性方程

$$v_1 A_1 = v_2 A_2$$
$$v_1 = v_2 \frac{A_2}{A_1} = v_2 \left(\frac{d_2}{d_1}\right)^2 = v_2 \times 0.25$$

### 压差测量

U型压差计（汞，ρ_Hg=13600kg/m³）：
$$\Delta p = (\rho_{Hg} - \rho_{水}) g \Delta h$$
$$= (13600 - 1000) \times 9.8 \times 0.5$$
$$= 61740 \text{ Pa}$$

### 流速计算

$$\Delta p = \frac{\rho(v_2^2 - v_1^2)}{2} = \frac{\rho v_2^2(1 - 0.25^2)}{2}$$
$$61740 = \frac{1000 \times v_2^2 \times 0.9375}{2}$$
$$v_2^2 = \frac{61740 \times 2}{1000 \times 0.9375} = 131.7$$
$$v_2 = 11.48 \text{ m/s}$$

$$v_1 = 11.48 \times 0.25 = 2.87 \text{ m/s}$$

### 流量

$$Q = v_1 A_1 = 2.87 \times \frac{\pi \times 0.2^2}{4}$$
$$= 2.87 \times 0.0314 = 0.0901 \text{ m}^3/\text{s}$$

### 答案

流量：**Q = 0.0901 m³/s = 90.1 L/s**

- 连续性方程（4分）
- 压差计算（5分）
- 流速推导（4分）
- 流量计算（2分）

## 📚 本节总结

### 三类水头

1. **压强水头**：p/(ρg)
2. **速度水头**：v²/(2g)
3. **位置水头**：z

**总水头守恒**（理想流体）  
**能量守恒**（实际流体扣除损失）

### 应用场景

- ✅ 流量测量（文丘里、皮托管）
- ✅ 供水设计（水塔、管网）
- ✅ 水电工程（机组、压力管道）
- ✅ 泵站设计（扬程计算）
