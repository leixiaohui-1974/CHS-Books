## 核心知识点

### 1. 水头损失分类

**沿程损失**（h_f）：摩擦阻力  
**局部损失**（h_j）：局部障碍

总损失：
$$h_w = h_f + \sum h_j$$

### 2. 沿程损失（达西公式）

$$h_f = \lambda \frac{L}{d} \frac{v^2}{2g}$$

- λ：沿程阻力系数
- L：管道长度
- d：管道直径
- v：平均流速

### 3. 阻力系数λ

**层流**（Re < 2300）：
$$\lambda = \frac{64}{Re}$$

**湍流光滑管**（Re > 2300）：
$$\frac{1}{\sqrt{\lambda}} = 2\lg(Re\sqrt{\lambda}) - 0.8$$

**湍流粗糙管**：
$$\frac{1}{\sqrt{\lambda}} = 2\lg\frac{d}{\Delta} + 1.14$$

（Δ：绝对粗糙度）

### 4. 局部损失

$$h_j = \zeta \frac{v^2}{2g}$$

ζ：局部阻力系数

| 局部装置 | ζ |
|---------|---|
| 突然扩大 | $(1-\frac{A_1}{A_2})^2$ |
| 突然缩小 | $0.5(1-\frac{A_2}{A_1})$ |
| 90°弯头 | 0.3~1.0 |
| 闸阀全开 | 0.2 |
| 进口 | 0.5 |
| 出口 | 1.0 |

## 详解示例：题191、193、198

## 题191：层流沿程损失

**【题目】**（18分，考频⭐⭐⭐⭐⭐）

水平圆管（d=20mm，L=10m），层流流动，Re=1500，平均流速v=0.075m/s。求沿程水头损失h_f。

**【解答】**

### 层流阻力系数

$$\lambda = \frac{64}{Re} = \frac{64}{1500} = 0.0427$$

### 达西公式

$$h_f = \lambda \frac{L}{d} \frac{v^2}{2g}$$

$$= 0.0427 \times \frac{10}{0.02} \times \frac{0.075^2}{2 \times 9.8}$$

$$= 0.0427 \times 500 \times \frac{0.005625}{19.6}$$

$$= 0.0427 \times 500 \times 0.000287 = 0.00613 \text{ m} = 6.13 \text{ mm}$$

### 压降计算

$$\Delta p = \rho g h_f = 1000 \times 9.8 \times 0.00613$$
$$= 60.1 \text{ Pa}$$

### 答案

- 沿程水头损失：**h_f = 6.13 mm**
- 压降：**Δp = 60.1 Pa**

### 踩分点（18分）

- λ公式（5分）
- λ计算（3分）
- 达西公式（5分）
- h_f计算（5分）

## 题193：突然扩大局部损失

**【题目】**（22分，考频⭐⭐⭐⭐⭐）

管道突然扩大，d₁=0.1m，d₂=0.2m，流量Q=0.05m³/s。求：
1. 扩大前后流速
2. 局部水头损失h_j
3. 如果改为渐扩（扩散角α=10°），损失如何变化？

**【解答】**

### 流速计算

$$v_1 = \frac{Q}{A_1} = \frac{0.05}{\pi \times 0.1^2/4} = \frac{0.05}{0.00785} = 6.37 \text{ m/s}$$

$$v_2 = \frac{Q}{A_2} = \frac{0.05}{\pi \times 0.2^2/4} = \frac{0.05}{0.0314} = 1.59 \text{ m/s}$$

### 突然扩大局部损失

**伯达公式**：
$$h_j = \frac{(v_1 - v_2)^2}{2g} = \left(1 - \frac{A_1}{A_2}\right)^2 \frac{v_1^2}{2g}$$

面积比：
$$\frac{A_1}{A_2} = \left(\frac{d_1}{d_2}\right)^2 = \left(\frac{0.1}{0.2}\right)^2 = 0.25$$

$$h_j = (1 - 0.25)^2 \times \frac{6.37^2}{2 \times 9.8}$$
$$= 0.75^2 \times \frac{40.58}{19.6}$$
$$= 0.5625 \times 2.07 = 1.164 \text{ m}$$

### 渐扩管损失

扩散角α=10°，查表得：ζ≈0.15

$$h_j' = \zeta \frac{v_1^2}{2g} = 0.15 \times \frac{6.37^2}{19.6}$$
$$= 0.15 \times 2.07 = 0.31 \text{ m}$$

### 损失对比

$$\frac{h_j'}{h_j} = \frac{0.31}{1.164} = 0.266 = 26.6\%$$

**渐扩损失仅为突扩的26.6%！**

### 答案

1. 流速：v₁=**6.37 m/s**，v₂=**1.59 m/s**
2. 突然扩大损失：**h_j = 1.164 m**
3. 渐扩损失：**h_j' = 0.31 m**（减少73.4%）

### 踩分点（22分）

- 流速计算（6分）
- 伯达公式（6分）
- 突扩损失（5分）
- 渐扩损失（5分）

## 题198：总水头损失计算

**【题目】**（25分，考频⭐⭐⭐⭐⭐）

管道系统：L=100m，d=0.15m，流量Q=0.08m³/s，λ=0.025。包含：2个90°弯头（ζ=0.5）、1个闸阀（ζ=0.2）、进口（ζ=0.5）、出口（ζ=1.0）。求总水头损失。

**【解答】**

### 流速

$$v = \frac{Q}{A} = \frac{0.08}{\pi \times 0.15^2/4} = \frac{0.08}{0.0177} = 4.52 \text{ m/s}$$

### 沿程损失

$$h_f = \lambda \frac{L}{d} \frac{v^2}{2g}$$
$$= 0.025 \times \frac{100}{0.15} \times \frac{4.52^2}{2 \times 9.8}$$
$$= 0.025 \times 666.7 \times \frac{20.43}{19.6}$$
$$= 0.025 \times 666.7 \times 1.042 = 17.37 \text{ m}$$

### 局部损失

$$\sum h_j = \sum \zeta \frac{v^2}{2g}$$

$$\sum \zeta = 2 \times 0.5 + 0.2 + 0.5 + 1.0 = 3.2$$

$$\sum h_j = 3.2 \times \frac{4.52^2}{2 \times 9.8} = 3.2 \times 1.042 = 3.33 \text{ m}$$

### 总水头损失

$$h_w = h_f + \sum h_j = 17.37 + 3.33 = 20.70 \text{ m}$$

### 损失比例

$$\frac{h_f}{h_w} = \frac{17.37}{20.70} = 83.9\%$$（沿程为主）

$$\frac{\sum h_j}{h_w} = \frac{3.33}{20.70} = 16.1\%$$（局部次之）

### 答案

- 沿程损失：**h_f = 17.37 m**
- 局部损失：**Σh_j = 3.33 m**
- 总损失：**h_w = 20.70 m**

### 踩分点（25分）

- 流速（3分）
- 沿程损失（8分）
- 局部损失（8分）
- 总损失（6分）

## 题目要点总结（199-215）

### 题199：串联管道
各段损失**累加**：
$$h_w = h_{f1} + h_{f2} + ... + \sum h_j$$

### 题200：并联管道
各支路水头损失**相等**：
$$h_{w1} = h_{w2}$$

流量分配：
$$Q_1/Q_2 = \sqrt{h_{w1}/h_{w2}} \times \sqrt{d_1^5/d_2^5} \times ...$$

### 题208：经济管径
**投资**与**运行费**的平衡

经济流速：v_经济≈1.0~3.0 m/s

### 题213：边界层
**层流边界层厚度**：
$$\delta \approx 5\sqrt{\frac{\nu x}{v}}$$

## 📚 本节总结

### 减小水头损失的措施

1. **减小沿程损失**
   - 增大管径
   - 降低流速
   - 使用光滑管
   - 减少管长

2. **减小局部损失**
   - 采用渐变管
   - 减少弯头数量
   - 大曲率半径弯头
   - 流线型设计

### 工程计算要点

- ✅ 准确查取λ和ζ
- ✅ 注意Re的判断
- ✅ 区分主要损失和次要损失
- ✅ 合理简化计算
