## 核心知识点

### 1. 连续性方程的物理意义

**质量守恒定律**：流体在流动过程中，质量既不能创造也不能消失。

### 2. 数学表达式

**通用形式**（微分形式）：
$$\frac{\partial \rho}{\partial t} + \frac{\partial(\rho u)}{\partial x} + \frac{\partial(\rho v)}{\partial y} + \frac{\partial(\rho w)}{\partial z} = 0$$

**不可压缩流体**（ρ=常数）：
$$\frac{\partial u}{\partial x} + \frac{\partial v}{\partial y} + \frac{\partial w}{\partial z} = 0$$

**一维稳定流**：
$$v_1 A_1 = v_2 A_2 = Q = \text{常数}$$

### 3. 流量Q的表达

$$Q = v \cdot A$$

单位：m³/s（立方米每秒）

## 题101：变截面管道流速计算

**【题目】**（15分，考频⭐⭐⭐⭐⭐）

某水平管道，管径由d₁=0.2m渐缩至d₂=0.1m。断面1处流速v₁=2m/s。求：
1. 断面2处流速v₂
2. 管道流量Q
3. 如果管径d₂增大到0.15m，v₂变为多少？

**【解答】**

### 连续性方程

$$Q = v_1 A_1 = v_2 A_2$$

### 断面面积

断面1：
$$A_1 = \frac{\pi d_1^2}{4} = \frac{\pi \times 0.2^2}{4} = 0.0314 \text{ m}^2$$

断面2：
$$A_2 = \frac{\pi d_2^2}{4} = \frac{\pi \times 0.1^2}{4} = 0.00785 \text{ m}^2$$

### 流速v₂

$$v_2 = v_1 \frac{A_1}{A_2} = 2 \times \frac{0.0314}{0.00785} = 2 \times 4 = 8 \text{ m/s}$$

**重要关系**：
$$v_2 = v_1 \left(\frac{d_1}{d_2}\right)^2 = 2 \times \left(\frac{0.2}{0.1}\right)^2 = 2 \times 4 = 8 \text{ m/s}$$

### 流量Q

$$Q = v_1 A_1 = 2 \times 0.0314 = 0.0628 \text{ m}^3/\text{s} = 62.8 \text{ L/s}$$

验证：
$$Q = v_2 A_2 = 8 \times 0.00785 = 0.0628 \text{ m}^3/\text{s}$$ ✓

### d₂=0.15m时

$$v_2' = v_1 \left(\frac{d_1}{d_2'}\right)^2 = 2 \times \left(\frac{0.2}{0.15}\right)^2$$
$$= 2 \times (1.333)^2 = 2 \times 1.778 = 3.56 \text{ m/s}$$

### 答案

1. 断面2流速：**v₂ = 8 m/s**
2. 流量：**Q = 0.0628 m³/s = 62.8 L/s**
3. d₂=0.15m时：**v₂' = 3.56 m/s**

### Python代码

```python
# 文件：problem_101_pipe_continuity.py
import numpy as np

def pipe_flow_velocity(d1, v1, d2):
    """
    管道流速计算（连续性方程）
    
    参数:
        d1: 断面1直径 (m)
        v1: 断面1流速 (m/s)
        d2: 断面2直径 (m)
    
    返回:
        v2: 断面2流速 (m/s)
        Q: 流量 (m³/s)
    """
    A1 = np.pi * d1**2 / 4
    A2 = np.pi * d2**2 / 4
    
    v2 = v1 * A1 / A2
    Q = v1 * A1
    
    return v2, Q

# 计算
v2, Q = pipe_flow_velocity(0.2, 2, 0.1)
print(f"断面2流速: {v2:.2f} m/s")
print(f"流量: {Q:.4f} m³/s = {Q*1000:.1f} L/s")

# 第3问
v2_new, _ = pipe_flow_velocity(0.2, 2, 0.15)
print(f"d2=0.15m时流速: {v2_new:.2f} m/s")
```

```

流量: 0.0628 m³/s = 62.8 L/s
d2=0.15m时流速: 3.56 m/s
```

## 题102：三支管分流

**【题目】**（20分，考频⭐⭐⭐⭐）

某管道系统，主管流量Q=0.5m³/s，在节点分为两支管：
- 支管1：流量Q₁=0.2m³/s，直径d₁=0.15m
- 支管2：流量Q₂=0.3m³/s，直径d₂=0.2m

求：
1. 验证流量守恒
2. 两支管的平均流速
3. 如果支管2直径增大到0.25m，流速如何变化？

**【解答】**

### 流量守恒（节点连续性）

**入流=出流**：
$$Q = Q_1 + Q_2$$

验证：
$$0.5 = 0.2 + 0.3 = 0.5$$ ✓

**流量守恒成立！**

### 支管流速

**支管1**：
$$A_1 = \frac{\pi d_1^2}{4} = \frac{\pi \times 0.15^2}{4} = 0.0177 \text{ m}^2$$

$$v_1 = \frac{Q_1}{A_1} = \frac{0.2}{0.0177} = 11.3 \text{ m/s}$$

**支管2**：
$$A_2 = \frac{\pi d_2^2}{4} = \frac{\pi \times 0.2^2}{4} = 0.0314 \text{ m}^2$$

$$v_2 = \frac{Q_2}{A_2} = \frac{0.3}{0.0314} = 9.55 \text{ m/s}$$

### d₂'=0.25m时

$$A_2' = \frac{\pi \times 0.25^2}{4} = 0.0491 \text{ m}^2$$

$$v_2' = \frac{Q_2}{A_2'} = \frac{0.3}{0.0491} = 6.11 \text{ m/s}$$

流速降低：
$$\frac{v_2'}{v_2} = \frac{6.11}{9.55} = 0.64 = 64\%$$

### 答案

1. 流量守恒：Q = Q₁ + Q₂ = **0.5 m³/s** ✓
2. 支管流速：
   - v₁ = **11.3 m/s**
   - v₂ = **9.55 m/s**
3. d₂=0.25m时：v₂' = **6.11 m/s**（降低36%）
