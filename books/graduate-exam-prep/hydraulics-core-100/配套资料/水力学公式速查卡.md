## 第一章：静水力学

### 1.1 静水压强 ⭐⭐⭐⭐⭐

#### 基本公式
```
p = p₀ + ρgh
```
**说明**：
- p：任意点压强（Pa）
- p₀：液面压强（Pa）
- ρ：流体密度（kg/m³）
- g：重力加速度（9.81 m/s²）
- h：深度（m）

**考点**：
- ⭐⭐⭐⭐⭐ 绝对压强 vs 相对压强
- ⭐⭐⭐⭐⭐ 压强分布图绘制
- ⭐⭐⭐⭐ 多层液体压强

---

### 1.2 静水总压力 ⭐⭐⭐⭐⭐

#### 平面壁总压力（大小）
```
F = ρg·h̄·A = p̄·A
```
**说明**：
- F：总压力（N）
- h̄：形心深度（m）
- A：受压面积（m²）
- p̄：形心处压强（Pa）

**记忆口诀**：总压力 = 形心压强 × 面积

---

#### 平面壁总压力（作用点）
```
yD = yC + Ic/(yC·A)
```
**说明**：
- yD：压力中心坐标（m）
- yC：形心坐标（m）
- Ic：对形心轴的惯性矩（m⁴）

**常用断面惯性矩**：
- 矩形（b×h）：Ic = bh³/12
- 圆形（半径r）：Ic = πr⁴/4
- 三角形（底b，高h）：Ic = bh³/36

**记忆要点**：
- 压力中心永远在形心下方（yD > yC）
- 深度越深，压力中心越接近形心

---

### 1.3 浮力与稳定性 ⭐⭐⭐⭐

#### 阿基米德原理
```
Fb = ρg·V排
```
**说明**：
- Fb：浮力（N）
- V排：排开液体体积（m³）

#### 浮体稳定性条件
```
稳定：M在G上方（GM > 0）
不稳定：M在G下方（GM < 0）
```
**说明**：
- M：稳心（Metacenter）
- G：重心（Center of Gravity）
- GM：稳心高度（Metacentric Height）

---

### 1.4 压强测量 ⭐⭐⭐

#### U型测压管
```
p = ρ₀gh₀ - ρgh
```
**适用**：小压强测量

#### 差压计
```
Δp = (ρ₀ - ρ)gh
```
**适用**：两点压差测量

## 第二章：流体动力学基础

### 2.1 连续性方程 ⭐⭐⭐⭐⭐

#### 不可压缩流体
```
Q = v₁A₁ = v₂A₂ = 常数
```
**说明**：
- Q：流量（m³/s）
- v：流速（m/s）
- A：过流断面面积（m²）

**物理意义**：质量守恒

**记忆口诀**：断面小，流速大

---

### 2.2 伯努利方程 ⭐⭐⭐⭐⭐

#### 理想流体（无能量损失）
```
z₁ + p₁/(ρg) + v₁²/(2g) = z₂ + p₂/(ρg) + v₂²/(2g)
```

#### 实际流体（有能量损失）
```
z₁ + p₁/(ρg) + α₁v₁²/(2g) = z₂ + p₂/(ρg) + α₂v₂²/(2g) + hw
```

**各项物理意义**：
- z：位置水头（位置能）
- p/(ρg)：压强水头（压强能）
- v²/(2g)：流速水头（动能）
- α：动能修正系数（层流2.0，紊流1.05-1.10）
- hw：水头损失（m）

**记忆口诀**：
> 位置+压强+速度 = 总水头（能量守恒）

**使用条件**：
1. ✅ 不可压缩流体
2. ✅ 沿同一流线
3. ✅ 恒定流
4. ✅ 质量力只有重力

**常见错误**：
- ❌ 忘记加水头损失hw
- ❌ 参考面选择不一致
- ❌ 压强单位不统一

---

### 2.3 动量方程 ⭐⭐⭐⭐

#### 动量方程（矢量形式）
```
ΣF = ρQ(v₂ - v₁)
```

#### 分量形式（常用）
```
ΣFx = ρQ(v₂x - v₁x)
ΣFy = ρQ(v₂y - v₁y)
```

**说明**：
- ΣF：外力合力（N）
- ρQ：质量流量（kg/s）
- v：流速（m/s）

**应用场景**：
- 弯管受力
- 射流冲击
- 闸门启闭力
- 喷水推进器

**记忆口诀**：动量变化 = 外力冲量

---

### 2.4 能量方程 ⭐⭐⭐⭐

#### 单位重量流体能量方程
```
z₁ + p₁/(ρg) + α₁v₁²/(2g) + Hm = z₂ + p₂/(ρg) + α₂v₂²/(2g) + hw
```

**说明**：
- Hm：水泵扬程（正）或水轮机作功水头（负）

#### 功率方程
```
N = ρgQH
```

**说明**：
- N：功率（W）
- H：水头（m）

**效率**：
```
η = 有效功率/轴功率
```

## 第三章：管道流动

### 3.1 雷诺数 ⭐⭐⭐⭐⭐

```
Re = vd/ν = ρvd/μ
```

**说明**：
- Re：雷诺数（无量纲）
- v：平均流速（m/s）
- d：管径（m）
- ν：运动粘度（m²/s）
- μ：动力粘度（Pa·s）

**流态判别**：
- Re < 2000：层流
- 2000 < Re < 4000：过渡区
- Re > 4000：紊流

**考点**：
- ⭐⭐⭐⭐⭐ 流态判别
- ⭐⭐⭐⭐ 临界雷诺数Rec = 2000
- ⭐⭐⭐ 影响因素分析

---

### 3.2 沿程水头损失 ⭐⭐⭐⭐⭐

#### 达西-魏斯巴赫公式（通用）
```
hf = λ·(L/d)·v²/(2g)
```

**说明**：
- hf：沿程损失（m）
- λ：沿程阻力系数（无量纲）
- L：管道长度（m）
- d：管径（m）

**记忆口诀**：损失 ∝ 长度/直径 × 速度²

---

#### 层流阻力系数
```
λ = 64/Re
```
**适用**：Re < 2000

---

#### 紊流阻力系数

**光滑管（Blasius公式）**：
```
λ = 0.3164/Re^0.25
```
**适用**：Re < 10⁵

**粗糙管（Colebrook公式）**：
```
1/√λ = -2lg(Δ/(3.7d) + 2.51/(Re√λ))
```
**说明**：
- Δ：绝对粗糙度（mm）
- Δ/d：相对粗糙度

**数值求解**：需要迭代（牛顿法或Python）

---

#### 谢才公式
```
v = C√(Ri)
```
**说明**：
- C：谢才系数（m^0.5/s）
- R：水力半径（m）
- i：水力坡度

**曼宁公式（计算C）**：
```
C = R^(1/6) / n
```
**说明**：
- n：曼宁粗糙系数
  - 混凝土：n = 0.012-0.014
  - 钢管：n = 0.011-0.012
  - 土渠：n = 0.020-0.025

---

### 3.3 局部水头损失 ⭐⭐⭐⭐

```
hj = ζ·v²/(2g)
```

**说明**：
- hj：局部损失（m）
- ζ：局部阻力系数（无量纲）

**常用ζ值**：
- 进口：ζ = 0.5
- 出口：ζ = 1.0
- 突然扩大：ζ = (1 - A₁/A₂)²
- 突然缩小：ζ = 0.5(1 - A₂/A₁)
- 90°弯头：ζ = 0.3-1.0
- 闸阀（全开）：ζ = 0.2

**记忆口诀**：
- 进口0.5，出口1.0
- 扩大损失大于缩小

---

### 3.4 管道计算三类问题 ⭐⭐⭐⭐⭐

#### 第一类：已知Q，求hw
```
直接计算：hw = (λL/d + Σζ)·v²/(2g)
```

#### 第二类：已知hw，求Q
```
迭代求解：假设λ → 算v → 算Re → 算λ → 迭代
```

#### 第三类：已知Q和hw，求d
```
迭代求解：假设d → 算v → 算Re → 算λ → 算hw → 调整d
```

**解题技巧**：
- 第一类：直接算
- 第二、三类：需要迭代（推荐Python）

---

### 3.5 经济管径 ⭐⭐⭐

```
d_经济 = k·Q^0.5
```

**常用经验值**：
- 输水管：k = 1.2-1.4
- 输油管：k = 1.0-1.2

**优化原则**：
- 直径大 → 能量损失小 → 运行费用低
- 直径小 → 管材费用低 → 初投资少
- 最优：总费用最小

## 第四章：明渠流动

### 4.1 明渠均匀流 ⭐⭐⭐⭐⭐

#### 谢才公式
```
Q = AC√(Ri)
v = C√(Ri)
```

**说明**：
- A：过流断面面积（m²）
- C：谢才系数
- R：水力半径 = A/χ（m）
- χ：湿周（m）
- i：渠底坡度

---

#### 曼宁公式
```
Q = (A/n)·R^(2/3)·i^(1/2)
v = (1/n)·R^(2/3)·i^(1/2)
```

**记忆口诀**：
> 流量 = 1/n × 面积 × 水力半径^(2/3) × 坡度^(1/2)

**考点**：
- ⭐⭐⭐⭐⭐ 正常水深计算
- ⭐⭐⭐⭐ 水力最优断面
- ⭐⭐⭐⭐ 允许流速验算

---

### 4.2 水力最优断面 ⭐⭐⭐⭐

#### 基本原理
给定Q、n、i，使A最小（或给定A，使Q最大）

#### 共同结论
```
R = h/2
```
**说明**：水力半径 = 水深/2

#### 矩形断面
```
最优宽深比：b/h = 2
最优水力半径：R = h/2
```

#### 梯形断面
```
最优边坡系数：m = 1/√3 (θ = 60°)
最优宽深比：b/h = 2(√3 - 1)/√3 ≈ 0.536
```

#### 三角形断面
```
最优边坡角：θ = 45° (m = 1)
```

**记忆口诀**：
> 不管什么断面，R = h/2！

---

### 4.3 临界水深 ⭐⭐⭐⭐⭐

#### 临界流条件
```
Fr = v/√(gA/B) = 1
```

**说明**：
- Fr：弗劳德数（无量纲）
- B：水面宽度（m）

**流态判别**：
- Fr < 1：缓流（h > hc）
- Fr = 1：临界流（h = hc）
- Fr > 1：急流（h < hc）

---

#### 矩形明渠临界水深
```
hc = ∛(q²/g)
```
**说明**：
- q = Q/b：单宽流量（m²/s）

**临界流速**：
```
vc = √(ghc)
```

---

#### 一般断面临界水深（迭代求解）
```
Q²B/gA³ = 1
```

**解题方法**：
1. 假设hc
2. 计算A、B
3. 检查是否满足Q²B/gA³ = 1
4. 调整hc，重复2-3

---

### 4.4 水跃 ⭐⭐⭐⭐⭐

#### 共轭水深公式（矩形）
```
h₂ = (h₁/2)(√(1 + 8Fr₁²) - 1)
```

**说明**：
- h₁：跃前水深（m）
- h₂：跃后水深（m）
- Fr₁：跃前弗劳德数

**反向计算**：
```
h₁ = (h₂/2)(√(1 + 8Fr₂²) - 1)
```

---

#### 水跃能量损失
```
ΔE = (h₂ - h₁)³/(4h₁h₂)
```

**相对能量损失**：
```
ΔE/E₁ = (Fr₁² - 1)³/(8Fr₁²(Fr₁² + 2))
```

---

#### 水跃长度（经验公式）
```
Lj = (4-7)·h₂
```
**常用**：Lj ≈ 5h₂

---

#### 水跃类型（按Fr₁）
- Fr₁ < 1：不可能发生
- 1 < Fr₁ < 1.7：波状水跃
- 1.7 < Fr₁ < 2.5：弱水跃
- 2.5 < Fr₁ < 4.5：稳定水跃 ⭐推荐
- 4.5 < Fr₁ < 9：强水跃
- Fr₁ > 9：猛水跃

**记忆要点**：
- 急流→缓流：必然发生水跃
- 跃后水深 > 跃前水深
- 能量大量损失（消能）

---

### 4.5 水面曲线 ⭐⭐⭐⭐

#### 基本方程
```
dh/ds = (i - if)/(1 - Fr²)
```

**说明**：
- i：渠底坡度
- if：摩阻坡度 = v²/(C²R)
- Fr：弗劳德数

---

#### 12种水面线类型

**陡坡（i > ic）**：
- S₁型：壅水曲线（h > h₀ > hc）
- S₂型：降水曲线（h₀ > h > hc）
- S₃型：背水曲线（h₀ > hc > h）

**缓坡（i < ic）**：
- M₁型：壅水曲线（h > h₀ > hc）⭐常见
- M₂型：降水曲线（h₀ > h > hc）
- M₃型：背水曲线（h₀ > hc > h）

**临界坡（i = ic）**：
- C₁型：壅水曲线（h > hc = h₀）
- C₃型：背水曲线（hc = h₀ > h）

**平坡（i = 0）**：
- H₂型、H₃型

**逆坡（i < 0）**：
- A₂型、A₃型

**记忆口诀**：
> S陡M缓C临界，H平A逆要记牢
> 1上2中3下行，缓流壅水急流降

---

#### 水面线计算（逐段法）
```
Δs = (E₂ - E₁)/(i - īf)
```

**步骤**：
1. 确定控制断面（hk已知）
2. 假设Δh
3. 计算h₁ = hk + Δh
4. 计算平均摩阻坡度īf
5. 计算Δs
6. 重复2-5，直到终点

## 第五章：水工建筑物

### 5.1 堰流 ⭐⭐⭐⭐⭐

#### 薄壁堰（自由出流）
```
Q = m·b·√(2g)·H^(3/2)
```

**说明**：
- m：流量系数（0.35-0.45）
- b：堰宽（m）
- H：堰上水头（m）

**记忆口诀**：流量 ∝ H^(3/2)

---

#### 宽顶堰（自由出流）
```
Q = m·b·√(2g)·H₀^(3/2)
```

**说明**：
- m：流量系数（0.32-0.38）
- H₀：行进水头 = H + v₀²/(2g)

**临界水深**：
```
hc = (2/3)·H₀
```

---

#### 实用堰（淹没出流）
```
Q = m·b·√(2g)·H₁^(3/2)·σs
```

**说明**：
- σs：淹没系数（< 1）
- H₁：上游水头
- H₂：下游水头

**淹没度**：
```
S = H₂/H₁
```

---

### 5.2 闸孔出流 ⭐⭐⭐⭐⭐

#### 平板闸门（自由出流）
```
Q = m·b·e·√(2g·H₀)
```

**说明**：
- m：流量系数（0.60-0.65）
- e：闸门开度（m）
- H₀：闸前水深（m）

**收缩系数**：
```
ε = hc/e ≈ 0.61
```

---

#### 平板闸门（淹没出流）
```
Q = m·b·e·√(2g·ΔH)·σs
```

**说明**：
- ΔH = H₁ - H₂：上下游水位差
- σs：淹没系数

**淹没条件**：
```
hs/hc > 0.8
```

---

### 5.3 水跃消能 ⭐⭐⭐⭐

#### 消能池设计

**池深计算**：
```
d = h₂ - Δz
```
**说明**：
- d：池深（m）
- h₂：共轭水深（m）
- Δz：池底高程差（m）

**池长计算**：
```
L = Lj + Ls
```
**说明**：
- Lj：水跃长度 ≈ 5h₂
- Ls：消能段长度

---

#### 辅助消能工

**消力墩**：
- 位置：跃前1-2h₁
- 高度：(0.8-1.0)h₁

**消力坎**：
- 位置：跃尾
- 高度：(0.05-0.15)h₂

---

### 5.4 掺气减蚀 ⭐⭐⭐

#### 掺气浓度
```
β = Qa/Qw
```
**说明**：
- Qa：掺气量（m³/s）
- Qw：水流量（m³/s）

**保护流速**：
- 无掺气：v < 30 m/s
- 有掺气：v < 50 m/s

## 第六章：非恒定流基础

### 6.1 水锤 ⭐⭐⭐⭐

#### 水锤压强（刚性水击）
```
Δp = ±ρcΔv
```

**说明**：
- Δp：水锤压强（Pa）
- c：波速（m/s）
- Δv：流速变化（m/s）

**波速**：
```
c = √(K/ρ)/(1 + (K/E)·(d/δ))
```
**说明**：
- K：水的弹性模量 ≈ 2.1×10⁹ Pa
- E：管壁弹性模量（钢：2.1×10¹¹ Pa）
- d：管径（m）
- δ：壁厚（m）

**水锤相**：
```
T = 2L/c
```
**说明**：
- T：水锤相（s）
- L：管道长度（m）

**水击类型**：
- 直接水击：tc < T（刚性）
- 间接水击：tc > T（弹性）
- tc：阀门关闭时间

---

### 6.2 圣维南方程组 ⭐⭐⭐

#### 连续方程
```
∂A/∂t + ∂Q/∂x = 0
```

#### 动量方程
```
∂v/∂t + v·∂v/∂x + g·∂h/∂x + gSf - gi = 0
```

**说明**：
- Sf：摩阻坡度
- i：渠底坡度

**特征线法求解**（数值方法）

---

### 6.3 洪水演进 ⭐⭐⭐

#### 马斯京根法
```
Q₂ = C₀I₂ + C₁I₁ + C₂Q₁
```

**系数**：
```
C₀ = (Δt - 2KX)/(2K(1-X) + Δt)
C₁ = (Δt + 2KX)/(2K(1-X) + Δt)
C₂ = (2K(1-X) - Δt)/(2K(1-X) + Δt)
```

**说明**：
- K：蓄量系数（h）
- X：比重系数（0-0.5）
- Δt：时间步长（h）

**验证**：C₀ + C₁ + C₂ = 1

## 💯 重要常数与经验值

### 物理常数
```
g = 9.81 m/s²（重力加速度）
ρ水 = 1000 kg/m³（水的密度，20℃）
ν水 = 1.01×10⁻⁶ m²/s（水的运动粘度，20℃）
patm = 101325 Pa = 1 atm（标准大气压）
```
