## 第一章：静水力学

### 1.1 静水压强

```
基本公式：
p = p₀ + ρgh          静水压强公式
p = γh                简化形式（p₀=0）
γ = ρg               重度

重要常数：
ρ_水 = 1000 kg/m³
g = 9.81 m/s²
γ_水 = 9810 N/m³
```

### 1.2 压力中心

```
平面上总压力：
P = γh_c·A            h_c为形心深度

压力中心深度：
D = h_c + I_c/(h_c·A)
  = h_c + e           e为偏心距

常见形状的I_c：
矩形：I_c = bh³/12
圆形：I_c = πd⁴/64
三角形：I_c = bh³/36
```

### 1.3 浮力

```
阿基米德原理：
F_浮 = γ_液·V_排

浮力作用点：
通过排开液体体积的形心
```

## 第二章：流体动力学

### 2.1 连续性方程

```
不可压缩流体：
Q = A₁v₁ = A₂v₂ = 常数
∂A/∂t + ∂Q/∂x = 0   （非恒定流）

质量守恒：
ρ₁A₁v₁ = ρ₂A₂v₂     （可压缩）
```

### 2.2 伯努利方程 ⭐⭐⭐

```
理想流体：
z₁ + p₁/(ρg) + v₁²/(2g) = z₂ + p₂/(ρg) + v₂²/(2g)

实际流体：
z₁ + p₁/(ρg) + α₁v₁²/(2g) = z₂ + p₂/(ρg) + α₂v₂²/(2g) + h_w

能量线：EL = z + p/(ρg) + v²/(2g)
水力坡度线：HGL = z + p/(ρg)

动能修正系数：
α = 1.0  （均匀流）
α ≈ 1.05-1.1 （紊流）
α ≈ 2.0  （层流）
```

### 2.3 动量方程

```
动量定理：
ΣF = ρQ(v₂ - v₁)

分量形式：
F_x = ρQ(v₂ₓ - v₁ₓ) + p₁A₁ - p₂A₂
F_y = ρQ(v₂ᵧ - v₁ᵧ)
```

### 2.4 孔口管嘴出流

```
小孔口：
v = φ√(2gh)          φ = 0.97-0.98
Q = μA√(2gh)         μ = 0.60-0.62

管嘴：
Q = μA√(2gh)         μ = 0.82
```

### 2.5 文丘里流量计

```
Q = (A₂/√(1-(A₂/A₁)²))·√(2gΔh)

简化（A₂<<A₁）：
Q ≈ A₂√(2gΔh)
```

## 第三章：管道流动

### 3.1 沿程损失 ⭐⭐⭐

```
达西-魏斯巴赫公式：
h_f = λ(L/d)(v²/2g)

λ的计算：
层流（Re<2300）：
  λ = 64/Re

紊流光滑管（Re>2300）：
  λ = 0.3164/Re^0.25         （Blasius）

Colebrook公式（隐式）：
  1/√λ = -2log₁₀(ε/(3.7d) + 2.51/(Re√λ))

Swamee-Jain显式：
  λ = 0.25/[log₁₀(ε/(3.7d) + 5.74/Re^0.9)]²
```

### 3.2 局部损失

```
h_j = ζ(v²/2g)

常见阻力系数：
突然扩大：ζ = (1 - A₁/A₂)²
突然收缩：ζ = 0.5(1 - A₂/A₁)
进口：ζ = 0.5
出口：ζ = 1.0
90°弯头：ζ = 0.3-1.0
闸阀全开：ζ = 0.2
```

### 3.3 长管短管

```
长管（忽略局部损失）：
Q = A√(2gH/(λL/d))

短管（考虑局部损失）：
Q = A√(2gH/(λL/d + Σζ))
```

### 3.4 管网计算

```
Hardy-Cross法：
ΔQ = -ΣhL/(n·Σ|hL|/Q)

节点法：
Σ Q_in = Σ Q_out    （节点流量平衡）
Σ hL = 0            （环路水头闭合）
```

## 第四章：明渠流动 ⭐⭐⭐

### 4.1 明渠均匀流

```
谢才公式：
v = C√(Ri)
Q = AC√(Ri)

曼宁公式：⭐⭐⭐
v = (1/n)R^(2/3)i^(1/2)
Q = (A/n)R^(2/3)i^(1/2)

常用糙率n：
混凝土：0.012-0.014
浆砌石：0.020-0.025
土渠：0.025-0.030
天然河道：0.030-0.050

断面几何要素：
矩形：A = bh,  P = b+2h,  R = bh/(b+2h)
梯形：A = (b+mh)h,  P = b+2h√(1+m²)
圆形：A = (θ-sinθ)d²/8,  P = θd/2
```

### 4.2 水力最优断面

```
条件：R最大，P最小

矩形：b = 2h, R = h/2
梯形：m = √3, b = 2h/√3, R = h/2
圆形：h/d = 0.938, R = 0.29d
```

### 4.3 临界水深 ⭐⭐⭐

```
临界流条件：
Fr = v/√(gA/B) = 1

矩形：
h_c = ³√(q²/g)
v_c = √(gh_c)

梯形（迭代求解）：
Q²B/(gA³) = 1

比能：
E = h + v²/(2g) = h + Q²/(2gA²)
E_min = 1.5h_c  （矩形）
```

### 4.4 水跃 ⭐⭐⭐

```
共轭水深（矩形）：
h₂ = (h₁/2)(√(1+8Fr₁²) - 1)

跃前Fr₁ = v₁/√(gh₁)

水跃长度：
L_j ≈ 6.9(h₂ - h₁)    （经验公式）

能量损失：
ΔE = (h₂ - h₁)³/(4h₁h₂)
```

### 4.5 水面曲线 ⭐⭐⭐

```
基本微分方程：
dh/dx = (i - if)/(1 - Fr²)

曲线分类：
缓坡（i < ic）：M₁, M₂, M₃
陡坡（i > ic）：S₁, S₂, S₃
临界坡（i = ic）：C₁, C₃

分段求和法：
Δx = Δh/(dh/dx)
```

## 第五章：水工建筑物

### 5.1 堰流 ⭐⭐

```
薄壁堰：
矩形：Q = (2/3)mb√(2g)H^(3/2)
       m = 0.40-0.49

三角堰：Q = (8/15)m_θtanθ√(2g)H^(5/2)
       θ = 90°, m = 0.58-0.62

实用堰：
Q = mb√(2g)H^(3/2)
m = 0.35-0.50
```

### 5.2 闸孔出流

```
孔口出流（e/H < 0.65）：
Q = εbe√(2gH₁)
ε = 0.60-0.65

堰流（e/H > 0.80）：
Q = mbe√(2g)H₁^(3/2)
m = 0.35-0.40

淹没出流：
Q = εbe√(2g(H₁-H₂))·σ_s
σ_s = f(H₂/H₁)
```

### 5.3 消能工

```
底流消能池：
L_pool = 5-6·h₂
深度：d = h₂ - h_t
齿坎高度：δ_s = 0.2h₂

水跃长度：
L_j = 5-7·(h₂-h₁)
```

## 第六章：非恒定流

### 6.1 Saint-Venant方程

```
连续方程：
∂A/∂t + ∂Q/∂x = 0

动量方程：
∂Q/∂t + ∂(Q²/A)/∂x + gA∂h/∂x = gA(S₀ - Sf)

简化形式：
运动波：∂Q/∂t + c∂Q/∂x = 0
扩散波：∂Q/∂t + c∂Q/∂x = D∂²Q/∂x²
```

### 6.2 水锤 ⭐⭐

```
水锤波速：
a = √(K/(ρ(1 + Kd/(Eδ))))

Joukowsky公式：
Δp = ρav₀
Δh = av₀/g

相位：
T = 2L/a
```

### 6.3 洪水演进

```
特征线法：
C⁺: dx/dt = v + c
C⁻: dx/dt = v - c

运动波法：
c = dQ/dA ≈ 5v/3  （宽浅矩形）
```

## 🎯 解题技巧

### 1. 判断流态

```
管道：计算Re，判断层流/紊流
明渠：计算Fr，判断缓流/急流
      比较i与ic，判断缓坡/陡坡
```

### 2. 选择公式

```
管道损失：先判断长管/短管
明渠流动：先判断均匀流/非均匀流
水工建筑物：先判断孔口流/堰流
```

### 3. 数值计算

```
临界水深：牛顿迭代法
正常水深：试算法或迭代法
水面线：分段求和法
管网：Hardy-Cross法
```

### 4. 单位换算

```
流量：1 m³/s = 1000 L/s
压强：1 atm = 101325 Pa ≈ 10.33 mH₂O
速度：1 m/s = 3.6 km/h
```

## ⚠️ 易错提醒

### 1. 单位统一
- ✅ 长度全用米(m)
- ✅ 时间全用秒(s)
- ✅ 压强用Pa或mH₂O

### 2. 伯努利方程
- ✅ 基准面要一致
- ✅ 压强用绝对压强或相对压强统一
- ✅ 不要忘记动能修正系数α

### 3. 明渠流动
- ✅ 区分i（坡度）和if（摩阻坡度）
- ✅ 注意h₀、hc的大小关系
- ✅ Fr的计算用A/B不是h

### 4. 管道流动
- ✅ λ和f的关系：f = 4λ（有些教材）
- ✅ Re的计算用v不是Q
- ✅ 局部损失系数ζ基于下游流速
