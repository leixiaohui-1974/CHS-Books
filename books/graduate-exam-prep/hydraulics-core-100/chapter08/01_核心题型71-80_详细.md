## 题型71：水泵基本参数计算

### 【题目】

离心泵抽水，流量$Q=100$L/s，扬程$H=30$m，轴功率$P_{轴}=40$kW。求：
(1) 有效功率$P_e$
(2) 效率$\eta$

### 【解答】

**(1) 有效功率**

$$P_e = \rho g Q H = 1000 \times 9.81 \times 0.1 \times 30 = 29430 \text{ W} = 29.43 \text{ kW}$$

**(2) 效率**

$$\eta = \frac{P_e}{P_{轴}} \times 100\% = \frac{29.43}{40} \times 100\% = 73.6\%$$

## 题型72：水泵扬程计算

### 【题目】

水泵站，吸水池水位$z_1=100$m，出水池水位$z_2=135$m，吸水管路损失$h_{f1}=1.5$m，压水管路损失$h_{f2}=3.8$m。求所需扬程$H$。

### 【解答】

**扬程定义**：

$$H = (z_2 - z_1) + (h_{f1} + h_{f2}) = H_{st} + h_f$$

其中：
- $H_{st} = z_2 - z_1 = 35$m（静扬程）
- $h_f = h_{f1} + h_{f2} = 5.3$m（管路损失）

$$H = 35 + 5.3 = 40.3 \text{ m}$$

**选泵**：选择扬程略大于40.3m的水泵。

## 题型73：相似定律应用

### 【题目】

某水泵在$n_1=1450$r/min时，$Q_1=200$L/s，$H_1=25$m，$P_1=60$kW。若转速改为$n_2=1750$r/min，求相应参数。

### 【解答】

**相似定律**（相似工况）：

$$\frac{Q_2}{Q_1} = \frac{n_2}{n_1}, \quad \frac{H_2}{H_1} = \left(\frac{n_2}{n_1}\right)^2, \quad \frac{P_2}{P_1} = \left(\frac{n_2}{n_1}\right)^3$$

转速比：
$$\frac{n_2}{n_1} = \frac{1750}{1450} = 1.207$$

**(1) 流量**

$$Q_2 = Q_1 \times 1.207 = 200 \times 1.207 = 241.4 \text{ L/s}$$

**(2) 扬程**

$$H_2 = H_1 \times 1.207^2 = 25 \times 1.456 = 36.4 \text{ m}$$

**(3) 功率**

$$P_2 = P_1 \times 1.207^3 = 60 \times 1.757 = 105.4 \text{ kW}$$

## 题型74：比转速计算

### 【题目】

水泵在最优工况下，$n=1450$r/min，$Q=150$L/s，$H=32$m。求比转速$n_s$，并判断水泵类型。

### 【解答】

**比转速定义**：

$$n_s = \frac{n\sqrt{Q}}{H^{3/4}}$$

单位：$n$(r/min)，$Q$(m³/s)，$H$(m)

$$n_s = \frac{1450 \times \sqrt{0.15}}{32^{3/4}} = \frac{1450 \times 0.387}{13.45} = 41.8$$

**水泵类型判断**：

| $n_s$范围 | 水泵类型 |
|----------|---------|
| 30 ~ 80 | 离心泵 |
| 80 ~ 150 | 混流泵 |
| 150 ~ 300 | 轴流泵 |

**判断**：$n_s = 41.8$，**离心泵**

## 题型75：工况点确定

### 【题目】

水泵特性曲线：$H = 50 - 0.002Q^2$（$H$单位m，$Q$单位L/s）  
管路特性曲线：$H = 25 + 0.001Q^2$

求工况点流量和扬程。

### 【解答】

**工况点**：水泵特性曲线与管路特性曲线交点

$$H_{泵} = H_{管}$$

$$50 - 0.002Q^2 = 25 + 0.001Q^2$$

$$25 = 0.003Q^2$$

$$Q^2 = 8333.3$$

$$Q = 91.3 \text{ L/s}$$

$$H = 25 + 0.001 \times 8333.3 = 33.3 \text{ m}$$

**工况点**：$(Q, H) = (91.3 \text{ L/s}, 33.3 \text{ m})$

## 题型76：串联与并联

### 【题目】

两台相同水泵，单泵特性：$Q_0=100$L/s，$H_0=40$m。求串联和并联时的性能。

### 【解答】

**(1) 串联**

**特点**：流量不变，扬程相加

$$Q_{串} = Q_0 = 100 \text{ L/s}$$

$$H_{串} = 2H_0 = 80 \text{ m}$$

**(2) 并联**

**特点**：扬程不变，流量相加

$$Q_{并} = 2Q_0 = 200 \text{ L/s}$$

$$H_{并} = H_0 = 40 \text{ m}$$

**应用场景**：
- 串联：提高扬程（高扬程系统）
- 并联：增大流量（大流量系统）

## 题型77：汽蚀判别

### 【题目】

离心泵安装高度$H_s=3$m，当地大气压$p_a=101.3$kPa，水温$20°C$时饱和蒸汽压$p_v=2.34$kPa，吸水管损失$h_{s}=1.2$m。已知必需汽蚀余量$NPSH_r=3.5$m。判断是否发生汽蚀。

### 【解答】

**有效汽蚀余量**：

$$NPSH_a = \frac{p_a}{\rho g} - H_s - h_s - \frac{p_v}{\rho g}$$

$$= \frac{101.3 \times 10^3}{1000 \times 9.81} - 3 - 1.2 - \frac{2.34 \times 10^3}{1000 \times 9.81}$$

$$= 10.33 - 3 - 1.2 - 0.24 = 5.89 \text{ m}$$

**判别**：

$$NPSH_a = 5.89 \text{ m} > NPSH_r = 3.5 \text{ m}$$

**结论**：$NPSH_a > NPSH_r$，**不发生汽蚀**，安装高度安全。

## 题型78：允许安装高度

### 【题目】

已知$NPSH_r=4.0$m，$p_a=101.3$kPa，$p_v=2.34$kPa，吸水管损失$h_s=0.8$m，安全裕量取$0.5$m。求允许安装高度$[H_s]$。

### 【解答】

**允许安装高度**：

$$[H_s] = \frac{p_a - p_v}{\rho g} - NPSH_r - h_s - \Delta h_{安全}$$

$$= 10.33 - 0.24 - 4.0 - 0.8 - 0.5$$

$$= 4.79 \text{ m}$$

**结论**：允许安装高度$[H_s] = 4.79$m，实际安装高度应$\leq 4.79$m。

## 题型79：水泵站能量分析

### 【题目】

水泵站装机3台，单泵流量$Q_0=80$L/s，扬程$H_0=35$m，效率$\eta=75\%$。年运行8000小时，电价0.6元/kWh。求年运行费用（3台全开）。

### 【解答】

**(1) 单泵轴功率**

$$P_{轴} = \frac{\rho g Q H}{\eta} = \frac{1000 \times 9.81 \times 0.08 \times 35}{0.75} = 36.7 \text{ kW}$$

**(2) 总功率**

$$P_{总} = 3 \times 36.7 = 110.1 \text{ kW}$$

**(3) 年耗电量**

$$W = P_{总} \times t = 110.1 \times 8000 = 880800 \text{ kWh}$$

**(4) 年运行费用**

$$C = W \times 单价 = 880800 \times 0.6 = 528480 \text{ 元} \approx 52.8 \text{ 万元}$$

## 题型80：切割定律

### 【题目】

水泵叶轮直径$D_1=300$mm，性能：$Q_1=120$L/s，$H_1=28$m。切割至$D_2=280$mm，求新性能。

### 【解答】

**切割定律**（小幅切割，$\frac{D_2}{D_1} > 0.8$）：

$$\frac{Q_2}{Q_1} \approx \frac{D_2}{D_1}, \quad \frac{H_2}{H_1} \approx \left(\frac{D_2}{D_1}\right)^2$$

直径比：
$$\frac{D_2}{D_1} = \frac{280}{300} = 0.933$$

**(1) 流量**

$$Q_2 = Q_1 \times 0.933 = 120 \times 0.933 = 112.0 \text{ L/s}$$

**(2) 扬程**

$$H_2 = H_1 \times 0.933^2 = 28 \times 0.871 = 24.4 \text{ m}$$

**注**：切割后效率略有下降，功率按$P \propto QH$计算。

## Python实现

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import fsolve

class PumpAnalysis:
    """水泵分析"""
    
    def __init__(self, g=9.81, rho=1000):
        self.g = g
        self.rho = rho
    
    def effective_power(self, Q, H):
        """
        有效功率
        
        P_e = ρgQH
        
        参数:
            Q: 流量 (m³/s)
            H: 扬程 (m)
        """
        P_e = self.rho * self.g * Q * H
        return P_e
    
    def efficiency(self, Q, H, P_shaft):
        """效率"""
        P_e = self.effective_power(Q, H)
        eta = P_e / P_shaft
        return eta
    
    def similarity_law(self, n1, n2, Q1, H1, P1):
        """
        相似定律
        
        Q2/Q1 = n2/n1
        H2/H1 = (n2/n1)²
        P2/P1 = (n2/n1)³
        """
        ratio = n2 / n1
        
        Q2 = Q1 * ratio
        H2 = H1 * ratio**2
        P2 = P1 * ratio**3
        
        return Q2, H2, P2
    
    def specific_speed(self, n, Q, H):
        """
        比转速
        
        n_s = n√Q / H^(3/4)
        
        参数:
            n: 转速 (r/min)
            Q: 流量 (m³/s)
            H: 扬程 (m)
        """
        n_s = n * np.sqrt(Q) / H**0.75
        return n_s
    
    def pump_type_classification(self, n_s):
        """根据比转速判断水泵类型"""
        if n_s < 80:
            return "离心泵"
        elif n_s < 150:
            return "混流泵"
        else:
            return "轴流泵"
    
    def operating_point(self, pump_curve, pipe_curve):
        """
        工况点计算
        
        参数:
            pump_curve: 水泵特性曲线函数 H=f(Q)
            pipe_curve: 管路特性曲线函数 H=f(Q)
        
        返回:
            Q_op, H_op: 工况点流量和扬程
        """
        def equation(Q):
            return pump_curve(Q) - pipe_curve(Q)
        
        Q_op = fsolve(equation, 100)[0]  # 初始猜测100 L/s
        H_op = pump_curve(Q_op)
        
        return Q_op, H_op
    
    def npsh_available(self, p_a, p_v, H_s, h_s):
        """
        有效汽蚀余量
        
        NPSH_a = (p_a - p_v)/(ρg) - H_s - h_s
        
        参数:
            p_a: 大气压 (Pa)
            p_v: 饱和蒸汽压 (Pa)
            H_s: 安装高度 (m)
            h_s: 吸水管损失 (m)
        """
        NPSH_a = (p_a - p_v) / (self.rho * self.g) - H_s - h_s
        return NPSH_a
    
    def allowable_installation_height(self, p_a, p_v, NPSH_r, h_s, safety_margin=0.5):
        """允许安装高度"""
        H_s_max = (p_a - p_v) / (self.rho * self.g) - NPSH_r - h_s - safety_margin
        return H_s_max
    
    def plot_pump_analysis(self):
        """绘制水泵分析图"""
        fig, axes = plt.subplots(2, 2, figsize=(16, 12))
        
        # 1. 水泵特性曲线与工况点
        ax1 = axes[0, 0]
        
        Q_range = np.linspace(0, 200, 200)
        
        # 水泵曲线（理想抛物线）
        H_pump = 50 - 0.002 * Q_range**2
        
        # 管路曲线（不同静扬程）
        H_st_values = [20, 25, 30]
        colors = ['blue', 'green', 'red']
        
        ax1.plot(Q_range, H_pump, 'k-', linewidth=3, label='水泵特性曲线')
        
        for H_st, color in zip(H_st_values, colors):
            H_pipe = H_st + 0.001 * Q_range**2
            ax1.plot(Q_range, H_pipe, color=color, linewidth=2,
                    linestyle='--', label=f'管路曲线(H_st={H_st}m)')
            
            # 工况点
            Q_op, H_op = self.operating_point(
                lambda Q: 50 - 0.002*Q**2,
                lambda Q: H_st + 0.001*Q**2
            )
            ax1.scatter([Q_op], [H_op], s=150, color=color, zorder=5, marker='o',
                       edgecolor='black', linewidth=2)
        
        ax1.set_xlabel('流量 Q (L/s)', fontsize=11)
        ax1.set_ylabel('扬程 H (m)', fontsize=11)
        ax1.set_title('水泵与管路特性曲线及工况点', fontsize=13, fontweight='bold')
        ax1.legend(fontsize=9)
        ax1.grid(True, alpha=0.3)
        ax1.set_xlim([0, 200])
        ax1.set_ylim([0, 60])
        
        # 2. 相似定律
        ax2 = axes[0, 1]
        
        n1 = 1450  # r/min
        Q1 = 100  # L/s
        H1 = 30  # m
        P1 = 35  # kW
        
        n_range = np.linspace(1000, 2000, 100)
        
        Q_values = []
        H_values = []
        P_values = []
        
        for n in n_range:
            Q, H, P = self.similarity_law(n1, n, Q1, H1, P1)
            Q_values.append(Q)
            H_values.append(H)
            P_values.append(P)
        
        ax2_twin1 = ax2.twinx()
        ax2_twin2 = ax2.twinx()
        ax2_twin2.spines['right'].set_position(('outward', 60))
        
        line1 = ax2.plot(n_range, Q_values, 'b-', linewidth=2.5, label='流量Q')
        line2 = ax2_twin1.plot(n_range, H_values, 'g-', linewidth=2.5, label='扬程H')
        line3 = ax2_twin2.plot(n_range, P_values, 'r-', linewidth=2.5, label='功率P')
        
        ax2.set_xlabel('转速 n (r/min)', fontsize=11)
        ax2.set_ylabel('流量 Q (L/s)', fontsize=11, color='b')
        ax2_twin1.set_ylabel('扬程 H (m)', fontsize=11, color='g')
        ax2_twin2.set_ylabel('功率 P (kW)', fontsize=11, color='r')
        ax2.set_title('相似定律：转速影响', fontsize=13, fontweight='bold')
        
        lines = line1 + line2 + line3
        labels = [l.get_label() for l in lines]
        ax2.legend(lines, labels, loc='upper left', fontsize=9)
        ax2.grid(True, alpha=0.3)
        
        # 3. 汽蚀余量
        ax3 = axes[1, 0]
        
        p_a = 101.3e3  # Pa
        p_v = 2.34e3  # Pa (20°C)
        h_s = 1.0  # m
        NPSH_r = 3.5  # m
        
        H_s_range = np.linspace(0, 8, 100)
        NPSH_a_values = [self.npsh_available(p_a, p_v, H_s, h_s) for H_s in H_s_range]
        
        ax3.plot(H_s_range, NPSH_a_values, 'b-', linewidth=2.5, label='NPSH_a(有效汽蚀余量)')
        ax3.axhline(NPSH_r, color='r', linestyle='--', linewidth=2,
                   label=f'NPSH_r={NPSH_r}m(必需汽蚀余量)')
        
        # 填充安全区和危险区
        ax3.fill_between(H_s_range, 0, NPSH_a_values,
                        where=(np.array(NPSH_a_values) > NPSH_r),
                        alpha=0.2, color='green', label='安全区')
        ax3.fill_between(H_s_range, 0, NPSH_a_values,
                        where=(np.array(NPSH_a_values) <= NPSH_r),
                        alpha=0.2, color='red', label='汽蚀危险区')
        
        # 标注允许安装高度
        H_s_max = self.allowable_installation_height(p_a, p_v, NPSH_r, h_s)
        ax3.axvline(H_s_max, color='orange', linestyle=':', linewidth=2,
                   label=f'[H_s]={H_s_max:.2f}m')
        
        ax3.set_xlabel('安装高度 H_s (m)', fontsize=11)
        ax3.set_ylabel('汽蚀余量 NPSH (m)', fontsize=11)
        ax3.set_title('汽蚀余量与安装高度', fontsize=13, fontweight='bold')
        ax3.legend(fontsize=9)
        ax3.grid(True, alpha=0.3)
        ax3.set_ylim([0, 12])
        
        # 4. 比转速与水泵类型
        ax4 = axes[1, 1]
        
        n_pump = 1450  # r/min
        Q_range_ns = np.linspace(0.01, 0.5, 100)
        
        H_values_ns = [20, 30, 40]
        colors_ns = ['blue', 'green', 'red']
        
        for H_ns, color in zip(H_values_ns, colors_ns):
            n_s_values = [self.specific_speed(n_pump, Q, H_ns) for Q in Q_range_ns]
            ax4.plot(Q_range_ns*1000, n_s_values, color=color, linewidth=2.5,
                    label=f'H={H_ns}m')
        
        # 分区标注
        ax4.axhspan(0, 80, alpha=0.2, color='lightblue', label='离心泵')
        ax4.axhspan(80, 150, alpha=0.2, color='lightgreen', label='混流泵')
        ax4.axhspan(150, 300, alpha=0.2, color='lightyellow', label='轴流泵')
        
        ax4.set_xlabel('流量 Q (L/s)', fontsize=11)
        ax4.set_ylabel('比转速 n_s', fontsize=11)
        ax4.set_title(f'比转速与水泵类型分类(n={n_pump}r/min)', fontsize=13, fontweight='bold')
        ax4.legend(fontsize=9)
        ax4.grid(True, alpha=0.3)
        ax4.set_ylim([0, 300])
        
        plt.tight_layout()
        plt.savefig('pump_analysis.png', dpi=300)
        plt.show()

# 实例化
```
