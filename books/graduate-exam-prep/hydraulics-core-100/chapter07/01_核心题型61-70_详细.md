## 题型61：Darcy定律基础应用

### 【题目】

砂土试样，长$L=30$cm，截面积$A=50$cm²，上下游水头差$\Delta H=15$cm，渗流量$Q=0.5$L/min。求渗透系数$K$。

### 【解答】

**Darcy定律**：

$$Q = KiA$$

其中水力梯度：
$$i = \frac{\Delta H}{L} = \frac{15}{30} = 0.5$$

单位转换：
- $Q = 0.5$L/min $= 0.5 \times 10^{-3}$m³/min $= 8.33 \times 10^{-6}$m³/s
- $A = 50$cm² $= 50 \times 10^{-4}$m² $= 0.005$m²

$$K = \frac{Q}{iA} = \frac{8.33 \times 10^{-6}}{0.5 \times 0.005} = 3.33 \times 10^{-3} \text{ m/s}$$

$$K = 3.33 \text{ mm/s} = 0.288 \text{ m/d}$$

## 题型62：渗透系数经验估算

### 【题目】

砂土有效粒径$d_{10}=0.3$mm，孔隙比$e=0.65$。用Hazen公式估算渗透系数。

### 【解答】

**Hazen经验公式**（细砂、粗砂）：

$$K = C \cdot d_{10}^2 \quad (\text{cm/s})$$

其中$C = 100 \sim 150$（常取$C = 100$）

$$K = 100 \times (0.03)^2 = 0.09 \text{ cm/s} = 9 \times 10^{-4} \text{ m/s}$$

或使用含孔隙比的公式：

$$K = C \frac{e^3}{1+e} d_{10}^2$$

$$K = 100 \times \frac{0.65^3}{1+0.65} \times 0.03^2 = 0.066 \text{ cm/s}$$

## 题型63：分层土渗流

### 【题目】

两层土坝，第1层：$H_1=5$m，$K_1=2 \times 10^{-4}$m/s；第2层：$H_2=3$m，$K_2=5 \times 10^{-5}$m/s。垂直渗流，总水头差$\Delta H=4$m。求：
(1) 各层水头损失
(2) 等效渗透系数

### 【解答】

**(1) 各层水头损失**

**串联渗流**：渗流速度相等

$$v = K_1 i_1 = K_2 i_2 = K_{等} i_{总}$$

$$i_1 = \frac{\Delta H_1}{H_1}, \quad i_2 = \frac{\Delta H_2}{H_2}$$

$$K_1 \frac{\Delta H_1}{H_1} = K_2 \frac{\Delta H_2}{H_2}$$

且：$\Delta H_1 + \Delta H_2 = 4$m

$$\frac{\Delta H_1}{H_1 / K_1} = \frac{\Delta H_2}{H_2 / K_2}$$

$$\Delta H_1 = \frac{H_1 / K_1}{H_1 / K_1 + H_2 / K_2} \times \Delta H$$

$$= \frac{5 / (2 \times 10^{-4})}{5 / (2 \times 10^{-4}) + 3 / (5 \times 10^{-5})} \times 4$$

$$= \frac{25000}{25000 + 60000} \times 4 = 1.18 \text{ m}$$

$$\Delta H_2 = 4 - 1.18 = 2.82 \text{ m}$$

**(2) 等效渗透系数**

$$K_{等} = \frac{H_1 + H_2}{\frac{H_1}{K_1} + \frac{H_2}{K_2}}$$

$$= \frac{8}{\frac{5}{2 \times 10^{-4}} + \frac{3}{5 \times 10^{-5}}} = \frac{8}{85000} = 9.41 \times 10^{-5} \text{ m/s}$$

## 题型64：承压水井流（Thiem公式）

### 【题目】

承压含水层厚度$M=20$m，渗透系数$K=15$m/d。两观测孔：$r_1=10$m，$s_1=2.5$m；$r_2=50$m，$s_2=1.2$m。求抽水流量$Q$。

### 【解答】

**Thiem公式**（稳定井流）：

$$Q = \frac{2\pi KM(s_1 - s_2)}{\ln(r_2 / r_1)}$$

代入数据：

$$Q = \frac{2\pi \times 15 \times 20 \times (2.5 - 1.2)}{\ln(50 / 10)}$$

$$= \frac{1884.96 \times 1.3}{1.609} = 1522 \text{ m}^3\text{/d}$$

$$Q = 1522 / 86400 = 0.0176 \text{ m}^3\text{/s} = 17.6 \text{ L/s}$$

## 题型65：潜水井流（Dupuit公式）

### 【题目】

潜水含水层，$K=10$m/d，影响半径$R=100$m，井半径$r_w=0.2$m，初始潜水面$H=15$m，井内水位$h_w=10$m。求出水量。

### 【解答】

**Dupuit公式**：

$$Q = \frac{\pi K(H^2 - h_w^2)}{\ln(R / r_w)}$$

$$Q = \frac{\pi \times 10 \times (15^2 - 10^2)}{\ln(100 / 0.2)}$$

$$= \frac{31.416 \times 125}{6.215} = 632 \text{ m}^3\text{/d}$$

$$Q = 632 / 86400 = 0.0073 \text{ m}^3\text{/s} = 7.3 \text{ L/s}$$

## 题型66：渗流量计算

### 【题目】

矩形堤坝，底宽$b=20$m，顶宽$b_0=5$m，坝高$H=10$m，上游水深$h_1=8$m，下游水深$h_2=2$m。坝体渗透系数$K=2 \times 10^{-5}$m/s。估算单宽渗流量（简化为一维）。

### 【解答】

**近似一维渗流**：

平均渗径：$L \approx \frac{b + b_0}{2} = 12.5$m

平均水头差：$\Delta H = h_1 - h_2 = 6$m

水力梯度：$i = \frac{\Delta H}{L} = \frac{6}{12.5} = 0.48$

平均过水断面：$A = H \times 1 = 10$m²（单宽）

单宽渗流量：
$$q = Ki \times H = 2 \times 10^{-5} \times 0.48 \times 10 = 9.6 \times 10^{-5} \text{ m}^2\text{/s}$$

$$q = 8.29 \text{ m}^3\text{/(d·m)}$$

## 题型67：渗流破坏判断

### 【题目】

砂土堤坝出逸点水力梯度$i_{出}=0.85$，砂土临界水力梯度$i_{cr}=0.90$。判断是否发生流土破坏，并计算安全系数。

### 【解答】

**临界水力梯度**：

$$i_{cr} = \frac{\gamma' }{\gamma_w} = \frac{(\gamma_s - \gamma_w)}{\gamma_w} \approx 1.0$$

（砂土一般$i_{cr} = 0.9 \sim 1.1$）

**流土判别**：

$$K_{流土} = \frac{i_{cr}}{i_{出}} = \frac{0.90}{0.85} = 1.06$$

**判断**：$K > 1.0$，**不发生流土**，但接近临界状态。

**工程要求**：$K \geq 1.5 \sim 2.0$

**结论**：安全系数偏低，需采取防渗措施。

## 题型68：管涌判别

### 【题目】

堤基砂层不均匀系数$C_u=8$，曲率系数$C_c=1.2$，粒径$d_{60}=0.5$mm，$d_{30}=0.25$mm，$d_{10}=0.06$mm。判断是否易发生管涌。

### 【解答】

**管涌判别准则**：

**(1) 不均匀系数**：

$$C_u = \frac{d_{60}}{d_{10}} = \frac{0.5}{0.06} = 8.33$$

$C_u > 5$，颗粒级配不良，细颗粒易流失。

**(2) 曲率系数**：

$$C_c = \frac{d_{30}^2}{d_{10} \times d_{60}} = \frac{0.25^2}{0.06 \times 0.5} = 2.08$$

$C_c$应在1~3之间，$C_c = 2.08$，级配连续性一般。

**(3) 综合判断**：

- $C_u > 10$且$d_5 < 0.075$mm：易管涌
- $5 < C_u < 10$：需进一步分析

**结论**：$C_u = 8.33$，处于临界状态，建议采取反滤措施。

## 题型69：井间干扰

### 【题目】

两口井距离$d=30$m，同时抽水，各井单独抽水时降深$s_1 = s_2 = 2.0$m。考虑干扰后，某点（距井1为$r_1=10$m，距井2为$r_2=25$m）的总降深。

### 【解答】

**叠加原理**：

$$s_{总} = s_{井1} + s_{井2}$$

假设各井降深沿对数分布：

$$s = s_0 \frac{\ln(R/r)}{\ln(R/r_w)}$$

**简化**：若各井单独作用在该点的降深为$s_1'$和$s_2'$，则：

$$s_{总} = s_1' + s_2'$$

## 题型70：渗流网格法

### 【题目】

简述渗流网格法的基本原理和绘制方法。

### 【解答】

**基本原理**：

1. **流线**：与流速方向相切的曲线
2. **等势线**：水头相等的点连线
3. **流网**：流线与等势线正交构成的网格

**绘制方法**：

1. 确定边界条件（不透水边界为流线，自由水面为等势线）
2. 等间距划分势降
3. 绘制流线，与等势线正交
4. 调整使网格近似为正方形

**渗流量计算**：

$$Q = K \Delta H \frac{n_f}{n_d}$$

其中：
- $n_f$：流槽数
- $n_d$：势降格数
- $\Delta H$：总水头差

**示例**：若$n_f=4$，$n_d=10$，$\Delta H=5$m，$K=10^{-4}$m/s

$$Q = 10^{-4} \times 5 \times \frac{4}{10} = 2 \times 10^{-4} \text{ m}^2\text{/s}$$

## Python实现

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import fsolve

class SeepageFlow:
    """渗流计算"""
    
    def __init__(self):
        self.g = 9.81
        self.gamma_w = 9.81  # kN/m³
    
    def darcy_law(self, K, i, A):
        """Darcy定律：Q = KiA"""
        Q = K * i * A
        return Q
    
    def permeability_hazen(self, d10, C=100):
        """
        Hazen公式估算渗透系数
        
        K = C * d10² (cm/s)
        
        参数:
            d10: 有效粒径 (cm)
            C: 系数（100~150）
        """
        K_cm_s = C * d10**2
        K_m_s = K_cm_s / 100
        return K_m_s
    
    def layered_soil_equivalent_K(self, K_list, H_list, direction='vertical'):
        """
        分层土等效渗透系数
        
        参数:
            K_list: 各层渗透系数列表
            H_list: 各层厚度列表
            direction: 'vertical'垂直流, 'horizontal'水平流
        """
        if direction == 'vertical':
            # 串联：K_eq = ΣH / Σ(H/K)
            H_total = sum(H_list)
            K_eq = H_total / sum(H / K for H, K in zip(H_list, K_list))
        else:  # horizontal
            # 并联：K_eq = Σ(KH) / ΣH
            K_eq = sum(K * H for K, H in zip(K_list, H_list)) / sum(H_list)
        
        return K_eq
    
    def thiem_formula(self, K, M, r1, r2, s1, s2):
        """
        Thiem公式（承压井）
        
        Q = 2πKM(s1-s2) / ln(r2/r1)
        """
        Q = 2 * np.pi * K * M * (s1 - s2) / np.log(r2 / r1)
        return Q
    
    def dupuit_formula(self, K, R, rw, H, hw):
        """
        Dupuit公式（潜水井）
        
        Q = πK(H²-hw²) / ln(R/rw)
        """
        Q = np.pi * K * (H**2 - hw**2) / np.log(R / rw)
        return Q
    
    def critical_hydraulic_gradient(self, gamma_s, gamma_w=9.81):
        """
        临界水力梯度
        
        i_cr = (γs - γw) / γw
        """
        i_cr = (gamma_s - gamma_w) / gamma_w
        return i_cr
    
    def piping_safety_factor(self, i_cr, i_exit):
        """流土安全系数"""
        K_piping = i_cr / i_exit
        return K_piping
    
    def plot_seepage_analysis(self):
        """绘制渗流分析图"""
        fig, axes = plt.subplots(2, 2, figsize=(16, 12))
        
        # 1. Darcy定律验证
        ax1 = axes[0, 0]
        
        K = 1e-3  # m/s
        A = 0.01  # m²
        
        i_range = np.linspace(0, 1, 100)
        Q_values = [self.darcy_law(K, i, A) * 1000 for i in i_range]  # L/s
        
        ax1.plot(i_range, Q_values, 'b-', linewidth=2.5)
        
        ax1.set_xlabel('水力梯度 i', fontsize=11)
        ax1.set_ylabel('流量 Q (L/s)', fontsize=11)
        ax1.set_title(f'Darcy定律：Q = KiA\nK={K}m/s, A={A}m²',
                     fontsize=13, fontweight='bold')
        ax1.grid(True, alpha=0.3)
        
        # 2. 分层土等效渗透系数
        ax2 = axes[0, 1]
        
        # 两层土
        K1 = 1e-3  # m/s
        H1_range = np.linspace(1, 10, 100)
        K2 = 1e-4
        H2 = 5
        
        K_eq_vertical = []
        K_eq_horizontal = []
        
        for H1 in H1_range:
            K_v = self.layered_soil_equivalent_K([K1, K2], [H1, H2], 'vertical')
            K_h = self.layered_soil_equivalent_K([K1, K2], [H1, H2], 'horizontal')
            K_eq_vertical.append(K_v * 1e3)  # mm/s
            K_eq_horizontal.append(K_h * 1e3)
        
        ax2.plot(H1_range, K_eq_vertical, 'b-', linewidth=2.5, label='垂直渗流')
        ax2.plot(H1_range, K_eq_horizontal, 'r-', linewidth=2.5, label='水平渗流')
        
        ax2.set_xlabel('第1层厚度 H₁ (m)', fontsize=11)
        ax2.set_ylabel('等效渗透系数 K_eq (mm/s)', fontsize=11)
        ax2.set_title(f'分层土等效渗透系数\nK₁={K1}m/s, K₂={K2}m/s, H₂={H2}m',
                     fontsize=13, fontweight='bold')
        ax2.legend()
        ax2.grid(True, alpha=0.3)
        
        # 3. 承压井vs潜水井
        ax3 = axes[1, 0]
        
        K_well = 10  # m/d
        M = 20  # m (承压)
        H = 20  # m (潜水)
        rw = 0.2  # m
        
        R_range = np.linspace(10, 200, 100)
        
        # 承压井（固定降深）
        s_confined = 5  # m
        Q_confined = []
        
        for R in R_range:
            # 简化：假设单井，用修正公式
            Q_c = 2 * np.pi * K_well * M * s_confined / np.log(R / rw)
            Q_confined.append(Q_c)
        
        # 潜水井
        hw = H - 5  # 井内水位
        Q_unconfined = []
        
        for R in R_range:
            Q_u = self.dupuit_formula(K_well, R, rw, H, hw)
            Q_unconfined.append(Q_u)
        
        ax3.plot(R_range, Q_confined, 'b-', linewidth=2.5, label='承压井')
        ax3.plot(R_range, Q_unconfined, 'r-', linewidth=2.5, label='潜水井')
        
        ax3.set_xlabel('影响半径 R (m)', fontsize=11)
        ax3.set_ylabel('出水量 Q (m³/d)', fontsize=11)
        ax3.set_title(f'承压井vs潜水井出水量\nK={K_well}m/d, rw={rw}m',
                     fontsize=13, fontweight='bold')
        ax3.legend()
        ax3.grid(True, alpha=0.3)
        
        # 4. 流土安全系数
        ax4 = axes[1, 1]
        
        i_cr = 0.95  # 临界水力梯度
        i_exit_range = np.linspace(0.1, 1.2, 100)
        
        K_piping_values = [self.piping_safety_factor(i_cr, i) for i in i_exit_range]
        
        ax4.plot(i_exit_range, K_piping_values, 'b-', linewidth=2.5)
        
        # 安全线
        ax4.axhline(1.0, color='r', linestyle='--', linewidth=2, label='K=1.0（临界）')
        ax4.axhline(1.5, color='orange', linestyle='--', linewidth=2, label='K=1.5（安全下限）')
        ax4.axhline(2.0, color='g', linestyle='--', linewidth=2, label='K=2.0（安全）')
        
        # 填充危险区
        ax4.fill_between(i_exit_range, 0, K_piping_values,
                        where=(np.array(K_piping_values) < 1.0),
                        alpha=0.3, color='red', label='流土危险区')
        
        ax4.set_xlabel('出逸点水力梯度 i_exit', fontsize=11)
        ax4.set_ylabel('流土安全系数 K', fontsize=11)
        ax4.set_title(f'流土安全系数分析\ni_cr={i_cr}', fontsize=13, fontweight='bold')
        ax4.legend(loc='upper right', fontsize=9)
        ax4.grid(True, alpha=0.3)
        ax4.set_ylim([0, 5])
        
        plt.tight_layout()
        plt.savefig('seepage_flow_analysis.png', dpi=300)
        plt.show()

# 实例化
```
