## 题1：压强计算

**【题目】**（10分）

水箱水深h=3m，大气压强p₀=101.3kPa，求：
1. 箱底绝对压强
2. 箱底表压

**【解答】**

**给分点**：
1. 公式 p = p₀ + γh（2分）
2. 绝对压强（3分）：
   $$p_{abs} = 101.3 + 9.81 \times 3 = 130.7 \text{ kPa}$$
3. 表压（3分）：
   $$p = \gamma h = 9.81 \times 3 = 29.43 \text{ kPa}$$
4. 单位正确（2分）

**常见错误**：
- 混淆绝对压强与表压
- 单位换算错误

## 题2：U型管压差计

**【题目】**（12分）

U型管压差计测量管道压差，测压液为水银（γ=133.4kN/m³），读数h=0.15m，管内为水，求压差Δp

**【解答】**

**思路**：等压面法

取U型管底部为等压面：
$$p_A + \gamma_{水} h_A = p_B + \gamma_{水} h_B + \gamma_{Hg} h$$

简化：
$$\Delta p = p_A - p_B = (\gamma_{Hg} - \gamma_{水}) h$$

**计算**（6分）：
$$\Delta p = (133.4 - 9.81) \times 0.15 = 18.54 \text{ kPa}$$

**Python实现**：

```python
class PressureMeasurement:
    """压强测量"""
    
    @staticmethod
    def u_tube_manometer(h, gamma_measuring, gamma_fluid=9810):
        """
        U型管压差计
        
        Δp = (γ_测 - γ_流) · h
        
        参数:
            h: 高差 (m)
            gamma_measuring: 测压液容重 (N/m³)
            gamma_fluid: 被测液体容重 (N/m³)
        """
        delta_p = (gamma_measuring - gamma_fluid) * h
        return delta_p

# 示例
gamma_Hg = 133400  # N/m³
h = 0.15  # m

delta_p = PressureMeasurement.u_tube_manometer(h, gamma_Hg)
print(f"压差: {delta_p/1000:.2f} kPa")
```

## 题3：平面闸门总压力

**【题目】**（15分）

矩形平面闸门，宽b=2m，高h=3m，顶部与水面齐平，求：
1. 闸门上总压力
2. 压力中心位置

**【解答】**

**1. 总压力**（7分）：

压强分布：$p = \gamma z$

$$P = \int_0^h \gamma z \cdot b \, dz = \gamma b \frac{h^2}{2}$$

代入：
$$P = 9810 \times 2 \times \frac{3^2}{2} = 88290 \text{ N} = 88.29 \text{ kN}$$

**2. 压力中心**（8分）：

$$z_D = \frac{2h}{3} = \frac{2 \times 3}{3} = 2 \text{ m}$$

**公式推导**：
$$z_D = \frac{\int_0^h z \cdot p \cdot b \, dz}{\int_0^h p \cdot b \, dz} = \frac{2h}{3}$$

## 题4：压强分布图

**【题目】**（8分）

绘制水深5m的矩形水箱侧壁压强分布图

**【解答】**

```python
import matplotlib.pyplot as plt
import numpy as np

# 参数
H = 5  # m
gamma = 9810  # N/m³

# 深度
z = np.linspace(0, H, 100)

# 压强
p = gamma * z

# 绘图
fig, ax = plt.subplots(figsize=(8, 6))

ax.plot(p/1000, z, 'b-', linewidth=2)
ax.fill_betweenx(z, 0, p/1000, alpha=0.3)
ax.invert_yaxis()

ax.set_xlabel('压强 (kPa)', fontsize=12)
ax.set_ylabel('深度 (m)', fontsize=12)
ax.set_title('静水压强分布', fontsize=14)
ax.grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('pressure_distribution_gate.png', dpi=300)
plt.show()

print(f"水面压强: 0 kPa")
print(f"底部压强: {gamma*H/1000:.1f} kPa")
```

## 题5：倾斜平面总压力

**【题目】**（18分）

倾角α=60°的矩形闸门，宽b=1.5m，长L=2m，顶部距水面h₀=1m，求总压力及作用点

**【解答】**

**1. 形心深度**（4分）：
$$h_C = h_0 + \frac{L \sin\alpha}{2} = 1 + \frac{2 \times \sin60°}{2} = 1.866 \text{ m}$$

**2. 总压力**（6分）：
$$P = \gamma h_C A = 9810 \times 1.866 \times (1.5 \times 2) = 54953 \text{ N}$$

**3. 压力中心**（8分）：

沿斜面距顶部：
$$y_D = y_C + \frac{I_C}{y_C A}$$

其中：
- $y_C = h_0/\sin\alpha + L/2 = 1.577$ m
- $I_C = \frac{bL^3}{12} = \frac{1.5 \times 2^3}{12} = 1$ m⁴
- $A = 1.5 \times 2 = 3$ m²

$$y_D = 1.577 + \frac{1}{1.577 \times 3} = 1.788 \text{ m}$$
