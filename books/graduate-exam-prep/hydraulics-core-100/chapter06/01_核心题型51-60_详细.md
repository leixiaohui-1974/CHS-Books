## 题型51：水跃共轭水深计算

### 【题目】

矩形水平渠道，$b=3$m，跃前水深$h_1=0.5$m，流速$v_1=8$m/s。求：
(1) 跃后水深$h_2$
(2) 水跃能量损失$\Delta E$
(3) 水跃长度$L_j$（取$L_j = 6h_2$）

### 【解答】

**(1) 跃后水深（共轭水深公式）**

$$\frac{h_2}{h_1} = \frac{1}{2}\left(\sqrt{1 + 8Fr_1^2} - 1\right)$$

计算跃前弗劳德数：
$$Fr_1 = \frac{v_1}{\sqrt{gh_1}} = \frac{8}{\sqrt{9.81 \times 0.5}} = \frac{8}{2.21} = 3.62$$

$$\frac{h_2}{h_1} = \frac{1}{2}\left(\sqrt{1 + 8 \times 3.62^2} - 1\right) = \frac{1}{2}(\sqrt{105.4} - 1) = \frac{1}{2}(10.27 - 1) = 4.63$$

$$h_2 = 4.63 \times 0.5 = 2.32 \text{ m}$$

**(2) 能量损失**

$$\Delta E = h_1 + \frac{v_1^2}{2g} - h_2 - \frac{v_2^2}{2g}$$

计算跃后流速：
$$v_2 = v_1 \frac{h_1}{h_2} = 8 \times \frac{0.5}{2.32} = 1.72 \text{ m/s}$$

$$E_1 = 0.5 + \frac{64}{2 \times 9.81} = 0.5 + 3.26 = 3.76 \text{ m}$$

$$E_2 = 2.32 + \frac{1.72^2}{2 \times 9.81} = 2.32 + 0.15 = 2.47 \text{ m}$$

$$\Delta E = 3.76 - 2.47 = 1.29 \text{ m}$$

相对能量损失：
$$\eta = \frac{\Delta E}{E_1} \times 100\% = \frac{1.29}{3.76} \times 100\% = 34.3\%$$

**(3) 水跃长度**

$$L_j = 6h_2 = 6 \times 2.32 = 13.9 \text{ m}$$

## 题型52：水跃类型判断

### 【题目】

矩形渠道水跃，判断以下水跃类型：
(1) $Fr_1 = 1.5$
(2) $Fr_1 = 3.0$
(3) $Fr_1 = 6.0$
(4) $Fr_1 = 10.0$

### 【解答】

**水跃分类标准**：

| $Fr_1$范围 | 水跃类型 | 特征 |
|-----------|---------|------|
| 1.0 ~ 1.7 | 波状水跃 | 水面起伏，能量损失<5% |
| 1.7 ~ 2.5 | 弱水跃 | 能量损失5%~15% |
| 2.5 ~ 4.5 | 稳定水跃 | 脉动小，能量损失15%~45% |
| 4.5 ~ 9.0 | 强水跃 | 脉动大，能量损失45%~70% |
| > 9.0 | 剧烈水跃 | 间歇性，能量损失>70% |

**(1) $Fr_1 = 1.5$**：波状水跃

**(2) $Fr_1 = 3.0$**：稳定水跃

**(3) $Fr_1 = 6.0$**：强水跃

**(4) $Fr_1 = 10.0$**：剧烈水跃

## 题型53：消能计算

### 【题目】

水闸下游发生水跃消能，闸下出流$Q=20$m³/s，收缩断面水深$h_c=0.8$m，渠宽$b=4$m。下游水深$h_2=3.0$m。求：
(1) 判断是否发生完全水跃
(2) 消能率

### 【解答】

**(1) 判断完全水跃**

收缩断面流速：
$$v_c = \frac{Q}{bh_c} = \frac{20}{4 \times 0.8} = 6.25 \text{ m/s}$$

$$Fr_c = \frac{v_c}{\sqrt{gh_c}} = \frac{6.25}{\sqrt{9.81 \times 0.8}} = 2.23$$

共轭水深：
$$\frac{h'_2}{h_c} = \frac{1}{2}(\sqrt{1 + 8Fr_c^2} - 1) = \frac{1}{2}(\sqrt{40.8} - 1) = 2.69$$

$$h'_2 = 2.69 \times 0.8 = 2.15 \text{ m}$$

比较：$h_2 = 3.0 > h'_2 = 2.15$m

**淹没水跃**（$\sigma = \frac{h_2 - h'_2}{h'_2} = \frac{3.0 - 2.15}{2.15} = 0.395 = 39.5\%$）

**(2) 消能率**

$$E_c = h_c + \frac{v_c^2}{2g} = 0.8 + \frac{6.25^2}{19.62} = 0.8 + 1.99 = 2.79 \text{ m}$$

$$v_2 = \frac{Q}{bh_2} = \frac{20}{4 \times 3.0} = 1.67 \text{ m/s}$$

$$E_2 = h_2 + \frac{v_2^2}{2g} = 3.0 + \frac{1.67^2}{19.62} = 3.0 + 0.14 = 3.14 \text{ m}$$

能量损失（考虑淹没）：
$$\Delta E \approx E_c - E_2 = 2.79 - 3.14 = -0.35 \text{ m}$$

**注**：出现负值说明下游水位过高，部分回流，实际消能效果降低。

消能率：
$$\eta = \frac{E_c - E_{末}}{E_c} \times 100\%$$

对于淹没水跃，消能效果显著降低。

## 题型54：远驱水跃

### 【题目】

陡坡末端$h_1=0.4$m，$v_1=10$m/s，平坡上共轭水深$h'_2=2.5$m，实际下游水深$h_2=2.0$m。判断水跃位置。

### 【解答】

计算$Fr_1$：
$$Fr_1 = \frac{10}{\sqrt{9.81 \times 0.4}} = 5.05$$

计算理论共轭水深：
$$\frac{h'_2}{h_1} = \frac{1}{2}(\sqrt{1 + 8 \times 5.05^2} - 1) = 7.24$$

$$h'_2 = 7.24 \times 0.4 = 2.90 \text{ m}$$

**对比**：
- 题目给定$h'_2 = 2.5$m（理论）
- 实际$h_2 = 2.0$m < $h'_2$

**判断**：$h_2 < h'_2$，**远驱水跃**（水跃向下游推移）

**物理意义**：下游水深不足，水跃无法在该位置稳定形成，向下游移动。

## 题型55：淹没水跃

### 【题目】

水跃共轭水深$h'_2=2.0$m，下游实际水深$h_2=2.4$m。求淹没度$\sigma$。

### 【解答】

**淹没度定义**：

$$\sigma = \frac{h_2 - h'_2}{h'_2}$$

$$\sigma = \frac{2.4 - 2.0}{2.0} = 0.20 = 20\%$$

**判断标准**：
- $\sigma < 0.05$：完全水跃
- $0.05 < \sigma < 0.10$：轻微淹没
- $0.10 < \sigma < 0.20$：中度淹没
- $\sigma > 0.20$：严重淹没

**结论**：$\sigma = 20\%$，**严重淹没**，消能效果降低。

## 题型56：水跃长度估算

### 【题目】

水跃跃后水深$h_2=3.5$m，估算水跃长度$L_j$。

### 【解答】

**经验公式**：

$$L_j = (5 \sim 7)h_2$$

常用：$L_j = 6h_2$

$$L_j = 6 \times 3.5 = 21 \text{ m}$$

**注**：
- 稳定水跃（$Fr_1 = 2.5 \sim 4.5$）：$L_j \approx 6h_2$
- 强水跃（$Fr_1 = 4.5 \sim 9.0$）：$L_j \approx 7h_2$

## 题型57：水跃方程推导

### 【题目】

推导矩形水平渠道水跃共轭水深公式。

### 【解答】

**动量方程**（水平渠道，忽略摩阻）：

$$\rho Q(v_2 - v_1) = P_1 - P_2$$

**静水压力**：

$$P_1 = \frac{1}{2}\rho g b h_1^2, \quad P_2 = \frac{1}{2}\rho g b h_2^2$$

**连续方程**：

$$Q = v_1 h_1 b = v_2 h_2 b$$

$$v_2 = v_1 \frac{h_1}{h_2}$$

代入动量方程：

$$\rho Q \left(v_1 \frac{h_1}{h_2} - v_1\right) = \frac{1}{2}\rho g b (h_1^2 - h_2^2)$$

$$Q v_1 \left(\frac{h_1 - h_2}{h_2}\right) = \frac{1}{2}g b (h_1^2 - h_2^2)$$

代入$Q = v_1 h_1 b$：

$$v_1^2 h_1 b \frac{h_1 - h_2}{h_2} = \frac{1}{2}g b (h_1 + h_2)(h_1 - h_2)$$

消去$(h_1 - h_2)b$：

$$\frac{v_1^2 h_1}{h_2} = \frac{1}{2}g(h_1 + h_2)$$

$$2v_1^2 h_1 = gh_2(h_1 + h_2)$$

$$2Fr_1^2 h_1 = h_2(h_1 + h_2)$$（其中$Fr_1^2 = \frac{v_1^2}{gh_1}$）

$$h_2^2 + h_1 h_2 - 2Fr_1^2 h_1^2 = 0$$

解二次方程：

$$h_2 = \frac{-h_1 + \sqrt{h_1^2 + 8Fr_1^2 h_1^2}}{2} = \frac{h_1}{2}(\sqrt{1 + 8Fr_1^2} - 1)$$

## 题型58：消能池设计

### 【题目】

泄水建筑物下游消能池，设计流量$Q=50$m³/s，池宽$b=10$m，收缩断面水深$h_c=1.0$m，下游水深$h_{下}=4.0$m。设计消能池深度。

### 【解答】

**(1) 计算共轭水深**

$$v_c = \frac{50}{10 \times 1.0} = 5.0 \text{ m/s}$$

$$Fr_c = \frac{5.0}{\sqrt{9.81 \times 1.0}} = 1.60$$

$$\frac{h'_2}{h_c} = \frac{1}{2}(\sqrt{1 + 8 \times 1.60^2} - 1) = 1.66$$

$$h'_2 = 1.66 \times 1.0 = 1.66 \text{ m}$$

**(2) 判断淹没**

$$\sigma = \frac{h_{下} - h'_2}{h'_2} = \frac{4.0 - 1.66}{1.66} = 1.41 = 141\%$$

严重淹没，需要降低池底。

**(3) 消能池深度**

设池底降深$\Delta z$，则池内水深$h_{池} = h_{下} + \Delta z$

要求：$h_{池} \approx h'_2$（完全水跃）

$$4.0 + \Delta z \approx 1.66$$

不合理！应提高收缩断面或采用其他措施。

**修正**：提高Fr1，或采用辅助消能工（如消力墩）。

## 题型59：斜坡水跃

### 【题目】

斜坡$i=0.02$上发生水跃，$h_1=0.6$m，$v_1=7$m/s，$b=4$m。考虑底坡影响，求跃后水深。

### 【解答】

**斜坡水跃修正**（近似）：

底坡影响较小时，仍可用水平水跃公式：

$$Fr_1 = \frac{7}{\sqrt{9.81 \times 0.6}} = 2.88$$

$$\frac{h_2}{h_1} = \frac{1}{2}(\sqrt{1 + 8 \times 2.88^2} - 1) = 3.71$$

$$h_2 = 3.71 \times 0.6 = 2.23 \text{ m}$$

**精确计算**需考虑重力分量：

$$\rho Q(v_2 - v_1) = P_1 - P_2 + G\sin\theta$$

（工程中底坡$i < 0.05$时，可忽略）

## 题型60：水跃综合应用

### 【题目】

矩形渠道，$b=5$m，$Q=25$m³/s。陡坡末端$h_1=0.8$m，缓坡正常水深$h_0=2.5$m。判断水跃位置并计算水跃参数。

### 【解答】

**(1) 计算Fr1**

$$v_1 = \frac{25}{5 \times 0.8} = 6.25 \text{ m/s}$$

$$Fr_1 = \frac{6.25}{\sqrt{9.81 \times 0.8}} = 2.23$$

**(2) 共轭水深**

$$\frac{h'_2}{h_1} = \frac{1}{2}(\sqrt{1 + 8 \times 2.23^2} - 1) = 2.69$$

$$h'_2 = 2.69 \times 0.8 = 2.15 \text{ m}$$

**(3) 判断水跃位置**

下游$h_0 = 2.5$m > $h'_2 = 2.15$m

**淹没水跃**，$\sigma = \frac{2.5 - 2.15}{2.15} = 16.3\%$

水跃在陡坡末端发生。

**(4) 能量损失**

$$E_1 = 0.8 + \frac{6.25^2}{19.62} = 0.8 + 1.99 = 2.79 \text{ m}$$

$$E_2 = 2.15 + \frac{(6.25 \times 0.8/2.15)^2}{19.62} = 2.15 + 0.28 = 2.43 \text{ m}$$

$$\Delta E = 2.79 - 2.43 = 0.36 \text{ m}$$

消能率：$\eta = \frac{0.36}{2.79} \times 100\% = 12.9\%$

## Python实现

```python
import numpy as np
import matplotlib.pyplot as plt

class HydraulicJump:
    """水跃计算"""
    
    def __init__(self, g=9.81):
        self.g = g
    
    def conjugate_depth(self, h1, Fr1):
        """
        共轭水深公式
        
        h2/h1 = 0.5*(√(1+8Fr1²) - 1)
        """
        ratio = 0.5 * (np.sqrt(1 + 8 * Fr1**2) - 1)
        h2 = ratio * h1
        return h2
    
    def energy_loss(self, h1, v1, h2, v2):
        """能量损失"""
        E1 = h1 + v1**2 / (2 * self.g)
        E2 = h2 + v2**2 / (2 * self.g)
        delta_E = E1 - E2
        return delta_E, E1, E2
    
    def jump_length(self, h2, Fr1):
        """
        水跃长度估算
        
        L_j = 6*h2 (稳定水跃)
        """
        if Fr1 < 2.5:
            factor = 5
        elif Fr1 < 4.5:
            factor = 6
        else:
            factor = 7
        
        L_j = factor * h2
        return L_j
    
    def jump_classification(self, Fr1):
        """水跃分类"""
        if Fr1 < 1.7:
            return "波状水跃"
        elif Fr1 < 2.5:
            return "弱水跃"
        elif Fr1 < 4.5:
            return "稳定水跃"
        elif Fr1 < 9.0:
            return "强水跃"
        else:
            return "剧烈水跃"
    
    def submergence_factor(self, h2_actual, h2_conjugate):
        """淹没度"""
        sigma = (h2_actual - h2_conjugate) / h2_conjugate
        return sigma
    
    def plot_jump_analysis(self):
        """绘制水跃分析图"""
        fig, axes = plt.subplots(2, 2, figsize=(16, 12))
        
        # 1. 共轭水深比vs Fr1
        ax1 = axes[0, 0]
        
        Fr1_range = np.linspace(1.0, 10, 200)
        h2_h1_ratio = [0.5 * (np.sqrt(1 + 8*Fr**2) - 1) for Fr in Fr1_range]
        
        ax1.plot(Fr1_range, h2_h1_ratio, 'b-', linewidth=2.5)
        
        # 分区标注
        Fr_boundaries = [1.7, 2.5, 4.5, 9.0]
        colors = ['lightblue', 'lightgreen', 'lightyellow', 'lightcoral', 'pink']
        labels = ['波状', '弱', '稳定', '强', '剧烈']
        
        for i, (Fr_low, Fr_high, color, label) in enumerate(
            zip([1.0] + Fr_boundaries, Fr_boundaries + [10], colors, labels)):
            ax1.axvspan(Fr_low, Fr_high, alpha=0.3, color=color, label=f'{label}水跃')
        
        ax1.set_xlabel('跃前弗劳德数 Fr₁', fontsize=11)
        ax1.set_ylabel('水深比 h₂/h₁', fontsize=11)
        ax1.set_title('共轭水深比与Fr₁关系', fontsize=13, fontweight='bold')
        ax1.legend(loc='upper left', fontsize=9)
        ax1.grid(True, alpha=0.3)
        
        # 2. 能量损失率vs Fr1
        ax2 = axes[0, 1]
        
        eta_values = []
        for Fr in Fr1_range:
            h1 = 1.0  # 假设
            h2 = self.conjugate_depth(h1, Fr)
            v1 = Fr * np.sqrt(self.g * h1)
            v2 = v1 * h1 / h2
            
            delta_E, E1, _ = self.energy_loss(h1, v1, h2, v2)
            eta = (delta_E / E1) * 100
            eta_values.append(eta)
        
        ax2.plot(Fr1_range, eta_values, 'r-', linewidth=2.5)
        
        ax2.set_xlabel('跃前弗劳德数 Fr₁', fontsize=11)
        ax2.set_ylabel('消能率 η (%)', fontsize=11)
        ax2.set_title('消能率与Fr₁关系', fontsize=13, fontweight='bold')
        ax2.grid(True, alpha=0.3)
        
        # 3. 水跃示意图
        ax3 = axes[1, 0]
        
        # 绘制水跃剖面
        h1_demo = 0.5
        Fr1_demo = 3.0
        h2_demo = self.conjugate_depth(h1_demo, Fr1_demo)
        L_j_demo = self.jump_length(h2_demo, Fr1_demo)
        
        x_jump = np.linspace(0, L_j_demo, 100)
        # 简化水面线（抛物线过渡）
        y_jump = h1_demo + (h2_demo - h1_demo) * (x_jump / L_j_demo)**0.5
        
        ax3.fill_between(x_jump, 0, y_jump, alpha=0.5, color='cyan')
        ax3.plot(x_jump, y_jump, 'b-', linewidth=2.5, label='水面线')
        
        # 标注
        ax3.plot([-2, 0], [h1_demo, h1_demo], 'b--', linewidth=2)
        ax3.plot([L_j_demo, L_j_demo+2], [h2_demo, h2_demo], 'r--', linewidth=2)
        
        ax3.text(-1, h1_demo/2, f'h₁={h1_demo}m\nFr₁={Fr1_demo}', fontsize=10,
                bbox=dict(boxstyle='round', facecolor='wheat', alpha=0.7))
        ax3.text(L_j_demo+1, h2_demo/2, f'h₂={h2_demo:.2f}m', fontsize=10,
                bbox=dict(boxstyle='round', facecolor='lightblue', alpha=0.7))
        
        ax3.arrow(L_j_demo/2, -0.5, 0, 0.3, head_width=0.5, head_length=0.1,
                 fc='green', ec='green')
        ax3.text(L_j_demo/2, -0.8, f'Lⱼ={L_j_demo:.1f}m', ha='center', fontsize=10,
                bbox=dict(boxstyle='round', facecolor='yellow', alpha=0.7))
        
        ax3.set_xlabel('距离 x (m)', fontsize=11)
        ax3.set_ylabel('水深 h (m)', fontsize=11)
        ax3.set_title(f'水跃剖面示意图\n{self.jump_classification(Fr1_demo)}',
                     fontsize=13, fontweight='bold')
        ax3.set_xlim([-3, L_j_demo+3])
        ax3.set_ylim([-1, h2_demo+0.5])
        ax3.grid(True, alpha=0.3)
        
        # 4. 淹没影响
        ax4 = axes[1, 1]
        
        sigma_range = np.linspace(0, 0.5, 100)
        # 消能率随淹没度的降低（经验）
        eta_reduction = 100 * (1 - sigma_range / 0.3)  # 假设30%淹没时消能完全失效
        eta_reduction = np.clip(eta_reduction, 0, 100)
        
        ax4.plot(sigma_range * 100, eta_reduction, 'orange', linewidth=2.5)
        
        # 标注淹没程度
        ax4.axvline(5, color='g', linestyle='--', linewidth=1.5, label='轻微淹没(5%)')
        ax4.axvline(10, color='y', linestyle='--', linewidth=1.5, label='中度淹没(10%)')
        ax4.axvline(20, color='r', linestyle='--', linewidth=1.5, label='严重淹没(20%)')
        
        ax4.set_xlabel('淹没度 σ (%)', fontsize=11)
        ax4.set_ylabel('相对消能率 (%)', fontsize=11)
        ax4.set_title('淹没对消能效果的影响', fontsize=13, fontweight='bold')
        ax4.legend()
        ax4.grid(True, alpha=0.3)
        
        plt.tight_layout()
        plt.savefig('hydraulic_jump_analysis.png', dpi=300)
        plt.show()

# 实例化
hj = HydraulicJump()

# 题型51计算
print("="*60)
print("题型51：水跃共轭水深计算")
print("="*60)

h1 = 0.5
v1 = 8.0
Fr1 = v1 / np.sqrt(9.81 * h1)

h2 = hj.conjugate_depth(h1, Fr1)
v2 = v1 * h1 / h2

delta_E, E1, E2 = hj.energy_loss(h1, v1, h2, v2)
L_j = hj.jump_length(h2, Fr1)

print(f"跃前: h1={h1}m, v1={v1}m/s, Fr1={Fr1:.2f}")
print(f"跃后: h2={h2:.2f}m, v2={v2:.2f}m/s")
print(f"能量损失: ΔE={delta_E:.2f}m ({delta_E/E1*100:.1f}%)")
print(f"水跃长度: Lj={L_j:.1f}m")
print(f"水跃类型: {hj.jump_classification(Fr1)}")

# 绘图
print("\n绘制水跃分析图...")
hj.plot_jump_analysis()
```
