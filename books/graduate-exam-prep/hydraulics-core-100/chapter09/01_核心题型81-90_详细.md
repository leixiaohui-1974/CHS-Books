## 题型81：水泵相似定律应用

### 问题描述
某离心泵在转速$n_1=1450$ r/min时，流量$Q_1=100$ m³/h，扬程$H_1=20$ m，轴功率$P_1=7.5$ kW。
若转速提高到$n_2=1750$ r/min，求新工况下的流量、扬程和功率。

### 理论基础

**相似定律**（保持几何相似）：
$$\frac{Q_2}{Q_1} = \frac{n_2}{n_1}$$
$$\frac{H_2}{H_1} = \left(\frac{n_2}{n_1}\right)^2$$
$$\frac{P_2}{P_1} = \left(\frac{n_2}{n_1}\right)^3$$

### 详细解答

**Step 1：计算转速比**
$$\frac{n_2}{n_1} = \frac{1750}{1450} = 1.207$$

**Step 2：计算新流量**
$$Q_2 = Q_1 \cdot \frac{n_2}{n_1} = 100 \times 1.207 = 120.7 \text{ m}^3/\text{h}$$

**Step 3：计算新扬程**
$$H_2 = H_1 \cdot \left(\frac{n_2}{n_1}\right)^2 = 20 \times 1.207^2 = 29.1 \text{ m}$$

**Step 4：计算新功率**
$$P_2 = P_1 \cdot \left(\frac{n_2}{n_1}\right)^3 = 7.5 \times 1.207^3 = 13.2 \text{ kW}$$

### Python实现

```python
import numpy as np
import matplotlib.pyplot as plt

class PumpSimilarityLaw:
    """泵相似定律计算"""
    
    def __init__(self, Q1, H1, P1, n1):
        """
        初始化泵参数
        
        参数:
            Q1: 初始流量(m³/h)
            H1: 初始扬程(m)
            P1: 初始功率(kW)
            n1: 初始转速(r/min)
        """
        self.Q1 = Q1
        self.H1 = H1
        self.P1 = P1
        self.n1 = n1
    
    def calculate_new_performance(self, n2):
        """
        计算新转速下的性能
        
        参数:
            n2: 新转速(r/min)
        
        返回:
            Q2, H2, P2: 新流量、扬程、功率
        """
        ratio = n2 / self.n1
        
        Q2 = self.Q1 * ratio
        H2 = self.H1 * ratio**2
        P2 = self.P1 * ratio**3
        
        return Q2, H2, P2
    
    def plot_performance_curves(self):
        """绘制性能曲线"""
        fig, axes = plt.subplots(2, 2, figsize=(14, 10))
        
        # 转速范围
        n_range = np.linspace(0.5*self.n1, 1.5*self.n1, 100)
        
        Q_curve = self.Q1 * (n_range / self.n1)
        H_curve = self.H1 * (n_range / self.n1)**2
        P_curve = self.P1 * (n_range / self.n1)**3
        
        # 流量曲线
        ax1 = axes[0, 0]
        ax1.plot(n_range, Q_curve, 'b-', linewidth=2.5)
        ax1.axvline(self.n1, color='r', linestyle='--', linewidth=1.5,
                   label=f'设计转速{self.n1} r/min')
        ax1.scatter([self.n1], [self.Q1], s=200, c='red', marker='o',
                   zorder=5)
        ax1.set_xlabel('转速 n (r/min)', fontsize=11)
        ax1.set_ylabel('流量 Q (m³/h)', fontsize=11)
        ax1.set_title('流量 Q ∝ n', fontsize=12, fontweight='bold')
        ax1.legend()
        ax1.grid(True, alpha=0.3)
        
        # 扬程曲线
        ax2 = axes[0, 1]
        ax2.plot(n_range, H_curve, 'g-', linewidth=2.5)
        ax2.axvline(self.n1, color='r', linestyle='--', linewidth=1.5)
        ax2.scatter([self.n1], [self.H1], s=200, c='red', marker='o',
                   zorder=5)
        ax2.set_xlabel('转速 n (r/min)', fontsize=11)
        ax2.set_ylabel('扬程 H (m)', fontsize=11)
        ax2.set_title('扬程 H ∝ n²', fontsize=12, fontweight='bold')
        ax2.grid(True, alpha=0.3)
        
        # 功率曲线
        ax3 = axes[1, 0]
        ax3.plot(n_range, P_curve, 'm-', linewidth=2.5)
        ax3.axvline(self.n1, color='r', linestyle='--', linewidth=1.5)
        ax3.scatter([self.n1], [self.P1], s=200, c='red', marker='o',
                   zorder=5)
        ax3.set_xlabel('转速 n (r/min)', fontsize=11)
        ax3.set_ylabel('功率 P (kW)', fontsize=11)
        ax3.set_title('功率 P ∝ n³', fontsize=12, fontweight='bold')
        ax3.grid(True, alpha=0.3)
        
        # 综合曲线（归一化）
        ax4 = axes[1, 1]
        
        n_norm = n_range / self.n1
        Q_norm = Q_curve / self.Q1
        H_norm = H_curve / self.H1
        P_norm = P_curve / self.P1
        
        ax4.plot(n_norm, Q_norm, 'b-', linewidth=2, label='Q/Q₁')
        ax4.plot(n_norm, H_norm, 'g-', linewidth=2, label='H/H₁')
        ax4.plot(n_norm, P_norm, 'm-', linewidth=2, label='P/P₁')
        ax4.axvline(1.0, color='r', linestyle='--', linewidth=1.5)
        ax4.axhline(1.0, color='r', linestyle='--', linewidth=1.5)
        ax4.set_xlabel('转速比 n/n₁', fontsize=11)
        ax4.set_ylabel('性能比', fontsize=11)
        ax4.set_title('归一化性能曲线', fontsize=12, fontweight='bold')
        ax4.legend()
        ax4.grid(True, alpha=0.3)
        
        plt.tight_layout()
        plt.savefig('pump_similarity_law.png', dpi=300)
        plt.show()

# 示例
pump = PumpSimilarityLaw(Q1=100, H1=20, P1=7.5, n1=1450)

# 计算新转速性能
n2 = 1750
Q2, H2, P2 = pump.calculate_new_performance(n2)

print(f"原转速 n₁ = {pump.n1} r/min")
print(f"  Q₁ = {pump.Q1} m³/h, H₁ = {pump.H1} m, P₁ = {pump.P1} kW")
print(f"\n新转速 n₂ = {n2} r/min")
print(f"  Q₂ = {Q2:.1f} m³/h, H₂ = {H2:.1f} m, P₂ = {P2:.1f} kW")

# 绘图
pump.plot_performance_curves()
```

**答案**：
- $Q_2 = 120.7$ m³/h
- $H_2 = 29.1$ m
- $P_2 = 13.2$ kW

## 题型82：水泵比转速计算与选型

### 问题描述
某泵站设计流量$Q=0.5$ m³/s，扬程$H=30$ m，转速$n=1450$ r/min。
求比转速并选择泵型。

### 理论基础

**比转速公式**：
$$n_s = \frac{3.65 n \sqrt{Q}}{H^{3/4}}$$

**泵型分类**：
| $n_s$ | 泵型 | 特点 |
|-------|------|------|
| $< 80$ | 低比速离心泵 | 高扬程、小流量 |
| $80 \sim 150$ | 中比速离心泵 | 中等扬程流量 |
| $150 \sim 300$ | 高比速离心泵 | 低扬程、大流量 |
| $300 \sim 500$ | 混流泵 | 很低扬程、很大流量 |
| $> 500$ | 轴流泵 | 极低扬程、极大流量 |

### 详细解答

**Step 1：单位换算**
$$Q = 0.5 \text{ m}^3/\text{s} = 1800 \text{ m}^3/\text{h}$$

**Step 2：计算比转速**
$$n_s = \frac{3.65 \times 1450 \times \sqrt{0.5}}{30^{3/4}}$$
$$= \frac{3.65 \times 1450 \times 0.707}{13.16}$$
$$= \frac{3739}{13.16} = 284$$

**Step 3：选择泵型**
$n_s = 284$，属于$150 \sim 300$范围，选择**高比速离心泵**或**混流泵**

### 工程建议
- 推荐：**混流泵**（$n_s \approx 300$）
- 型号参考：$500$HW系列
- 优点：效率高、气蚀性能好

**答案**：$n_s = 284$，选混流泵

## 题型83：水泵串联与并联

### 问题描述
两台相同水泵，单泵性能：$Q_0=100$ m³/h，$H_0=20$ m。
求串联和并联时的流量和扬程。

### 理论基础

**串联**（增大扬程）：
$$Q_{串} = Q_0$$
$$H_{串} = 2H_0$$

**并联**（增大流量）：
$$Q_{并} = 2Q_0$$
$$H_{并} = H_0$$

**注意**：实际中需考虑管路特性曲线的影响

### 详细解答

**1. 串联工况**
- 流量：$Q_{串} = 100$ m³/h（不变）
- 扬程：$H_{串} = 2 \times 20 = 40$ m

**应用场合**：需要高扬程的场合（如高层建筑供水）

**2. 并联工况**
- 流量：$Q_{并} = 2 \times 100 = 200$ m³/h
- 扬程：$H_{并} = 20$ m（不变）

**应用场合**：需要大流量的场合（如灌溉、排水）

### Python实现

```python
class PumpCombination:
    """水泵串并联分析"""
    
    @staticmethod
    def series_pumps(Q0, H0, n=2):
        """
        串联泵组
        
        参数:
            Q0: 单泵流量
            H0: 单泵扬程
            n: 泵数量
        """
        Q_series = Q0
        H_series = n * H0
        return Q_series, H_series
    
    @staticmethod
    def parallel_pumps(Q0, H0, n=2):
        """
        并联泵组
        
        参数:
            Q0: 单泵流量
            H0: 单泵扬程
            n: 泵数量
        """
        Q_parallel = n * Q0
        H_parallel = H0
        return Q_parallel, H_parallel
    
    @staticmethod
    def plot_combination_curves():
        """绘制串并联特性曲线"""
        fig, axes = plt.subplots(1, 2, figsize=(14, 6))
        
        # 单泵特性曲线（简化为抛物线）
        Q = np.linspace(0, 150, 100)
        H_single = 25 - 0.001 * Q**2
        
        # 串联特性
        H_series = 2 * H_single
        
        # 并联特性
        Q_parallel = np.linspace(0, 300, 100)
        H_parallel = 25 - 0.001 * (Q_parallel/2)**2
        
        # 串联图
        ax1 = axes[0]
        ax1.plot(Q, H_single, 'b-', linewidth=2.5, label='单泵')
        ax1.plot(Q, H_series, 'r-', linewidth=2.5, label='两泵串联')
        ax1.set_xlabel('流量 Q (m³/h)', fontsize=11)
        ax1.set_ylabel('扬程 H (m)', fontsize=11)
        ax1.set_title('串联泵组特性曲线', fontsize=12, fontweight='bold')
        ax1.legend()
        ax1.grid(True, alpha=0.3)
        ax1.set_xlim([0, 150])
        ax1.set_ylim([0, 60])
        
        # 并联图
        ax2 = axes[1]
        ax2.plot(Q, H_single, 'b-', linewidth=2.5, label='单泵')
        ax2.plot(Q_parallel, H_parallel, 'g-', linewidth=2.5,
                label='两泵并联')
        ax2.set_xlabel('流量 Q (m³/h)', fontsize=11)
        ax2.set_ylabel('扬程 H (m)', fontsize=11)
        ax2.set_title('并联泵组特性曲线', fontsize=12, fontweight='bold')
        ax2.legend()
        ax2.grid(True, alpha=0.3)
        ax2.set_xlim([0, 300])
        ax2.set_ylim([0, 30])
        
        plt.tight_layout()
        plt.savefig('pump_combination.png', dpi=300)
        plt.show()

# 示例
pc = PumpCombination()

Q0, H0 = 100, 20

Q_series, H_series = pc.series_pumps(Q0, H0, n=2)
Q_parallel, H_parallel = pc.parallel_pumps(Q0, H0, n=2)

print("单泵性能: Q₀ = 100 m³/h, H₀ = 20 m")
print(f"\n串联: Q = {Q_series} m³/h, H = {H_series} m")
print(f"并联: Q = {Q_parallel} m³/h, H = {H_parallel} m")

pc.plot_combination_curves()
```

**答案**：
- 串联：$Q=100$ m³/h，$H=40$ m
- 并联：$Q=200$ m³/h，$H=20$ m

## 题型84：泵的汽蚀与允许安装高度

### 问题描述
某泵NPSH$_r = 4$ m，装置在海拔$z=1000$ m处，水温$t=20°$C。
大气压$p_a = 90$ kPa，水的饱和蒸汽压$p_v = 2.34$ kPa，
吸水管路损失$h_s = 1$ m。求允许安装高度。

### 理论基础

**汽蚀现象**：泵内局部压力降至饱和蒸汽压以下，液体汽化形成气泡

**必需汽蚀余量**：NPSH$_r$（required），泵的固有特性

**有效汽蚀余量**：
$$\text{NPSH}_a = \frac{p_a - p_v}{\rho g} - h_s - H_s$$

**允许安装高度**：
$$H_s^{允许} = \frac{p_a - p_v}{\rho g} - \text{NPSH}_r - h_s - \Delta h$$

其中$\Delta h$为安全裕量，通常取$0.3 \sim 0.5$ m

### 详细解答

**Step 1：计算大气压水头**
$$H_a = \frac{p_a}{\rho g} = \frac{90 \times 10^3}{1000 \times 9.81} = 9.17 \text{ m}$$

**Step 2：计算饱和蒸汽压水头**
$$H_v = \frac{p_v}{\rho g} = \frac{2.34 \times 10^3}{1000 \times 9.81} = 0.24 \text{ m}$$

**Step 3：计算允许安装高度**
取安全裕量$\Delta h = 0.5$ m

$$H_s^{允许} = (9.17 - 0.24) - 4 - 1 - 0.5$$
$$= 8.93 - 5.5 = 3.43 \text{ m}$$

**工程建议**：
- 实际安装高度应$< 3.43$ m
- 海拔越高，允许安装高度越小
- 水温越高，饱和蒸汽压越大，允许安装高度越小

### Python实现

```python
class CavitationAnalysis:
    """汽蚀分析"""
    
    def __init__(self, g=9.81, rho=1000):
        self.g = g
        self.rho = rho
    
    def allowable_installation_height(self, p_a, p_v, NPSH_r, h_s,
                                     safety_margin=0.5):
        """
        计算允许安装高度
        
        参数:
            p_a: 大气压(kPa)
            p_v: 饱和蒸汽压(kPa)
            NPSH_r: 必需汽蚀余量(m)
            h_s: 吸水损失(m)
            safety_margin: 安全裕量(m)
        
        返回:
            H_s_allow: 允许安装高度(m)
        """
        H_a = p_a * 1000 / (self.rho * self.g)
        H_v = p_v * 1000 / (self.rho * self.g)
        
        H_s_allow = (H_a - H_v) - NPSH_r - h_s - safety_margin
        
        return H_s_allow
    
    def plot_cavitation_limits(self):
        """绘制汽蚀限制曲线"""
        fig, axes = plt.subplots(1, 2, figsize=(14, 6))
        
        # 不同海拔的允许安装高度
        ax1 = axes[0]
        
        altitudes = np.linspace(0, 3000, 50)  # 海拔(m)
        # 大气压随海拔降低（简化）
        p_a_values = 101.3 * np.exp(-altitudes / 8000)
        
        NPSH_r = 4  # m
        h_s = 1  # m
        p_v = 2.34  # kPa (20°C)
        
        H_s_allows = []
        for p_a in p_a_values:
            H_s = self.allowable_installation_height(p_a, p_v, NPSH_r, h_s)
            H_s_allows.append(H_s)
        
        ax1.plot(altitudes, H_s_allows, 'b-', linewidth=2.5)
        ax1.axhline(0, color='r', linestyle='--', linewidth=1.5,
                   label='零安装高度')
        ax1.fill_between(altitudes, H_s_allows, 0, alpha=0.3, color='green',
                        label='允许区域')
        ax1.set_xlabel('海拔高度 (m)', fontsize=11)
        ax1.set_ylabel('允许安装高度 Hs (m)', fontsize=11)
        ax1.set_title('海拔对允许安装高度的影响', fontsize=12,
                     fontweight='bold')
        ax1.legend()
        ax1.grid(True, alpha=0.3)
        
        # 不同水温的影响
        ax2 = axes[1]
        
        temperatures = np.linspace(0, 100, 50)  # 温度(°C)
        # 饱和蒸汽压随温度增加（简化Antoine方程）
        p_v_values = 0.6105 * np.exp(17.27 * temperatures / (temperatures + 237.3))
        
        p_a = 101.3  # kPa (标准大气压)
        
        H_s_allows_temp = []
        for p_v_temp in p_v_values:
            H_s = self.allowable_installation_height(p_a, p_v_temp, NPSH_r, h_s)
            H_s_allows_temp.append(H_s)
        
        ax2.plot(temperatures, H_s_allows_temp, 'r-', linewidth=2.5)
        ax2.axhline(0, color='b', linestyle='--', linewidth=1.5)
        ax2.fill_between(temperatures, H_s_allows_temp, 0, alpha=0.3,
                        color='orange', label='允许区域')
        ax2.set_xlabel('水温 (°C)', fontsize=11)
        ax2.set_ylabel('允许安装高度 Hs (m)', fontsize=11)
        ax2.set_title('水温对允许安装高度的影响', fontsize=12,
                     fontweight='bold')
        ax2.legend()
        ax2.grid(True, alpha=0.3)
        
        plt.tight_layout()
        plt.savefig('cavitation_analysis.png', dpi=300)
        plt.show()

# 示例
ca = CavitationAnalysis()

H_s_allow = ca.allowable_installation_height(
    p_a=90, p_v=2.34, NPSH_r=4, h_s=1, safety_margin=0.5
)

print(f"允许安装高度 Hs = {H_s_allow:.2f} m")

ca.plot_cavitation_limits()
```

**答案**：$H_s^{允许} = 3.43$ m

**问题**：管道长$L=1000$ m，流速$v_0=2$ m/s，
关阀时间$T_c=5$ s，水击波速$c=1000$ m/s。

**解**：
相长$T_r = 2L/c = 2$ s $< T_c$，属间接水锤

$$\Delta p = \frac{\rho g L \Delta v}{g T_c} = \frac{1000 \times 9.81 \times 1000 \times 2}{9.81 \times 5 \times 1000}$$
$$= 400 \text{ kPa} = 40.8 \text{ m水柱}$$
