## 📖 项目简介

《水力学考研核心100题》是一本专门针对水利、土木、环境类考研学生的**分数导向型**教材。

### 核心特色

🎯 **真题为王**：100题全部改编自985/211名校真题（2016-2021年）  
📊 **分值明确**：每题标注考频（★）和预计分值  
🐍 **Python可视化**：40个代码示例，可交互修改参数  
📚 **分层设计**：基础40题 + 强化40题 + 综合20题

## 📚 学习路径

### 路径A：基础薄弱型（60-80分目标）
```
4周计划：
Week 1: 第一章（15题）+ 第二章前10题
Week 2: 第二章后15题 + 第三章前10题
Week 3: 第三章后10题 + 第四章前15题
Week 4: 第四章后10题 + 第五章 + 复习

策略：只做40道基础题，熟练掌握
```

### 路径B：中等水平型（80-110分目标）
```
3周计划：
Week 1: 第一章 + 第二章（全部40题）
Week 2: 第三章 + 第四章（全部45题）
Week 3: 第五章 + 第六章 + 错题回顾
```

```
Week 1: 直接做强化题（40题）+ 综合题（20题）
```

## 🚀 快速开始

### 安装依赖

```bash
cd hydraulics-core-100/code
pip install -r requirements.txt
```

### 运行示例代码

```bash
# 示例1：矩形断面均匀流
cd examples/ch04_open_channel
python3 uniform_flow_rectangular.py

# 示例2：水力最优断面设计（6子图）
python3 uniform_flow_optimal_design.py

# 示例3：水跃共轭水深与消能
python3 hydraulic_jump.py
```

