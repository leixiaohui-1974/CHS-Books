## 📖 项目简介

《水力学考研核心100题》是一本专门针对水利、土木、环境类考研学生的**分数导向型**教材。

### 核心特色

🎯 **真题为王**：100题全部改编自985/211名校真题（2016-2021年）  
📊 **分值明确**：每题标注考频（★）和预计分值  
🐍 **Python可视化**：40个代码示例，可交互修改参数  
📚 **分层设计**：基础40题 + 强化40题 + 综合20题

### 安装依赖

```bash
cd hydraulics-core-100/code
pip install -r requirements.txt
```

### 运行示例代码

```bash
# 示例1：矩形断面均匀流
cd examples/ch04_open_channel

# 示例2：水力最优断面设计（6子图）
```
