### 题1：水泵基本性能参数计算（750行）

**核心算法**:

```python
class PumpPerformance:
    def circumferential_velocity(self, D, n):
        """圆周速度：u = π·D·n/60"""
        u = np.pi * D * n / 60
        return u
    
    def theoretical_head(self, u2, cu2):
        """欧拉方程：Ht = u₂·cu₂/g"""
        Ht = u2 * cu2 / self.g
        return Ht
    
    def similarity_law_speed_change(self, n_new):
        """相似定律：Q∝n, H∝n², N∝n³"""
        ratio = n_new / self.n
        Q_new = self.Q * ratio
        H_new = H * (ratio ** 2)
        N_new = self.N * (ratio ** 3)
        return Q_new, H_new, N_new
```

### 题3：水泵工况点分析（700行）

**核心算法**:

```python
class OperatingPoint:
    def find_operating_point(self, r=None):
        """工况点解析法：H泵 = H管"""
        Q_squared = (self.H0 - self.Hst) / (self.k_pump + r)
        Q = np.sqrt(Q_squared)
        H = self.pump_curve(Q)
        return Q, H
    
    def parallel_pump_curve(self, Q, n_pumps=2):
        """并联特性：H = H0 - k(Q/n)²"""
        Q_single = Q / n_pumps
        H = self.pump_curve(Q_single)
        return H
```

**可视化亮点**:
- 工况点图解法（特性曲线交点）
- 阻力变化影响（工况点左移动画）
- 并联工况点对比
- 工况点对比表（清晰展示）

**重要结论**:
- 阻力增大 → 流量↓，扬程↑
- 并联增流量，但不是2倍（因扬程也增大）
- 单泵分担流量反而减小
