#### 3.2 性能参数参考
```python
# P&O标准配置
step_size = 1.0          # 步长(V)
适用: 成本敏感、无遮挡

# INC标准配置
step_size = 1.0
threshold = 0.01         # 判断阈值
适用: 高精度要求

# Fuzzy标准配置
step_size_max = 3.0
适用: 复杂环境、鲁棒性

# PSO标准配置
n_particles = 10
w = 0.7, c1 = 1.5, c2 = 1.5
适用: 部分遮挡、多峰

# Hybrid标准配置
pso_particles = 10
switch_threshold = 0.95
适用: 所有场景 (推荐!)
```
