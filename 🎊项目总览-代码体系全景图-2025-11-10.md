## 📚 知识体系总览

### 核心公式体系（50+）

#### 静水力学（8个）
1. p = p₀ + ρgh
2. P = ρghcA
3. yp = yc + Ic/(ycA)
4. Px = ρghcAx, Pz = ρgV
5. F浮 = ρgV排
6. GM = I₀/V - BG
7. M稳 = C稳 - G稳
8. ΔGM = ΔI₀/V

#### 水动力学（12个）
1. Q₁v₁ = Q₂v₂
2. z + p/(ρg) + v²/(2g) = H
3. ΣF = ρQ(v₂ - v₁)
4. Q = μA√(2gH)
5. μ = φε
6. Π = f(Π₁, Π₂, ...)
7. Re = vd/ν
8. Fr = v/√(gL)
9. Eu = Δp/(ρv²)
10. σ = (pa - pv)/(ρgH)
11. NPSH = (pa - pv)/(ρg) - [hs]
12. [hs] = pa/(ρg) - pv/(ρg) - Σh - v²/(2g)

#### 管流（10个）
1. Re = vd/ν, Re_cr = 2320
2. λ层流 = 64/Re
3. 1/√λ = -2lg(Δ/(3.7d) + 2.51/(Re√λ))
4. h_f = λ(L/d)(v²/2g)
5. h_j = ζv²/(2g)
6. Q = πd²√(2gH)/(4√(λL/d + Σζ))
7. ΣQᵢ = 0 (节点)
8. Σh_f = 0 (回路)
9. c = √(K/ρ)/√(1 + Kd/(Eδ))
10. ΔH = cv/g

#### 明渠流（8个）
1. Fr = v/√(gh)
2. h_c = (Q²/(gb²))^(1/3)
3. E = h + v²/(2g)
4. h₂ = (h₁/2)(√(1+8Fr₁²)-1)
5. Q = AC√(Ri)
6. Q = (1/n)AR^(2/3)i^(1/2)
7. dh/dx = (i-J)/(1-Fr²)
8. Q = mbH^(3/2)

#### 渗流（6个）
1. v = ki
2. Q = kiA
3. i_cr = (γₛ - γw)/γw
4. H(r) = H₀ + Q/(2πkM)ln(r/R) (承压)
5. h²(r) = h₀² + Q/(πk)ln(r/R) (潜水)
6. k = Qln(r₂/r₁)/(2πM(s₁-s₂))

#### 水泵（6个）
1. H = H₀ - aQ²
2. η = bQ - cQ²
3. P = ρgQH/η
4. H_pump(Q) = H_pipe(Q)
5. n'/n = Q'/Q = √(H'/H)
6. P'/P = (n'/n)³
