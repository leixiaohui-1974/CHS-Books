```bash
cd /workspace/platform/backend
python3 main.py
```

2. **访问前端页面**
```
http://localhost:3000
```

3. **浏览书籍目录**
```
http://localhost:3000/books
```

### API端点

1. **获取所有书籍**
```
GET /api/v1/books
```

2. **获取单本书详情**
```
GET /api/v1/books/{book_slug}
```

3. **获取章节内容**
```
GET /api/v1/books/{book_slug}/chapters/{chapter_slug}
```

4. **获取案例详情**
```
GET /api/v1/books/{book_slug}/chapters/{chapter_slug}/cases/{case_slug}
```
