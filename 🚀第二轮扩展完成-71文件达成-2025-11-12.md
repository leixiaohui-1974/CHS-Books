### 1. 静水总压力（30days-sprint）
```python
class HydrostaticForce:
    def rectangular_gate_vertical(self, b, h, h0=0):
        """竖直矩形闸门总压力与压力中心"""
        h_C = h0 + h / 2
        P = self.gamma * h_C * b * h
        I_C = b * h**3 / 12
        y_D = h0 + h/2 + I_C/(h_C * b * h)
        return P, y_D
```

### 2. 边界层理论（hydraulics-advanced）
```python
class BoundaryLayerTheory:
    def laminar_thickness(self, x):
        """层流边界层厚度 δ = 5x/√Re_x"""
        Re_x = self.U * x / self.nu
        return 5 * x / np.sqrt(Re_x)
```

### 3. Theis井流（groundwater）
```python
def theis_drawdown(r, t, Q, T, S):
    """Theis非稳定井流降深"""
    u = (r**2 * S) / (4 * T * t)
    W_u = exp1(u)  # 井函数
    s = (Q / (4 * np.pi * T)) * W_u
    return s
```
